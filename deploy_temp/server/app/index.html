📋 行业标准 (126条)

  • 3GPP Release 15
    5G NR Phase 1 (第一阶段5G标准)
    3GPP · 2018年6月 · 状态：现行
    5G NR首个商用标准版本，定义NSA（非独立组网）和SA（独立组网）架构，支持6GHz以下频段（FR1）和毫米波频段（FR2），引入新型波形Numerology、LDPC信道编码、Polar码控制信道等关键技术
  • 3GPP Release 16
    5G NR Phase 2 / 5G-Advanced起点
    3GPP · 2020年7月 · 状态：现行
    增强URLLC（超可靠低时延通信）、V2X（车联网）、工业IoT、定位增强、终端节能、IAB（集成接入回传）等技术；引入NR Unlicensed（NR-U，免授权频段）；MIMO进一步增强
  • 3GPP Release 17
    5G NR Phase 2 增强
    3GPP · 2022年3月 · 状态：现行
    引入NR-Light（轻量化5G，替代eMTC/NB-IoT增强）、mmWave频段扩展（上行256QAM、更宽信道带宽）、MIMO OTA测试标准化（TS 38.551）、广播/多播增强、卫星接入NR（NTN）、IIoT（工业物联网）进一步增强
  • 3GPP Release 18
    5G-Advanced (5.5G)
    3GPP · 2024年3月 · 状态：现行
    AI驱动的空口（AI for NR）、XR（扩展现实）增强、上行增强（UL Tx切换、更多载波聚合）、覆盖增强、ELAA（超大孔径天线）概念引入、沉浸式通信（IC）、更高阶MIMO（256天线单元）、智能波束管理、RedCap进一步增强
  • 3GPP Release 19
    5G-Advanced 第二阶段
    3GPP · 2025年（规划中） · 状态：现行
    通感融合（ISAC）、AI/ML原生空口、XR持续增强、XL-6G（更高频段研究）、非地面网络（NTN）增强、轻量化IoT进一步演进
  • 3GPP Release 20
    6G标准化起点（规划中）
    3GPP · 预计2027年 · 状态：现行
    6G标准研究开始，预计引入智能通信、空天海地一体化、太赫兹（THz）通信等前沿技术
  • TS 38.101-1
    NR; User Equipment (UE) radio transmission and reception; Part 1: Range 1 Standalone

  • TS 38.101-2
    NR; User Equipment (UE) radio transmission and reception; Part 2: Range 2 Standalone

  • TS 38.101-4
    NR; User Equipment (UE) radio transmission and reception; Part 4: Performance requirements

  • TS 38.104

  • TS 38.141-1

  • TS 38.141-2
