天线基板 Substrate Substrate

天线基板是承载天线辐射体和馈电网络的"画布"——它的介电常数（ε_r）、损耗角正切（tanδ）和厚度直接决定了天线的尺寸、效率和带宽。Sub-6GHz 常用 FR4/Rogers，毫米波常用 LCP/LTCC，封装级用 Si/GBS。

类比入口

天线基板就像画家的画布——画布的质量决定了颜料（射频信号）能保留多少。便宜的画布（FR4）吸水（损耗大），贵的画布（Rogers）不吸水（损耗小）。毫米波频率越高，对画布的要求越苛刻——就像高清印刷需要更光滑的纸张。

一、定义与功能

1.1 基板的角色

基板承载：
      1. 天线辐射体（patch、dipole、slot 等）
      2. 馈电网络（microstrip、stripline 等）
      3. 匹配电路（capacitor、inductor）
      4. 接地层（ground plane）

1.2 基板的关键参数

三、介电常数和损耗

3.1 介电常数（ε_r）的影响

基板中的波长：λ_sub = λ_0 / √ε_r
    
    ε_r = 4 → λ_sub = λ_0 / 2（天线尺寸减半）
    ε_r = 10 → λ_sub = λ_0 / 3.16（天线尺寸大幅缩小）

ε_r 的优缺点：

高 ε_r：天线尺寸小，但带宽窄、损耗大
    低 ε_r：带宽宽、损耗小，但天线尺寸大

3.2 损耗角正切（tanδ）

基板损耗 = tanδ × f × 体积
    
    tanδ 越小，损耗越低

4.2 厚度对带宽的影响

h/λ 越大 → 带宽越宽
    h/λ 越小 → 带宽越窄
    
    薄基板（h < 0.1mm）：适合毫米波，但制造困难
    厚基板（h > 1mm）：适合 Sub-6GHz，带宽好

五、表面粗糙度

5.1 集肤效应（Skin Effect）

高频电流只在导体表面流动：
    δ_skin = √(2ρ / ωμ)
    
    f = 10GHz → δ_skin ≈ 0.66μm（铜）
    f = 60GHz → δ_skin ≈ 0.27μm（铜）

关键：当表面粗糙度 R_z > δ_skin 时，导体损耗显著增加。

5.2 低轮廓铜箔（HVLP/RTF）

标准铜箔 R_z ≈ 3-5μm → 不适合 > 20GHz
    HVLP 铜箔 R_z ≈ 1-2μm → 适合 28GHz
    RTF 铜箔 R_z ≈ 0.5-1μm → 适合 60GHz+

六、基板选择指南

6.1 按频段选择

f < 3GHz → FR4（消费级）/ Rogers RO4000（高端）
    3-10GHz → Rogers RO4350B / TLY
    10-30GHz → LCP / Rogers RO3000
    30-60GHz → LCP / GBS / Si
    > 60GHz → Si / GaAs / InP

七、工艺与制造

7.1 PCB 制造工艺

FR4：标准 IPC 工艺（线宽/间距 ≥ 50μm）
    Rogers：特殊工艺（需专用层压板）
    LCP：精密蚀刻（线宽/间距 ≥ 10μm）
    mSAP：半加成法（线宽/间距 ≥ 5μm）
    AIP：晶圆级工艺（线宽/间距 ≥ 1μm）

7.2 层压工艺

Sub-6GHz：标准层压（100-150°C）
    Rogers：专用层压（180-200°C，高压）
    LCP：低温层压（200-250°C，需防潮）
    LTCC：共烧（850°C，陶瓷生带）

八、一句话总结

基板是天线的"地基"——ε_r 决定天线尺寸（越高越小），tanδ 决定损耗（越低越好），厚度决定阻抗和带宽。Sub-6GHz 用 FR4 或 Rogers，毫米波用 LCP 或 GBS，封装级用 Si。记住：高频下基板不再是"绝缘体"，而是射频电路的一部分——它的每一个参数都会影响天线性能。
