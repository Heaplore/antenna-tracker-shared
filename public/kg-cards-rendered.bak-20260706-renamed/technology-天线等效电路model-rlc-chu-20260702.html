天线等效电路模型 Equivalent Circuit Models for Antennas

任何天线都可以用一个等效电路来描述——最简单的就是 RLC 串联或并联谐振回路，其中电阻代表辐射电阻和损耗电阻，电感和电容代表储能在天线周围的电磁场。更复杂的宽带天线可以用多模 RLC 网络或传输线模型来描述。

类比入口

天线本质上是一个"谐振器"——就像吉他弦在特定频率振动最强一样，天线也在特定频率辐射最强。这个谐振过程可以用 RLC 电路来完美模拟：电阻 R 消耗能量（辐射出去或变成热量），电感 L 储存磁场能量，电容 C 储存电场能量。当你调谐 RLC 电路到谐振频率时，就像调准吉他弦一样——天线达到了最佳辐射状态。

一、基本等效电路模型

1.1 串联 RLC 模型（最基础）

        ┌───[R_r]───[L]───[C]───┐
            │                        │
           [R_s]                     │
            │                        │
            └────────────────────────┘
                  端口

元件含义：

元件物理意义频率特性
R_r辐射电阻与频率相关
R_s损耗电阻（导体+介质）基本恒定
L磁储能等效电感∝ √f
C电储能等效电容∝ 1/√f

输入阻抗：

Z_in(f) = R_r + R_s + j(2πfL - 1/(2πfC))

谐振频率：

f₀ = 1/(2π√(LC))

在谐振时：

Z_in(f₀) = R_r + R_s（纯电阻）
    Q = f₀/Δf = (1/R)·√(L/C)

其中 Δf 是 -3dB 带宽。

1.2 并联 RLC 模型

        ┌────[R_r]────┐
            │             │
           [L]           [C]
            │             │
            └────[R_s]────┘
                  端口

输入导纳：

Y_in(f) = 1/R_r + 1/R_s + j(2πfC - 1/(2πfL))

适用场景：

  • 并联馈电天线（如微带贴片）
  • 阵列天线的单元等效

1.3 频率相关电阻模型

实际天线的辐射电阻 R_r 和损耗电阻 R_s 都随频率变化：

R_r(f) ≈ R_r(f₀) · (f/f₀)²    （电小天线）
    R_r(f) ≈ R_r(f₀)               （谐振天线附近）
    R_s(f) ≈ R_s(f₀) · √(f/f₀)    （趋肤效应）

二、宽带天线的多模等效电路

2.1 多模 RLC 串联模型

宽带天线（如 Vivaldi、螺旋、对数周期）可以用多个 RLC 支路并联来模拟不同"模"的响应：

                    ┌──[R₁]-[L₁]-[C₁]──┐
                        │                    │
    Z_in =              +──[R₂]-[L₂]-[C₂]──┤
                        │                    │
                        +──...               │
                        │                    │
                        +──[Rₙ]-[Lₙ]-[Cₙ]──┘

每个 RLC 支路对应天线的一个谐振模。

提取方法：

  1. 全波仿真或测量获得 Z_in(f)
  2. 在多个谐振频率附近拟合 RLC 参数
  3. 优化所有参数使电路响应与仿真一致

2.2 分布参数模型

对于长线天线（如偶极子、单极子），可以用传输线模型：

         ┌───[Z₀]───[θ]───┐
             │                 │
             +───[Z₀]───[θ]───+
             │                 │
            [负载]            [馈电]

其中 Z₀ 是特性阻抗，θ = β·l 是电长度。

偶极子的传输线模型：

Z_in = Z₀ · (Z_L + jZ₀·tanθ) / (Z₀ + jZ_L·tanθ)

对于开路终端（Z_L → ∞）：

Z_in = -jZ₀ · cotθ

2.3 非均匀分段模型

2023 年 ACES 期刊提出的宽带偶极子分布参数模型：

将偶极子沿轴向分为 N 段，每段用 RLC 表示：

段 1: [R₁]-[L₁]-[C₁]
    段 2: [R₂]-[L₂]-[C₂]
    ...
    段 N: [R_N]-[L_N]-[C_N]

优势：

  • 宽带（DC 到 THz）
  • 频率无关的参数
  • 物理意义清晰
  • 可用于瞬态响应分析

三、Chu 电路模型与 Chu-Harrington 极限

3.1 Chu 球形天线模型

Chu (1948) 将天线放在半径为 a 的球壳内，用电磁球模展开：

TM_mn 模的等效电路：
        ┌──[R_rad,mn]──[L_mn]──[C_mn]──┐
        │                               │
        +──[R_loss,mn]                  │
        │                               │
        └──────────── 端口 ─────────────┘

基模（TM₁₁）的 Q 值下限：

Q_min = 1/(ka)³ + 1/(ka)

其中 k = 2π/λ, a = 天线所在球壳半径。

物理意义：

  • ka << 1（电小天线）：Q_min ≈ 1/(ka)³ → Q 很大，带宽很窄
  • ka >> 1（电大天线）：Q_min ≈ 1/(ka) → Q 很小，带宽很宽

3.2 Chu 极限的工程解读

Q_min · BW_max ≈ 常数

即：天线的 Q 值和带宽的乘积有理论下限。

天线尺寸kaQ_min相对带宽
a = λ/100.63~8~12%
a = λ/200.31~64~1.5%
a = λ/500.13~610~0.16%
a = λ/1000.063~4900~0.02%

3.3 Harrington 直接性-带宽极限

Harrington (1960) 证明了给定球壳内可以激发的最大直接性：

D_max ≤ (2l_max + 1)² / l_max

其中 l_max 是可以有效激发的球模阶数。

与 Chu 极限的关系：

  • Chu 限制带宽（Q 值）
  • Harrington 限制直接性（增益）
  • 两者结合 → 直接性-带宽-效率的"Pozar 三角"

3.4 2024 年 Chu 极限更新

2024 年新研究对 Chu 极限进行了修正：

  • 建立了精确的 1D 传输线模型替代 Chu 电路
  • 重新定义了"存储能量"和"辐射能量"的边界
  • 发现 Chu 极限在电大天线（ka > 10）时过于保守
  • 新的直接性-带宽极限比 Harrington 高 20-40%

四、特性模分析（Characteristic Mode Analysis, CMA）

4.1 基本原理

CMA 将天线的表面电流分解为一系列正交的"特性模"：

J_total = Σ_n α_n · J_n_characteristic

每个特性模有自己的谐振频率、辐射电阻和 Q 值。

4.2 特性模的等效电路

每个特性模可以用独立的 RLC 串联电路表示：

Z_in = Σ_n |α_n|² · Z_n_RLC
    Z_n_RLC = R_n + j(ωL_n - 1/(ωC_n))

特征值 β_n：

β_n = 0 → 谐振
    β_n > 0 → 感性（频率高于谐振）
    β_n < 0 → 容性（频率低于谐振）

4.3 CMA 的工程应用

  • 确定天线的谐振模式
  • 优化多频天线设计
  • 分析 MIMO 天线的模式隔离度
  • 指导表面形状优化

五、天线等效电路的参数提取

5.1 从 S 参数提取

使用 VNA 测量 S₁₁(f)，然后：

Z_in = Z₀ · (1+S₁₁)/(1-S₁₁)
    R = Re(Z_in)
    X = Im(Z_in)

拟合 RLC 参数：

X(f) = 2πfL - 1/(2πfC)

通过最小二乘法拟合 L 和 C。

5.2 从脉冲响应提取

时域反射计（TDR）或超宽带脉冲测量：

I(t) → FFT → I(f)
    Z_in(f) = V(f)/I(f)

5.3 从全波仿真提取

HFSS/CST 仿真得到 Z_in(f)，然后：

1. 找到谐振频率 f₀（Im(Z) = 0）
    2. 在 f₀ 处，R = Re(Z)
    3. 在 f₀ 附近，Q = f₀/(f₂-f₁) = ω₀·W/P_rad
    4. L = Im(Z)/(2πf)，C = -1/(Im(Z)·2πf)

六、不同类型天线的典型等效电路

6.1 偶极子天线

串联 RLC：
    R_r ≈ 73Ω（半波）
    L ≈ λ/2 对应的电感
    C ≈ λ/2 对应的电容
    
    多模模型（宽带偶极子）：
    ┌──[73Ω]-[L₁]-[C₁]──┐
    ├──[65Ω]-[L₂]-[C₂]──┤  （第二模）
    ├──[58Ω]-[L₃]-[C₃]──┤  （第三模）
    └────────────────────┘

6.2 微带贴片天线

并联 RLC：
    R ≈ 120-300Ω（取决于馈电位置和基板）
    L ≈ 基板介质中的磁储能
    C ≈ 贴片- Ground 之间的电容
    
    辐射电阻与 Q：
    R_r ∝ (h/λ)² · (W/λ)
    Q ≈ (λ/h) · √(ε_r)

6.3 螺旋天线

多模串联 RLC：
    每个圈对应一个 RLC 模
    模与模之间有互感耦合
    
    特征阻抗：
    Z₀ ≈ 10π·(D/h) Ω

6.4 Vivaldi 天线

传输线模型：
    渐变的特性阻抗 Z(x) 沿槽线变化
    
    等效为多级阻抗变换器：
    Z₁ → Z₂ → ... → Z_N → Z₀

七、工程挑战与坑

7.1 集总参数模型的局限性

RLC 串联/并联模型只适用于窄带（< 20% 带宽）天线。宽带天线需要多模模型或分布参数模型。

7.2 频率相关电阻

实际天线的 R_r、L、C 都随频率变化，固定参数的 RLC 模型在频带边缘误差很大。

7.3 互耦的电路表示

阵列中单元之间的互耦需要在等效电路中增加互感 M 和互电阻 R_m：

Z_array = 
    ┌[Z₁₁] [Z₁₂] ... [Z₁N]┐
    │[Z₂₁] [Z₂₂] ... [Z₂N]│
    │[...]  [...]  [...]  │
    └[ZN1] [ZN2] ... [ZNN]┘

其中 Z_nm = R_nm + jωL_nm - j/(ωC_nm)。

7.4 温度和环境的影响

  • 温度变化 → 基板 ε_r 变化 → C 变化 → 谐振频率偏移
  • 湿度变化 → 介质损耗变化 → R_s 变化
  • 机械应力 → 几何变形 → L、C 变化

八、技术演进

  • 1948年：Chu 球形天线模型和 Q 极限

  • 2000s：分布参数等效电路

  • 2020-2023年：宽带分布参数模型、非均匀分段模型

九、技术成熟度

  • 成熟度等级：成熟
  • 说明：RLC 等效电路和 Chu 极限是天线理论的标准内容，已写入几乎所有天线教科书。CMA 和分布参数模型在工程实践中也广泛应用。

十、一句话总结

天线等效电路是把电磁问题转化为电路问题的桥梁——简单的 RLC 谐振模型解释了天线的带宽极限，多模模型描述了宽带天线的行为，Chu 极限给出了任何天线都无法逾越的物理约束。理解等效电路，就能从电路的角度设计和优化天线。
