Rogers 系列高频板材 Rogers

Rogers 是美国罗杰斯公司生产的高频微波基板品牌——Dk（介电常数）稳定在 2.2-10.2 之间，Df（损耗角正切）低至 0.0009，是 5G 基站天线、毫米波天线和卫星通信的核心材料。

类比入口

Rogers 板材就像"航空级铝合金"——普通 PCB 用的 FR4 是"建筑钢材"，能用但性能一般；Rogers 是"航空铝"，贵但轻、强、稳定。在 3.5GHz 以上，FR4 的损耗会让信号"跑不动"，Rogers 能让信号"高速公路"畅通无阻。

1.2 材料结构

Rogers 层压板结构：
      ┌──────────────────┐ ← 铜箔（EP/RA）
      ├──────────────────┤ ← 介质层（PTFE+陶瓷填料）
      └──────────────────┘
      
    铜箔类型：
      EP（Electroplated）：粗糙度 Ra=3-5μm
      RA（Rolled Annealed）：粗糙度 Ra=1-2μm
      
    介质层：
      PTFE（聚四氟乙烯）基体
      + 陶瓷填料（控制 Dk）
      + 玻璃纤维布（增强）

二、关键参数详解

2.1 介电常数 Dk

Dk 定义：
      Dk = ε_r = C_material / C_vacuum
      
    Dk 稳定性：
      Rogers：ΔDk < ±0.05（-40°C to +85°C）
      FR4：ΔDk = ±0.5（温度敏感）
      
    Dk 对天线的影响：
      λ_g = λ_0 / √Dk
      Dk 越稳定 → 中心频率越稳定
      Dk 漂移 → 天线谐振频率偏移

2.2 损耗角正切 Df

Df 定义：
      Df = tanδ = 介质损耗 / 存储能量
      
    信号衰减：
      α = (2πf/c)·√Dk·Df （Np/m）
      
    5GHz 时衰减对比（1m 微带线）：
      FR4（Df=0.02）：~3dB
      RO4350B（Df=0.0037）：~0.5dB
      RO3003（Df=0.0013）：~0.2dB
      RT5880（Df=0.0009）：~0.1dB

三、Rogers 在天线中的应用

3.1 5G Sub-6GHz 天线

RO4350B / RO3003 应用：
      - 微带贴片天线阵列
      - 馈电网络（Wilkinson/分支线）
      - 巴伦（Balun）
      
    典型设计：
      频段：3.5GHz（n78）
      基板：RO4350B，厚度 0.8mm
      铜厚：35μm（1oz）
      天线：64 单元贴片阵列
      增益：22-24dBi

3.2 毫米波天线

RO3003 / RT5880 应用：
      - 28GHz/39GHz 贴片阵列
      - 波导缝隙天线
      - AiP 封装基板
      
    典型设计：
      频段：28GHz
      基板：RO3003，厚度 0.254mm
      铜厚：17μm（0.5oz）
      天线：256 单元 AiP 阵列
      增益：23-26dBi

3.3 馈电网络

RO4350B 馈电网络：
      - Wilkinson 功分器
      - 分支线耦合器
      - 移相器
      
    优势：
      - 低损耗（Df=0.0037）
      - 相位稳定（Dk 温度系数小）
      - 幅度平衡好（±0.2dB）

五、一句话总结

Rogers 是高频天线的"黄金材料"——RO4350B 性价比高（Sub-6GHz 基站主流），RO3003 性能更好（毫米波/AiP），RT5880 最低损耗（测试/雷达）。记住：3.5GHz 以上，FR4 的损耗会让信号"跑不动"，Rogers 是必选项。
