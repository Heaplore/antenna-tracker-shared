工参工程参数 Installation Parameters

工参是基站天线的"身份证"——包括经纬度、高度、方位角、下倾角、PCI、频点等核心参数，直接决定了网络的覆盖、容量和干扰水平。一个错误的工参可能导致覆盖空洞或越区覆盖。

一、工参分类

1.1 机械参数

参数单位典型范围说明
经度°100-120基站位置
纬度°20-40基站位置
海拔高度m0-500天线安装高度
挂高m15-50天线离地高度
方位角°0-360天线朝向
机械下倾°0-15物理下倾角
电调下倾°0-15RET 下倾角

1.2 射频参数

二、工参配置流程

1. 站点勘察
       - 确定经纬度、挂高、方位
       - 测量机械下倾基准
       - 拍摄现场照片
    
    2. 工参设计
       - 规划 PCI（避免冲突）
       - 设置方位角和下倾角
       - 配置波束参数
    
    3. 工参录入
       - 网管系统录入
       - 自动化下发到 RCU
       - 验证 RET 角度
    
    4. 现场验证
       - DT 路测
       - 检查覆盖和干扰
       - 调整工参（必要时）

三、常见工参问题

3.1 PCI 冲突

问题：相邻小区 PCI 相同
    影响：切换失败、干扰增加
    解决：重新规划 PCI
    
    冲突检测：
      - PCI mod 3 冲突（RSRP）
      - PCI mod 6 冲突（CSI-RS）
      - PCI mod 30 冲突（SRS）

3.2 方位角错误

问题：工参方位角与实际不符
    影响：波束指向错误、覆盖偏移
    解决：现场校准
    
    典型偏差：
      - 罗盘误差：±5°
      - 安装误差：±3°
      - 总误差：±8°

3.3 下倾角错误

问题：电调下倾角设置错误
    影响：覆盖过远或过近
    解决：RET 校准
    
    校准方法：
      1. 使用 RET 测试仪
      2. 比对工参与实际角度
      3. 修正工参或 RET 设置

四、工参自动化

4.1 5G 工参自动化

自动化流程：
      1. 规划系统自动生成工参
      2. CSV/Excel 批量导入网管
      3. 网管校验（PCI 冲突、覆盖重叠）
      4. 自动下发到 AAU
      5. 自动验证（DT 路测）

4.2 工参管理工具

主流工具：
      - 华为 UMG（统一管理网关）
      - 中兴 NetNumen
      - 爱立信 ENM
      - 诺基亚 NetAct
      
    功能：
      - 工参批量管理
      - 变更追踪
      - 版本控制
      - 自动化校验

五、一句话总结

工参是基站天线的"灵魂"——一个错误的方位角或下倾角就能让整个小区的覆盖偏离预期。5G AAU 的工参更复杂（多了波束参数），但自动化程度也更高。记住：工参管理不是"录完就完"，需要定期核查和校准。
