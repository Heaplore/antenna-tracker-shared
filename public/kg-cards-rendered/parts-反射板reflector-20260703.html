反射板 Reflector（抛物面反射器） Reflector

反射板是天线的"聚光灯反光碗"——它把振子发出的球面波反射成平面波，大幅提高增益和方向性。抛物面反射器是卫星通信、雷达和射电天文的核心部件。

类比入口

反射板就像手电筒的反光碗——灯泡（振子）向四面八方发光，反光碗（反射板）把光集中到一个方向。抛物面反射器的精妙之处在于它的几何形状——所有从焦点发出的光线，反射后都平行于轴线射出。

二、抛物面反射器

2.1 几何原理

抛物线方程：y² = 4fx
    
    f = 焦距
    D = 口径直径
    F/D = 焦距口径比（通常 0.3-0.6）
    
    所有从焦点 F 发出的射线，经抛物面反射后平行于轴

2.2 增益计算

G = η_aperture · (πD/λ)²
    
    η_aperture = 口径效率（通常 0.55-0.75）
    D = 口径直径
    λ = 波长
    
    G_dBi = 10·log₁₀(η_aperture) + 20·log₁₀(D/λ) + 9.5

典型值：

D/ληG (dBi)

5000.6551.3

2.3 口径效率

η_aperture = η_illumination × η_spillage × η_phase × η_polarization
    
    η_illumination = 照明效率（馈源方向图匹配）：0.7-0.9
    η_spillage = 溢出效率（能量不溢出反射面）：0.7-0.9
    η_phase = 相位误差效率：0.9-1.0
    η_polarization = 极化效率：0.95-1.0
    
    总效率：0.55-0.75（典型）

2.4 馈源选择

馈源必须覆盖整个反射面，但不能溢出太多：
    
    F/D = 0.3 → 馈源波束窄，溢出少，但照明不均匀
    F/D = 0.5 → 平衡设计（最常见）
    F/D = 0.6 → 馈源波束宽，溢出多，但照明均匀

馈源类型：

三、栅格反射器

3.1 结构

完整反射面 → 金属网格（栅格）
    栅条间距 s < λ/10 时，等效于完整反射面

3.2 优势

重量减轻：70-90%
    风阻降低：80%
    成本降低：50%

3.3 设计约束

栅条间距 s < λ/10
    
    例如：30GHz → s < 1mm
          10GHz → s < 3mm
          3GHz → s < 10mm

四、平面反射板（Reflector Plate）

4.1 微带天线 + 反射板

贴片天线上方加反射板：
      - 抑制后向辐射 → FBR ↑
      - 反射能量向前 → 增益 ↑ 3-6dB
      - 形成谐振腔 → 带宽 ↓

4.2 反射板距离

反射板与贴片的距离 h：
    
    h = λ/4 → 最佳增益提升（谐振）
    h < λ/4 → 增益 ↓，但带宽 ↑
    h > λ/2 → 出现栅瓣
    
    典型值：h = 0.15-0.25λ

五、龙勃透镜 vs 抛物面反射器

抛物面反射器：
      优点：增益高、方向性好
      缺点：体积大、需要精确对准
    
    龙勃透镜：
      优点：宽波束、多目标跟踪
      缺点：增益较低、体积大

六、工程注意事项

6.1 制造精度

表面精度要求：σ < λ/16（Ruze 公式）
    
    σ = 表面均方根误差
    
    例如：30GHz（λ = 10mm）→ σ < 0.6mm
          100GHz（λ = 3mm）→ σ < 0.2mm

6.2 热变形

温度变化 → 反射面变形 → 增益下降
    
    ΔG ≈ -8·(4πσ/λ)² [dB]
    
    σ = α·ΔT·L
    α = 热膨胀系数
    ΔT = 温差
    L = 特征尺寸

6.3 积雪/结冰

户外反射器需考虑：
      积雪覆盖 → 表面粗糙度 ↑ → 增益 ↓
      结冰 → 介电常数变化 → 相位误差 ↑

七、一句话总结

反射器是天线的"能量聚光灯"——抛物面反射器把振子发出的球面波变成平面波，增益可达 30-50dBi。栅格反射器减轻重量，平面反射板增强微带天线。记住三个关键：① 增益与 (D/λ)² 成正比 ② 口径效率 55-75% ③ 表面精度必须 < λ/16。
