天线测试与测量方法

天线测试分三大类：远场测试（最直接但暗室大）、近场测试（暗室小但要数学变换）、混响室测试（最快但只能测总量）。5G AAU 出厂用远场/混响室，研发用近场。校准是精度保障——VNA 校准用 SOLT/TRL 方法。

类比入口

天线测试就像"体检"——远场测试是"全身 CT"（看得最全但设备贵），近场测试是"局部超声"（便宜但要医生解读），混响室测试是"量血压"（快但信息少）。校准就是"校准血压计"——不准的话测什么都没用。

一、远场测试（Far-Field Measurement）

1.1 原理

远场条件：
      距离 R > 2D²/λ
      
      D = 天线最大尺寸
      λ = 波长
      
      例如 64T64R AAU（D=1.2m）@ 3.5GHz（λ=86mm）：
      R > 2 × 1.2² / 0.086 = 33.5m
      
      → 远场暗室需要 35m 以上！

1.2 测试配置

1. 自由空间法（Two-Antenna Method）
       DUT ↔ 参考天线
       测量 S21 → 计算增益
       
       G_DUT = G_REF + 20log₁₀(4πd/λ) - 20log₁₀(|S21|)
       
    2. 三天线法（Three-Antenna Method）
       三对天线两两测量 S21
       解方程组求各天线增益
       
    3. 已知增益法
       用标准增益喇叭（SGB）作参考
       精度最高

1.3 测试项目

项目设备方法
增益VNA/SG+频谱仪比较法/绝对法
方向图转台+探头旋转扫描
极化隔离VNA交叉极化测量
VSWR/回波损耗VNAS11 测量
前后比转台+探头180° 旋转
旁瓣电平转台+探头全方向扫描

1.4 远场暗室类型

1. 开阔场（OATS - Open Area Test Site）
       半电波暗室/全电波暗室
       地面反射板+吸波材料
       频率：30MHz-1GHz
       
    2. 电波暗室（AWR - Anechoic Windown Room）
       全吸波材料覆盖
       频率：1GHz-40GHz
       尺寸：10m-50m
       
    3. 球面近远场变换暗室
       近场扫描 + 数学变换 → 远场
       适合大型天线

1.5 QMS 关键点

测量不确定度来源：
      - 距离误差（±0.1m → ±0.1dB 增益误差）
      - 对准误差（±0.5° → ±0.2dB 增益误差）
      - 多径反射（吸波材料老化 → ±0.5dB）
      - 电缆损耗变化（弯曲/温度 → ±0.1dB）
      - VNA 校准残余（SOLT 残余误差 → ±0.05dB）
      
    控制措施：
      - 定期校准暗室（标准喇叭验证）
      - 电缆固定防弯曲
      - 温湿度控制（23±2°C, 50±5%RH）
      - 测量人员持证上岗

二、近场测试（Near-Field Measurement）

2.1 原理

近场扫描：
      探头靠近 DUT（距离 < 2D²/λ）
      测量近场电磁分布
      通过数学变换 → 远场方向图
      
    变换方法：
      1. 球面近场变换（Spherical NF-FF）
         最准确，适用所有天线
         复杂度：高
         
      2. 柱面近场变换（Cylindrical NF-FF）
         适用长线阵/柱面对称天线
         复杂度：中
         
      3. 平面近场变换（Planar NF-FF）
         适用平面天线（如 AAU）
         速度快，最常用

2.2 平面近场扫描

扫描方式：
      探头在 DUT 前方平面扫描（X-Y 网格）
      
    网格间距：
      Δx, Δy ≤ λ/2（采样定理）
      
    扫描距离：
      z = 0.5-2m（取决于天线尺寸）
      
    变换公式（FFT）：
      E_far(θ,φ) = FFT{E_near(x,y)}
      
    优势：
      - 暗室只需 3-5m
      - 扫描速度快（电机直线运动）
      - 适合平面天线（AAU/面板天线）
      
    劣势：
      - 变换引入误差
      - 不适用于所有天线类型

2.3 近场测试设备

设备规格作用
探头双极化喇叭/小环接收近场电磁场
扫描平台X-Y-Z 三轴探头精确定位
VNA4-8 端口S 参数测量
控制器PC + 软件数据采集+NF-FF 变换
暗室3-5m 电波暗室屏蔽+吸波

2.4 QMS 关键点

测量不确定度来源：
      - 探头定位误差（±0.1mm → ±0.05dB）
      - 探头校准误差（±0.1dB）
      - 网格间距不足（欠采样 → 混叠）
      - NF-FF 变换截断误差（±0.2dB）
      
    控制措施：
      - 探头定期校准（标准源）
      - 网格间距 ≤ λ/2
      - 扫描区域 > 天线尺寸 2 倍
      - 多次扫描取平均

三、混响室测试（Reverberation Chamber）

3.1 原理

混响室：
      金属封闭腔体 + 搅拌器（Mode Stirrer）
      
    工作原理：
      搅拌器旋转 → 改变腔内边界条件
      → 形成"统计均匀"的多径环境
      → 天线在"虚拟自由空间"中测试
      
    测量：
      - TRP（Total Radiated Power）：总辐射功率
      - TIS（Total Isotropic Sensitivity）：总接收灵敏度
      - MIMO 容量
      - 通道互易性

3.2 测试流程

1. 空载校准：无 DUT 时测量搅拌特性
    2. 加载 DUT：安装被测天线
    3. 搅拌扫描：搅拌器旋转 N 次（N=100-500）
    4. 数据采集：每次搅拌位置记录 S 参数
    5. 统计分析：平均 → TRP/TIS

3.3 混响室 vs 远场 vs 近场

特性远场近场混响室
暗室尺寸大（15-50m）中（3-5m）小（2-3m）
测试速度慢（30-60min）中（10-30min）快（5-15min）
测试内容全参数全参数TRP/TIS 为主
精度最高高中
成本高中低
适用场景研发/认证研发产线/批量

3.4 QMS 关键点

测量不确定度来源：
      - 搅拌器位置不足（< 200 位置 → ±1dB）
      - 腔体模式密度不足（频率太低 → 统计不均匀）
      - DUT 加载影响（金属支架/电缆 → 偏差）
      - 温度漂移（腔体尺寸变化 → 频率偏移）
      
    控制措施：
      - 搅拌器位置 ≥ 200
      - 工作频率 > 最低有效频率（FREQ）
      - 使用非导电支架
      - 温度控制 23±2°C

四、VNA 校准方法

4.1 SOLT 校准（Standard Open-Load-Through）

校准件：
      1. Open（开路）：已知电容模型
      2. Load（短路）：50Ω 负载
      3. Short（短路）：理想短路
      4. Through（直通）：零长度连接
      
    校准步骤：
      1. 连接 Open 到端口 1
      2. 连接 Load 到端口 1
      3. 连接 Short 到端口 1
      4. 连接 Through（端口 1 ↔ 端口 2）
      5. 执行校准算法
      
    精度：
      - 单端口：±0.02dB / ±1°
      - 双端口：±0.05dB / ±2°
      
    适用：
      - Sub-6GHz 天线测试
      - 频率 < 20GHz

4.2 TRL 校准（Thru-Reflect-Line）

校准件：
      1. Thru（直通）：零长度
      2. Reflect（反射）：开路/短路/非对称
      3. Line（延迟线）：已知长度传输线
      
    优势：
      - 不需要精确负载（Load）
      - 高频精度更高
      - 适合波导/同轴测试
      
    精度：
      - 频率 < 67GHz：±0.02dB / ±1°
      
    适用：
      - 毫米波天线测试
      - 波导器件测试

4.3 LRM 校准（Line-Reflect-Match）

校准件：
      1. Line（延迟线）
      2. Reflect（反射）
      3. Match（匹配负载）
      
    优势：
      - 比 SOLT 简单（不需要 Short）
      - 比 TRL 精度高
      
    适用：
      - 中频天线测试（1-40GHz）

五、5G AAU 天线测试流程

5.1 出厂测试流程

1. 外观检查（AOI）
       - 尺寸/颜色/标识
       - 表面缺陷
       
    2. S 参数测试（VNA）
       - S11（回波损耗）
       - S21（插入损耗）
       - 端口隔离度
       
    3. 有源测试（远场/近场）
       - 增益（比较法）
       - 方向图（全扫描）
       - 波束宽度
       - 前后比
       - 旁瓣电平
       
    4. MIMO 测试
       - 通道一致性（幅度/相位）
       - 空间流数
       - 预编码验证
       
    5. PIM 测试
       - 双三载波 PIM
       
    6. 极化测试
       - 交叉极化隔离度
       
    7. 环境测试（抽样）
       - 高低温循环
       - 盐雾测试
       - 振动测试

5.2 测试时间

测试项目时间自动化
S 参数10min全自动
远场有源30min半自动
近场扫描15min全自动
MIMO10min全自动
PIM5min全自动
极化5min半自动
总计75min

六、一句话总结

天线测试方法选择取决于"精度-速度-成本"的权衡——远场最直接但暗室大，近场暗室小但要数学变换，混响室最快但信息少。5G AAU 出厂测试用"远场+混响室"组合：研发用远场/近场验证全参数，产线用混响室快速测 TRP/TIS。记住：测试精度取决于校准质量——VNA 校准是基础，暗室校准是保障，探头校准是关键。
