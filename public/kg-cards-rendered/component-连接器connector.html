射频连接器 RF Connector Connector

射频连接器是天线的"接口"——它把天线和馈线、天线和测试设备连接在一起。核心指标是工作频率、插入损耗、VSWR 和功率容量。频率越高，连接器越小（SMA → 2.92mm → 2.4mm → 1.85mm → 1.0mm）。

类比入口

射频连接器就像 USB 接口——不同尺寸的 USB 插头对应不同的数据传输速度。射频连接器也一样：SMA 就像 USB 2.0（够用但有限），2.92mm 就像 USB 3.0（更快），1.0mm 就像 USB 4（极致速度但娇贵）。

一、定义与功能

1.1 射频连接器的作用

天线 ↔ 连接器 ↔ 馈线/电缆 ↔ 连接器 ↔ 测试设备/滤波器/LNA/PA

核心功能：

  1. 电气连接：传递射频信号
  2. 阻抗匹配：保持 50Ω 连续
  3. 机械固定：连接天线和电缆/设备
  4. 可拆卸：方便维护和更换

1.2 连接器 vs 焊接

连接器：
      优点：可拆卸、方便测试、可更换
      缺点：增加损耗（+0.1-0.3dB）、可能松动
      
    焊接：
      优点：损耗低、可靠
      缺点：不可拆卸、维修困难

2.2 螺纹 vs 推拉式

螺纹连接器（SMA/N 型）：
      优点：连接可靠、VSWR 好
      缺点：拧螺丝麻烦、多次插拔后磨损
      
    推拉式连接器（Push-on SMA）：
      优点：快速连接
      缺点：可靠性稍差、不适合振动环境
      
    卡口式（BNC）：
      优点：快速旋转锁定
      缺点：频率上限低（< 4GHz）

2.3 平板连接器（Board Mount）

适用于 PCB 直接焊接：
      SMPM / SSMC：≤ 12GHz
      2.4mm board mount：≤ 50GHz
      1.85mm board mount：≤ 67GHz

2.4 无连接器设计（Wire Bond / Flip-Chip）

毫米波 AiP 中常用：
      天线直接 wire bond 到芯片引脚
      省去连接器损耗（0.1-0.3dB）
      但不可拆卸

三、关键指标

3.1 插入损耗（Insertion Loss）

IL = IL_conductor + IL_dielectric + IL_reflection
    
    典型值：
      SMA：0.1-0.3dB（每对连接器）
      2.92mm：0.05-0.15dB
      1.85mm：0.03-0.08dB

3.2 电压驻波比（VSWR）

VSWR = (1+|Γ|)/(1-|Γ|)
    Γ = (Z_L - Z_0)/(Z_L + Z_0)

3.5 频率上限的决定因素

频率上限主要由以下因素决定：
      1. 高阶模激发（TE11 模）
      2. 连接器公差累积
      3. VSWR 恶化
      4. 插入损耗增大

高阶模截止频率：

同轴线的 TE11 模截止：
    f_c ≈ 1.586 · c / (π · D · √ε_r)
    
    D = 外径
    c = 光速
    ε_r = 介质常数

四、5G 连接器应用

4.1 基站 AAU 连接器

Sub-6GHz 基站：
      N 型 / SMA 连接器
      频率：700MHz - 6GHz
      功率：10-100W
      
    毫米波基站：
      2.92mm / 2.4mm 连接器
      频率：24-40GHz
      功率：0.1-1W
      或无连接器设计（板载焊接）

4.2 手机天线连接器

手机内部：
      IPEX / U.FL 连接器
      频率：DC - 6GHz（Sub-6GHz）
      尺寸：1.0×1.5mm（极小）
      
    毫米波手机（未来）：
      板载焊盘 / flip-chip
      频率：24-40GHz
      无连接器（直接焊接到 PCB）

4.3 测试连接器

5G 测试设备：
      2.92mm（≤ 40GHz）
      2.4mm（≤ 50GHz）
      1.85mm（≤ 67GHz）
      
    关键要求：
      - 低 VSWR（< 1.05）
      - 低 IL（< 0.1dB）
      - 高重复性（多次插拔后性能不变）

五、连接器选择指南

5.1 按频率选择

f < 3GHz → N 型 / SMA
    3GHz < f < 18GHz → SMA / 3.5mm
    18GHz < f < 40GHz → 2.92mm
    40GHz < f < 67GHz → 2.4mm / 1.85mm
    f > 67GHz → 1.85mm / 1.0mm / 波导

5.2 按功率选择

P < 1W → 任何连接器
    1W < P < 10W → SMA / 2.92mm
    10W < P < 100W → N 型 / SMA（大尺寸）
    P > 100W → N 型 / 7/16 DIN

5.3 按环境选择

室内 → SMA / 2.92mm
    室外 → N 型（防水密封）
    振动环境 → 螺纹连接器（非推拉式）
    洁净室 → 卡口式（BNC）

六、工程注意事项

6.1 连接器的维护

清洁：
      使用无尘布 + 异丙醇清洁中心针和外导体
      灰尘 → VSWR 恶化 → 信号反射
      
    扭矩：
      SMA：2-5 in-lbs（过紧损坏螺纹）
      N 型：15-20 in-lbs
      使用扭矩扳手，不要凭手感！

6.2 转接器和延长线

每个转接头增加：
      IL：0.02-0.1dB
      VSWR 恶化：0.02-0.05
      
    5G 毫米波测试中：
      尽量减少转接头数量
      使用低损耗电缆（如 LMR-400）

6.3 连接器与 PCB 的过渡

SMA 板载连接器：
      中心针 → 微带线过渡
      外导体 → 接地平面
      
    关键设计：
      - 过渡区域阻抗匹配
      - 避免阶跃（step discontinuity）
      - 足够的接地过孔

七、一句话总结

射频连接器是天线的"接口"——SMA 是最通用的（≤ 18GHz），毫米波用 2.92mm/2.4mm/1.85mm（越来越小、越来越精密）。核心指标就四个：频率上限（决定用哪种）、VSWR（决定匹配好坏）、IL（决定损耗多少）、功率容量（决定能不能承受）。记住：连接器越多，损耗越大；毫米波连接器比 SMA 贵 10-100 倍，但不可替代。
