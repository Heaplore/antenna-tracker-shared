天线阵列 Antenna Array

天线阵列是将多个独立天线单元按一定几何排列组合，通过控制各单元的幅度和相位，使辐射方向图按需求成形和指向的综合性天线系统。它是现代通信、雷达、卫星导航等领域几乎所有高级天线技术的基础。

类比入口

想象一群人站在广场上各自喊话——如果每个人都随意喊，声音杂乱无章；但如果大家按指挥的节奏同时开口，声波就能叠加成一个方向的声音，让对面的人听得特别清楚。这就是天线阵列的原理：每个天线单元"喊"一个电磁波，通过精确控制它们的相位关系，让波在需要的方向"叠加增强"，在不需要方向"抵消减弱"。

一、基本定义与原理

1.1 定义

天线阵列（Antenna Array）是由两个或多个相同或不同的辐射单元按照一定的几何形状和排列方式组成的天线系统。阵列的核心思想是空间信号处理——利用多个天线单元的空间分布特性，通过幅度加权（Amplitude Weighting）和相位控制（Phase Control），实现对辐射方向图的灵活调控。

2.2 基本原理：阵列因子理论

天线阵列的总辐射方向图由两个部分组成：

总方向图 = 单元方向图 × 阵列因子（Array Factor, AF）
  • 单元方向图（Element Pattern）：单个天线单元自身的辐射方向特性
  • 阵列因子（AF）：由多个单元的空间排列、激励幅度和相位决定的方向图

阵列因子的数学表达式（均匀线性阵列）：

AF(θ) = Σ(n=0 to N-1) [An · exp(j·n·ψ)]

其中：

  • N = 阵列单元数量
  • An = 第n个单元的激励幅度
  • ψ = kd·cos(θ) + α，k为波数（2π/λ），d为单元间距，α为相邻单元间的相位差

这个公式揭示了天线阵列的本质：它是一个空间采样系统，单元间距d决定了空间采样率，相位差α决定了波束指向。

1.3 核心物理机制

  1. 相长干涉：当电磁波到达某方向时相位相同，信号叠加增强
  2. 相消干涉：当电磁波到达某方向时相位相反，信号相互抵消
  3. 波束扫描：通过改变馈电相位梯度，电子控制波束指向，无需机械转动

二、核心技术与公式

2.1 关键公式

公式含义说明
波束指向角θ₀ = arccos(-α / kd)α为相邻单元相位差，d为单元间距
半功率波束宽度HPBW ≈ 0.886·λ / (N·d·cosθ₀)N为单元数，HPBW越小波束越窄
阵列增益G ≈ N·Ge理想情况下增益与单元数成正比（Ge为单元增益）
栅瓣条件d/λ ≥ 1/(1+sinθₛ)超过此间距会出现栅瓣（Grating Lobe）
第一零点波束宽度FNBW ≈ 2λ / (Nd)主瓣两侧第一个零点之间的角度

2.2 阵列权重函数（窗函数）

与光学中的傅里叶变换类似，阵列单元的幅度分布决定了旁瓣电平：

  • 均匀分布（所有单元等幅）：波束最窄，但旁瓣最高（-13.2 dB）
  • 泰勒分布（Taylor Weighting）：旁瓣可控（-20 至 -40 dB），波束稍宽
  • 切比雪夫分布（Chebyshev）：等旁瓣电平最优分布
  • 汉宁/海明分布（Hanning/Hamming）：旁瓣抑制好，但波束展宽明显

2.3 互耦效应（Mutual Coupling）

阵列中相邻单元之间通过近场耦合相互影响，导致：

  • 单元输入阻抗随扫描角变化（Active Impedance Variation）
  • 方向图畸变
  • 增益下降

互耦效应在密集阵列（d < 0.5λ）中尤为显著，是工程设计的核心挑战之一。

三、实现方式与分类

3.1 按几何排列分类

类型结构特征典型应用
线性阵列（Linear Array）单元沿直线排列基站扇区覆盖、雷达扫描
平面阵列（Planar Array）单元在二维平面上排列卫星通信、5G Massive MIMO
圆形阵列（Circular Array）单元分布在圆周上全向扫描、测向系统
球形阵列（Spherical Array）单元分布在球面上360°全覆盖、近场测量
共形阵列（Conformal Array）贴合载体曲面飞机/导弹外壳天线

3.2 按波束形成方式分类

类型原理优缺点
模拟波束赋形（Analog BF）用移相器和功分器在RF域形成波束硬件简单、成本低，但波束灵活性差
数字波束赋形（Digital BF）每个单元配独立RF链，在基带数字域处理波束灵活、可多波束，但成本高、功耗大
混合波束赋形（Hybrid BF）模拟+数字两级处理，折中方案5G mmWave主流方案，平衡成本和性能

3.3 按有源/无源分类

  • 无源阵列（Passive Array）：只有一个发射机/接收机，通过移相网络分配到各单元
  • 有源阵列（Active Array / AESA）：每个单元或子阵配有独立的发射/接收模块（T/R Module）
  • T/R模块通常采用GaAs或GaN工艺
  • 支持真正的多波束同时工作
  • 单点故障不影响整体性能（冗余设计）

六、工程挑战与坑

6.1 栅瓣（Grating Lobe）问题

当单元间距 d > λ/2 时，在波束扫描过程中会出现与主瓣强度相当的"镜像瓣"，导致能量浪费和方向混淆。工程铁律：在最大扫描角θ_max处，需满足 d ≤ λ/(1+sinθ_max)。对于±60°扫描的阵列，间距需≤λ/2。

6.2 互耦引起的阻抗失配

阵列中单元越密，互耦越强，导致：

  • 单元输入阻抗偏离设计值（通常从50Ω漂移）
  • VSWR恶化，反射增大
  • 实际增益低于理论值

解决方案：去耦结构设计（如defected ground structure、电磁带隙EBG结构）、单元间距优化。

6.3 相位误差累积

实际系统中，每个T/R通道的相位和幅度不可能完全一致：

  • 电缆长度差异引入相位偏差
  • 温度变化导致移相器漂移
  • ADC/DAC量化精度有限

相位误差会导致波束指向偏移、旁瓣升高。校准是关键环节，通常在出厂时做一次全通道校准，运行中定期自校准。

6.4 散热问题

有源阵列（尤其AESA）的T/R模块功耗巨大。一个64T64R的5G AAU，整机功耗可达数百瓦，散热设计直接影响可靠性和寿命。

6.5 校准复杂度

大规模阵列的校准是一个"鸡生蛋"问题：要校准就需要知道真实方向图，但方向图本身又受校准精度影响。业界通常采用基于近场测量的迭代校准方案。

七、技术演进

  • 1905年：K.F. Braun首次描述定向天线概念
  • 1937年：K. Küpfmüller首次提出通过相位控制实现波束扫描的理论
  • 1944年：德国Mammut雷达建成首批相控阵天线（3D防空雷达）
  • 1956-1961年：MIT林肯实验室开发AN/FPS-85（首批大型相控阵雷达），Hughes研发T/R模块概念
  • 1970s-1980s：AESA技术成熟，AN/APG-63成为首款量产战斗机AESA雷达；AN/SPY-1相控阵雷达装备宙斯盾战舰
  • 1990s：卫星通信相控阵终端商业化；自适应阵列（LMS算法）广泛用于抗干扰
  • 2000s：标准化T/R模块（SMTR®）出现，GaN技术大幅提升效率
  • 2010s：5G Massive MIMO兴起，相控阵从军事走向民用大规模商用
  • 2016年：诺基亚和爱立信推出首批5G相控阵基站
  • 2019年：SpaceX Starlink相控阵用户终端量产
  • 2020s：6G太赫兹阵列、AI辅助波束管理、智能超表面（RIS）与阵列融合
  • 2025-2026年：数字智能阵列、全平面覆盖天线、共享孔径技术成为研究热点

八、技术成熟度

  • 成熟度等级：大规模商用
  • 说明：天线阵列技术本身已高度成熟（军事领域应用超过60年）。在5G基站Massive MIMO、卫星相控阵终端、汽车毫米波雷达等领域已进入大规模商用阶段。6G太赫兹阵列和AI辅助波束管理处于原型验证向小规模商用过渡阶段。

九、未来趋势

  • 短期（1-2年）：
  • 5G-Advanced（Rel-18/19）推动更大规模Massive MIMO（256T256R）
  • 6G太赫兹频段阵列设计成为研究重点
  • AI驱动的波束预测和管理在5G-6G中落地
  • 中期（3-5年）：
  • 6G标准确立，通感一体化（ISAC）阵列成为标配
  • 共享孔径技术（同一组天线同时用于通信和 sensing）
  • 封装阵列天线（Package-on-Antenna）集成到芯片封装中
  • 智能超表面（RIS）与相控阵融合，实现低成本波束控制
  • 长期（5年以上）：
  • 全息阵列天线（Holographic MIMO）：连续孔径替代离散单元
  • 量子阵列：利用量子纠缠实现超高灵敏度接收
  • 可编程物质天线：通过材料属性动态变化实现波束控制
  • 6G-7G太赫兹/光频段大规模集成阵列

十、一句话总结

天线阵列是现代天线技术的"操作系统"——从5G Massive MIMO到Starlink相控阵终端，从战斗机AESA雷达到自动驾驶毫米波传感器，几乎所有高性能天线系统都建立在阵列理论之上。掌握阵列因子、波束赋形和互耦控制，就等于掌握了打开现代天线世界的钥匙。
