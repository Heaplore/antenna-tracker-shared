功率放大器 PA Power Amplifier

PA 是发射链路的"发动机"——它把微弱的射频信号放大到足以辐射出去的功率水平。核心指标是输出功率、效率和线性度，三者之间是一个"不可能三角"。

类比入口

PA 就像音响系统的功率放大器——麦克风传来的声音信号只有几毫瓦，PA 把它放大到几十瓦驱动扬声器。但 PA 不只是"放大音量"那么简单：它还要保证放大后的声音不失真（线性度），还要省电（效率），还要能在不同音量下都能正常工作（back-off）。

一、定义与作用

1.1 PA 在发射链路中的位置

基带信号 → DAC → 上变频 → 驱动级 PA → 末级 PA → 滤波器 → 天线
                                        ↑
                                  核心放大
                                  （功率最大）

1.2 PA 的核心功能

  1. 功率放大：将 mW 级信号放大到 W 级
  2. 保持线性：不扭曲调制信号的幅度和相位
  3. 高效率：减少电能浪费和发热
  4. 宽带工作：覆盖 5G 多个频段

3.2 按拓扑结构

Doherty PA（5G 基站标配）

        ┌──[Offset Line]──┐
    Main PA ──→ ──→ Combiner ──→ Output
            │                   │
            └──[Aux PA]──→──────┘
                  ↑
             回退 6dB 时激活

工作原理：

  • Main PA 始终工作
  • Aux PA 在 Main PA 回退 6dB 时激活
  • 通过阻抗调制，Aux 激活时 Main 的效率翻倍

优势：

  • 回退效率高（5G 高峰值比信号的关键）
  • 平均效率比传统 PA 高 2-3 倍

LINC（Linear Amplifier with Non-Constant Envelope）

  输入信号 → 分解为两个恒包络信号
                ↓
      恒包络信号 1 → Class D/SAWPA → 合成 → 输出
      恒包络信号 2 → Class D/SAWPA →

优势：

  • 可以用最高效的开关放大器
  • 线性度好

Outphase（Chireix 组合）

  输入 → 相位分解 → 两个 Class C PA → 组合网络 → 输出

六、热管理

七、工程注意事项

7.1 PA 回退（Back-off）

P_backoff = P_sat - BO

5G 信号的 PAPR 高（10-12dB），PA 必须回退运行：

P_sat = 40dBm
    PAPR = 10dB
    Back-off = 6dB
    P_avg = 40 - 10 - 6 = 24dBm

7.2 预失真（DPD）

数字预失真（DPD）是 5G 基站 PA 的标配：

输入信号 → DPD → PA → 输出
                  ↑
             在线校准

效果：

  • ACLR 改善 10-20dB
  • 允许 PA 更接近饱和工作
  • 效率提升 3-5dB

7.3 PA 与天线的匹配

VSWR > 2:1 → PA 可能损坏（反射功率）
    VSWR > 3:1 → PA 必须关闭保护

保护机制：

  • 检波器监测反射功率
  • 自动回退输出功率
  • 紧急情况下切断 PA

八、技术成熟度

  • 成熟度等级：高度成熟（Sub-6GHz）→ 快速演进（毫米波）
  • 说明：Sub-6GHz PA 技术非常成熟（4G 积累）。毫米波 PA 随 5G FR2 快速成熟。Doherty PA 是 5G 基站的核心技术，GaN Doherty 已大规模商用。

九、一句话总结

PA 是发射链路的"发动机"——输出功率决定信号能传多远，效率决定费多少电，线性度决定信号质量好不好。5G 基站 PA 的核心挑战是高峰值比信号（PAPR 10dB+）下的效率问题，Doherty 拓扑 + DPD 预失真是目前的最优解。记住：PA 和 LNA 是一对镜像——LNA 追求最低噪声，PA 追求最高效率；LNA 在前端，PA 在后端。
