半波偶极子深度解读 Half-Wave Dipole

偶极子是"所有天线的祖先"——两根各 λ/4 的金属棒，中间馈电，增益 2.15dBi，阻抗 73+j42.5Ω（谐振时约 70Ω）。它是 Yagi-Uda、八木天线、基站扇区天线、WiFi 天线的"基本单元"。5G 毫米波把它做成印刷电路板上的弯曲形状，堆成阵列，增益从 2dBi 干到 25dBi。

一、基本结构与原理

1.1 几何结构

        λ/4          λ/4
        ──────────┬──────────
                  │
               馈电点
             (50Ω/75Ω)
                  
        ←────── λ/2 ──────→

总长度：L = λ/2 = c/(2f)

其中 c 是光速（考虑速度因子 v < c，实际长度要打 0.95 左右的折扣系数）。

1.2 电流分布

电流幅度:
        Imax
         │
         │   ╱╲
         │  ╱  ╲
         │ ╱    ╲
         │╱      ╲
    ─────┼────────┼─────
        端点     端点
       (I=0)    (I=0)

正弦分布：

I(z) = I₀ × sin(k(L/2 - |z|))

其中 z 是从中心到端点的距离，k = 2π/λ。

关键特性：

  • 中心电流最大（I₀）
  • 两端电流为零（节点）
  • 电压分布与电流相反（中心电压最小，两端最大）

1.3 辐射方向图

三维形状：甜甜圈（toroid）

二维截面：

  • E 面（含偶极子轴线的平面）：∝ |cos(π/2 × cosθ)| / sinθ
  • H 面（垂直于偶极子轴线的平面）：圆形（全向）

HPBW（半功率波束宽度）：78°（E 面）

方向性：D = 1.64 = 2.15 dBi = 0 dBd

1.4 输入阻抗

在 λ/2 处：

Z_in = 73.1 + j42.5 Ω

谐振时（缩短到约 0.48λ）：

Z_in ≈ 70 + j0 Ω

关键洞察：

  • 73Ω 非常接近 75Ω 同轴电缆 → 可以直接连接，无需匹配
  • 但 73Ω ≠ 50Ω → 与 50Ω 系统连接时 VSWR ≈ 1.46（可接受）
  • 有 +j42.5Ω 感性电抗 → 需要缩短长度来谐振

二、偶极子的主要变种

2.1 折叠偶极子（Folded Dipole）

结构：两根平行的 λ/2 偶极子，两端连接，中间馈电。

    ────────┬────────
                │
             馈电点
                │
        ────────┴────────

阻抗变换：

Z_folded = 4 × Z_dipole ≈ 292 Ω

为什么用 4 倍？：电流在两根导体上平分，电压不变 → 阻抗变为 4 倍（Z = V²/P，功率变为 1/4）。

特点：

  • 阻抗 292Ω → 需要 4:1 巴伦（Balun）变换到 50Ω
  • 带宽比单偶极子宽 2-3 倍（15-30% vs 8-15%）
  • 抗干扰能力强（差分结构）
  • TV 天线、WiFi 天线常用

2.2 V 型偶极子（V-Dipole / Inverted-V）

结构：两根臂向下倾斜一定角度（通常 90-120°）。

        ╲    ╱
             ╲  ╱
              ╳  ← 馈电点
             ╱  ╲
            ╱    ╲

特点：

  • 不需要桅杆支撑中间点
  • 阻抗降至约 50-60Ω（接近 50Ω 系统）
  • 方向图从"甜甜圈"变为"扁甜甜圈"（水平方向增益略有增加）
  • 便携电台、业余无线电常用

2.3 蝴蝶结天线（Bowtie / Biconical）

结构：两根三角形"翅膀"面对面放置。

    △       △
       △△       △△
      △△△     △△△
     ────────┬───────
             │
           馈电点

特点：

  • 带宽极宽（3:1 到 10:1 频率比）
  • 增益仍为 2.15dBi
  • 常用于 UHF/VHF 宽带接收
  • NASA 深空通信使用双锥天线（biconical）变种

2.4 磁电偶极子（Magnetoelectric Dipole, MED）

结构：偶极子 + 微带贴片（E-field + H-field 同时辐射）。

特点：

  • 带宽极宽（> 100%）
  • 增益 6-8dBi（比普通偶极子高）
  • 前后比好（> 20dB）
  • 5G 毫米波阵列的主力单元（IEEE TAP 2019-2021 多篇论文）

2023 IEEE TAP 突破：平行折叠偶极子（PFD）+ 耦合槽馈电

  • 阻抗带宽 64%（17-33.75GHz）
  • 4×4 阵列峰值增益 17.5dBi
  • 总效率 69-82%

2.5 印刷偶极子（Printed Dipole）

结构：在 PCB 上蚀刻出的偶极子形状。

特点：

  • 成本低，易于批量生产
  • 可与微带馈线直接集成
  • 尺寸受基板介电常数影响（L ∝ 1/√ε_r）
  • 5G 毫米波 AiP 的核心单元

2026 前沿：弯曲印刷偶极子（Curved Printed Dipole）

  • RO5880 基板，0.5mm 厚
  • 4 元端射阵列，峰值增益 8.85dBi
  • 覆盖 n257/n258/n261（22.79-29.7GHz）

三、偶极子作为天线阵列的基本单元

3.1 Yagi-Uda 天线（八木天线）

结构：偶极子（有源）+ 反射器 + 1-N 个引向器。

    ← 引向器 ← [偶极子] → 反射器 →

增益：

单元数增益
3 单元（1 反射+1 有源+1 引向）7-8 dBi
5 单元10-11 dBi
8 单元12-13 dBi

特点：

  • 方向性强，前后比好
  • 带宽窄（~5%）
  • 电视天线、业余无线电经典选择

3.2 共线阵列（Collinear Array）

结构：多个偶极子沿轴向堆叠。

    ┌───┐
        │ λ/2│
        └───┘
         │
        ┌───┐
        │ λ/2│
        └───┘
         │
        ┌───┐
        │ λ/2│
        └───┘

增益叠加：

  • 2 单元：5.15 dBi（+3dB）
  • 4 单元：8.15 dBi（+6dB）
  • 8 单元：11.15 dBi（+9dB）

特点：

  • 水平全向，垂直方向图变窄
  • 基站扇区天线常用结构
  • 每个偶极子间距约 0.4-0.5λ

3.3 基站扇区天线

结构：8-16 个偶极子（或其变种）堆叠 + 水平排列。

典型参数（5G AAU）：

参数数值
单元数64（4×16 阵列）
单个偶极子增益2-3 dBi
阵列增益20-25 dBi
HPBW（水平）6-10°
HPBW（垂直）3-6°
前后比> 30dB

五、偶极子 vs 单极子（Monopole）

单极子 = 偶极子 + 镜像

单极子:     偶极子:
        │           │
        │           │
    ─── ───        ─── ───
        │           │
        │           │

关系：

  • 单极子 = 偶极子的一半
  • 地面/金属平面提供镜像
  • 单极子长度 = λ/4（偶极子 = λ/2）

参数对比：

参数半波偶极子λ/4 单极子
长度λ/2λ/4
阻抗73Ω36.5Ω
增益2.15dBi5.15dBi（相对各向同性）
方向图甜甜圈半球
需要地面否是（至少 λ/4）

关键洞察：单极子增益更高（5.15dBi vs 2.15dBi），不是因为辐射更强，而是因为能量被"压"到了半球空间——和偶极子比，它只是少了一半的"浪费方向"。

六、5G 毫米波偶极子阵列设计

6.1 挑战

  1. 尺寸小：28GHz 时 λ/2 ≈ 5.4mm → 加工精度要求高
  2. 损耗大：PCB 基板损耗在毫米波频段显著
  3. 互耦强：单元间距小（~0.5λ），互耦影响阻抗和方向图
  4. 馈电网络复杂：64 路馈电，插入损耗累积

6.2 解决方案

技术方法效果
MED（磁电偶极子）偶极子+贴片带宽>100%，增益 6-8dBi
弯曲偶极子弧形单元端射辐射，前后比>20dB
平行折叠偶极子双平行线+耦合槽带宽 64%，阻抗平坦
3D 打印波导馈电步进波导低损耗，适合 Ka 波段
超表面增强顶部加载 metamaterial增益提升 2-3dB

6.3 典型案例

MDPI 2024 超表面偶极子阵列：

  • 5×7 超表面单元加载
  • 23.2-45.3GHz 带宽
  • 峰值增益 8.02dBi（平均 6.59dBi）
  • 端射辐射，前后比>15dB

IEEE TAP 2023 平行折叠偶极子：

  • 4×4 阵列，17-33.75GHz
  • 峰值增益 17.5dBi
  • 总效率 69-82%
  • 扫描角 ±50°

七、给非专业读者的一句话总结

偶极子是"天线界的基元"——就像乐高积木的最小块。一根 λ/2 的金属棒，增益 2.15dBi，结构简单到任何时代都能造。但把它折叠、弯曲、堆叠、印刷在电路板上，就能从 2dBi 干到 25dBi，从 900MHz 干到 300GHz。5G 基站里 64 根天线，每根的本质都是一个"进化版"的偶极子。

金句：偶极子告诉你一个道理——最简单的结构往往最有生命力。一根棒子 + 一根棒子，解决了人类无线通信 100 年的基本问题。
