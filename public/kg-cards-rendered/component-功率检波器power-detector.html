功率检波器 Power Detector Power-Detector

功率检波器是 T/R 模块功率控制闭环的"传感器"——它将射频信号的幅度转换为直流电压，让基带芯片能实时监测和调整发射功率。核心指标是线性动态范围、精度和响应速度。

类比入口

功率检波器就像汽车的速度表——发动机（PA）在狂飙，但你需要一个仪表告诉你实际速度（功率）是多少，然后踩油门或刹车（调整增益）来保持目标速度。

一、定义与功能

1.1 功率检波器的角色

射频信号 → 功率检波器 → 直流电压（与功率成正比）
                  ↑
             ADC 读取 → 基带芯片 → PID 控制 PA 增益

1.2 主要功能

功能说明
功率监测实时检测发射功率
功率控制闭环为 PID 控制器提供反馈
过载保护功率超限 → 自动降低 PA 增益
温度补偿检测 PA 功率随温度的变化
接收信号强度指示（RSSI）接收通道信号强度监测

二、检波器类型

2.1 峰值检波器（Peak Detector）

原理：
      射频信号 → 二极管 → 电容充电 → 峰值电压
      
    特点：
      - 响应速度快
      - 线性动态范围窄（10-20dB）
      - 适合调制信号的峰值监测

2.2 均值检波器（RMS Detector）

原理：
      射频信号 → 平方律器件 → 低通滤波 → RMS 电压
      
    特点：
      - 精度高（±1dB）
      - 线性动态范围宽（40-60dB）
      - 响应速度慢（需要低通滤波）
      - 适合 CW 信号和功率控制

2.3 对数检波器（Logarithmic Detector）

原理：
      射频信号 → 多级放大器 → 检波级 → 对数输出
      
    特点：
      - 对数输出（dB 线性）
      - 动态范围极宽（60-80dB）
      - 精度 ±1-2dB
      - 5G 功率控制首选

三、关键指标

3.1 动态范围（Dynamic Range）

DR = P_max - P_min [dB]
检波器类型动态范围
峰值检波10-20dB
均值检波40-60dB
对数检波60-80dB

3.2 精度（Accuracy）

精度 = |V_measured - V_true| / V_true
检波器类型精度
峰值检波±3-5dB
均值检波±1-2dB
对数检波±1-2dB

3.3 响应时间（Response Time）

τ = 1 / (2π · f_cutoff)
    
    f_cutoff = 低通滤波器截止频率
检波器类型响应时间
峰值检波< 10ns
均值检波1-10μs
对数检波1-5μs

3.4 工作频率范围

检波器频率范围
峰值检波DC - 6GHz
均值检波100MHz - 6GHz
对数检波10MHz - 8GHz
毫米波检波24-40GHz

四、在 T/R 模块中的应用

4.1 发射功率控制闭环

PA 输出 → 耦合器（-10dB） → 功率检波器 → DC 电压
                                                        ↓
                                                  ADC（基带）
                                                        ↓
                                                  PID 控制器
                                                        ↓
                                                  DAC → PA 增益控制

4.2 功率控制精度要求

5G NR 功率控制精度：
      Sub-6GHz：±1.5dB（3GPP 要求）
      毫米波：±2dB
    
    实现方法：
      1. 高精度对数检波器（±0.5dB）
      2. 温度补偿查找表
      3. 校准补偿（出厂校准）

4.3 过载保护

功率超限检测：
      检波器输出 > 阈值 → 触发保护
      保护动作：
        1. 降低 PA 增益（快速）
        2. 关闭 PA（严重过载）
        3. 报警通知基带
        
    响应时间要求：
      快速保护：< 1μs
      慢速保护：< 10μs

六、工程注意事项

6.1 温度漂移

检波器温度漂移：
      峰值检波：±0.02dB/°C
      均值检波：±0.01dB/°C
      对数检波：±0.015dB/°C
    
    补偿方法：
      1. 内置温度传感器
      2. 查找表补偿
      3. 出厂校准

6.2 负载牵引效应

检波器并联在耦合器上 → 负载效应
    负载效应 → 耦合度变化 → 测量误差
    
    解决方法：
      1. 高输入阻抗设计
      2. 缓冲放大器隔离
      3. 耦合度设计时预留余量

6.3 调制信号的影响

CW 信号：检波器输出稳定
    OFDM 信号：峰值/均值比 = PAPR（峰均比）
      5G NR PAPR = 9-10dB
      检波器需要能处理高峰均比
    
    解决方法：
      1. 使用 RMS 检波器（不受 PAPR 影响）
      2. 峰值检波器 + 保持电路
      3. 对数检波器（天然对数压缩）

七、一句话总结

功率检波器是 T/R 模块功率控制的"眼睛"——它将射频功率转换为直流电压，让基带芯片能实时监测和调整发射功率。对数检波器动态范围最宽（60-80dB），是 5G 功率控制的首选。记住：检波器的精度决定了功率控制的精度，5G NR 要求 ±1.5dB，选检波器时要预留余量。
