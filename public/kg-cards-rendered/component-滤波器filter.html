滤波器 Filter Filter

滤波器是射频前端的"安检门"——只让目标频段的信号通过，把带外干扰和噪声挡在外面。5G 滤波器的主流技术是 SAW（Sub-6GHz）和 BAW/FBAR（高频段），毫米波频段则用波导滤波器或 SIW 滤波器。

类比入口

滤波器就像咖啡滤纸——热水（目标信号）能通过，咖啡渣（带外干扰）被拦住。但射频滤波器的"滤纸"精细得多：它不仅要挡住杂质，还要同时挡住几百个不同频率的干扰信号，而且损耗要小到几乎感觉不到。

一、定义与功能

1.1 滤波器的基本功能

输入信号 → 滤波器 → 输出信号
                ↑
            频率选择性

核心作用：

  1. 选频：只让目标频段通过
  2. 抑制干扰：阻挡带外信号
  3. 保护前端：防止强干扰损坏 LNA/PA
  4. 满足频谱掩模：符合法规要求

二、关键指标

2.1 插入损耗（Insertion Loss, IL）

IL = -10·log₁₀(P_out / P_in) [dB]
滤波器类型IL（典型）说明
波导0.1-0.3dB最低
陶瓷0.3-0.8dB很低
BAW/FBAR0.5-1.5dB低
SAW1-3dB中等
SIW0.5-2dB中等
LC0.5-2dB低频

IL 对系统的影响：

IL = 1dB → 信号损失 20%
    IL = 3dB → 信号损失 50%
    IL = 0.5dB → 信号损失 10%

2.2 带宽（Bandwidth）

相对带宽 = (f_high - f_low) / f_center × 100%
滤波器带宽说明
窄带< 5%基站双工器
中宽5-20%5G Sub-6GHz
宽带> 20%WiFi、SDR

2.3 带外抑制（Rejection）

带外抑制 = 阻带最小衰减 [dB]
应用带外抑制要求
5G 手机> 60dB @ 相邻频段
基站> 80dB @ 相邻基站

卫星通信> 60dB

群时延波动影响数字信号的相位一致性：

5G NR 要求：τ_g_fluctuation < 0.1μs

三、SAW vs BAW 对比

3.1 SAW（Surface Acoustic Wave）

原理：
      电信号 → IDT（叉指换能器）→ 表面声波 → IDT → 电信号
      声波在材料表面传播（速度 ~3000m/s）
参数典型值
频段100MHz - 3GHz
IL1-3dB
Q100-500
功率容量< 23dBm
温度系数较大
成本低

3.2 BAW/FBAR（Bulk Acoustic Wave / Film Bulk Acoustic Resonator）

原理：
      电信号 → 谐振器 → 体声波 → 谐振器 → 电信号
      声波在薄膜中垂直传播（速度 ~5000m/s）
参数典型值
频段1GHz - 10GHz
IL0.5-1.5dB
Q1000-5000
功率容量23-30dBm
温度系数较小
成本中

3.3 选择指南

f < 3GHz → SAW（成本低，性能够用）
    3GHz < f < 7GHz → BAW/FBAR（SAW 性能不够）
    f > 7GHz → 波导/介质滤波器（声学滤波器不适用）

六、工程注意事项

6.1 温度漂移

SAW 温度系数：-20 to -50 ppm/°C
    BAW 温度系数：-5 to -15 ppm/°C
    
    温度变化 50°C → 频率偏移：
      SAW：-0.5% to -1.25%
      BAW：-0.125% to -0.375%

6.2 功率容量

手机滤波器：< 23dBm（200mW）
    基站滤波器：> 40dBm（10W）
    
    高功率下滤波器可能：
      1. 谐振频率漂移
      2. 插入损耗增大
      3. 永久损坏

6.3 滤波器与天线的匹配

滤波器输出阻抗必须与天线输入阻抗匹配
    不匹配 → 反射 → 插入损耗增大 → 抑制恶化

七、技术成熟度

  • 成熟度等级：高度成熟（SAW/BAW）→ 快速演进（毫米波）
  • 说明：Sub-6GHz SAW/BAW 滤波器技术非常成熟（4G 积累）。5G n77/n78/n79 滤波器已大规模商用。毫米波滤波器随 5G FR2 快速成熟。

八、一句话总结

滤波器是射频前端的"安检门"——SAW 便宜但带宽窄（< 3GHz），BAW/FBAR 性能好但贵（3-10GHz），波导滤波器功率容量大但体积大（基站用）。5G 带来了滤波器需求量的暴增（每个频段一个滤波器，Sub-6GHz 手机需要 10-20 个滤波器）。记住：IL 越低越好，抑制越高越好，但两者通常是 trade-off。
