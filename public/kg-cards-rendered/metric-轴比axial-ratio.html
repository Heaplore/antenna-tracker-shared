轴比 Axial Ratio AR

轴比是衡量圆极化或椭圆极化纯度的指标——定义为极化椭圆长轴与短轴之比，理想圆极化 AR = 1（0dB），线极化 AR = ∞。

类比入口

想象一个呼啦圈。如果它正对着你转，你看到的是完美的圆形（AR = 1，理想圆极化）。如果你歪着头看，圆就变成了椭圆——椭圆越扁，短轴越短，轴比越大。当椭圆被压成一条线时（AR = ∞），就变成了线极化。轴比就是告诉你"这个圆有多圆"。

一、定义与物理意义

1.1 数学定义

轴比定义为极化椭圆的长轴（E_major）与短轴（E_minor）之比：

AR = E_major / E_minor

其中：

  • E_major 是极化椭圆长轴幅值
  • E_minor 是极化椭圆短轴幅值

1.3 轴比的 dB 表示

AR(dB) = 20·log₁₀(AR) = 20·log₁₀(E_major / E_minor)

或者用功率表示：

AR(dB) = 10·log₁₀(P_major / P_minor)

工程上常用的验收标准：

  • 卫星通信天线：AR ≤ 3dB（即 AR ≤ 1.41）
  • GPS 天线：AR ≤ 2dB（在波束中心 ±15° 范围内）
  • 雷达圆极化天线：AR ≤ 1.5dB
  • 6G 太赫兹圆极化：AR ≤ 1dB（更严格要求）

二、计算公式

2.1 从电场分量计算

给定两个正交电场分量 E_x 和 E_y：

AR = √[(|Ex|² + |Ey|² + 2|Ex||Ey||cosΔφ]) / (|Ex|² + |Ey|² - 2|Ex||Ey||cosΔφ])]

简化形式（τ 为倾角）：

AR = |cot(τ)|  （当 |τ| ≤ 45°）

2.2 轴比与相位差/幅度差的关系

圆极化的两个条件：

  1. 幅度相等：|E_x| = |E_y|
  2. 相位差 90°：Δφ = ±π/2

当这两个条件被破坏时：

幅度不平衡度：δ = |E_x| / |E_y| - 1
    相位误差：ε = Δφ - π/2
    
    AR ≈ √[(1 + |δ| + |ε|) / (1 - |δ| - |ε|)]

经验法则：

  • 幅度误差 ±1dB → AR ≈ 0.4dB
  • 相位误差 ±5° → AR ≈ 0.6dB
  • 两者叠加时 AR ≈ √(0.4² + 0.6²) ≈ 0.72dB

2.3 轴比与增益的关系

圆极化天线的总增益分解为：

G_total = G_RHCP + G_LHCP

其中 RHCP 分量是期望极化，LHCP 分量是交叉极化。

AR = √(G_RHCP / G_LHCP)

轴比 3dB 时：G_RHCP / G_LHCP = 2，交叉极化分量占 33%。

四、测量方法

4.1 旋转天线法（Two-Antenna Method）

最经典、最常用的方法：

设置：
    - 发射天线：线极化（固定）
    - 接收天线（AUT）：可绕传播轴旋转
    - 距离：远场条件（2D²/λ）
    
    步骤：
    1. 旋转 AUT 360°，记录接收功率 P(θ)
    2. P_max = 最大功率（长轴方向）
    3. P_min = 最小功率（短轴方向）
    4. AR = √(P_max / P_min)  （功率比开方）
       或 AR(dB) = 10·log₁₀(P_max/P_min) / 2

优点：简单直观 缺点：需要远场暗室，旋转机构精度要求高

4.2 两点法（Two-Antenna Cross-Pol Method）

适用于卫星信号测试：

设置：
    - 两个已知 AR 的圆极化参考天线
    - 共同照射同一个圆极化源（卫星）
    
    步骤：
    1. 测量参考天线 A 与 AUT 的共极化和交叉极化隔离度
    2. 测量参考天线 B 与 AUT 的共极化和交叉极化隔离度
    3. 联立方程求解 AUT 的 AR

4.3 叉模法（Cross-Dual Mode Method）

使用双圆极化天线（同时接收 RHCP 和 LHCP）：

AUT（双圆极化）← 照射来自线极化源
    
    测量：
    - P_RHCP：RHCP 端口接收功率
    - P_LHCP：LHCP 端口接收功率
    
    AR(dB) = 10·log₁₀(P_RHCP / P_LHCP)

优点：一次测量得到 AR，无需旋转 缺点：需要双圆极化参考天线

4.4 仅幅度法（Amplitude-Only Method）

2024 年 IEEE 提出的新方法：

仅需测量幅度信息，无需相位：
    1. 固定线极化源
    2. 旋转 AUT 记录 P_max 和 P_min
    3. AR = √(P_max / P_min)

优点：无需相位参考信号，简化测量流程 缺点：精度略低于叉模法

六、工程注意事项

6.1 AR 随角度变化

轴比不是常数——它随观察角度变化：

AR(θ,φ) = f(角度)

通常在波束中心（θ = 0°）AR 最小，偏离中心后 AR 恶化。

6.2 温度对 AR 的影响

温度变化会导致天线结构膨胀/收缩，改变极化纯度：

ΔAR/ΔT ≈ 0.01-0.05 dB/°C（典型值）

对于卫星天线（太空温差 -150°C 到 +120°C），AR 漂移可达 5-10dB，必须在设计时考虑。

6.3 安装误差对 AR 的影响

圆极化天线安装时如果旋转了角度 α：

AR_effective ≈ AR_intrinsic + 20·log₁₀(tan(45° + α))

安装偏差 5° → AR 恶化约 0.87dB 安装偏差 10° → AR 恶化约 1.8dB

6.4 AR 与频响

圆极化天线的 AR 通常随频率变化：

低频 → AR 恶化（相位差偏离 90°）
    中心频率 → AR 最佳
    高频 → AR 恶化（幅度不平衡）

圆极化带宽通常只有 VSWR 带宽的 30-60%。

七、一句话总结

轴比是圆极化天线的"纯度体检报告"——AR = 1 就是满分（理想圆极化），AR 越大说明椭圆越扁，偏离圆极化越远。工程上 AR ≤ 3dB 是及格线，≤ 1dB 是优秀。记住：轴比和 VSWR 是两码事，VSWR 好不代表 AR 好，圆极化天线必须两个都测。
