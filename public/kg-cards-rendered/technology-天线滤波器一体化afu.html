天线滤波器一体化 AFU

AFU（Antenna Filter Unit）= 滤波器直接集成在天线腔体内，省去馈线连接——少一根电缆 = 少两个连接器 = 少两个 PIM 源 + 少 1dB 损耗。5G 多频段基站标配。

类比入口

传统方案：天线 → 馈线 → 滤波器 → 馈线 → 基站（4 个连接器，3 段馈线）。AFU：天线 + 滤波器一体（0 个额外连接器）。就像"咖啡机自带滤杯"vs"咖啡机+外接滤杯"。

一、为什么需要 AFU？

1.1 传统方案的痛点

传统基站架构：
      天线阵列
        │
        ├─→ 馈线（1/2" 低损电缆）
        │     ↓
        │   滤波器（双工器/双频滤波器）
        │     ↓
        │   馈线
        │     ↓
        │   RRU/AAU
        
    问题：
      1. 连接器 PIM：每个连接器 = PIM 源
         - 2 个连接器 → 2 个 PIM 源
         - 双三载波 PIM 累积可能超标
         
      2. 馈线损耗：10m 馈线 @3.5GHz ≈ 2dB
         - 2 段馈线 = 4dB 损耗
         - 4dB = 发射功率降低 60%!
         
      3. 安装复杂度：2 段馈线 + 4 个连接器
         - 安装时间长
         - 安装质量不一致
         
      4. 重量：馈线 + 连接器 ≈ 5-10kg
         - 塔装风载增加

1.2 AFU 的解决方案

AFU 架构：
      天线阵列
        │
        └─→ 滤波器（集成在天线腔体内）
              │
              └─→ 短跳线（< 0.5m）
                    │
                    └─→ RRU/AAU
      
    优势：
      1. 连接器减少 2 个 → PIM 源减半
      2. 馈线减少 2 段 → 损耗减少 ~3dB
      3. 重量减轻 5-10kg → 风载降低
      4. 安装简化 → 一致性提高

二、AFU 结构设计

2.1 滤波器类型

AFU 中常用的滤波器：
    
    1. 双频滤波器（Dual-Band Filter）
       用途：同一天线支持两个频段
       例如：700MHz + 2.6GHz 合一
       结构：两个谐振腔并联
       
    2. 双工器（Duplexer）
       用途：Tx/Rx 频率分离
       例如：FDD 模式，Tx 和 Rx 不同频段
       结构：Tx 滤波器 + Rx 滤波器 + 环形器
       
    3. 多频滤波器（Multi-Band Filter）
       用途：支持 3-4 个频段
       例如：700/900/1800/2600MHz 合一
       结构：多个谐振腔 + 耦合网络
       
    4. 带通滤波器（Band-Pass Filter）
       用途：带外干扰抑制
       例如：抑制邻频干扰

2.2 集成方式

方式 1：腔体集成（主流）
      天线腔体 + 滤波器腔体 = 同一个金属壳
      内部用隔板分隔
      优点：屏蔽好，PIM 低
      缺点：体积大
      
    方式 2：PCB 集成
      滤波器做在 PCB 上（LTCC/SAW）
      贴在天线馈电网络旁边
      优点：体积小，成本低
      缺点：功率容量低，只适合小功率
      
    方式 3：SiP 集成
      滤波器 + 天线 + 馈电网络 = 一个封装
      优点：极致小型化
      缺点：设计复杂，只适合 Small Cell/AiP

2.3 AFU 内部结构

典型 64T64R AFU（双频 700+2600MHz）：
    
    ┌─────────────────────────────────────────┐
    │  天线罩（Radome）                         │
    │  ┌───────────────────────────────────┐   │
    │  │  天线阵列（双频振子）               │   │
    │  │   700MHz 振子（底部）               │   │
    │  │   2600MHz 振子（中部）              │   │
    │  │   3500MHz 振子（顶部）              │   │
    │  └───────────────────────────────────┘   │
    │  ┌───────────────────────────────────┐   │
    │  │  滤波器腔体（集成）                  │   │
    │  │   700MHz 滤波器                    │   │
    │  │   2600MHz 滤波器                    │   │
    │  │   3500MHz 滤波器                    │   │
    │  └───────────────────────────────────┘   │
    │  ┌───────────────────────────────────┐   │
    │  │  T/R 模块 + 馈电网络                │   │
    │  └───────────────────────────────────┘   │
    └─────────────────────────────────────────┘
    
    关键设计：
      - 天线腔体与滤波器腔体之间：电磁隔离
      - 滤波器输入到天线振子：< 0.1m 短距离
      - 滤波器输出到 RRU：1 根 7/16" 电缆

四、一句话总结

AFU 的核心价值就是"少一根线少一个坑"——把滤波器塞进天线腔体，少两段馈线、少四个连接器，损耗降 3dB，PIM 减半，安装快一倍。5G 多频段基站（双频/三频）几乎全部采用 AFU 方案。记住：AFU 不是简单地把滤波器和天线放在一起，而是要解决"电磁隔离"和"热耦合"两个核心问题。
