T/R 模块 Transmit-Receive Module

T/R 模块是 AAU 的"最小作战单元"——每个模块包含 PA、LNA、移相器、衰减器、开关和耦合器，负责一根天线的收发切换、相位控制和信号放大。64T64R AAU 有 128 个 T/R 模块，是 AAU 成本最高的部分（占 BOM 40-50%）。

类比入口

T/R 模块就像士兵的"单兵作战系统"——每个士兵（天线单元）都有自己的武器（PA）、耳朵（LNA）、通信设备（移相器）和护甲（隔离器）。64T64R AAU 就是 128 个士兵组成的连队，指挥员（基带芯片）通过控制每个士兵的相位来实现"齐步走"（波束赋形）。

一、定义与功能

1.1 T/R 模块的组成

TX 路径：
    数字信号 → DAC → 混频器 → 可变衰减器 → PA → 移相器 → 天线
    
    RX 路径：
    天线 → LNA → 移相器 → 可变衰减器 → 混频器 → ADC → 数字信号
    
    控制路径：
    基带芯片 → 控制总线 → 每个 T/R 模块的寄存器

1.2 核心功能

功能说明
发射放大PA 将小信号放大到所需功率（0-25dBm）
接收放大LNA 放大微弱信号，添加最小噪声
相位控制移相器调节信号相位，实现波束赋形
幅度控制可变衰减器调节信号幅度
收发切换开关或环形器分离 TX/RX 路径
校准内部环路校准，保证通道一致性
功率监测耦合器 + 功率检波器反馈

1.3 T/R 模块 vs 分立器件

分立方案：
      PA + LNA + 移相器 + 开关 + 连接器
      体积大、成本高、一致性差
      
    T/R 模块（MMIC 集成）：
      所有功能集成在 2×2mm 芯片上
      体积小、一致性高、适合大规模阵列

3.3 技术路线对比

方案优点缺点适用场景
分立 GaAs成熟、低成本体积大Sub-6GHz 基站
SiP 集成体积小、一致性好成本高毫米波 AAU
AiP（天线在封装内）最小体积、最低损耗散热难手机/小型基站
BFIC（波束赋形 IC）全集成、数字控制功率低毫米波大规模阵列

四、AAU 中的 T/R 模块部署

4.1 64T64R AAU

64 发射通道 × 64 接收通道 = 128 个 T/R 模块
    
    每个 T/R 模块驱动 1 个天线单元（贴片）
    天线阵列：8 列 × 16 行 = 128 个单元
    
    总功耗：
      发射：128 × 3W = 384W
      接收：128 × 1W = 128W
      合计：~500W（不含基带）

4.2 成本结构

AAU BOM 成本分配：
      T/R 模块：40-50%
      天线阵列：10-15%
      散热系统：10-15%
      基带芯片：10-15%
      PCB/连接器：5-10%
      其他：5-10%

4.3 散热挑战

Sub-6GHz 64T：
      单通道功耗：3-5W
      总功耗：384-640W
      散热方式：铝合金外壳 + 导热垫 + 风扇
    
    毫米波 64T：
      单通道功耗：1-3W
      总功耗：64-192W
      散热方式：金属基板 + 导热过孔 + 自然对流

五、T/R 模块校准

5.1 为什么需要校准

通道不一致的原因：
      1. 器件公差（PA增益、LNA噪声差异）
      2. 温度漂移（不同通道温升不同）
      3. 频率响应（不同通道频率响应不同）
      4. 老化（长期使用后性能退化）
    
    不校准的后果：
      - 波束指向偏差
      - 旁瓣抬高
      - MIMO 容量下降
      - 干扰增加

5.2 校准方法

内部校准（推荐）：
      1. 每个 T/R 模块有耦合器/环形器
      2. 主通道 → 从通道采样
      3. 基带计算幅度/相位误差
      4. 更新移相器和衰减器
      
    外部校准（OTA）：
      1. 暗室中用标准天线测量
      2. 计算全通道误差矩阵
      3. 下发校准系数

5.3 校准精度要求

幅度误差：< 0.1dB
    相位误差：< 1°
    校准周期：开机时 + 定期（如每 6 小时）

六、工程注意事项

6.1 热设计

T/R 模块是 AAU 的最大热源
    散热设计要点：
      1. MMIC 直接焊接在金属基板上
      2. 导热过孔连接到外壳
      3. 外壳与散热器紧密接触
      4. 考虑风冷/液冷（大功率 AAU）

6.2 电磁兼容

TX 和 RX 通道之间的隔离：
      Sub-6GHz：> 60dB（环形器）
      毫米波：> 40dB（开关）
    
    隔离不足的影响：
      - TX 泄漏到 RX → 接收机饱和
      - 通道间耦合 → 波束失真

6.3 可靠性

MTBF（平均无故障时间）：
      Sub-6GHz T/R：> 100,000 小时
      毫米波 T/R：> 50,000 小时
      
    环境要求：
      温度：-40°C to +65°C
      湿度：0-100% RH
      振动：符合 Telcordia GR-452

七、技术成熟度

  • 成熟度等级：高度成熟（Sub-6GHz GaAs）→ 快速演进（毫米波 SiP/BFIC）
  • 说明：Sub-6GHz T/R 模块已大规模商用（4G/5G 基站），GaAs pHEMT 工艺成熟。毫米波 T/R 模块随 5G FR2 快速成熟，SiP 集成和 AiP 是主流方向。BFIC（波束赋形 IC）将移相器和天线集成，是未来趋势。

八、一句话总结

T/R 模块是 AAU 的心脏——每个模块负责一根天线的收发，包含 PA、LNA、移相器、衰减器和开关。64T64R AAU 有 128 个 T/R 模块，占 BOM 成本 40-50%。Sub-6GHz 用 GaAs MMIC（$2-5/片），毫米波用 SiP/BFIC（$5-15/片）。记住：T/R 模块的一致性决定了波束赋形的精度，校准是保证一致性的关键。
