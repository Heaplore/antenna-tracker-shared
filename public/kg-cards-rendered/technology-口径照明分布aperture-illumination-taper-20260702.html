天线口径照明分布 Aperture Illumination / Taper Distribution

口径照明分布描述了天线孔径上各点的场强（幅度 + 相位）分布，它直接决定了天线方向图的三大核心指标：主瓣宽度、旁瓣电平和增益。没有最好的分布，只有最合适的权衡——压低旁瓣就会加宽主瓣、降低增益，反之亦然。

类比入口

想象一排人举着手电筒照向同一个方向。如果每个人都以同样的亮度（均匀分布），光束最亮但边缘模糊不清（旁瓣高）。如果中间的人最亮、边缘的人逐渐变暗（余弦或泰勒分布），光束边缘干净了（旁瓣低），但整体亮度下降了（增益降低）。这就是口径照明的核心权衡。

一、基本定义与原理

1.1 什么是口径照明分布？

口径照明分布（Aperture Illumination / Taper Distribution）是指天线辐射孔径上场强的空间分布函数：

E_aperture(x, y) = A(x, y) · exp[j·φ(x, y)]

其中：

  • A(x, y) 是幅度分布（taper）
  • φ(x, y) 是相位分布

方向图是口径场的傅里叶变换：

F(θ, φ) = ∬ E_aperture(x, y) · exp[jk·(x·sinθ·cosφ + y·sinθ·sinφ)] dx dy

关键洞察：改变口径上的 A(x, y)，就是在改变方向图的"形状"。

1.2 均匀分布（Uniform / Rectangular）

孔径上场强处处相等：

A_uniform(x, y) = 1,  对于所有 (x, y) ∈ 孔径

方向图（矩形孔径）：

F(θ) = sinc(k·L·sinθ/2) = sin(πL·sinθ/λ) / (πL·sinθ/λ)
指标值说明
峰值旁瓣电平-13.2 dB第一旁瓣
旁瓣总数无穷sinc 函数有无穷多个旁瓣
3dB 波束宽度0.886·λ/L最窄
口径效率100%理论最大
增益最大所有能量集中

优点：增益最大、波束最窄。 缺点：旁瓣太高（-13.2dB），容易受到杂散信号干扰。

1.3 余弦分布（Cosine / Cosine-Tapered）

A_cosine(x) = cos(πx/L),  |x| ≤ L/2
指标值
峰值旁瓣电平-23.0 dB
3dB 波束宽度1.21·λ/L（比均匀宽 37%）
口径效率58%
增益损失-2.37 dB

1.4 余弦平方分布（Cosine-Squared）

A_cos2(x) = cos²(πx/L),  |x| ≤ L/2
指标值
峰值旁瓣电平-32.2 dB
3dB 波束宽度1.50·λ/L（比均匀宽 69%）
口径效率40%
增益损失-3.92 dB

1.5 高斯分布（Gaussian）

A_gaussian(x) = exp[-(x/w)²]

其中 w 是高斯宽度参数，控制 taper 的陡峭程度。

指标值（w = L/2.5）
峰值旁瓣电平-40 dB 或更低
旁瓣特性旁瓣快速滚降（无振荡）
3dB 波束宽度~1.28·λ/L
口径效率~45-55%

特点：高斯分布的傅里叶变换仍然是高斯函数——旁瓣没有振荡，而是单调递减。这在雷达应用中很有优势（旁瓣不会"反弹"）。

1.6 二项式分布（Binomial）

A[n] = C(N-1, n) = (N-1)! / (n! · (N-1-n)!)

对于 N = 4 个单元：A = [1, 3, 3, 1]

指标值
峰值旁瓣电平< -40 dB（理论无旁瓣）
3dB 波束宽度最宽
口径效率最低（20-30%）

特点：二项式分布完全没有旁瓣（方向图只有主瓣），但代价是波束很宽、效率很低。

1.7 Dolph-Chebyshev 分布

Chebyshev 多项式构造的分布，在所有单元等旁瓣的前提下实现最窄波束：

A_CE(θ) = T_N(cos(ψ)·cos(ψ₀)) / T_N(cos(θ₀)·cos(ψ₀))

其中 T_N 是 N 阶 Chebyshev 多项式，ψ₀ 控制旁瓣电平。

旁瓣目标波束展宽因子增益损失口径效率
-20 dB1.15-0.8 dB78%
-30 dB1.35-1.5 dB65%
-40 dB1.60-2.5 dB50%

核心优势：在给定旁瓣电平下，波束最窄。

1.8 Taylor 分布

Taylor 分布在前 N 个旁瓣保持等高水平，然后按 sinc 函数滚降：

A_Taylor(θ) = [sin(π·N·u) / (π·N·u)] · [1 + 2·Σ(n=1 to N_bar) cos(2πn·u/S)]

其中 S 是旁瓣控制参数，N_bar 是保持等高的旁瓣数量。

指标值
旁瓣特性内圈等高，外圈滚降
波束宽度介于 Chebyshev 和高斯之间
口径效率60-75%
灵活性高（可调 N_bar 和 S）

特点：Taylor 是工程中最常用的分布之一——兼顾旁瓣抑制和口径效率。

三、幅度-旁瓣-波束宽度的根本权衡

3.1 物理定律：不可能三角

口径照明分布存在一个不可逾越的权衡三角：

       低旁瓣
             /\
            /  \
           /    \
      窄波束 ------ 高增益

你只能同时满足其中两个，第三个必然妥协：

选择牺牲代表分布
窄波束 + 高增益旁瓣高均匀
低旁瓣 + 高增益波束宽高斯
窄波束 + 低旁瓣增益低Chebyshev/Taylor

3.2 数学本质：不确定性原理

口径照明分布的权衡本质上是傅里叶变换的不确定性原理：

Δx · Δk ≥ 1/2

其中 Δx 是口径上的"宽度"（能量集中度），Δk 是方向图上的"宽度"（波束宽度）。

  • 口径能量越集中（均匀分布）→ 方向图主瓣越窄，但旁瓣越高
  • 口径能量越分散（tapered 分布）→ 方向图旁瓣越低，但主瓣越宽

3.3 pedestal（平台）效应

实际工程中，经常在 taper 分布中加入一个"平台"（pedestal）分量：

A_tapered(x) = (1-p) · A_distribution(x) + p · A_uniform(x)

其中 p 是 pedestal 比例（0 ≤ p ≤ 1）。

效果：

  • p = 0：纯 taper 分布（旁瓣低，波束宽）
  • p = 1：均匀分布（旁瓣高，波束窄）
  • p = 0.1-0.2：折中方案（旁瓣稍高，波束稍窄）

应用：卫星通信天线常用 p ≈ 0.15 的 Taylor-pedestal 组合。

四、相位分布的影响

除了幅度分布，相位分布同样重要：

4.1 均匀相位（理想情况）

φ(x, y) = 0,  所有 (x, y)

方向图由幅度分布唯一决定。

4.2 锥形相位（Tilt）

φ(x, y) = k·x·sinθ₀

导致波束指向偏移 θ₀。

4.3 像差相位（Aberration）

实际系统中，相位不均匀是常态：

φ_actual(x, y) = φ_desired(x, y) + φ_error(x, y)

相位误差对方向图的影响：

RMS 相位误差增益损失旁瓣升高
0°0dB无
10°0.05dB+1dB
20°0.2dB+3dB
30°0.5dB+6dB
45°1.2dB+10dB
60°2.5dB+15dB

Ruze 公式（表面精度与增益的关系）：

η_phase = exp[-(4π·σ/λ)²]

其中 σ 是表面 RMS 误差。

对于 λ = 3cm（10GHz），σ = 1mm：

η_phase = exp[-(4π·0.1/3)²] = exp[-0.175] ≈ 84%  (-0.76dB)

五、圆形孔径的特殊考虑

抛物面天线通常是圆形孔径，其口径照明分布用贝塞尔函数描述：

5.1 圆形孔径的均匀分布

F(θ) = 2·J₁(k·a·sinθ) / (k·a·sinθ)

其中 J₁ 是一阶贝塞尔函数，a 是圆孔径半径。

指标值
峰值旁瓣电平-17.6 dB（第一旁瓣）
3dB 波束宽度1.02·λ/D
口径效率100%

圆形孔径的第一旁瓣（-17.6dB）比矩形孔径（-13.2dB）更高，这是因为贝塞尔函数的振荡特性。

5.2 Hansen 圆形孔径分布

R.C. Hansen 提出的单参数圆形孔径分布，在波束宽度、旁瓣和效率之间取得最优折中：

A_Hansen(ρ) = J₀(v·ρ) · exp(-α·ρ²)

其中 v 和 α 是两个可调参数。

六、应用场景选择指南

6.1 卫星通信

  • 要求：低旁瓣（减少干扰）、适中增益
  • 选择：Taylor 分布（旁瓣 -25 到 -30dB）+ pedestal
  • 原因：卫星之间距离远，旁瓣干扰是主要问题

6.2 雷达

  • 要求：低旁瓣（减少虚警）、窄波束（高分辨率）
  • 选择：Chebyshev 或 Taylor
  • 原因：旁瓣会产生虚假目标回波

6.3 基站天线

  • 要求：适中旁瓣、适中波束宽度、高增益
  • 选择：Cosine 或 Taylor（旁瓣 -20 到 -25dB）
  • 原因：基站天线需要平衡覆盖和干扰

6.4 射电天文

  • 要求：极低旁瓣（减少地面噪声）
  • 选择：高斯或二项式
  • 原因：需要尽可能减少来自地平线方向的干扰

6.5 点对点微波

  • 要求：最大增益、最窄波束
  • 选择：均匀或接近均匀
  • 原因：点对点链路距离固定，旁瓣不重要

七、工程挑战与坑

7.1 馈源照明不均匀

抛物面天线的口径照明由馈源决定：

A_aperture(θ_feed) = G_feed(θ_feed)

馈源的方向图通常不是理想的 taper 分布，导致：

  • 边缘溢出（spillover）：馈源能量照射到反射面之外
  • 中心过亮或过暗：导致口径效率下降

优化方法：选择合适焦距直径比（f/D）和馈源方向图。

7.2 相位中心偏移

馈源的相位中心偏移会导致口径相位误差：

Δφ = k·Δz·(1 - cosθ)

其中 Δz 是相位中心沿轴向的偏移。

影响：波束指向偏移、增益下降、旁瓣升高。

7.3 制造公差

口径上的物理缺陷（面板变形、馈源位置误差）会破坏理想的照明分布：

  • 面板变形 → 相位误差
  • 馈源偏移 → 幅度 + 相位误差
  • 遮挡（副反射面支撑）→ 幅度凹陷 + 相位畸变

7.4 极化纯度

双极化天线中，交叉极化口径分布会污染主极化方向图：

E_total = E_co · p_co + E_cross · p_cross

交叉极化分量会在主瓣附近产生额外的旁瓣结构。

八、技术演进

  • 1940s：均匀分布、余弦分布经典分析
  • 1946年：Dolph-Chebyshev 分布提出
  • 1960年：Taylor 分布提出（NASA）
  • 1976年：Hansen 圆形孔径最优分布
  • 1990s：Pedestal 技术实用化
  • 2000s：AI 辅助照明分布优化
  • 2020s：可重构照明分布（通过 RIS 动态调整）

九、技术成熟度

  • 成熟度等级：成熟
  • 说明：口径照明分布理论已完全成熟，Chebyshev 和 Taylor 分布是工程标准工具。现代进展主要集中在可重构照明和 AI 优化上。

十、一句话总结

口径照明分布是天线的"调色板"——均匀分布追求最大亮度（增益），taper 分布追求干净边缘（低旁瓣），Chebyshev 和 Taylor 是在亮度和干净之间找到最优平衡的艺术。选择哪种分布，取决于你的应用场景更看重什么。
