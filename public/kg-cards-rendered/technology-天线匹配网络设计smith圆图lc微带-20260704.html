天线匹配网络设计 Smith

天线匹配网络 = 把天线阻抗（通常不是 50Ω）变换到 50Ω 的系统阻抗。核心工具是 Smith 圆图——在圆图上画"阻抗轨迹"，用 LC 元件（电感/电容）沿着等电阻/等电抗圆走，走到中心点（50Ω）为止。

类比入口

匹配网络就像"变压器"——天线是 30Ω，系统要 50Ω，中间加个 LC 网络把 30Ω 变成 50Ω。Smith 圆图就是"导航地图"——告诉你往哪个方向走几步能到目的地（50Ω）。

一、匹配网络基础

1.1 为什么需要匹配？

阻抗不匹配的后果：
      1. 反射功率 → 驻波比升高 → 功率回传到 PA
      2. PA 效率下降 → 发热增加 → 寿命缩短
      3. 信号失真 → EVM/ACLR 超标
      4. 互调产物 → PIM 恶化
      
    理想匹配：
      Z_antenna = Z_system = 50Ω
      Γ = 0（反射系数为零）
      VSWR = 1（驻波比为 1）

1.2 反射系数与 VSWR

反射系数：
      Γ = (Z_L - Z_0) / (Z_L + Z_0)
      
      Z_L = 负载阻抗（天线）
      Z_0 = 系统阻抗（50Ω）
      
    回波损耗：
      RL = -20log₁₀|Γ| (dB)
      
    VSWR 与 Γ 的关系：
      VSWR = (1 + |Γ|) / (1 - |Γ|)
      
    换算表：
      ┌─────────────────────────────────┐
      │ VSWR  │  Γ      │  RL(dB) │ 反射功率 │
      ├─────────────────────────────────┤
      │ 1.0   │  0.000  │  ∞      │   0%    │
      │ 1.2   │  0.091  │  20.8   │   0.8%  │
      │ 1.5   │  0.200  │  14.0   │   4.0%  │
      │ 2.0   │  0.333  │   9.5   │  11.1%  │
      │ 3.0   │  0.500  │   6.0   │  25.0%  │
      │ 5.0   │  0.667  │   3.5   │  44.4%  │
      └─────────────────────────────────┘
      
    5G AAU 要求：
      VSWR < 1.5（多数端口）
      RL > 14dB

二、Smith 圆图

2.1 圆图结构

Smith 圆图 = 归一化阻抗圆 + 归一化导纳圆
    
    核心要素：
      1. 中心点：归一化阻抗 z = 1+j0（即 50Ω）
      2. 等电阻圆（R 恒定）：右侧半圆
      3. 等电抗圆（X 恒定）：上下弧线
      4. 纯电抗弧：上半圆（感性）/ 下半圆（容性）
      5. 纯电导弧：下半圆（导纳）
      
    归一化：
      z = Z / Z_0 = (R + jX) / 50
      
      例如 Z = 30 + j40Ω：
      z = 0.6 + j0.8

2.2 在 Smith 圆图上移动阻抗

串联元件（Series）：
      - 串联电感 L → 沿等 R 圆向上（+jX）
      - 串联电容 C → 沿等 R 圆向下（-jX）
      
    并联元件（Shunt）：
      - 并联电感 L → 沿等 G 圆向上（导纳圆）
      - 并联电容 C → 沿等 G 圆向下
      
    操作口诀：
      "串 L 上，串 C 下；并 L 上，并 C 下"
      （导纳圆方向相反）

2.3 匹配步骤（Smith 圆图法）

目标：将 Z_L = 25 - j50Ω 匹配到 50Ω
      
    步骤：
      1. 归一化：z_L = 0.5 - j1.0
      2. 在 Smith 圆图上标出 z_L 点
      3. 选择匹配拓扑（L 型 / π 型 / T 型）
      4. 沿等 R 圆移动，加串联元件
      5. 沿等 G 圆移动，加并联元件
      6. 到达中心点 → 匹配完成
      7. 反归一化 → 计算 L/C 值
      
    L 型匹配（最简单，2 个元件）：
      - 串联 L + 并联 C
      - 或串联 C + 并联 L
      - 仅适用于 Z_L 在特定区域
      
    π 型匹配（3 个元件，更灵活）：
      - 并联 C + 串联 L + 并联 C
      - 可调 Q 值，带宽可控
      
    T 型匹配（3 个元件）：
      - 串联 L + 并联 C + 串联 L
      - 适合低阻抗匹配

三、匹配网络实现方式

3.1 集总参数 LC 匹配

适用：Sub-6GHz（< 6GHz）
    元件：电感（L）+ 电容（C）
    
    计算公式：
      X_L = 2πfL
      X_C = 1/(2πfC)
      
      例如 f = 3.5GHz, 需要 X = j25Ω:
      L = X/(2πf) = 25/(2π×3.5×10⁹) = 1.14nH
      
      需要 X = -j25Ω:
      C = 1/(2πfX) = 1/(2π×3.5×10⁹×25) = 1.82pF
      
    Q 值（品质因数）：
      Q = f₀ / BW = X_L / R = R / X_C
      
      高 Q → 窄带宽 → 选择性好
      低 Q → 宽带宽 → 匹配范围大
      
    5G AAU 匹配网络 Q 值：
      - 单频点：Q = 5-10
      - 多频段：Q = 2-5

3.2 微带线匹配

适用：毫米波（> 6GHz）、PCB 集成
    
    实现方式：
      1. 串联微线段（阻抗变换器）
      2. 并联微带枝节（Stub）
      3. λ/4 阻抗变换器
      4. 阶梯阻抗变换器
      
    λ/4 变换器：
      Z_in = Z₀² / Z_L
      
      例如 Z_L = 100Ω, Z₀ = 50Ω:
      需要 Z_transform = √(50×100) = 70.7Ω
      长度 = λ/4 @ f₀
      
    微带枝节匹配：
      并联短路枝节 → 提供可调电纳
      长度控制 B 值
      
    优势：
      - PCB 集成，无需分立元件
      - 毫米波损耗低
      - 可批量生产
      
    劣势：
      - 带宽较窄
      - 对加工精度敏感

3.3 多频匹配

5G 多频段天线挑战：
      700MHz、2600MHz、3500MHz 三个频段
      → 每个频段阻抗不同
      → 需要宽频或多频匹配
      
    方案 1：宽带匹配
      - 低 Q 值网络（Q < 3）
      - 多节 λ/4 变换器
      - 带宽覆盖 3 个频段
      
    方案 2：多频匹配网络
      - 每个频段独立匹配支路
      - 并联在主线路上
      - 支路在其它频段呈现高阻抗（不影响）
      
    方案 3：自适应匹配
      - 可变电容/电感（MEMS/变容管）
      - 根据频段自动切换
      - 5G-A/6G 方向

四、匹配网络 QMS 关键点

4.1 制造偏差影响

元件公差：
      - 电容：±5%（C0G）/ ±10%（X7R）
      - 电感：±5%（高频专用）
      
    对 VSWR 的影响：
      5% 偏差 → VSWR 从 1.3 劣化到 1.5-1.8
      
    频率偏移：
      1% 元件偏差 → 谐振频率偏移 1%
      3.5GHz → 3.535GHz（影响不大）
      但宽带天线可能影响边缘频段
      
    对策：
      - 生产时微调（trimming）
      - 使用高精度元件（±1%）
      - 匹配网络预留可调范围

4.2 温度漂移

温度系数：
      - C0G 电容：±30ppm/°C（极好）
      - X7R 电容：±15%（-55°C ~ +125°C）
      - 高频电感：±50ppm/°C
      
    3.5GHz 天线 @ -40°C ~ +85°C：
      X7R 电容容值变化 → 匹配偏移 → VSWR 变差
      
    对策：
      - 使用 C0G/NP0 电容（温度稳定）
      - 匹配网络设计预留温度裕量
      - 高温/低温测试验证

4.3 测试验证

出厂测试项目：
      1. S11 测试（VNA）
         - 各频段 VSWR < 1.5
         - 回波损耗 > 14dB
         
      2. 插损测试
         - 匹配网络插入损耗 < 0.5dB
         
      3. 温度循环后复测
         - -40°C → +85°C → 常温
         - VSWR 变化 < 0.2
         
      4. 振动后复测
         - IEC 60068-2-64
         - 元件无位移/脱落

五、一句话总结

匹配网络设计的核心是"用最少元件达到最佳匹配"——L 型 2 元件最简单，π 型 3 元件最灵活，T 型适合低阻抗。Smith 圆图是设计师的"导航地图"，LC 值是计算器给的"坐标"。记住：5G AAU 匹配网络最怕的不是"匹配不好"，而是"温度变了匹配就跑了"——C0G 电容 + 预留裕量是标配。
