天线方向图深度解读 Radiation Pattern

天线方向图是天线在三维空间中的"指纹"——它告诉你信号往哪个方向最强、往哪个方向最弱、旁瓣会不会干扰别人。5G Massive MIMO 的 3D 波束赋形、干扰消除、null steering，全建立在对方向图的精确控制之上。

一、方向图的几何解剖

三大瓣区

天线方向图在球坐标系（θ, φ）中展开，分为三个主要区域：

区域名称含义重要性
主瓣Main Lobe / Beam能量最集中的方向核心关注区域
旁瓣Side Lobe主瓣之外的次要辐射方向干扰源，需抑制
后瓣Back Lobe与主瓣相反方向的辐射通常为旁瓣的一部分

类比：你站在操场中央喊话。正前方声音最大（主瓣），侧面也能听到一些（旁瓣），背后几乎听不到（后瓣/零点）。

关键参数定义

1. HPBW（Half-Power Beamwidth，半功率波束宽度）

主瓣峰值两侧功率下降到一半（-3dB）之间的夹角。

HPBW ≈ 70° × λ / D  （D 为天线孔径，λ 为波长）

典型值：

天线类型频率HPBW（水平/垂直）
全向天线2.4GHz360° × 15°（甜甜圈形）
4G 扇区天线2GHz65° × 8°
5G AAU (64T64R)3.5GHz6° × 3°
5G mmWave28GHz3° × 2°
Starlink 终端14GHz2° × 2°

关键判断：HPBW 越窄，方向性越强，增益越高，但覆盖范围越小。5G Massive MIMO 的波束宽度（6°×3°）比 4G（65°×8°）窄了一个数量级——这是"精准投送"的代价。

2. SLL（Side Lobe Level，旁瓣电平）

旁瓣峰值与主瓣峰值的比值，通常用 dB 表示。

SLL = 20 × log₁₀(E_side / E_main)

典型值：

SLL含义应用场景
-13dB均匀照射（Taylor 分布极限）基础天线阵列
-20dBChebyshev 分布5G 基站广播波束
-25dBDolph-Chebyshev 优化卫星通信
-30dB 以下高精度雷达军用雷达

5G 广播波束设计约束：

  • 水平 3dB 波束宽度：约 36°（约为 4G 的 0.55 倍）
  • 垂直面旁瓣抑制：≥ 13.8dB
  • 增益损失：随波束宽度增大而增加，最大约 13dB

3. Front-to-Back Ratio（前后比）

主瓣方向功率与正后方（180°）功率的比值。

FBR = 10 × log₁₀(P_front / P_back)

典型值：

前后比含义
< 10dB较差，适合全向场景
10-20dB一般，适合定向场景
> 20dB优秀，适合点对点链路

4. Null Depth（零点深度）

方向图中最小值与主瓣峰值的比值。零点越深，干扰抑制能力越强。

Null Depth = 20 × log₁₀(E_null / E_main)

典型值：

零点深度含义
-10dB轻度干扰抑制
-20dB中度干扰抑制
-30dB 以下强干扰抑制（null steering）

二、方向图与增益、方向性的关系

方向性（Directivity）vs 增益（Gain）

很多人混淆这两个概念，它们的区别很关键：

维度方向性 D增益 G
定义天线在最大辐射方向的功率密度与理想各向同性天线的比值考虑了天线效率后的实际增益
公式D = 4π × U_max / P_radG = η × D（η 为效率）
单位dBidBi
关系G ≤ D（因为 η ≤ 1）G = D - 损耗(dB)

典型损耗来源：

  • 导体损耗：0.5-1dB
  • 介质损耗：0.2-0.5dB
  • 驻波比损耗：0.1-0.5dB
  • 总损耗：0.8-2dB

举例：一个 64T64R 5G AAU，方向性 25dBi，效率 90%（-0.45dB），增益 = 25 - 0.45 = 24.55dBi。

方向图的积分关系

D = 4π / ∫∫ F(θ,φ)² dΩ

其中 F(θ,φ) 是归一化方向图函数，dΩ 是立体角元。

物理意义：方向性与方向图的"集中度"成反比——方向图越窄（能量越集中在一个方向），方向性越高。

三、3D 方向图：5G FD-MIMO 的革命

从 2D 到 3D

4G 天线方向图通常是 2D 的——只关心水平面（方位角 φ），垂直面（俯仰角 θ）是固定的机械下倾角。

5G Massive MIMO 引入了 FD-MIMO（Full Dimension MIMO），方向图变成了真正的 3D：

F(θ, φ) = F_azimuth(φ) × F_elevation(θ)

3D 波束赋形的能力：

  • 水平波束指向：±90°（左右各 90°）
  • 垂直波束指向：±15°（动态电下倾）
  • 同时控制水平和垂直波束，实现"3D 精准投送"

3GPP 3D 信道模型

3GPP TR 38.901 的 3D 信道模型考虑了：

  • 到达角（AoA）和 Departure Angle（AoD）的二维分布（方位角 + 俯仰角）
  • 天线方向图的三维响应
  • 交叉极化功率比（XPR）

计算复杂度：相比 2D 模型，3D 模型的计算复杂度增加 3 倍以上，且随天线单元数线性增长。

电下倾 vs 机械下倾

维度机械下倾电下倾（3D 波束赋形）
调整方式物理倾斜天线改变相位权重
调整速度手动，分钟级软件定义，毫秒级
灵活性固定每个用户独立
旁瓣影响旁瓣也跟着倾斜可独立控制旁瓣
典型范围2°-15°±15° 动态可调

动态电下倾的收益：

  • 高层建筑场景：对不同楼层的用户分配不同俯仰角波束
  • 干扰抑制：在干扰方向形成零点（null steering）
  • 容量提升：同一扇区可服务 2-3 倍用户

四、Null Steering：方向图的"雕刻刀"

什么是 Null Steering？

在方向图的干扰方向"雕刻"出深零点（-30dB 以下），让能量不往那个方向辐射。

类比：你用手电筒照一个人，同时用另一只手在他身后放一块挡板，挡住散射光。Null steering 就是用算法在干扰方向"放一块挡板"。

3GPP Rel-18 EIRP 评估

2026 年 arXiv 论文评估了 3GPP Rel-18 标准下的 5G gNB 3D EIRP 轮廓：

核心发现：

  1. 基站干扰不只取决于最差波束方向，还取决于旁瓣产生的各种波束方向
  2. AAS 架构和天线端口配置对 3D EIRP 有决定性影响
  3. 提出的两种 beam nulling 方法可实现 11dB 的目标方向功率降低
  4. 代价：UE 链路 SNR 损失 3.5-4.5dB（在 10⁻⁴ BER 下）

Null Steering 的工程实现

相位-only 控制（MDPI 2019）：

  • 只调整相位，不调整幅度，降低馈电网络复杂度
  • 闭式解（closed-form）估算每个单元的相位
  • 计算成本低，适合实时部署
  • 支持任意几何形状的平面阵列和共形阵列

多用户场景：

  • 同时形成 P 个主瓣和 Q 个零点
  • 目标：在 Ξ = {ξ₁, ..., ξₚ} 方向有最大值，在 Χ = {χ₁, ..., χ_Q} 方向有零点
  • 加权代价函数最小化

五、旁瓣抑制：从 Taylor 分布到神经网络

Taylor 分布

经典的旁瓣抑制方法，通过调整阵列激励幅度分布来控制旁瓣：

α（Taylor 参数）：控制主瓣宽度和旁瓣电平的 trade-off
    - α 大 → 旁瓣更低，但主瓣更宽
    - α 小 → 主瓣更窄，但旁瓣更高

典型值：

αSLL主瓣展宽
2.5-13dB1.0×
3.5-20dB1.2×
4.5-25dB1.5×
5.5-30dB1.8×

Raised Cosine 成形函数（PMC 2025）

2025 年 PMC 论文提出了一种混合解析-优化框架：

核心思想：用 Raised Cosine 函数的滚降因子控制主瓣和旁瓣之间的过渡平滑度。

  • 滚降因子大 → 空间渐变平滑 → 旁瓣更强但主瓣更窄
  • 滚降因子小 → 过渡锐利 → 主瓣更窄但旁瓣更高

实验结果：15 元均匀线性阵列（ULA），半波长间距，初始 SLL = -13.1dB。优化后 SLL = -25dB 以下，同时保持主瓣宽度不变。

神经网络辅助阵列综合

MDPI Antennas 2025：将 Taylor 级数展开与神经网络优化结合。

流程：

  1. 用 Taylor 级数初始化阵列综合参数
  2. 神经网络优化剩余自由度
  3. 目标：指定主瓣宽度 + 指定 SLL + 指定零点位置

优势：比纯数值优化快 10-100 倍，收敛更稳定。

六、方向图在五大场景中的应用

6.1 5G 基站广播波束设计

约束条件：

  • 水平 3dB 波束宽度：~36°（4G 的 0.55 倍）
  • 垂直面旁瓣抑制：≥ 13.8dB
  • 全局垂直面旁瓣抑制：≥ 20dB
  • 增益损失：最大 ~13dB

设计权衡：

  • 波束越宽 → 覆盖越好 → 增益损失越大
  • 旁瓣越低 → 干扰越小 → 主瓣越宽

6.2 毫米波稀疏阵列

挑战：稀疏排列导致栅瓣（grating lobe）出现。

解决方案：

  • 优化单元位置（非均匀间距）
  • 旁瓣抑制算法（如本文提到的 RC 成形函数）
  • 第一旁瓣电平降至 -25dB 以下

6.3 卫星通信

要求：

  • 高方向性（29.9 dBi）
  • 低旁瓣（-14.8dB）
  • 窄波束宽度（Ka-band 波导阵列）

典型配置：

  • 频率：Ka-band（27-30GHz）
  • 孔径：波导阵列
  • 方向性：~30dBi
  • SLL：<-15dB

6.4 雷达系统

要求：

  • 极低旁瓣（<-30dB）
  • 极窄主瓣（<1°）
  • 快速波束切换（微秒级）

Taylor 分布的典型 α 值：4.5-5.5（对应 SLL = -25 到 -30dB）

6.5 6G XL-MIMO

新挑战：

  • 孔径 10m+，远场边界 200m+
  • 方向图不再是简单的平面波响应
  • 需要考虑球面波方向图（spherical wave pattern）
  • 空间非平稳性：不同孔径位置的方向图不同

七、方向图的测量：暗室与近场扫描

暗室测量（Anechoic Chamber）

原理：在无反射的暗室中，旋转天线并测量各方向的辐射强度。

典型流程：

  1. 将 DUT（被测天线）安装在旋转台上
  2. 用标准增益天线作为参考
  3. 在方位角 0°-360°、俯仰角 0°-180° 范围内采样
  4. 绘制极坐标方向图

局限：需要足够大的远场距离 dF = 2D²/λ。对于 6G 的 10m 孔径天线，dF = 200m，暗室需要至少 200 米长——造价极高。

近场扫描（Near-Field Measurement）

原理：在近场区域扫描，通过傅里叶变换转换到远场。

优势：

  • 暗室尺寸要求低（只需近场距离）
  • 适合大孔径天线（6G XL-MIMO）
  • 精度高

流程：

  1. 探针在近场平面（或圆柱面）上扫描
  2. 记录每个采样点的场强和相位
  3. 通过近远场变换（Near-Far Field Transformation）计算远场方向图

八、给非专业读者的一句话总结

天线方向图就是天线在三维空间的"立体照片"——主瓣是它"看"的方向，旁瓣是它"漏看"的方向，零点是它"故意不看"的方向。5G 的 3D 波束赋形，本质上就是在实时"转动"这张照片，让主瓣永远对准用户，零点永远对准干扰。

金句：方向图告诉你一个道理——有时候，"不看哪里"比"看哪里"更重要。null steering 的本质就是学会忽略干扰。
