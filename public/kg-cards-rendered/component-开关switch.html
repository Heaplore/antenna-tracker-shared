开关 Switch

开关是射频电路的"十字路口"——控制信号走哪条路。在 T/R 模块中，收发切换开关（Tx/Rx Switch）是最关键的开关，它在发射时把 PA 连接到天线，在接收时把 LNA 连接到天线，切换速度需要 < 1μs。

类比入口

开关就像铁路道岔——火车（信号）来了，道岔往左扳就走左边轨道（PA→天线），往右扳就走右边轨道（天线→LNA）。5G TDD 系统每秒切换几千次，道岔必须又快又可靠。

一、射频开关类型

1.1 FET 开关（主流）

原理：
      利用 MOSFET/JFET/GaN HEMT 的开关特性
      
    工作状态：
      导通（ON）：Vgs > Vth → 低阻抗（~0.1Ω）
      截止（OFF）：Vgs < Vth → 高阻抗（~1kΩ）
      
    类型：
      SPDT（单刀双掷）：1 输入，2 输出
      SP4T（单刀四掷）：1 输入，4 输出
      2×2/4×4 矩阵开关
      
    优势：
      - 速度快（< 10ns）
      - 体积小
      - 可集成
      - 成本低
      
    劣势：
      - 插入损耗较高（0.5-2dB）
      - 功率容量有限（< 1W）

1.2 MEMS 开关

原理：
      微机电系统——静电驱动可动梁
      
    工作状态：
      导通：静电吸合，金属接触
      截止：释放，断开
      
    优势：
      - 插入损耗极低（< 0.1dB）
      - 隔离度高（> 60dB）
      - 功率容量大（> 1W）
      - 适合毫米波
      
    劣势：
      - 速度慢（1-10ms）
      - 寿命有限（10^7-10^9 次）
      - 成本高
      - 需要高压驱动（50-100V）

1.3 继电器开关

原理：
      电磁继电器——线圈驱动机械触点
      
    优势：
      - 功率容量大（> 10W）
      - 插入损耗低（< 0.1dB）
      - 隔离度高（> 80dB）
      
    劣势：
      - 速度慢（1-10ms）
      - 寿命短（10^5-10^6 次）
      - 体积大
      - 不适合 T/R 模块
      
    适用场景：
      - 测试设备
      - 天线切换（非 T/R 模块）

二、T/R 模块中的开关

2.1 收发切换开关（Tx/Rx Switch）

TDD 系统中的收发切换：
      发射时：PA → Tx/Rx Switch → 天线
      接收时：天线 → Tx/Rx Switch → LNA
      
    切换时序（TDD）：
      保护间隔（Guard Period）：~100ns
      切换速度：< 1μs（必须）
      
    关键指标：
      插入损耗（TX）：< 1dB
      插入损耗（RX）：< 0.5dB
      隔离度（TX→RX）：> 40dB
      切换速度：< 1μs
      功率容量：> 30dBm（PA 输出）

2.2 校准开关

内部校准中的开关：
      校准模式时：
        TX 采样 → 开关 → 耦合器 → 从通道 RX
        从通道 TX → 开关 → 耦合器 → 主通道 RX
      
    开关要求：
      低损耗（< 0.5dB）
      高隔离（> 30dB）
      快速切换（< 10μs）
      
    实现：SP4T/SP8T 多路开关矩阵

2.3 通道选择开关

多通道 AAU 中的开关：
      64T64R → 64 个 Tx/Rx 开关
      每个开关控制 1 个通道的收发
      
    集成方案：
      开关 + 衰减器 + 移相器 → 集成到 T/R 模块
      减少外部开关数量
      降低损耗和尺寸

四、一句话总结

开关是射频电路的"十字路口"——T/R 模块中最关键的是收发切换开关（Tx/Rx Switch），需要 < 1μs 切换速度和 > 40dB 隔离度。FET 开关是 Sub-6GHz 主流（快、便宜），MEMS 开关是毫米波趋势（低损耗、高隔离）。记住：开关的插入损耗直接贡献到系统噪声系数和发射功率预算，不能忽视。
