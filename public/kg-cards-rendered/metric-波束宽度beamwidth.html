波束宽度 Beamwidth HPBW / BW

波束宽度是天线主瓣在功率下降到峰值一半（-3dB）处的张角——它告诉你天线"指得有多准"，波束越窄，方向性越强，但扫描范围可能受限。

类比入口

波束宽度就像手电筒的光斑角度——聚光模式（窄波束）照亮远处但范围小，泛光模式（宽波束）照亮近处但范围广。5G 毫米波基站用窄波束（5-10°）对准手机，就像用激光笔；而 WiFi 路由器用宽波束（60-90°）覆盖整个房间，就像用台灯。

一、定义与物理意义

1.1 半功率波束宽度（HPBW）

HPBW = 两个功率下降到峰值一半（-3dB）的点之间的夹角

在方向图中找到 E 面和 H 面：

1.2 零点波束宽度（BWFN）

BWFN = 两个功率为零（或最小）的点之间的夹角
    BWFN ≈ 2 × HPBW（近似）

1.3 波束宽度与方向性的关系

D ≈ 4π / (θ_E · θ_H)
    D(dBi) ≈ 10.5 + 10·log₁₀(3600 / (θ_E · θ_H))

其中 θ_E 和 θ_H 分别是 E 面和 H 面的 HPBW（单位：度）。

简化公式（圆形对称波束）：

D(dBi) ≈ 10.5 + 20·log₁₀(360 / θ) - 10·log₁₀(θ²)
    D(dBi) ≈ 10.5 + 20·log₁₀(360/θ) - 20·log₁₀(θ)

更常用的近似：

D(dBi) ≈ 10.5 + 10·log₁₀(41253 / (θ_E · θ_H))

1.4 波束宽度与天线尺寸的关系

θ ≈ k · λ / D

其中 k 是常数（约 70° 对于均匀口径，约 50° 对于均匀照射），D 是天线口径尺寸。

典型值：

θ(度) ≈ 65 · λ / D（均匀照射）
    θ(度) ≈ 70 · λ / D（均匀口径）
    θ(度) ≈ 50 · λ / D（抛物面天线，高口径效率）

二、计算公式

2.1 阵列天线波束宽度

N 元均匀线阵：
    HPBW ≈ 0.886 · λ / (N · d · cosθ₀)（窄束近似）
    
    其中：
      λ = 波长
      d = 单元间距
      θ₀ = 波束指向角
      N = 单元数

示例（64 单元，d = λ/2，θ₀ = 0°）：

HPBW ≈ 0.886 · λ / (64 · λ/2) = 0.886 / 32 = 0.0277 rad = 1.59°

扫描到 60° 时：

HPBW(60°) = HPBW(0°) / cos(60°) = 1.59° / 0.5 = 3.18°
    波束展宽 2 倍！

四、测量方法

4.1 近场测量

设备：近场扫描系统（平面/圆柱/球形）
    方法：
      1. 探头在距离天线 λ-2D²/λ 处扫描
      2. 记录每个点的幅度和相位
      3. 傅里叶变换到远场
      4. 提取 HPBW
    精度：±0.1°

4.2 远场测量

设备：转台 + 标准增益天线 + 矢量网络分析仪
    方法：
      1. 被测天线安装在转台上
      2. 旋转转台，记录不同角度下的功率
      3. 找到 -3dB 点
      4. 计算两点夹角 = HPBW
    精度：±0.5°
    场地要求：d > 2D²/λ

4.3 OTA 测量（5G 毫米波）

设备：暗室 + 多探头阵列 + 矢量网络分析仪
    方法：
      1. 被测天线放在暗室转台上
      2. 多探头同时接收
      3. 提取 HPBW
    优势：无需线缆连接，适合终端天线

五、与其他指标的关系

关联指标关系类型说明
天线增益反比波束越窄 → 增益越高
旁瓣电平制约旁瓣越低 → 波束越宽（口径效率↓）
扫描范围制约扫描角越大 → 波束展宽越严重
前后比间接前后比高 → 主瓣更纯净 → HPBW 更准确
方向性系数正比方向性越高 → 波束越窄
口径效率反比效率越高（均匀照射）→ 波束越窄

六、工程注意事项

6.1 波束宽度与覆盖范围的权衡

覆盖半径 R ≈ h · cot(θ/2)
    其中 h 是天线高度，θ 是垂直波束宽度

基站覆盖设计：

场景垂直 HPBW水平 HPBW覆盖半径
城市宏站6-8°65-70°500m-1km
郊区宏站3-6°65-70°1-3km
室内分布30-60°60-90°30-100m
高铁覆盖2-4°15-30°沿轨道

6.2 电调下倾（Mechanical/Electrical Tilt）

电调下倾 = 通过移相器改变波束垂直指向
    机械下倾 = 物理倾斜天线
    
    电调下倾优势：
      - 远程调整
      - 每个扇区独立
      - 不影响水平波束宽度
    机械下倾优势：
      - 便宜
      - 不改变波束形状

6.3 波束宽度随频率变化

θ ∝ λ/D ∝ 1/f

频率升高 2 倍 → 波束宽度缩小 2 倍。

影响：

  • 宽带天线在不同频段的波束宽度不同
  • 毫米波天线比 Sub-6GHz 窄得多
  • 设计时需考虑频带内最坏情况

七、一句话总结

波束宽度是天线"指得有多准"的度量——HPBW 越小，方向性越强，增益越高，但覆盖范围越窄。记住三个关键：①波束宽度与天线尺寸成反比（天线越大波束越窄）②扫描时波束展宽（60° 扫描时展宽 2 倍）③波束宽度与增益是跷跷板关系（窄波束 = 高增益 = 小覆盖）。
