前后比 Front-to-Back Ratio FBR

前后比是天线主瓣方向功率与背面方向功率之比——它衡量天线"指向前方时后面泄露了多少"，FBR 越高，天线的方向性越纯粹，对后方干扰的抑制能力越强。

类比入口

前后比就像你站在马路边喊话——你面向前方大声喊（主瓣），但嘴巴也会向后漏声音（后瓣）。FBR 就是"前面喊的声音"和"后面漏的声音"的比值。FBR 越高，后面的人越听不到你在喊什么。

一、定义与物理意义

1.1 前后比的定义

FBR = 20·log₁₀(E_front / E_back) [dB]
    FBR = 10·log₁₀(P_front / P_back) [dB]

其中：

  • E_front = 主瓣最大值处的电场强度
  • E_back = 后瓣最大值处的电场强度

1.2 前后比的变体

主瓣/后瓣比（FBR）：

FBR = 20·log₁₀(E_max / E_rear_max)

3dB 后瓣宽度：

后瓣中功率下降 3dB 的范围

10dB 后瓣宽度：

后瓣中功率下降 10dB 的范围

1.3 前后比与旁瓣的区别

旁瓣电平（SLL）：主瓣外的次大辐射
    前后比（FBR）：主瓣与后瓣的比值
    
    旁瓣可能在任何方向（30°、60°等）
    后瓣特指 180° 附近的辐射

二、计算公式

2.1 阵列天线的 FBR

N 元均匀线阵（端射）：
    FBR ≈ 10·log₁₀(N) [dB]
    
    N = 4 → FBR ≈ 6dB
    N = 8 → FBR ≈ 9dB
    N = 16 → FBR ≈ 12dB
    N = 32 → FBR ≈ 15dB
    N = 64 → FBR ≈ 18dB

2.2 抛物面天线的 FBR

FBR ≈ 32·log₁₀(D/λ) - 8 [dB]
    
    D/λ = 100 → FBR ≈ 56dB
    D/λ = 50 → FBR ≈ 44dB
    D/λ = 20 → FBR ≈ 32dB
    D/λ = 10 → FBR ≈ 20dB

2.3 八木天线的 FBR

FBR 取决于引向器数量和长度精度
    
    典型值：
      3 单元 → 15-20dB
      5 单元 → 20-25dB
      8 单元 → 25-30dB
      12 单元 → 30-35dB

2.4 FBR 与方向系数的关系

高 FBR 不一定意味着高方向性
    高方向性通常需要高 FBR + 低旁瓣

四、测量方法

4.1 远场测量

设备：转台 + 标准增益天线 + VNA
    方法：
      1. 旋转被测天线 360°
      2. 记录 0°（主瓣）和 180°（后瓣）的功率
      3. FBR = 20·log₁₀(P_0° / P_180°)
    精度：±1dB

4.2 暗室测量

设备：微波暗室 + 转台
    方法：
      1. 在暗室中旋转天线
      2. 记录全方向图
      3. 提取 FBR
    优势：无反射干扰，精度高

4.3 OTA 测量

设备：暗室 + 多探头阵列
    方法：
      1. 被测天线放在暗室中央
      2. 多探头同时采集
      3. 提取 FBR
    适用：终端天线、手机

五、与其他指标的关系

关联指标关系类型说明

方向性系数相关高方向性通常需要高 FBR

六、工程注意事项

6.1 FBR 恶化的常见原因

设计 FBR：30dB
    实测 FBR：20dB（恶化 10dB）
    
    原因：
      1. 安装支架反射（+5dB 恶化）
      2. 馈线耦合（+3dB 恶化）
      3. 天线罩影响（+2dB 恶化）
      4. 地面反射（+2dB 恶化）

6.2 改善 FBR 的方法

方法效果代价
加反射板/背腔FBR ↑ 10-15dB体积 ↑
优化馈源设计FBR ↑ 5-10dB设计复杂
使用吸波材料FBR ↑ 5-10dB重量 ↑
背腔贴片天线FBR ↑ 15-20dB带宽 ↓
端射阵列优化FBR ↑ 5-10dB增益 ↓

6.3 FBR 与覆盖设计的权衡

高 FBR（> 30dB）：
      优点：后方干扰小，定位精度高
      缺点：可能需要更大天线，成本 ↑
    
    低 FBR（< 15dB）：
      优点：成本低，体积小
      缺点：后方干扰大，系统容量 ↓

七、一句话总结

前后比 FBR 是天线"指向前方时后面泄露了多少"的度量——FBR 越高，天线越"专一"。记住三个关键：① FBR 与旁瓣不同，它特指主瓣与后瓣（180°）的比值 ② 抛物面天线 FBR 可达 30-40dB，微带贴片只有 15-25dB ③ 实际 FBR 受安装环境影响很大，设计时要预留余量。
