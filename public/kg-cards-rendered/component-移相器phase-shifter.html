移相器 Phase Shifter Phase-Shifter

移相器是改变射频信号相位但不改变其幅度的无源或有源器件——它是相控阵天线的"方向盘"，通过精确控制每个天线单元的相位来改变波束指向。

类比入口

移相器就像水管上的"阀门延时器"——水流（射频信号）的大小不变，但你可以通过调节阀门的开合时间来改变水流的"节奏"（相位）。在相控阵中，每个天线单元的移相器就像一排水龙头，你精确控制每个龙头的开启时间，所有水流汇聚到一起时就形成了一股定向的"水柱"（波束）。

一、定义与作用

1.1 移相器的基本功能

输入信号：A·cos(ωt + φ_in)
    输出信号：A·cos(ωt + φ_in + Δφ)

其中 Δφ 是移相器引入的相移量。

1.2 在相控阵中的作用

波束指向角 θ₀ 由移相器相位决定：
    Δφ_n = n · (2πd/λ) · sin(θ₀)

其中 n 是单元序号，d 是单元间距。

举例（64 单元阵列，θ₀ = 30°）：

  • 单元 0：Δφ = 0°
  • 单元 1：Δφ = 5.6°
  • 单元 32：Δφ = 179°
  • 单元 63：Δφ = 353°

1.3 移相器的关键指标

指标说明典型值
相移范围可实现的总相移0-360°（模拟）或 0-180°（数字）
相移精度实际相移与目标相移的偏差±2-5°
插入损耗信号通过移相器的功率损失0.5-3dB
工作频率中心频率和带宽1-40GHz

二、移相器的主要类型

2.1 数字移相器（Binary Phase Shifter）

用 N 位二进制控制相移量：

N 位 → 2^N 个离散相位状态
    3 bit → 8 个状态（0°, 45°, 90°, ..., 315°）
    4 bit → 16 个状态（22.5° 步进）
    5 bit → 32 个状态（11.25° 步进）
    6 bit → 64 个状态（5.625° 步进）

常见相位码：

位相位码最小步进
3 bit180°, 90°, 45°45°
4 bit180°, 90°, 45°, 22.5°22.5°
5 bit180°, 90°, 45°, 22.5°, 11.25°11.25°

优点：简单、便宜、数字接口 缺点：量化误差、相移不连续

2.2 模拟移相器（Analog/Continuous Phase Shifter）

连续可调相移：

Δφ = 0° to 360°（连续可调）

2.3 反射型移相器（Reflection Type）

        ┌───[可变反射负载]──┐
    Input ──┤                   ├─── Output
            └───[可变反射负载]──┘
               90° Hybrid Coupler

信号经 hybrid coupler 分成两路，各自被可变负载反射回来，在 coupler 处重新组合，相位差由两个负载的反射系数决定。

优点：体积小、易于集成 缺点：隔离度有限

2.4 传输线型移相器（Transmission Line Type）

通过改变传输线的电长度来实现相移：

Δφ = β·Δl = (2π/λ)·Δl

其中 Δl 是传输线长度的变化。

实现方式：

  • 开关切换不同长度的传输线（数字式）
  • 可变介质常数（模拟式）

2.5 PLL 型移相器（基于锁相环）

VCO → Phase Detector → Loop Filter → VCO
                  ↑
              Reference Signal

优点：相位连续可调、精度高 缺点：复杂、功耗高

五、应用场景

5.1 相控阵雷达

  • 每个天线单元配一个移相器
  • 64 单元 = 64 个移相器
  • 波束扫描速度：微秒级
  • 相位精度要求：±2° RMS

5.2 5G 基站（Massive MIMO）

  • 64T64R = 128 个移相器
  • 模拟波束赋形：每通道 1 个移相器链
  • 混合波束赋形：数字 + 模拟混合

5.3 卫星通信

  • 馈电网络中的相位校准
  • 多波束卫星的波束指向控制
  • 相位精度要求：±5°

5.4 毫米波通信（28/39GHz）

  • BFIC 内置移相器（3-6 bit）
  • 集成于封装内
  • 插入损耗是关键指标

六、工程注意事项

6.1 温度稳定性

Δφ/ΔT ≈ 0.1-1°/°C（典型值）

对于 64 单元阵列，温度变化 30°C 可能导致波束偏移 2-5°。

解决方案：

  • 温度补偿电路
  • 数字域自适应校准
  • 使用温度稳定的材料（LTCC、SiGe）

6.2 相位-幅度耦合

移相器在改变相位的同时，幅度也会波动：

ΔA/Δφ ≈ 0.05-0.2 dB/°（典型值）

解决方案：

  • 选择幅度平坦度好的移相器
  • 在数字域进行幅度补偿

6.3 互易性

TDD 系统中，移相器的上行和下行相位应该一致：

φ_TX ≠ φ_RX → 波束指向偏差

解决方案：

  • 使用互易性好的移相器
  • 定期校准

七、技术成熟度

  • 成熟度等级：高度成熟
  • 说明：数字移相器（PIN/MOSFET）技术非常成熟，广泛用于雷达和通信。MEMS 移相器在高性能场景中成熟。CMOS 毫米波移相器随 5G BFIC 快速成熟。

八、一句话总结

移相器是相控阵天线的"方向盘"——通过精确控制每个单元的相位来"拧"出想要的波束指向。核心指标就三个：相位精度（决定了波束准不准）、插入损耗（决定了信号剩多少）、切换速度（决定了波束转多快）。数字移相器便宜但量化误差大，模拟移相器连续可调但复杂，MEMS 损耗最低但速度慢。
