天线方向图函数与数学工具（Directional Functions & Math Tools） Directional-Functions

天线方向图的背后是一套丰富的数学工具——sinc 函数描述矩形孔径、贝塞尔函数描述圆形孔径、Chebyshev 多项式控制旁瓣、Legendre 多项式描述球面波。掌握这些函数的数学特性，就能理解和预测天线方向图的行为。

类比入口

天线方向图就像一首乐曲的"声音指纹"，而这些特殊函数就是乐谱上的音符。sinc 函数是矩形天线的"主旋律"，贝塞尔函数是抛物面天线的"和弦"，Chebyshev 多项式是旁瓣控制的"调音师"。理解了这些数学工具，你就理解了天线方向图的"乐理"。

一、核心方向图函数

1.1 sinc 函数（抽样函数）

定义：

sinc(x) = sin(x)/x    （数学定义）
    或
    sinc(x) = sin(πx)/(πx)  （工程定义，零点在整数处）

在天线中的应用：

  • 矩形孔径均匀照明 → 方向图 = sinc 函数
  • 线性阵列均匀激励 → 阵列因子 = Dirichlet 核（周期 sinc）

关键性质：

零点：x = ±π, ±2π, ±3π, ...
    第一旁瓣：-13.2 dB
    旁瓣滚降：∝ 1/x（缓慢）

图形特征：

主瓣宽度（-3dB）：Δx ≈ 0.886·π
    主瓣宽度（零点到零点）：Δx = 2π

1.2 贝塞尔函数（Bessel Function）

定义（一阶）：

J₁(x) = Σ(k=0 to ∞) [(-1)ᵏ · (x/2)^(2k+1)] / [k! · (k+1)!]

在天线中的应用：

  • 圆形孔径均匀照明 → 方向图 = 2·J₁(x)/x
  • 抛物面天线的馈源方向图

关键性质：

第一零点：x ≈ 3.832
    第一旁瓣：-17.6 dB（比矩形孔径更高）
    旁瓣滚降：∝ 1/√x（比 sinc 慢）

圆形孔径方向图：

F(θ) = 2·J₁(ka·sinθ) / (ka·sinθ)

其中 a = 孔径半径，k = 2π/λ。

3dB 波束宽度：

HPBW ≈ 1.02·λ/D    （D = 2a 是直径）

对比矩形孔径：HPBW ≈ 0.886·λ/L

1.3 Chebyshev 多项式

定义：

T_n(x) = cos(n·arccos(x)),  |x| ≤ 1
    T_n(x) = cosh(n·arccosh(x)), |x| > 1

递推关系：

T_0(x) = 1
    T_1(x) = x
    T_{n+1}(x) = 2x·T_n(x) - T_{n-1}(x)

关键性质：

  • 在 [-1, 1] 区间内，|T_n(x)| ≤ 1（等振荡）
  • 在 |x| > 1 时，T_n(x) 单调递增
  • 等旁瓣特性 → 用于 Dolph-Chebyshev 天线设计

在天线中的应用： Dolph-Chebyshev 阵列的方向图：

AF(ψ) = T_{N-1}[cos(ψ₀)·cos(ψ/2)] / T_{N-1}[cos(ψ₀)]

其中 ψ₀ 控制旁瓣电平：

SLL = 20·log₁₀(T_{N-1}(cos(ψ₀)))

1.4 Legendre 多项式

定义：

P_n(x) = (1/(2ⁿ·n!)) · dⁿ/dxⁿ[(x²-1)ⁿ]

前几个：

P_0(x) = 1
    P_1(x) = x
    P_2(x) = (3x²-1)/2
    P_3(x) = (5x³-3x)/2

在天线中的应用：

  • 球面波展开
  • 球面天线（如卫星天线）的方向图分析
  • 球坐标系中的角度函数

1.5 Dirichlet 核（周期 sinc）

定义：

D_N(x) = sin((N+½)x) / sin(x/2)

在天线中的应用：

  • 均匀线性阵列的方向图（有限单元数 N）
  • 当 N → ∞ 时，D_N(x) → 周期 sinc 函数

关键性质：

主瓣宽度：4π/N
    第一旁瓣：-13.2 dB
    旁瓣数量：N-1 个

二、辅助数学工具

2.1 傅里叶变换对

天线方向图计算的核心——方向图是口径场的傅里叶变换：

口径分布方向图
矩形（均匀）sinc
余弦sinc²
高斯高斯
三角形sinc²
Bartlettsinc²
圆形均匀2·J₁(x)/x

2.2 渐近展开

当 x → ∞ 时：

sinc(x) ≈ 1/x          （缓慢滚降）
    J₁(x) ≈ √(2/πx)·cos(x-3π/4)  （振荡滚降）
    T_n(x) ≈ (x+√(x²-1))ⁿ / (2ⁿ·√(x²-1))  （指数增长）

2.3 积分恒等式

∫₀^∞ J₁(ax)/x dx = 1
    ∫_{-∞}^∞ sinc²(x) dx = π
    ∫₀^π sinⁿ(x) dx = √π · Γ((n+1)/2) / Γ((n+2)/2)

2.4 特殊函数关系

J_{n+1}(x) = (2n/x)·J_n(x) - J_{n-1}(x)    （递推）
    d/dx[J₀(x)] = -J₁(x)
    ∫x·J₀²(x)dx = x²/2 · [J₀²(x) + J₁²(x)]

五、技术演进

  • 1700s：傅里叶分析建立
  • 1800s：贝塞尔函数、Legendre 多项式系统化
  • 1940s：Chebyshev 多项式应用于天线
  • 1960s：特殊函数在天线方向图中的系统整理
  • 2000s：计算机代数系统（Mathematica/Matlab）加速计算
  • 2020s：AI 辅助特殊函数近似和优化

六、一句话总结

天线方向图不是魔法——它背后是 sinc、贝塞尔、Chebyshev 这些数学函数的精确表达。理解这些函数的性质（零点位置、旁瓣电平、滚降速度），就能在不跑仿真的情况下预估天线的方向图行为。
