G/T 值 Figure of Merit

G/T 值是接收系统的"综合性能指标"——它把天线增益 G 和系统噪声温度 T 压缩成一个数字，数字越高表示接收能力越强。它是卫星通信和雷达接收机性能的核心评判标准。

类比入口

G/T 就像汽车的"燃油经济性"——不是看发动机多大（G），也不是看油耗多低（T），而是把两者结合起来看"每升油能跑多远"。G 越高（天线越大越聚光），T 越低（噪声越小越安静），G/T 就越高，接收弱信号的能力就越强。

一、定义与物理意义

1.1 G/T 值的定义

G/T = G_r / T_sys

其中：

  • G_r = 接收天线增益 [dBi 或线性]
  • T_sys = 接收系统总噪声温度 [K]

1.2 G/T 的 dB 表示

G/T(dB/K) = G_r(dBi) - 10·log₁₀(T_sys)

注意：G/T 的单位是 dB/K（分贝每开尔文），不是 dB。

1.3 为什么 G/T 很重要？

在卫星通信链路预算中：

C/N₀ = EIRP + G/T - L_fs - L_other + 10·log₁₀(k)

其中：

  • C/N₀ = 载噪谱密度比 [dB-Hz]
  • EIRP = 发射端等效全向辐射功率
  • L_fs = 自由空间路径损耗
  • k = 玻尔兹曼常数

核心洞察：G/T 是唯一由接收端决定的链路参数。发射端靠 EIRP，接收端靠 G/T。要提高链路质量，要么加大发射功率/天线（EIRP↑），要么加大接收天线/降低噪声（G/T↑）。

二、G/T 的计算

2.1 系统噪声温度 T_sys

T_sys = T_ant + T_feeder + T_LNA/G_feeder + T_other/(G_feeder·G_LNA)

2.2 天线增益 G_r

G_r = η_a · (4π·A_e / λ²)

其中 η_a 是天线效率。

2.3 G/T 综合计算

G/T = G_r / T_sys
    G/T(dB/K) = G_r(dBi) - 10·log₁₀(T_sys)

四、G/T 优化

4.1 提高 G_r

方法效果代价
增大天线直径G ∝ D²体积/重量/成本↑
提高口径效率η: 50%→70%加工精度↑
降低工作频率λ↑ → G↑天线尺寸↑

抛物面天线增益公式：

G = η·(πD/λ)²
    G(dBi) = 20·log₁₀(D/λ) + 10·log₁₀(η) + 9.5

4.2 降低 T_sys

方法效果代价
使用低温 LNAT_LNA: 100K→20K冷却成本↑
缩短馈线L_feeder↓安装限制
低损耗馈线波导替代同轴成本/重量↑
优化天线指向T_ant↓（避开地面）跟踪系统

4.3 G/T 权衡

增大天线直径 D：
    - G ∝ D² → G(dBi) ∝ 40·log₁₀(D)
    - 但成本 ∝ D²，重量 ∝ D²
    
    所以 G/T 的增长是"对数级"的，而成本是"平方级"的。

边际效益递减：

5.3 雨衰对 G/T 的影响

雨衰不仅降低 EIRP，还增加 T_ant：

T_ant_rain = T_phys · (1 - e^(-τ_rain))
    T_sys_rain = T_ant_rain + T_other
    
    G/T_rain = G_r / T_sys_rain < G_r / T_sys_clear

六、测量方法

6.1 两天线法（标准方法）

步骤：
    1. 用已知 EIRP 的卫星信号
    2. 测量接收功率 P_1
    3. 换一个已知 EIRP 的不同卫星
    4. 测量接收功率 P_2
    5. 联立方程求解 G/T

6.2 Y 因子法（LNA 部分）

步骤：
    1. 用冷热负载测量 LNA 的噪声温度 T_LNA
    2. 测量天线增益 G_r
    3. G/T = G_r / T_LNA（忽略其他噪声源）

6.3 噪声源注入法

步骤：
    1. 在天线输入端注入已知 ENR 噪声源
    2. 测量接收功率变化
    3. 计算系统噪声温度 T_sys
    4. 独立测量 G_r
    5. G/T = G_r / T_sys

八、一句话总结

G/T 是接收系统的"综合战斗力指数"——天线增益越高、系统噪声越低，G/T 就越高，能接收的信号就越弱。它是卫星通信链路预算中接收端的唯一代表，单位 dB/K。记住公式：G/T(dB/K) = G_r(dBi) - 10·log₁₀(T_sys[K])。深空站的 G/T 能达到 +50dB/K 以上，而手机 5G 接收的 G/T 只有 -15dB/K——差了 65dB，也就是 300 万倍。
