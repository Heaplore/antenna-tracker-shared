圆极化天线深度解读 Circularly Polarized Antenna

圆极化天线发出的电磁波像螺旋钻——电场矢量端点在空间画出一个圆。好处是接收端不管卫星怎么翻转都能收到信号，GPS 卫星全用圆极化就是这个原因。衡量圆极化质量的指标叫"轴比"（Axial Ratio），越接近 0dB 越圆。5G 毫米波也开始用圆极化来降 RCS（雷达截面积），让隐身效果更好。

一、圆极化的物理本质

1.1 线极化 vs 圆极化 vs 椭圆极化

线极化：电场矢量只在一个平面内振动。

    ↑
        │ E
        ↓

右旋圆极化（RHCP）：电场矢量端点在空间顺时针画圆（从波源看）。

      ↻
        ↵   ↶
       ← E →
        ↶   ↵
          ↻

左旋圆极化（LHCP）：逆时针画圆。

椭圆极化：介于圆和线之间，电场矢量端点画椭圆。

1.2 圆极化的产生条件

两个正交的线极化波，幅度相等，相位差 90°：

E_x = E₀ cos(ωt)
    E_y = E₀ cos(ωt ± π/2) = ±E₀ sin(ωt)

合成电场：

|E|² = E_x² + E_y² = E₀²(cos²ωt + sin²ωt) = E₀²

结论：电场幅度恒定，方向匀速旋转 → 圆极化。

1.3 轴比（Axial Ratio, AR）

定义：椭圆长轴与短轴之比（线性单位），或以 dB 表示。

AR_linear = a/b ≥ 1
    AR_dB = 20 × log₁₀(AR_linear)

理想圆极化：AR = 1（0dB）

工程标准：

AR (dB)极化质量应用场景
0-1优秀高精度卫星导航
1-3良好GNSS 接收
3-6一般卫星通信
6-10较差宽带应用可接受
> 10接近线极化不认为是 CP

3dB AR 带宽：AR < 3dB 的频率范围，是圆极化天线的有效工作带宽。

二、圆极化天线的实现方法

2.1 正交馈电贴片天线（Quadrature-Fed Patch）

结构：两个正交的微带贴片，馈电相位差 90°。

        ┌─────┐
            │  ▓  │  ← 贴片（十字形或方形）
            └─────┘
             │   │
           90°  0°
             │   │

实现 90° 相位差的方法：

  1. quadrature hybrid（3dB 电桥）：如分支线耦合器、Lange 耦合器
  2. 不等长馈线：利用传输线相位延迟
  3. 激励端口位置偏移：在贴片对角线两侧激励

特点：

  • 优点：平面结构，易于集成
  • 缺点：AR 带宽窄（通常 2-5%）
  • 适用：卫星通信终端、GPS 接收

2.2 螺旋天线（Helical Antenna / Turnstile）

结构：导线绕成螺旋形，沿轴向辐射。

      ╭───╮
        ╭─╯   ╰─╮
      ╭─╯       ╰─╮
      │           │
      │  螺旋线圈  │
      │           │
        ╰─╮   ╮─╯
          ╰───╯
           轴向辐射

参数：

周长 C ≈ λ
    匝间距 S ≈ λ/4
    总匝数 N

增益：

G ≈ 15 × N × (C/λ)² × SA  （dBi）

其中 SA 是轴比效率（SA = 1/[1+(AR²)]，理想圆极化时 SA ≈ 0.81）。

特点：

  • 优点：纯圆极化（AR < 1dB），增益高（N=3-5 时 12-15dBi）
  • 缺点：体积大，轴向安装
  • 适用：卫星通信、深空探测（NASA 经典选择）

2014 清华突破：贴片-螺旋混合天线

  • 寄生贴片 + 10 匝螺旋
  • 小地平面下增益 13.35dBic
  • 阻抗带宽 22%（5.05-6.30GHz）
  • 螺旋电流均匀分布，不受地平面限制

2.3 缝隙天线（Slot Antenna）

结构：在金属面上蚀刻特定形状的缝隙。

常见形状：

  • 十字形缝隙（cross-slot）：天然正交
  • S 形缝隙：紧凑设计
  • 阶梯形缝隙：多频谐振

2024 PMC 空气填充 SIW 圆极化缝隙阵列：

  • 1×8 阵列，34.5-41.5GHz
  • 阻抗带宽 18.4%
  • 3dB AR 带宽 16.4%（36.5-43GHz）
  • 峰值增益 18dBic
  • 金属腔体辐射，S 形缝隙产生 CP

2.4 螺旋线天线（Quadrifilar Helix）

结构：四根导线绕成螺旋形，交替馈电。

    ╭──╮  ╭──╮
      ╭─╯  ╰─╯  ╰─╮
      │   四线螺旋   │
      ╰─╮  ╭──╮  ╭─╯
        ╰─╯    ╰─╯

特点：

  • 半球形覆盖（hemispherical）
  • AR 带宽宽（> 20%）
  • 增益 6-9dBi
  • GNSS 天线经典选择（GPS/GLONASS/BDS 接收）

2.5 螺旋天线（Travelling-Wave Spiral）

结构：平面阿基米德螺旋或对数螺旋。

特点：

  • 超宽带（> 10:1 频率比）
  • 圆极化纯度好
  • 增益恒定（~8dBi）在整个频带
  • 适用：频谱监测、电子战

三、圆极化的极化损失（Polarization Mismatch Loss）

3.1 极化失配公式

Polarization Loss Factor (PLF) = |ρ̂_w · ρ̂_a|²

其中 ρ̂_w 是入射波极化单位矢量，ρ̂_a 是天线极化单位矢量。

3.2 圆极化 - 圆极化匹配

发射接收PLF (dB)
RHCPRHCP0dB（完美匹配）
LHCPLHCP0dB（完美匹配）
RHCPLHCP-∞dB（完全隔离！）
LHCPRHCP-∞dB（完全隔离！）

关键洞察：圆极化的左右旋之间是完全隔离的！这意味着可以在同一频段、同一位置用 RHCP 发射、LHCP 接收，实现频率复用。

3.3 圆极化 - 线极化匹配

PLF = 0.5（-3dB）

无论线极化方向如何，圆极化天线接收线极化波总是损失 3dB。

实际意义：

  • GPS 卫星用 RHCP 发射
  • 手机天线是线极化 → 理论损失 3dB
  • 但好处是：手机怎么拿都不会信号骤降（因为线极化方向随机变化时，圆极化接收稳定）

3.4 极化损失总结

发射极化接收极化损失
RHCPRHCP0dB
LHCPLHCP0dB
RHCPLHCP> 20dB（隔离）
线极化RHCP3dB
线极化线极化（同向）0dB
线极化线极化（正交）> 20dB（隔离）

四、圆极化天线的应用场景

4.1 GNSS 卫星导航（GPS/BDS/GLONASS/Galileo）

为什么用圆极化？

  • 卫星在轨道上会翻滚/姿态变化 → 线极化会失效
  • 圆极化不受卫星姿态影响
  • 多径抑制：地面反射波会反转旋向（RHCP → LHCP），被天线天然抑制

典型参数：

参数数值
频率L1: 1575.42MHz, L2: 1227.60MHz, L5: 1176.45MHz
极化卫星发 RHCP，接收机天线 RHCP
增益6-9dBi
3dB AR 波束宽度> 200°（半球覆盖）
前后比> 20dB
多径抑制10-15dB

2025 高精度海洋定位 GNSS 天线：

  • 八根短路销钉围绕贴片 → 扩宽 3dB AR 波束
  • 两组弯曲金属分支在接地层 → 前后比提升
  • 重叠带宽 > 45%（10dB 回波损耗 + 3dB AR）
  • 覆盖 GPS L1/L2, BDS B1/B2, GLONASS, Galileo

4.2 卫星通信

应用场景：

  • 北斗/GPS 导航卫星发射天线
  • 通信卫星星载天线
  • 地面站接收天线
  • 无人机/导弹导引头

特点：

  • 卫星姿态不确定 → 必须圆极化
  • 远距离传输 → 线极化旋转（Faraday 旋转）→ 圆极化不受影响

4.3 5G 毫米波通信

2024 研究热点：

  • 圆极化 MIMO：降低通道相关性，提升容量
  • 圆极化阵列：RCS 降低（隐身应用）
  • 正交圆极化复用：RHCP + LHCP = 双倍频谱效率

2025 5G mmWave 圆极化 MIMO：

  • 频段：25.45-32.92GHz
  • AR 带宽：26.24-29.88GHz（3.64GHz）
  • 增益：> 5.8dBi
  • 辐射效率：> 91.72%
  • ECC < 10⁻²（极低相关性）
  • 设计：圆形贴片 + 扇形槽 + 半月形槽

4.4 雷达与隐身（RCS Reduction）

原理：圆极化天线发射 RHCP，目标反射后变为 LHCP → 接收天线只收 RHCP → 回波被抑制。

RCS 降低量：

ΔRCS = 20 × log₁₀(AR)  （dB）

对于 AR = 3dB 的天线，RCS 可降低 ~12dB（功率降低 98.4%）。

2019-2025 研究进展：

  • 超表面（Metasurface）加载贴片阵列 → 带外 RCS 降低 15-20dB
  • 圆极化序贯旋转阵列 → 宽频 RCS 降低
  • 极化转换超表面 + CP 阵列 → 联合优化 RCS 和辐射性能

4.5 可穿戴设备

优势：

  • 用户头部/身体转动不影响天线极化匹配
  • 降低人体组织吸收导致的极化畸变
  • MIMO + CP → 高隔离、低相关性

五、圆极化 vs 线极化对比

特性线极化圆极化
极化纯度高中（取决于 AR）
极化对准要求严格（正交时完全隔离）宽松（旋转不变）
多径抑制差好（反射反转旋向）
极化损失0dB（对准）/ >20dB（正交）3dB（对线极化）/ 0dB（同旋向）
带宽宽窄（AR 带宽通常 < 阻抗带宽）
复杂度低中（需要 90° 相位差）
典型增益高略低（~0.5-1dB 极化效率损失）
应用WiFi/基站/大多数地面通信GNSS/卫星/隐身/可穿戴

六、给非专业读者的一句话总结

圆极化天线发出的电磁波是"螺旋钻"——电场矢量在空间画圆圈。GPS 卫星用圆极化是因为卫星在天上翻滚，线极化的话地面天线就收不到了。圆极化还有一个神奇特性：左右旋完全隔离，互不干扰。这让科学家想到用它来做隐身——雷达发射 RHCP，飞机收到后反射变成 LHCP，雷达天线只收 RHCP，于是回波被大幅削弱。圆极化就是物理学送给工程师的"免费礼物"：一个结构，两种能力（通信+隐身）。

金句：圆极化告诉你一个道理——有时候转一圈不是绕远路，而是为了在任何角度都能接到。
