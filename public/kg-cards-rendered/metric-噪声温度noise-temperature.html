噪声温度 Noise Temperature T

噪声温度是用"温度"来量化噪声功率的等效方法——任何有噪声的设备都可以用一个等效温度 T_n 来表示它的噪声贡献，T_n 越高，噪声越大。接收系统的总噪声温度决定了最小可检测信号和链路质量。

类比入口

想象你在一个安静的图书馆里说话（信号），但旁边有人在装修（噪声）。"噪声温度"就像是在说"这个装修噪音相当于把室温提高了 500°C 所产生的嗡嗡声有多大"。温度越高，嗡嗡声越大，你说的话就越难被听见。噪声温度就是把"这噪音有多大"换算成一个直观的"温度数字"。

一、定义与物理意义

1.1 噪声功率与噪声温度的关系

P_n = k · T_n · B

其中：

  • P_n = 噪声功率 [W]
  • k = 玻尔兹曼常数 = 1.38×10⁻²³ J/K
  • T_n = 噪声温度 [K]
  • B = 带宽 [Hz]

反过来：

T_n = P_n / (k · B)

1.2 为什么用噪声温度而不是噪声系数？

特性噪声温度（T_n）噪声系数（NF）
适用范围极低噪声系统（卫星、射电）一般 RF 系统
单位开尔文（K）分贝（dB）
线性度可以直接相加需要转换
低温系统更直观不准确（NF 在低温下失真）
转换关系T_n = T_0·(10^(NF/10) - 1)NF = 10·log₁₀(1 + T_n/T_0)

其中 T_0 = 290K（标准参考温度）。

1.3 噪声温度的组成

接收系统的总噪声温度由多部分串联组成：

T_sys = T_ant + T_feeder + T_LNA + T_mixer + T_IF

各部分的等效噪声温度通过 Friis 噪声公式 叠加：

T_total = T_1 + T_2/G_1 + T_3/(G_1·G_2) + T_4/(G_1·G_2·G_3) + ...

关键洞察：第一级的噪声温度最重要——因为它后面的各级都被第一级的增益"稀释"了。

三、计算公式

3.1 Friis 噪声公式（级联系统）

T_sys = T_1 + T_2/G_1 + T_3/(G_1·G_2) + ...

实际计算示例（卫星接收链路）：

馈线损耗：L_f = 0.5dB → G_f = 0.89, T_f = 290·(1/0.89 - 1) = 36K
    LNA：G_1 = 40dB (10000x), T_1 = 20K
    混频器：G_2 = 10dB (10x), T_2 = 500K
    IF AMP：G_3 = 30dB, T_3 = 2000K
    
    T_sys = 36 + 20/0.89 + 500/(0.89×10000) + 2000/(0.89×10000×10)
          = 36 + 22.5 + 0.056 + 0.002
          ≈ 58.6K

关键洞察：LNA 的增益（40dB）把后续级的噪声贡献压缩到了 0.06K——几乎可以忽略。这就是为什么 LNA 要放在最前面且增益要高。

3.2 天线噪声温度

T_ant = ∫∫_4π T_b(θ,φ) · F_n(θ,φ) dΩ / ∫∫_4π F_n(θ,φ) dΩ

其中 T_b(θ,φ) 是亮温分布，F_n 是归一化方向图。

简化理解：天线噪声温度 = 天线方向图"看到"的平均环境温度。

T_atm = T_phys · (1 - e^(-τ))

四、测量方法

4.1 Y 因子法（标准方法）

步骤：
    1. 连接"冷"负载（液氮 77K 或室温 290K）→ 测得功率 P_cold
    2. 连接"热"负载（室温 290K 或高温负载）→ 测得功率 P_hot
    3. Y = P_hot / P_cold
    4. T_n = (T_hot - Y·T_cold) / (Y - 1)

设备：噪声源（ENR 已知）+ 功率计/VNA

4.2 ENR 法（噪声源标定法）

T_n = T_0 · (ENR / (Y - 1 - ENR))

其中 ENR = Excess Noise Ratio（噪声源标称值）。

4.3 接收机法（系统级测量）

步骤：
    1. 用已知噪声温度的负载
    2. 测量接收机输出功率
    3. 计算系统总噪声温度

六、工程注意事项

6.1 LNA 位置至关重要

方案 A：馈线 → LNA → 混频器
    方案 B：LNA → 馈线 → 混频器
    
    方案 A 的 T_sys 比方案 B 低 10-50 倍！

规则：LNA 必须尽可能靠近天线（最好在室外/天线罩内）。

6.2 馈线损耗的影响

T_feeder = T_phys · (L - 1)

其中 L 是馈线损耗因子（> 1）。

6.3 温度对 T_sys 的影响

T_ant 随环境温度变化：
    - 夏天：T_ant ≈ 50-100K（卫星链路）
    - 冬天：T_ant ≈ 20-50K

链路预算需要预留温度裕量（Temperature Margin）：3-6dB。

七、一句话总结

噪声温度是用"开尔文"来度量噪声的——它把看不见的电磁噪声翻译成"这噪声相当于多少度热辐射"。核心公式 T_sys = T_1 + T_2/G_1 + T_3/(G_1·G_2)... 告诉我们一个朴素道理：第一级最重要——LNA 的噪声和增益决定了整个接收链路的下限。记住：噪声温度越低越好，290K（室温）是基准线，卫星接收系统追求 20-50K，射电天文追求 5-10K。
