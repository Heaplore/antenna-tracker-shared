天线罩深度解读 Radome

天线罩就是天线的"防弹衣+隐形斗篷"——保护天线不被风吹雨打日晒，同时要让无线电波几乎无损耗地穿过去。核心指标就两个：介电常数 Dk 要低（接近空气的 1.0），损耗角正切 Df 要低（< 0.001）。5G 毫米波时代，天线罩厚度必须精确到半波长的整数倍，否则波束会偏移、增益会下降。6G 太赫兹频段，Dk=1.19 的泡沫材料能让传输距离提升 9 倍。

一、天线罩的基本功能

1.1 保护功能

威胁防护措施
风雨侵蚀防水涂层 + 密封结构
紫外线老化UV 稳定剂 + 外层保护漆
冰雹冲击高强度复合材料外壳
沙尘堆积光滑曲面设计（自清洁）
温度变化低热膨胀系数材料
盐雾腐蚀防腐涂层

1.2 电磁功能

天线罩对电磁波的影响：

  1. 反射损耗：波在天线罩内外表面反射 → VSWR 恶化
  2. 穿透损耗：材料吸收 + 散射 → 增益下降
  3. 相位畸变：厚度不均匀 → 波束偏移（BSE）
  4. 方向图畸变：边缘效应 → 旁瓣升高

理想天线罩：对电磁波"透明"，仿佛不存在。

  • 保持工程塑料的强度/耐候性/阻燃性
  • 5G 基站天线罩首选材料之一

三、天线罩结构设计

3.1 单层板天线罩（Monolithic Radome）

结构：单一材料平板。

    ┌─────────────┐
        │   均匀厚度    │  ← 单一材料
        └─────────────┘

特点：

  • 最简单，成本最低
  • 仅适用于低频或小口径天线
  • 毫米波频段需要精确控制厚度

3.2 A 型夹层天线罩（A-Sandwich Radome）

结构：两层薄表皮 + 一层厚泡沫芯。

    ┌─────┐  ← 外层表皮（高强度）
        │ 芯材  │  ← 低 Dk 泡沫
        └─────┘  ← 内层表皮

原理：

  • 两个内部介电界面各产生反射波
  • 芯材厚度 = λ_c / 2（芯材中的半波长）的整数倍
  • 两反射波相位差 180° → 相消干涉 → 反射抵消

特点：

  • 5G 基站天线罩标准方案
  • 兼顾机械强度和电磁性能
  • 适用于双频（低频"电薄"，高频优化）

3.3 C 型夹层天线罩（C-Sandwich / Double A）

结构：两层表皮 + 两层芯材 + 中间刚性层。

    ┌─────┐  ← 外层表皮
        │ 芯1  │
        ├─────┤  ← 中间层
        │ 芯2  │
        └─────┘  ← 内层表皮

特点：

  • 比 A 型更强的结构刚度
  • 可针对不同频段分别优化两层芯材
  • 成本高，用于大型/高性能场景

3.4 变厚度流线型天线罩（Variable-Thickness Streamlined）

结构：厚度随位置变化，呈流线型（如切线抛物线、正切卵形）。

         ╭─────╮
           ╭─╯     ╰─╮
         ╭─╯         ╰─╮
        │               │
         ╰─╮         ╭─╯
           ╰─╮     ╭─╯
             ╰─────╯

应用场景：

  • 飞机/导弹机头雷达罩
  • 需要满足空气动力学要求

设计挑战：

  • 厚度变化 → 相位畸变 → 波束指向误差（BSE）
  • 需用遗传算法（GA）+ 射线追踪（RT）优化厚度分布
  • 2021 IEEE TAP：渐变介电多层壁 + GA 优化

3.5 TI 毫米波雷达天线罩设计指南

德州仪器 2024 官方指南核心要点：

  1. 天线与天线罩间距 H：
  2. 理想：H = m × λ₀/2（m 为正整数）
  3. 推荐：H = 1λ₀ 或 1.5λ₀
  4. 例：24GHz 毫米波雷达，λ₀/2 ≈ 6.25mm → 推荐 H = 12.5mm 或 18.75mm
  5. 受限情况下 H = λ₀/2 ≈ 6.25mm，误差控制在 ±1.25mm
  1. 天线罩厚度 D：
  2. 理想：D = m × λ_c/2（λ_c = λ₀/√ε_r，材料中的波长）
  3. 无法满足半波长时：D ≤ λ_c/8
  4. 目的：保持前后阻抗一致，减少反射和穿透损耗
  1. 材料选择：
  2. 低 Dk（< 3.0），低 Df（< 0.005）
  3. PBT、发泡塑料优先
  4. 避免金属/导电涂层
  1. 结构设计：
  2. 表面平滑均匀，无突变
  3. 涂层不含金属成分
  4. 天线罩表面平行于天线平面
  5. 视场（FOV）内避免多层介质或遮挡

四、天线罩对天线性能的影响

4.1 插入损耗（Insertion Loss）

组成：

IL_total = IL_reflection + IL_absorption + IL_misalignment
损耗类型来源典型值
反射损耗界面不连续0.1-0.5dB
吸收损耗材料 tanδ0.05-0.3dB
失配损耗波束偏移0.2-1.0dB

毫米波频段：由于频率高，同样的厚度会产生更大的相位偏移 → 插入损耗更大。

4.2 波束指向误差（Boresight Error, BSE）

原因：天线罩厚度不均匀 → 不同入射角的相位延迟不同 → 波束偏移。

影响：

  • 单脉冲雷达：差波零点偏移 → 测角误差
  • 相控阵：波束指向偏差 → 覆盖缺口

2019 Wiley 研究：

  • 插入相位延迟（IPD）校正方法
  • 基于平均入射角设计厚度分布
  • IPD 对称校正 → BSE 降低 50%+

4.3 方向图畸变

现象：

  • 主瓣展宽或分裂
  • 旁瓣升高
  • 前后比下降

典型案例（IntechOpen 2010）：

  • 5mm 特氟龙天线罩 → 主瓣衰减 0.4dB，旁瓣从 -3.27dB 升至 -4.19dB
  • 加金属防护帽（DC=12mm）→ 额外衰减 1.1dB，旁瓣显著升高
  • DC=18mm → 额外衰减 6.0dB

4.4 极化纯度下降

原因：天线罩应力双折射 → 线极化波获得椭圆分量。

影响：轴比恶化，圆极化纯度下降。

五、5G 毫米波天线罩的特殊挑战

5.1 高频 = 更薄的"半波长"

频段λ₀λ₀/2λ_c/2 (Dk=3.0)
3.5GHz (n78)85.7mm42.9mm24.8mm
28GHz10.7mm5.4mm3.1mm
60GHz5.0mm2.5mm1.4mm
140GHz (6G)2.1mm1.1mm0.6mm

关键洞察：毫米波天线罩厚度容差只有几十微米！

5.2 多层介质的累积效应

2020 IEEE Access 研究：W 频段（75-110GHz）多层天线罩：

  • 表皮 + 胶合层 + 芯材 + 胶合层 + 内表皮
  • 胶合层和涂层在毫米波频段不可忽略
  • 需计算有效介电常数 ε_eff 来修正芯材尺寸

5.3 散热问题

5G AAU 功率密度高：

  • 天线罩内部温度可达 60-80°C
  • 高温 → 材料 Dk 变化 → 谐振频率偏移
  • 需要热-电磁协同设计

5.4 外观与城市融合

城市 5G 小基站天线罩：

  • 伪装成路灯/广告牌/建筑构件
  • 需要美观 + 电磁透明
  • 曲面 + 变色涂层

六、天线罩设计流程

6.1 标准设计步骤

  1. 确定工作频段和带宽
  2. 选择材料（Dk/Df/强度/成本权衡）
  3. 计算理论厚度（半波长法则）
  4. 结构设计（单层/夹层/变厚度）
  5. 电磁仿真（HFSS/CST 全波仿真）
  6. 优化（GA/RT 算法优化厚度分布）
  7. 原型制作与测试（VNA 测量 S11/S21）
  8. 环境测试（高低温/湿热/风载/冰雹）

6.2 仿真工具

工具用途特点
HFSS全波电磁仿真精确，适合毫米波
CST Studio3D 频域/时域适合复杂结构
FEKO矩量法/物理光学适合大型天线罩

七、给非专业读者的一句话总结

天线罩就是天线的"透明盔甲"——它要扛得住台风冰雹，又要让无线电波毫无阻碍地穿过。在 5G 毫米波时代，这个"盔甲"的厚度必须精确到毫米甚至亚毫米级别，差一点点就会让信号质量下降。6G 太赫兹频段更夸张，新材料 Dk=1.19 的泡沫可以让信号传输距离提升 9 倍。天线罩告诉我们：最好的保护不是"挡在外面"，而是"让你感觉不到我的存在"。

金句：天线罩的最高境界——隐身于无形，护之于有形。
