定向耦合器 Directional Coupler Directional-Coupler

定向耦合器是 T/R 模块校准的"眼睛"——它从主信号路径中采样一小部分功率，供监测和校准使用。耦合度决定采样比例（通常 -10dB 到 -30dB），方向性决定它能区分正向和反向信号的能力。

类比入口

定向耦合器就像水管上的"取样阀"——主水管里水流很大，取样阀只取一点点用来检测水质（信号功率/相位），而且能区分水是往前流还是往后返（正向/反向）。

一、定义与功能

1.1 基本结构

Port 1（输入）───┬─── Port 2（直通/输出）
                      │
                   耦合区
                      │
    Port 4（隔离）────┴─── Port 3（耦合/采样）
    
    信号流向：
      Port 1 → Port 2（主通路，低损耗）
      Port 1 → Port 3（耦合端口，采样信号）
      Port 2 → Port 1（反射信号到 Port 4）
      Port 4（端接负载）

1.2 核心功能

功能说明
功率采样从主通路采样一小部分信号
方向性检测区分正向和反向信号
VSWR 测量通过正向/反向功率计算反射系数
通道校准T/R 模块内部校准的参考信号
功率监测实时监测发射功率

二、关键指标

2.1 耦合度（Coupling Factor）

C = -10·log₁₀(P_coupled / P_input) [dB]
耦合度用途
3dB等分功率（合路器）
6dB不等分功分
10dBT/R 模块校准采样
20dB功率监测
30dB微弱采样（高功率系统）

2.2 方向性（Directivity）

D = C - ISO = 10·log₁₀(P_forward / P_reverse)

方向性越高，正向和反向信号的区分能力越强。

耦合器类型方向性（典型）
微带耦合器15-25dB
带状线耦合器20-30dB
波导耦合器25-35dB
分支线耦合器20-30dB
Lange 耦合器20-25dB

2.3 插入损耗（Insertion Loss）

IL = IL_coupling + IL_conductor + IL_dielectric
    
    理想 3dB 耦合器的最小损耗：3dB（耦合损耗）
    实际插入损耗：3.5-4.5dB（3dB 耦合器）
                  0.5-1.5dB（10dB 耦合器）
                  0.2-0.8dB（20dB 耦合器）

2.4 隔离度（Isolation）

ISO = -10·log₁₀(P_isolated / P_input)
指标典型值
Port 1 → Port 4（隔离端口）> 25dB
Port 2 → Port 3（反向耦合）> 25dB
Port 1 → Port 2（直通）主通路，低损耗

2.5 带宽（Bandwidth）

BW = (f_high - f_low) / f_center
耦合器类型带宽
分支线10-20%
Lange20-40%
多节耦合> 50%
波导10-15%

三、耦合器类型

3.1 分支线耦合器（Branch-Line Coupler）

结构：
      4 条传输线组成矩形环
      每条线长 λ/4
      
    特点：
      - 90° 相位差（Quadrature）
      - 耦合度：3-20dB
      - 带宽：10-20%
      - 易于集成（微带/带状线）

3.2 Lange 耦合器

结构：
      4 条交错 interleaved 的细微带线
      
    特点：
      - 紧密耦合（3dB）
      - 带宽：20-40%
      - 需要金线键合连接
      - 毫米波常用

3.3 波导耦合器

结构：
      波导壁上的耦合孔（单孔/多孔/Bethe耦合）
      
    特点：
      - 极低损耗（0.1-0.3dB）
      - 高方向性（25-35dB）
      - 高功率容量
      - 体积大、成本高

3.4 耦合线耦合器（Coupled-Line Coupler）

结构：
      两条平行传输线，通过边缘场耦合
      
    特点：
      - 反向耦合器（耦合端口在隔离侧）
      - 带宽宽（30-50%）
      - 微带/带状线均可

四、在 T/R 模块中的应用

4.1 校准采样

T/R 模块内部校准链路：
      PA 输出 → 耦合器（-10dB） → 校准接收机
                                  → PA 主输出 → 天线
      
    LNA 输入 → 耦合器（-20dB） ← 校准信号源
      ↑
    从天线反向采样

4.2 VSWR 监测

正向功率 P_fwd ← 耦合器 Port 3
    反向功率 P_rev ← 耦合器 Port 4
    
    Γ = √(P_rev / P_fwd)
    VSWR = (1 + Γ) / (1 - Γ)
    
    实时监控 → 过载保护

4.3 功率控制闭环

目标功率 → DAC 设定 → PA 增益控制
                        ↑
                  耦合器采样 → 检波器 → ADC → 基带
                                            ↓
                                  PID 控制器 → 调整 PA 增益

六、工程注意事项

6.1 功率容量

耦合器额定功率：
      微带：1-10W（Sub-6GHz）
      波导：100W-1kW（基站）
      
    过载后果：
      - 耦合端口烧毁
      - 方向性恶化
      - 永久性损伤

6.2 温度稳定性

耦合度温度漂移：
      微带：±0.1dB/°C
      波导：±0.02dB/°C
      
    方向性温度漂移：
      微带：±0.5dB/°C
      波导：±0.1dB/°C

6.3 集成化趋势

Sub-6GHz：
      耦合器独立元件 → 集成到 T/R 模块 PCB
      
    毫米波：
      耦合器集成到 SiP 封装内
      与 PA/LNA 共用基板

七、一句话总结

定向耦合器是 T/R 模块校准和功率监测的关键组件——它从主信号路径中采样一小部分功率，同时区分正向和反向信号。耦合度决定采样比例（10-20dB 最常用），方向性决定正反向信号的区分能力（> 20dB 为佳）。记住：耦合器不是功分器——功分器把功率分成几路，耦合器只采样一小路，主通路几乎不受影响。
