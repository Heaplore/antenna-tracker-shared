互调 Intermodulation IMD / PIM

互调是两个或多个频率的信号在天线系统中"碰撞"产生的寄生频率——三阶互调（IM3）最危险，因为它离有用信号最近。无源互调（PIM）是基站天线的核心指标，劣化的 PIM 会直接降低接收灵敏度、引发掉话。

类比入口

互调就像两个人在不同房间唱歌——理想情况下你应该只听到他们各自的声音。但如果墙壁有缺陷（非线性），你的耳朵会"听到"他们歌声的合成音（比如一个人唱 900MHz，另一个人唱 1800MHz，你会额外听到 100MHz 和 2700MHz 的"幽灵音"）。

一、互调的基本原理

1.1 互调产物

两个频率 f1 和 f2 通过非线性器件：
    
    输出包含：
      f1（有用）
      f2（有用）
      2f1 - f2  ← 三阶互调（IM3，最危险）
      2f2 - f1  ← 三阶互调（IM3）
      3f1 - 2f2  ← 五阶互调
      3f2 - 2f1  ← 五阶互调
      ...
    
    IM3 离 f1/f2 最近，最难滤除

1.2 三阶截点（IP3）

IP3 是衡量线性度的关键指标：
    
    输入 IP3（IIP3）：
      IIP3 = Pout + ΔP
      其中 ΔP = 输出功率与 IM3 产物的差值
    
    输出 IP3（OIP3）：
      OIP3 = IIP3 + Gain
    
    IP3 越高 → 线性度越好 → IM3 越小
器件类型IIP3 典型值
优质 PA+30 to +40dBm
一般 PA+20 to +30dBm
低噪声放大器+20 to +30dBm
连接器> +100dBm（无源）
劣质接头+50 to +80dBm PIM

1.3 无源互调（PIM）

PIM 定义：
      无源器件（天线、连接器、馈线）中的非线性效应
      产生的互调产物
    
    PIM 单位：dBc（相对于载波）
    
    典型要求：
      基站天线 PIM：≤ -150dBc（双载波 2×43dBm）
      5G 多载波 PIM：≤ -167dBc

二、PIM 的来源

2.1 主要来源

1. 连接器（最大来源）
       - 不同金属接触面氧化
       - 螺纹松动
       - 清洁度差
    
    2. 天线辐射单元
       - 金属表面氧化/污染
       - 不同金属接触（Galvanic）
    
    3. 馈线/跳线
       - 弯曲半径过小
       - 接头不良
    
    4. 安装环境
       - 生锈的铁丝/金属物靠近天线
       - 螺栓松动

2.2 材料 PIM 敏感性

材料PIM 影响
铜/镀锡铜低（优秀）
不锈钢中等
镀锌钢较高
铁/碳钢高（避免使用）
铝中等（表面氧化后升高）
不同金属接触高（Galvanic PIM）

三、5G 多载波对 PIM 的要求

3.1 多载波叠加效应

2G/3G 单载波：PIM 影响较小
    4G CA（2载波）：PIM 要求 -150dBc
    5G NSA（多载波）：PIM 要求 -167dBc
    
    原因：
      载波越多 → 互调产物越多
      功率越高 → PIM 越大（PIM ∝ P²）

3.2 5G 频段 PIM 挑战

频段典型功率PIM 要求
700MHz（n28）43dBm≤ -150dBc
1.8GHz（n3）43dBm≤ -150dBc
2.6GHz（n41）41dBm≤ -157dBc
3.5GHz（n78）40dBm≤ -167dBc
4.9GHz（n79）38dBm≤ -167dBc

3.3 PIM 与功率的关系

PIM 功率增量：
      输入功率 ↑ 3dB → PIM ↑ 6dB（二阶）
      输入功率 ↑ 3dB → PIM ↑ 10dB（三阶）
    
    所以 PIM 是"功率的倍数"恶化

四、PIM 测试与测量

4.1 双音 PIM 测试

测试方法：
      1. 两个大功率信号 f1 和 f2（如 43dBm）
      2. 通过 PIM 测试仪注入天线
      3. 测量 f1、f2 产生的 PIM 产物
      4. 报告最差 PIM 值
    
    典型测试条件：
      f1 = 930MHz, f2 = 960MHz（GSM 频段）
      P1 = P2 = 43dBm（2×20W）
      测量 IM3 = 2f1 - f2 和 2f2 - f1

4.2 现场 PIM 测试

现场测试步骤：
      1. 断开天线与 RRU 连接
      2. 连接 PIM 测试仪
      3. 施加双音信号
      4. 记录 PIM 值
      5. 逐个测试馈线/接头
    
    PIM 告警阈值：
      > -150dBc：警告
      > -145dBc：严重
      > -140dBc：必须整改

五、PIM 控制措施

5.1 天线设计

低 PIM 天线设计要点：
      1. 使用单一金属材料（优选铜或镀锡铜）
      2. 避免不同金属接触
      3. 辐射单元间隙 > 0.5mm
      4. 表面处理：镀锡/镀银
      5. 结构对称，减少磁滞效应

5.2 连接器

低 PIM 连接器要求：
      1. 扭矩扳手紧固（按规格值）
      2. 清洁接触面
      3. 使用防水胶泥
      4. 避免过度弯曲馈线
    
    连接器 PIM 等级：
      普通：-100 to -120dBc
      低 PIM：-140 to -150dBc
      超低 PIM：≤ -167dBc

5.3 安装规范

安装 PIM 禁忌：
      ✗ 铁丝/扎带靠近天线
      ✗ 生锈的金属物在 2m 范围内
      ✗ 不同金属螺栓混用
      ✗ 馈线弯曲半径 < 10×外径
      ✓ 使用非金属扎带
      ✓ 保持天线周围清洁
      ✓ 按扭矩规范紧固

六、一句话总结

互调是多载波系统的"隐形杀手"——PIM 每恶化 10dB，接收灵敏度就下降 1dB。5G NSA 组网下，PIM 要求达到 -167dBc。记住三个关键点：① PIM 与功率平方成正比 ② 连接器是最大来源 ③ 安装规范比天线本身更重要——一根生锈的铁丝就能让 PIM 恶化 20dB。
