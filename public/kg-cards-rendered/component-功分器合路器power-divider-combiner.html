功分器/合路器 Power Divider / Combiner Power-Divider-Combiner

功分器把一路信号分成多路（等幅或不等幅），合路器把多路信号合成一路——它们是馈电网络、PA 阵列、LNA 阵列的"交通枢纽"。Wilkinson 是最经典的 2 路功分器，Rat-Race 是 3dB 电桥，Hybrid Coupler 提供 90° 或 180° 相位差。

类比入口

功分器就像公路的分岔口——一辆车（信号）从主干道来，分成两条支路出去。合路器则是两条支路汇成主干道。Wilkinson 功分器的精妙之处在于它中间加了个"隔离桥"（电阻）——如果一条支路堵车（阻抗失配），车流不会倒灌回另一条支路。

一、定义与功能

1.1 功分器（Power Divider / Splitter）

         ┌──→ Port 2（等幅同相）
    Input ──→│
             └──→ Port 3（等幅同相）
             ↑
          Port 1（隔离电阻）

理想 2 路等分功分器：

  • 输出功率：P_out = P_in / 2（理论）
  • 功率损失：3dB（一半功率去了一路）
  • 插入损耗：3dB + 导体损耗 ≈ 3.1-3.5dB
  • 隔离度：> 20dB（Port 2 和 Port 3 之间）

1.2 合路器（Combiner）

Port 2 ──→ ──┐
                  ├──→ Output（合成）
    Port 3 ──→ ──┘

理想合路器：

  • 与功分器互为逆过程
  • 插入损耗：3dB + 导体损耗
  • 隔离度：> 20dB（两输入端口之间）

1.3 N 路功分器

2 路 → 1 级
    4 路 → 2 级（2→4 = 2×2）
    8 路 → 3 级
    16 路 → 4 级
    N 路 → log₂(N) 级

级联损耗：

N 路总损耗 = 10·log₁₀(N) + IL_per_stage × log₂(N)
N级数理论分配损耗实际总损耗（含插入损耗）
213.0dB3.2-3.5dB
426.0dB6.8-7.5dB
839.0dB10.5-11.8dB
16412.0dB14.5-16.5dB

二、主要类型

2.1 Wilkinson 功分器（最经典）

        Z₀√2, λ/4
    Port 1 ──┤╬├── Port 2
             │R│
             │ │    R = 2Z₀ = 100Ω（50Ω 系统）
    Port 3 ──┤╬├── 
            Z₀√2, λ/4

原理：

  • 两段 λ/4 阻抗变换线（Z = Z₀√2）
  • 中心隔离电阻 R = 2Z₀
  • 当两输出端都匹配时，电阻上无电流 → 不耗能
  • 当一端失配时，电阻吸收反射 → 保护另一端

性能：

参数典型值
插入损耗3.1-3.5dB
隔离度20-30dB
VSWR1.1-1.3
带宽10-20%
幅度平衡±0.5°
相位平衡±0.1dB

2.2 T 结功分器（T-Junction）

最简单的结构：

Port 1 ──┬── Port 2
             └── Port 3

性能：

参数典型值
插入损耗3.0dB（理论）+ 反射损耗
隔离度-3dB（很差！）
VSWR2.0-3.0
带宽很宽（DC 到 GHz）
成本最低

适用场景：对隔离度要求不高的场合

2.3 Rat-Race 合路器（Race-Track Combiner）

        Σ Port（求和）
             │
        ┌────┴────┐
        │         │
        │  λ/4   λ/2   λ/4
        │         │
        └────┬────┘
             │
            Δ Port（差值）

特点：

  • 3dB 合路/功分
  • 90° 相位差（Σ 和 Δ 端口）
  • 隔离度：Σ 和 Δ 端口之间隔离
  • 带宽：10-20%

2.4 混合耦合器（Hybrid Coupler）

90° 混合耦合器（Branch-Line Coupler）

Port 1 ──→ Port 2（0°）
    Port 1 ──→ Port 3（-90°）
    Port 1 ──→ Port 4（-180°，隔离）

180° 混合耦合器（Magic-T / Ring Hybrid）

Port 1 ──→ Port 2（0°）
    Port 1 ──→ Port 3（-180°）
    Port 1 ──→ Port 4（隔离）

2.5 定向耦合器（Directional Coupler）

Port 1 ──→ Port 2（直通，主信号）
                  ↑
             Port 3（耦合端口，取样信号）
                  ↑
             Port 4（隔离端口，端接）

耦合度：

耦合度用途
3dB等分功分
6dB不等分功分
10dB功率监测
20dB采样/检测

三、关键指标

3.1 插入损耗

IL = -10·log₁₀(P_out / P_in)
类型IL（典型）
Wilkinson3.1-3.5dB
T-Junction3.0dB + 反射
Rat-Race3.5-4.0dB
90° Hybrid3.5-4.5dB
定向耦合器取决于耦合度

3.2 隔离度

ISO = -10·log₁₀(P_coupled / P_input)
类型隔离度
Wilkinson20-30dB
T-Junction-3dB（无隔离）
Rat-Race20-25dB
90° Hybrid20-30dB

3.3 幅度/相位平衡

幅度不平衡：ΔP = |P_out1 - P_out2| < 0.5dB
    相位不平衡：Δφ = |φ_out1 - φ_out2| < 5°

3.4 带宽

BW = (f_max - f_min) / f_center
类型带宽
Wilkinson10-20%
T-Junction> 100%（DC 到 GHz）
Rat-Race10-15%
90° Hybrid10-20%
多节 Wilkinson> 50%

四、多路扩展

4.1 树状扩展（Corporate Feed）

1→2 → 2→2 → 4→2 → 8→2 → 16 路

优势：设计简单 劣势：级数多时损耗大

4.2 并行扩展（Bus Feed）

Input ──────────────────────
            │     │     │     │
          Unit 1 Unit 2 Unit 3 Unit 4

优势：结构简单 劣势：幅度不均匀（远端弱）

4.3 多节阻抗变换（Broadband Wilkinson）

1→2 宽频带设计：
      使用多级 λ/4 变换线
      每级阻抗不同
      带宽可扩展到 > 50%

六、工程注意事项

6.1 功率容量

高功率下电阻发热 → Wilkinson 的中心电阻是关键瓶颈
类型功率容量
微带 Wilkinson1-10W
波导 Wilkinson100W-1kW
T-Junction10-100W（无电阻，功率容量大）

6.2 温度稳定性

电阻值随温度变化 → 隔离度变化
    ΔR/R ≈ 0.01%/°C（金属膜电阻）

6.3 毫米波设计挑战

挑战解决方案
导体损耗增大使用波导或厚铜
尺寸缩小光刻/精密 CNC
加工公差敏感仿真优化 + 容差分析
辐射损耗屏蔽/接地

七、一句话总结

功分器/合路器是天线的"十字路口"——Wilkinson 是最经典的等分方案（加了隔离电阻所以好使），Rat-Race 和 Hybrid Coupler 提供相位关系（90°/180°），定向耦合器用于功率采样。N 路功分器通过 log₂(N) 级 2 路级联实现。记住：功分器的"分"不是免费的——2 路等分就有 3dB 理论损耗，16 路就是 12dB。
