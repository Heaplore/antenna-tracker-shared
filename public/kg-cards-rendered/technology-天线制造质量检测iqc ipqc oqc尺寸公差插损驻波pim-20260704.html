天线制造质量检测 IQC IPQC OQC

天线制造质检分三道关：IQC（来料检——基板/元器件合格吗）、IPQC（制程检——尺寸/插损/VSWR 达标吗）、OQC（出货检——整天线测试全通过吗）。毫米波天线最关键是线宽公差和表面粗糙度——差 0.01mm，阻抗就差 2Ω。

类比入口

天线质检就像"体检三关"——IQC 是"验血"（查原材料），IPQC 是"B超"（查加工过程），OQC 是"全身CT"（查成品性能）。

一、IQC 来料检验

1.1 基板材料检验

检验项目：
      1. 外观检查
         - 表面划痕/气泡/杂质
         - 颜色/厚度均匀性
         - 标准：GB/T 1770
          
      2. 尺寸测量
         - 长度/宽度/厚度
         - 公差：±0.1mm（厚度）
         - 工具：千分尺/影像仪
          
      3. 介电常数 Dk
         - 谐振腔法/微波法
         - Rogers RO4350B：Dk = 3.48 ± 0.05
         - 偏差 > 0.05 → 拒收
          
      4. 损耗角正切 Df
         - Df = 0.0037 ± 0.0004（RO4350B）
         - 偏差 > 0.001 → 降级使用
          
      5. 铜箔附着力
         - 剥离强度 > 0.8N/mm
         - 划格法测试
          
      6. 铜箔厚度
         - 标准：35μm（1oz）/ 18μm（0.5oz）
         - 公差：±3μm

1.2 射频元器件检验

PA/LNA/滤波器/连接器检验：
      1. 外观：引脚氧化/封装破损
      2. 尺寸：引脚间距/封装尺寸
      3. 电气参数：
         - PA：输出功率/增益/效率
         - LNA：噪声系数/增益
         - 滤波器：中心频率/带宽/插损
      4. 批次一致性：抽样 10%，CPK > 1.33

二、IPQC 制程检验

2.1 冲压工艺检验

检验项目：
      1. 尺寸公差
         - 长度/宽度：±0.1mm
         - 厚度：±0.05mm
         - 孔位：±0.05mm
         - 工具：CMM/影像仪
          
      2. 表面质量
         - 毛刺高度 < 0.05mm
         - 裂纹：0 容忍
         - 划伤深度 < 0.02mm
          
      3. 镀层质量
         - 镀层厚度：镍 5-10μm，铬 0.3-0.5μm
         - 附着力：百格法 5B
         - 盐雾测试：24h 无锈蚀
          
      4. 角度/平整度
         - 折弯角度：±0.5°
         - 平面度：< 0.5mm/m

2.2 蚀刻工艺检验

检验项目：
      1. 线宽/线距
         - Sub-6GHz：线宽 ≥ 0.1mm，公差 ±0.02mm
         - 毫米波：线宽 ≥ 0.05mm，公差 ±0.01mm
         - 工具：AOI/显微镜
          
      2. 铜厚
         - 标准：35μm（1oz）
         - 蚀刻后：≥ 28μm（保留 80%）
         - 工具：涡流测厚仪
          
      3. 阻抗测试
         - 微带线阻抗：50Ω ± 5Ω
         - 工具：TDR（时域反射计）
         - 抽样：每批次 5%
          
      4. 插损测试
         - 10cm 微带线 @ 3.5GHz：插损 < 0.5dB
         - 10cm 微带线 @ 28GHz：插损 < 2dB
         - 工具：VNA
          
      5. 孔质量
         - 孔径公差：±0.05mm
         - 孔壁铜厚：≥ 10μm
         - 工具：切片分析/显微镜

2.3 CNC 加工检验

检验项目：
      1. 尺寸精度
         - 定位精度：±0.005mm
         - 表面粗糙度：Ra 0.4-1.6μm
         - 工具：CMM/粗糙度仪
          
      2. 形位公差
         - 平面度：< 0.02mm
         - 平行度：< 0.01mm
         - 同轴度：< 0.01mm
          
      3. 表面质量
         - 刀纹深度：< 0.01mm
         - 无烧伤/裂纹
         - 工具：显微镜/白光干涉仪

2.4 组装工艺检验

检验项目：
      1. 胶水/密封胶
         - 涂胶量：按工艺规范
         - 固化时间/温度：按胶水规格
         - 拉力测试：抽样验证
          
      2. 螺栓紧固
         - 扭矩：按规格（25-60Nm）
         - 防松标记：涂 Loctite 蓝胶
         - 工具：扭矩扳手
          
      3. 防水密封
         - O 型圈安装：无扭曲/翻转
         - 防水接头：拧紧 + 防水胶泥
         - IP 等级验证：抽样做防水测试

三、OQC 出货检验

3.1 5G AAU 出厂测试

测试项目（100% 全检）：
      1. S 参数测试
         - S11（回波损耗）：> 14dB（VSWR < 1.5）
         - S21（插入损耗）：< 0.5dB
         - 端口隔离度：> 25dB
         - 设备：VNA（4-8 端口）
         - 时间：~10min
          
      2. 有源测试（远场/近场）
         - 增益：标称值 ± 0.5dB
         - 方向图：主瓣指向 ± 1°
         - 波束宽度：标称值 ± 1°
         - 前后比：> 30dB
         - 旁瓣电平：> 13dB 抑制
         - 设备：暗室 + 转台 + VNA
         - 时间：~30min
          
      3. MIMO 通道一致性
         - 幅度一致性：< 0.5dB
         - 相位一致性：< 5°
         - 空间流数：验证 64T64R
         - 时间：~10min
          
      4. PIM 测试
         - 双三载波 PIM：< -150dBc
         - 设备：PIM 测试仪
         - 时间：~5min
          
      5. 极化隔离度
         - ±45° 极化隔离：> 25dB
         - 时间：~5min
          
      6. 功能测试
         - RET 功能：电调下倾正常
         - 告警功能：温度/驻波告警
         - 时间：~5min
    
      总测试时间：~75min/台

3.2 测试不合格处理

不合格分类：
      1. A 类（致命）
         - S11 不达标（VSWR > 2.0）
         - 增益偏差 > 1dB
         - 方向图严重畸变
         - 处理：返工或报废
         
      2. B 类（严重）
         - 增益偏差 0.5-1dB
         - 前后比偏低 3-5dB
         - 通道一致性略超
         - 处理：调试后复测
         
      3. C 类（轻微）
         - 外观瑕疵
         - 标识不清
         - 包装破损
         - 处理：返修后放行
    
    直通率目标：
      - 5G AAU：> 95%
      - 毫米波 AAU：> 90%
      - 传统 PAS：> 98%

五、自动化质检

5.1 AOI（自动光学检测）

应用场景：
      - PCB 蚀刻后线宽检测
      - 组装后外观缺陷检测
      - 焊点质量检测
      
    精度：
      - 线宽检测：±0.01mm
      - 缺陷识别率：> 99%
      
    设备：
      - Omron FA 系列
      - KLA 系列
      - 国产：新益昌/大族光电子

5.2 AI 视觉检测

应用场景：
      - 表面缺陷分类（划痕/凹坑/锈迹）
      - 镀层质量评估
      - 组装完整性检查
      
    技术：
      - CNN 图像分类
      - 目标检测（YOLO）
      - 异常检测（AutoEncoder）
      
    效果：
      - 缺陷检出率：> 99.5%
      - 误报率：< 0.5%
      - 检测速度：< 3s/件

六、一句话总结

天线制造质检是"来料→制程→出货"三道防线——IQC 把关原材料，IPQC 监控加工过程，OQC 验证成品性能。5G AAU 出厂测试要过 S 参数/有源/MIMO/PIM/极化隔离六道关，总测试时间约 75 分钟/台。记住：天线制造最怕的不是"测不出来"，而是"测出来了但来不及返工"——所以 IPQC 的过程控制比 OQC 的事后检验更重要。QMS 的核心价值就是用数据驱动过程控制，把缺陷消灭在出厂之前。
