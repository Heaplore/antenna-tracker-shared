EIRP / ERP（等效全向辐射功率 / 等效辐射功率） EIRP-ERP

EIRP 是"假设天线各向同性辐射时需要的功率"——它是发射功率和天线增益的综合指标，决定了信号在自由空间中的"杀伤力"。ERP 类似但参考基准是偶极子而非各向同性体。

类比入口

EIRP 就像"子弹的杀伤力"——不仅看枪膛里的弹药量（发射功率），还要看枪管有没有瞄准器（天线增益）。一把 1W 的手枪（各向同性天线）和一把 1W 但装了瞄准镜的手枪（10dBi 增益天线），后者能打得更远更准。EIRP 就是"装了瞄准镜后的等效杀伤功率"。

一、定义

1.1 EIRP（Effective Isotropic Radiated Power）

EIRP = P_t · G_t
    EIRP(dBW) = P_t(dBW) + G_t(dBi)
    EIRP(dBm) = P_t(dBm) + G_t(dBi)

其中：

  • P_t = 发射机输出功率（不含天线增益）
  • G_t = 发射天线增益（dBi）

物理意义：如果用一个各向同性天线（G = 1）替代现有天线，需要多大的功率才能达到相同的功率密度？

1.2 ERP（Effective Radiated Power）

ERP = P_t · G_t_dipole
    ERP(dBm) = P_t(dBm) + G_t(dBd)

其中 G_t(dBd) 是相对于半波偶极子的增益。

与 EIRP 的关系：

EIRP(dBW) = ERP(dBW) + 2.15
    ERP(dBW) = EIRP(dBW) - 2.15

因为半波偶极子的增益是 2.15dBi。

1.3 EIRP vs ERP vs 发射功率

2.2 5G 毫米波 EIRP 计算

P_t per channel = 20-30dBm (100mW-1W)
    G_t per panel = 20-30dBi
    EIRP = P_t + G_t = 40-60dBm (10W-1kW)

实际限制：

  • 中国：28GHz EIRP ≤ 40dBm（功率密度限制）
  • 美国：28GHz EIRP ≤ 41dBm
  • 欧盟：28GHz EIRP ≤ 40dBm

2.3 功率密度限制（PSD）

毫米波频段更关注功率密度而非总 EIRP：

PSD = EIRP / A_beam
    PSD(dBm/mm²) = EIRP(dBm) - 10·log₁₀(A_beam_mm²)

FCC 限制（>6GHz）：

PSD ≤ 10dBm/mm²（1cm² 平均）
    或 PSD ≤ 23dBm/600MHz（带宽归一化）

2.4 SAR（Specific Absorption Rate，比吸收率）

手机等贴近人体的设备受 SAR 限制：

SAR = σ · E² / ρ

其中 σ 是组织电导率，E 是电场强度，ρ 是组织密度。

三、EIRP 计算示例

3.1 5G 基站 AAU

配置：
    P_t per RRU = 40W = 46dBm
    G_t (64T64R) = 28dBi
    馈线损耗 = 1dB
    
    EIRP = 46 + 28 - 1 = 73dBm = 73W ≈ 20kW

注意：73dBm 是峰值 EIRP，实际平均 EIRP 取决于调制方式和功率回退。

3.2 卫星下行链路

配置：
    转发器功率 = 100W = 50dBW
    天线增益 = 30dBi
    馈线损耗 = 1dB
    
    EIRP = 50 + 30 - 1 = 79dBW ≈ 80MW

3.3 WiFi 路由器

配置：
    P_t = 20dBm（100mW）
    G_t = 5dBi
    
    EIRP = 20 + 5 = 25dBm = 316mW

3.4 GPS 卫星上行

配置：
    P_t = 50W = 47dBW
    G_t = 40dBi（L1 频段）
    
    EIRP = 47 + 40 = 87dBW ≈ 5GW

四、EIRP 与链路预算

4.1 完整链路方程

P_r = EIRP + G_r - L_fs - L_other
    P_r = (P_t + G_t) + G_r - L_fs - L_other

4.2 信噪比

SNR = P_r - P_n
    SNR = EIRP + G_r - L_fs - L_other - P_n
    SNR = EIRP + G_r/T_sys - L_fs - L_other - 10·log₁₀(k·B)

4.3 链路裕量

Margin = EIRP + G_r - L_fs - L_other - P_min

工程建议：

  • 5G/卫星/雷达 → 用 EIRP（dBi 基准）
  • FM 广播/电视 → 用 ERP（dBd 基准）
  • 法规合规 → 按当地法规要求选择

六、一句话总结

EIRP 是"发射功率 × 天线增益"的综合指标，告诉你信号到底有多"猛"。它不是真实的物理功率，而是一个等效值——假设你用一根什么都不聚光的天线（各向同性），需要多大功率才能达到同样的效果。记住：EIRP = P_t + G_t，单位 dB；ERP = EIRP - 2.15dB（因为偶极子比各向同性强 2.15dB）。法规管的就是 EIRP 上限，不是单纯的发射功率。
