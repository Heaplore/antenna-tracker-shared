巴伦 Balun Balance-Unbalance Converter

巴伦是平衡信号与不平衡信号之间的"翻译官"——它将单端信号（如 coax 电缆）转换为差分信号（如微带线），反之亦然。核心指标是幅度平衡、相位平衡和插入损耗。

类比入口

想象你有一根水管（同轴电缆，单端）要连接到两个对称的水管（差分线，平衡）。巴伦就是这个"三通接头"——它把一股水分成两股对称的水流，而且两股水流的方向相反（差分）。如果巴伦做得不好，两股水流就不对称（幅度不平衡）或者不同时到达（相位不平衡）。

一、定义与功能

1.1 平衡 vs 不平衡

不平衡（Unbalanced）：
      信号线 + 地线（参考面）
      例：同轴电缆、单端微带线
      
    平衡（Balanced）：
      两条对称的信号线
      例：差分微带线、对称振子天线

1.2 巴伦的功能

单端输入 → 巴伦 → 差分输出
      (50Ω)      ↑      (2 × 50Ω 差分)
      
    或反过来：
    差分输入 → 巴伦 → 单端输出

核心作用：

  1. 阻抗变换：单端 50Ω → 差分 100Ω
  2. 信号对称：输出两路幅度相等、相位差 180°
  3. 抑制共模：阻止共模信号通过
  4. 防止共模电流：在同轴电缆外皮上产生共模电流时会辐射干扰

1.3 巴伦 vs 功分器

功分器：1 路 → N 路（同相输出）
    巴伦：1 路 → 2 路（180° 反相输出）
    
    功分器：幅度相等，相位相同
    巴伦：幅度相等，相位相反（180°）

二、巴伦的主要类型

2.1 传输线巴伦（Transmission Line Balun）

180° 传输线巴伦

结构：
      两条平行传输线，一端交叉连接
      
    信号流向：
      输入 → 传输线 A → 输出 +
      输入 → 传输线 B（交叉）→ 输出 -
      
    相位差：180°

Marconi Balun（折叠式）

结构：
      同轴电缆绕芯（共模扼流圈）
      
    原理：
      共模电流被扼流圈抑制
      差模信号正常通过

2.2 分支线耦合器巴伦（Branch-Line Coupler）

        Port 1（输入）
             │
        ┌────┴────┐
        │         │
        │  90°    │
        │ Hybrid  │
        │         │
        └────┬────┘
        Port 2    Port 3（90° 相位差）
        
    注意：这是 90° 巴伦，不是 180°！

2.3 环形桥巴伦（Rat-Race Coupler）

        Port 1（Σ 端口）
             │
        ┌────┴────┐
        │         │
        │ 180°    │
        │ Hybrid  │
        │         │
        └────┬────┘
        Port 2    Port 3（180° 相位差）
             │
        Port 4（隔离端口）

2.4 变压器巴伦（Transformer Balun）

结构：
      磁芯 + 初级绕组 + 次级绕组（中心抽头）
      
    原理：
      电磁感应实现平衡-不平衡转换
      
    优点：
      宽带（DC 到 GHz）
      阻抗变换灵活

2.5 功率合成巴伦（Power Combiner Balun）

结构：
      两个晶体管差分对 + 传输线
      
    用途：
      PA 中的差分功率合成
      LNA 中的差分输入匹配

三、关键指标

3.1 相位平衡（Phase Balance）

Δφ = |φ_+ - φ_- - 180°|
巴伦类型相位平衡
传输线巴伦±5-10°
变压器巴伦±2-5°
分支线耦合器±5°
Rat-Race±3-5°

3.2 幅度平衡（Amplitude Balance）

ΔA = |A_+ - A_-| [dB]
巴伦类型幅度平衡
传输线巴伦±0.5-1dB
变压器巴伦±0.2-0.5dB
分支线耦合器±0.5dB
Rat-Race±0.3dB

3.3 插入损耗（Insertion Loss）

IL = IL_balun + IL_matching
巴伦类型IL（典型）
传输线巴伦0.2-0.5dB
变压器巴伦0.1-0.3dB
分支线耦合器0.3-0.8dB
Rat-Race0.5-1dB

3.4 带宽（Bandwidth）

BW = (f_high - f_low) / f_center
巴伦类型带宽
传输线巴伦20-40%
变压器巴伦1-2 个十倍频程
分支线耦合器10-20%
Rat-Race15-25%

3.5 隔离度（Isolation）

隔离度 = 两个输出端口之间的衰减
巴伦类型隔离度
传输线巴伦15-25dB
变压器巴伦20-30dB
Rat-Race20-30dB

四、典型应用

4.1 差分 LNA 输入

单端天线 → 巴伦 → 差分 LNA
                  ↑
             抑制共模噪声

4.2 差分 PA 输出

差分 PA → 巴伦 → 单端天线
                  ↑
             功率合成

4.3 平衡混频器

LO（单端）→ 巴伦 → 差分 LO 输入
    RF（单端）→ 巴伦 → 差分 RF 输入

4.4 平衡天线馈电

单端馈线 → 巴伦 → 对称振子天线
                  ↑
             防止共模电流

4.5 5G MIMO 天线

MIMO 天线阵列中每个通道都需要巴伦：
      单端馈线 → 巴伦 → 差分天线单元

五、巴伦设计要点

5.1 频率选择

f < 1GHz → 变压器巴伦（宽带好）
    1-6GHz → 传输线巴伦（紧凑）
    > 6GHz → 微带/共面波导巴伦（集成）

5.2 阻抗匹配

单端阻抗 Z_0 = 50Ω
    差分阻抗 Z_diff = 100Ω
    
    巴伦阻抗变换比：
      1:2（50Ω → 100Ω 差分）
      1:4（50Ω → 200Ω 差分，较少用）

5.3 共模抑制比（CMRR）

CMRR = 20·log₁₀(|A_diff| / |A_common|)
CMRR效果
> 30dB良好
> 40dB优秀
> 50dB极佳

六、工程注意事项

6.1 巴伦的功率容量

PA 输出端巴伦：
      手机：P < 23dBm
      基站：P < 40dBm
      
    LNA 输入端巴伦：
      功率很小，主要考虑线性度

6.2 温度稳定性

变压器巴伦的温度漂移：
      Δf/f/ΔT ≈ 10-50 ppm/°C
    
    传输线巴伦的温度漂移：
      Δf/f/ΔT ≈ 5-20 ppm/°C

6.3 尺寸与集成

Sub-6GHz 巴伦：1-3mm²（芯片级）
    毫米波巴伦：0.5-1mm²（集成于 BFIC）

七、一句话总结

巴伦是平衡与不平衡信号之间的"翻译官"——把单端信号变成差分信号（或反过来），核心指标是相位平衡（±2-10°）、幅度平衡（±0.2-1dB）和插入损耗（0.1-1dB）。变压器巴伦宽带最好，传输线巴伦最紧凑，Rat-Race 和分支线耦合器常用于特定频率。记住：巴伦不是功分器——功分器输出同相，巴伦输出反相（180°）。
