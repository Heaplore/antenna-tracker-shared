天线互耦 Mutual Coupling in Antenna Arrays

互耦是天线阵列中相邻单元之间的电磁相互作用——当一个单元辐射时，一部分能量会通过空间耦合或表面波耦合被邻近单元"偷走"，导致阻抗偏移、方向图畸变和 MIMO 容量下降。它是阵列设计中无处不在的物理现象，无法消除只能管理。

类比入口

想象一排人站在广场上喊话。第一个人喊的时候，声音不仅传向远方，还会被旁边的人听到并产生回声——这些人听到的声音和自己嘴里发出的声音叠加在一起，改变了他们对自己"声音大小"的判断。天线阵列中的互耦就是这样：每个单元不仅辐射自己的信号，还会"听到"邻居的信号，从而改变自己的阻抗和辐射特性。

一、基本定义与原理

1.1 什么是互耦？

互耦（Mutual Coupling）是指天线阵列中两个或多个单元之间的电磁能量交换。当一个单元（端口 n）被激励时，不仅该单元辐射能量，部分能量还会通过以下方式耦合到其他单元（端口 m）：

  1. 空间耦合（Space Wave）：直接通过自由空间传播
  2. 表面波耦合（Surface Wave）：通过基板或接地面传播
  3. 导波耦合（Guide Wave）：通过馈电网络传播

1.2 互耦的数学描述

阻抗矩阵形式

[V₁]   [Z₁₁  Z₁₂  ... Z₁N]   [I₁]
    [V₂]   [Z₂₁  Z₂₂  ... Z₂N]   [I₂]
    [...] = [...            ...] · [...]
    [VN]   [ZN1  ZN2  ... ZNN]   [IN]

其中：

  • Z_nn 是自阻抗（self-impedance）
  • Z_nm (n≠m) 是互阻抗（mutual impedance）

关键关系：

Z₁₂ = Z₂₁  （互易定理，无源阵列）

S 参数形式

[b₁]   [S₁₁  S₁₂  ... S₁N]   [a₁]
    [b₂]   [S₂₁  S₂₂  ... S₂N]   [a₂]
    [...] = [...            ...] · [...]
    [bN]   [SN1  SN2  ... SNN]   [aN]

其中：

  • S_nm = b_n / a_m |_{a_k=0, k≠m} 是端口 m 到端口 n 的传输系数
  • |S_nm|² 表示从端口 m 耦合到端口 n 的功率比例

工程意义：

  • S₁₂ = -20dB → 5% 的能量从端口 2 耦合到端口 1
  • S₁₂ = -30dB → 0.1% 的能量耦合
  • S₁₂ = -40dB → 0.01% 的能量耦合

1.3 互耦的物理机制

激励端口 m
        │
        ▼
    ┌─────────┐     空间耦合      ┌─────────┐
    │  单元 m  │ ───────────────→ │  单元 n  │
    │ (发射)   │                  │ (接收)   │
    └─────────┘                  └─────────┘
        │                              ▲
        │         表面波               │
        └──────────────────────────────┘
             通过基板/接地面传播

空间耦合：

  • 距离衰减：∝ 1/r²（远场）或 1/r³（近场）
  • 与波长相关：d > λ/2 时显著降低
  • 与极化相关：正交极化耦合更低

表面波耦合：

  • 在介质基板中传播，衰减较慢
  • 在高介电常数基板（如硅、LTCC）中更强
  • 与基板厚度、介电常数、损耗角正切相关

馈电网络耦合：

  • 通过公共馈电线传播
  • 在功分器/合路器中形成间接耦合路径
  • 与馈电网络的隔离度相关

二、互耦的影响

2.1 对阻抗的影响

互耦导致每个单元的输入阻抗随阵列状态变化：

Z_in,n = Z_nn + Σ(m≠n) (I_m/I_n) · Z_nm

后果：

  • 单个单元的 50Ω 匹配在阵列中可能变成 30+j20Ω
  • 波束扫描时阻抗随扫描角变化（scan impedance variation）
  • VSWR 恶化，反射损耗增加

典型数据：

单元间距S₁₂ (dB)阻抗偏移
0.5λ-10dB±15Ω
0.7λ-18dB±8Ω
1.0λ-25dB±3Ω
1.5λ-35dB±1Ω

2.2 对方向图的影响

互耦改变有源单元方向图（AEP）：

AF_active(θ) = Σ w_n · exp(jk·r_n) · F_element(θ) · (1 + Σ_m≠n Γ_nm)

其中 Γ_nm 是单元 m 对单元 n 的耦合系数。

后果：

  • 主瓣增益变化（通常降低 0.5-2dB）
  • 旁瓣电平升高（3-10dB）
  • 波束指向偏移（0.1-1°）
  • 扫描盲区（scan blindness）

2.3 对 MIMO 容量的影响

互耦降低 MIMO 信道矩阵的条件数：

H_MIMO = H_free_space + H_coupling
    H_coupling = K_c · H_free_space

其中 K_c 是耦合矩阵。

后果：

  • 信道矩阵列之间相关性增加
  • 空间复用增益下降
  • MIMO 容量降低 10-30%

但：在某些情况下，互耦可以增加相关性从而提高容量（当空间自由度受限时）。

2.4 对波束赋形的影响

互耦导致波束赋形向量偏离预期：

w_actual = (I + K_c) · w_desired

后果：

  • 零陷深度恶化
  • 旁瓣不对称
  • 波束指向误差

四、互耦的减小方法

4.1 增加单元间距

最简单的办法，但有局限性：

S₁₂ ∝ exp(-π·d/λ)  （近似）
间距S₁₂阵列尺寸
0.5λ-10dB最小
0.7λ-18dB紧凑
1.0λ-25dB适中
1.5λ-35dB较大

代价：阵列尺寸增大，可能出现栅瓣（d > λ 时）。

4.2 去耦结构

详见 技术概念-MIMO天线去耦Decoupling：

  • DGS（缺陷Ground结构）
  • EBG（电磁带隙）
  • 中和线（Neutralization Line）
  • 超材料/超表面
  • 隔离腔体

4.3 极化正交

利用正交极化降低互耦：

S₁₂ (co-pol) = -10dB
    S₁₂ (cross-pol) = -30dB

4.4 数字域互耦补偿

在数字波束赋形中，通过预矫正矩阵消除互耦影响：

w_compensated = (I + K_c)^{-1} · w_desired

优势：

  • 不需要物理修改
  • 可以自适应更新
  • 补偿高阶互耦

挑战：

  • 需要精确测量 K_c
  • 计算复杂度 O(N³)
  • 温度/频率漂移需要重新校准

4.5 去耦馈电网络（DFN）

2024 年提出的新方法——在馈电网络中引入额外的耦合路径，抵消单元之间的空间耦合：

S_DFN = -S_space

效果：互耦降低 7-24dB。

五、互耦的利用

5.1 互耦增强 MIMO 容量

当空间自由度受限时，互耦可以增加通道间的相关性，反而提升容量：

C = log₂ det(I + ρ/N · H·Hᴴ)

当 H 的列相关性高时，互耦可以"混叠"信号，创造新的自由度。

5.2 互耦用于传感

利用互耦对环境的敏感性实现传感功能：

  • 生物传感：介电常数变化 → 互耦变化
  • 湿度传感：水汽吸收 → 基板 Dk 变化 → 互耦变化
  • 形变监测：单元间距变化 → 互耦变化

5.3 互耦用于小型化

利用互耦的"电磁近邻效应"，可以在小于 λ/2 的间距下实现可接受的隔离度：

  • 寄生单元（parasitic elements）通过互耦被激励
  • 等效孔径增大，增益不降

六、互耦测量与仿真

6.1 S 参数测量

使用 VNA（矢量网络分析仪）测量多端口 S 参数矩阵：

步骤：
    1. 将所有端口接 50Ω 负载
    2. 逐端口激励，测量所有端口的 S 参数
    3. 得到 N×N 的 S 矩阵

注意：测量时需要确保所有端口正确端接（a_k = 0, k ≠ m）。

6.2 全波仿真

使用 HFSS、CST 等工具仿真阵列的 S 参数：

优势：
    - 可以分离空间耦合和表面波耦合
    - 可以研究互耦随扫描角的变化
    - 可以优化去耦结构
    
    劣势：
    - 大规模阵列（N > 100）计算成本高
    - 需要网格收敛性验证

6.3 半解析方法

使用矩量法（MoM）或等效电流法快速估算互耦：

Z_nm = (1/I_n·I_m) · ∬∬ J_n · E_m dS

其中 J_n 是单元 n 的等效电流，E_m 是单元 m 产生的场。

七、工程挑战与坑

7.1 互耦的频率依赖性

互耦随频率变化，在中心频率匹配良好，但在频带边缘可能恶化：

S₁₂(f) = S₁₂(f₀) · (f/f₀)²

对于宽带阵列（> 20% 带宽），需要在整个频带上评估互耦。

7.2 互耦的扫描角依赖性

波束扫描时，互耦随扫描角变化：

S₁₂(θ) = S₁₂(0°) · cos(θ)

扫描角越大，互耦越强（因为单元间的投影距离变短）。

7.3 互耦的温度依赖性

温度变化改变基板介电常数和金属导电率，从而影响互耦：

Δε_r/ε_r ≈ -0.001/°C（FR4）
    Δσ/σ ≈ 0.004/°C（铜）

对于高精度阵列（如雷达），需要进行温度补偿。

7.4 互耦与表面波的耦合

在高介电常数基板（如硅 ε_r = 11.9）上，表面波耦合可能占主导地位：

η_surface ≈ 1 - exp(-2π·h/λ·√(ε_r - 1))

其中 h 是基板厚度。

对策：

  • 使用薄基板
  • 使用低介电常数材料（LCP ε_r = 3.5）
  • 使用 EBG/DGS 抑制表面波

八、技术演进

  • 1950s：互耦的阻抗矩阵理论建立
  • 1970s：互耦对方向图影响的定量分析
  • 1990s：互耦在 MIMO 中的作用被认识
  • 2000s：DGS/EBG 去耦技术提出
  • 2010s：数字域互耦补偿
  • 2020-2022年：互耦在 XL-MIMO 中的新特性研究
  • 2023-2024年：去耦馈电网络（DFN）、AI 辅助互耦建模

九、技术成熟度

  • 成熟度等级：成熟
  • 说明：互耦理论已完全成熟，S 参数测量是天线阵列的标准测试项目。去耦技术和数字补偿方法已广泛商用。

十、一句话总结

互耦是天线的"邻里效应"——每个单元都不是孤立的，它会受到邻居的电磁影响，也会影响到邻居。它无法消除（这是麦克斯韦方程决定的），但可以通过增加间距、去耦结构、数字补偿等方法控制在可接受范围内。在 6G XL-MIMO 中，互耦的影响更加显著，需要更精细的建模和补偿。
