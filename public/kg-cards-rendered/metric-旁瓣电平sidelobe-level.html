旁瓣电平 Sidelobe Level SLL

旁瓣电平是主瓣之外最大辐射方向的功率与主瓣最大功率之比——它衡量天线的"干净程度"，旁瓣越低，天线指向越纯粹，抗干扰能力越强。

类比入口

旁瓣就像手电筒除了主光束外那些散出来的光——你本想把光照向一个方向，但余光（旁瓣）会照亮其他地方。如果余光太强，不仅浪费能量，还可能被敌人（干扰源）发现你的位置。SLL 就是衡量"余光有多强"的指标。

一、定义与物理意义

1.1 旁瓣电平（SLL）

SLL = 20·log₁₀(E_max_sidelobe / E_mainlobe) [dB]
    SLL = 10·log₁₀(P_max_sidelobe / P_mainlobe) [dB]

注意：SLL 是负值（旁瓣功率低于主瓣），但工程上常说"SLL = -20dB"表示旁瓣比主瓣低 20dB。

1.2 栅瓣（Grating Lobe）

栅瓣 = 由于单元间距过大而产生的等大旁瓣

当 d > λ/2 时（相控阵），会出现栅瓣——它与主瓣功率几乎相等，是严重的旁瓣问题。

栅瓣条件：
    sin(θ_grating) = sin(θ₀) ± n·λ/d
    
    当 d > λ 时，即使 θ₀ = 0° 也有栅瓣（n=1）

1.3 方向图结构

            ╱    ╲
           ╱      ╲    旁瓣
          ╱   ★    ╲  ╱

    栅瓣 = 与主瓣等大的旁瓣（d > λ 时出现）

二、计算公式

2.1 SLL 与口径照明分布的关系

均匀照射：SLL = 0dB（最差，第一旁瓣与主瓣等高）
    三角形照射：SLL = -23dB
    二次方（Cos²）照射：SLL = -32dB
    切比雪夫照射：SLL = -20 到 -40dB（可调）
    泰勒照射：SLL = -20 到 -30dB（可调）

2.2 均匀照射的旁瓣电平

N 元均匀线阵：
    SLL ≈ -13.5dB（恒定，与 N 无关）
    第一旁瓣在主瓣两侧 ±λ/(Nd) 处

2.3 切比雪夫阵列的 SLL 设计

给定 SLL 目标值（如 -20dB）：
      设计参数 R = 10^(SLL_dB/20)⁻¹
      R = 10^(-20/20)⁻¹ = 10
    
    切比雪夫多项式系数由 R 决定

典型设计：

SLL 目标R 值增益损失备注
-10dB3.160.1dB低隔离
-20dB100.3dB通用
-30dB31.60.8dB高隔离
-40dB1001.5dB极端隔离

2.4 旁瓣与主瓣宽度的关系

SLL 越低 → 波束越宽（口径效率越低）
    SLL 越高 → 波束越窄（口径效率越高）

Trade-off 曲线：

-101.1×5%
-201.2×10%
-301.4×18%
-401.6×25%

四、测量方法

4.1 暗室测量

设备：暗室 + 转台 + 标准增益天线 + VNA
    方法：
      1. 旋转被测天线 360°
      2. 记录每个角度的功率
      3. 找到主瓣峰值和最大旁瓣
      4. SLL = 20·log₁₀(P_sidelobe / P_mainlobe)
    精度：±1dB

4.2 OTA 测量

设备：暗室 + 多探头阵列
    方法：
      1. 被测天线放在暗室中央
      2. 多探头同时采集方向图
      3. 提取 SLL
    优势：适合终端天线（手机、AiP 模组）

4.3 近场扫描

设备：近场扫描系统
    方法：
      1. 二维扫描（平面/圆柱/球形）
      2. FFT 变换到远场
      3. 计算 SLL
    优势：精度高（±0.5dB），速度快

六、工程注意事项

6.1 栅瓣的避免

相控阵最大扫描角 θ_max：
    d/λ ≤ 1 / (1 + |sin(θ_max)|)
    
    θ_max = 0° → d ≤ λ/2
    θ_max = 30° → d ≤ 0.67λ
    θ_max = 60° → d ≤ 0.5λ
    θ_max = 90° → d ≤ 0.5λ

经验法则：相控阵单元间距 d ≤ λ/2 以避免栅瓣。

6.2 旁瓣对系统性能的影响

干扰抑制比 SIR = -SLL
    
    SLL = -20dB → SIR = 20dB
    SLL = -10dB → SIR = 10dB

5G 基站旁瓣影响：

  • 旁瓣过高 → 邻小区干扰 ↑ → 系统容量 ↓
  • 旁瓣过高 → 定位精度 ↓（雷达）
  • 旁瓣过高 → 被敌方测向定位 ↑（军用雷达）

6.3 旁瓣抑制技术

技术效果代价
口径照明 taperSLL ↓ 5-10dB增益 ↓ 1-3dB
切比雪夫分布SLL 可调（-20~-40dB）复杂度 ↑
泰勒分布SLL 可调 + 波束窄复杂度 ↑↑
单元振幅加权SLL ↓ 3-5dB馈电网络复杂
单元位置扰动SLL ↓ 3-8dB设计复杂
吸收材料环境旁瓣 ↓重量 ↑

6.4 实际系统中的旁瓣恶化

理想 SLL：-20dB
    实际 SLL：-15dB（恶化 5dB）
    
    原因：
      1. 单元间距误差（加工公差）
      2. 相位误差（移相器精度）
      3. 幅度误差（馈电网络不一致）
      4. 互耦效应
      5. 安装环境反射

七、一句话总结

旁瓣电平是天线"干净程度"的度量——主瓣之外的辐射越低越好，旁瓣太高会导致干扰、容量下降和被定位风险。记住三个关键：①均匀照射的旁瓣最差（-13.5dB），切比雪夫/泰勒分布可设计到 -30dB 以下 ②栅瓣是最危险的旁瓣（d > λ/2 时出现）③SLL 和波束宽度是跷跷板（旁瓣越低，波束越宽，增益越低）。
