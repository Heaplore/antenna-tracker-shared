低噪声放大器 LNA Low Noise Amplifier

LNA 是接收链路的第一级放大器——它的任务是"在信号被噪声淹没之前，先把信号放大"。噪声系数（NF）和增益是两个最核心的指标，NF 越低越好，增益要足够高以压制后续级的噪声贡献。

类比入口

LNA 就像录音棚里的"前置放大器"——麦克风拾取的信号只有几微伏，LNA 要在不引入明显底噪的前提下把它放大到几百毫伏。如果前置放大器本身的底噪太大，录出来的声音就算再大也是"沙沙"的。LNA 就是射频世界的"高保真前置放大器"。

一、定义与作用

1.1 LNA 在接收链路中的位置

天线 → 滤波器 → LNA → 混频器 → 滤波器 → VGA → ADC
             ↑          ↑
          抑制带外    第一级放大
          干扰        （最关键！）

Friis 噪声公式的启示：

T_sys = T_ant + T_LNA/G_feeder + T_mixer/(G_feeder·G_LNA) + ...

LNA 的噪声温度 T_LNA 直接加到系统噪声中，而后面的混频器等被 G_LNA 除——所以 LNA 的噪声贡献最大，必须做到最低。

1.2 LNA 的核心功能

  1. 放大微弱信号：将 μV 级信号放大到 mV 级
  2. 低噪声：自身引入的噪声尽可能小
  3. 阻抗匹配：与天线/滤波器良好匹配（VSWR < 1.5）
  4. 线性度：在强干扰下不失真
  5. 稳定性：不会自激振荡

二、关键指标

2.1 噪声系数（Noise Figure, NF）

NF = 10·log₁₀(F) = 10·log₁₀(T_sys/T_0)
    T_sys = T_0·(F - 1) = 290·(10^(NF/10) - 1)
NF (dB)T_n (K)说明
0.322超低噪声（低温 HEMT）
0.536高性能 HEMT LNA
1.072优秀 LNA
1.5107典型 GaAs LNA
2.0159普通 LNA
3.0253入门级 LNA
5.0466消费级 LNA

2.2 增益（Gain, S21）

G(dB) = 10·log₁₀(P_out/P_in) = S21(dB)
应用典型增益说明
Sub-6GHz LNA15-25dB适中增益
毫米波 LNA20-30dB需要高增益压制后续级
卫星接收 LNA30-40dB极低噪声要求
射电天文 LNA40-50dB极致性能

增益选择原则：

  • 太低 → 后续级噪声贡献大
  • 太高 → 线性度下降、功耗增加、可能不稳定

2.3 1dB 压缩点（P1dB）

P1dB = 增益下降 1dB 时的输出功率
LNA 类型P1dB (dBm)说明
毫米波 CMOS5-10集成度高，功率低
GaAs MMIC10-18典型值
GaN MMIC18-25高功率
低温 HEMT0-5极致噪声，功率低

2.4 三阶截点（IP3）

OIP3 = Output Third-Order Intercept Point
    IIP3 = OIP3 - G

重要性：IP3 决定了 LNA 在多个信号同时存在时的线性度。

LNA 类型IIP3 (dBm)说明
毫米波 CMOS-5 to 5线性度一般
GaAs MMIC5-15典型值
GaN MMIC15-25高线性度

2.5 输入/输出回波损耗（S11/S22）

VSWR = (1+|Γ|)/(1-|Γ|)
    S11(dB) = 20·log₁₀(|Γ|)
指标要求
S11< -10dB（VSWR < 1.22）
S22< -10dB
隔离度 S12< -25dB（单向化）

2.6 稳定性因子（K-factor）

K = (1 - |S11|² - |S22|² + |Δ|²) / (2|S12·S21|)
    Δ = S11·S22 - S12·S23

要求 K > 1 且 |Δ| < 1 才能保证无条件稳定。

三、LNA 的主要拓扑结构

3.1 共源/共发射极（Common Source/Gate）

最简单但噪声和增益都不够好：

输入 → Gate/Base → Drain/Collector → 输出

3.2 共源共栅（Cascode）

最主流的 LNA 拓扑：

M1 (共源) → 低噪声
    M2 (共栅) → 高隔离度、高稳定性

优点：

  • 高增益（20-30dB）
  • 高隔离度（S12 < -30dB）
  • 高稳定性
  • 良好的噪声性能

3.3 差分 LNA

Input+ → ┬→ M1 → Output+
              │
    Input- → ┴→ M2 → Output-

优点：

  • 抑制共模噪声
  • 抑制偶次谐波
  • 更好的线性度
  • 适合直接下变频架构

3.4 多级 LNA

Stage 1: 最低噪声（NF = 0.5dB, G = 15dB）
    Stage 2: 中等噪声（NF = 1.5dB, G = 15dB）
    Stage 3: 高增益（NF = 3dB, G = 10dB）

Friis 级联：

T_total = T_1 + T_2/G_1 + T_3/(G_1·G_2)

第一级最关键——它的噪声和增益决定了整体性能。

六、应用场景

6.1 5G 基站

  • 每个接收通道配一个 LNA
  • 64T64R = 64 个 LNA
  • NF < 2dB, Gain > 20dB
  • 通常集成在 T/R 模块中

6.2 卫星通信

  • 室外 LNB 中的 LNA
  • NF < 0.5dB（低温 HEMT）
  • 增益 30-40dB
  • 工作在 -40°C 到 +70°C

6.3 GPS 接收机

  • NF < 1dB
  • 增益 25-35dB
  • 带宽 10-20MHz（L1 频段）
  • 低功耗（< 20mW）

6.4 雷达接收

  • 高线性度（IIP3 > 15dBm）
  • 宽动态范围
  • 高功率容量

七、工程注意事项

7.1 LNA 必须靠近天线

馈线损耗 1dB → LNA 的噪声贡献放大 2.5 倍
    馈线损耗 3dB → LNA 的噪声贡献放大 5.6 倍

规则：LNA 越靠近天线越好，最好集成在天线罩内。

7.2 输入匹配 vs 噪声匹配

S11 最佳匹配 ≠ NF 最小值

设计 LNA 时需要在输入匹配和噪声匹配之间做权衡：

优化目标结果
最佳 S11VSWR 最小，但 NF 可能不是最低
最佳 NFNF 最低，但 S11 可能较差
最佳噪声系数增益积 NGP综合最优

7.3 电源噪声对 LNA 的影响

电源噪声 → 调制到 RF 信号 → 相位噪声恶化

解决方案：

  • LDO 供电（而非 DC-DC）
  • 去耦电容（0.1μF + 10μF）
  • 屏蔽 LNA 芯片

八、技术成熟度

  • 成熟度等级：高度成熟
  • 说明：Sub-6GHz LNA 技术非常成熟（几十年积累）。毫米波 LNA 随 5G 快速成熟。CMOS 毫米波 LNA 已在 5G 手机和基站中大规模商用。

九、一句话总结

LNA 是接收链路的"守门员"——它决定了整个系统能听到的最小信号有多小。核心指标就三个：NF（噪声越小越好）、Gain（足够高以压制后续级）、P1dB/IP3（线性度要好）。记住 Friis 公式的教诲：第一级的噪声贡献最大，所以 LNA 的 NF 是接收系统性能的天花板。
