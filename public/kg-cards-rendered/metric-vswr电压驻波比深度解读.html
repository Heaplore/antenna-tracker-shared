天线 VSWR（电压驻波比）深度解读 VSWR

VSWR 是射频系统的"反射警报器"——它告诉你有多少功率被天线反射回来了。VSWR = 1 是完美匹配（0 反射），VSWR = 2 意味着 11% 的功率被反射，VSWR = 6 意味着一半功率全丢了。5G AAU 要求 VSWR ≤ 1.5，因为 64 根天线的每一根都不能浪费功率。

一、VSWR 的物理本质：驻波的形成

驻波是怎么来的？

当传输线末端（天线）的阻抗 Z_L 不等于传输线的特征阻抗 Z_0（通常 50Ω）时，部分入射波会被反射。入射波和反射波在传输线上叠加，形成驻波：

V(z) = V⁺(e^(-γz) + Γ·e^(+γz))

其中 Γ 是反射系数：

Γ = (Z_L - Z_0) / (Z_L + Z_0)

驻波的最大值和最小值：

V_max = |V⁺| × (1 + |Γ|)
    V_min = |V⁺| × (1 - |Γ|)

VSWR 的定义

VSWR = V_max / V_min = (1 + |Γ|) / (1 - |Γ|)

关键理解：

  • VSWR = 1 → Γ = 0 → 无反射（完美匹配）
  • VSWR = ∞ → |Γ| = 1 → 全反射（开路或短路）
  • VSWR 永远是 ≥ 1 的正实数

驻波在传输线上的分布

位置:  |← λ/2 →|← λ/2 →|← λ/2 →|
           V_min    V_max    V_min    V_max
           
    V(z) = |      /\      /\      /\      |
           |     /  \    /  \    /  \     |
           |    /    \  /    \  /    \    |
           |___/______\/______\/______\___|
           0    λ/4   λ/2  3λ/4   λ    距离

重要特性：

  • 相邻 V_max 和 V_min 间距 = λ/4
  • 相邻 V_max 间距 = λ/2
  • VSWR 是标量，不包含相位信息

三、失配损耗（Mismatch Loss）

什么是失配损耗？

失配损耗是指由于阻抗不匹配导致的可用功率损失——不是功率被"消耗"了，而是被反射回去了，没有传送到天线。

Mismatch Loss (dB) = -10 × log₁₀(1 - |Γ|²)
                         = -10 × log₁₀(1 - ((VSWR-1)/(VSWR+1))²)

物理意义：

  • 失配损耗 = 0dB → 无反射，全部功率传输
  • 失配损耗 > 0dB → 部分功率被反射

失配损耗 vs 插入损耗

维度插入损耗失配损耗
原因导体/介质发热消耗阻抗不匹配反射
功率去向转化为热反射回源端
可否消除材料决定可通过匹配网络消除
计算公式测量 S21-10×log₁₀(1-Γ²)

关键判断：失配损耗不是"真正的损耗"，而是"没传出去的功率"。这部分功率可能在源端被吸收、被重反射、或损坏功放。

四、VSWR 对系统的影响

4.1 功率效率下降

功率传输效率 = 1 - |Γ|² = 1 - ((VSWR-1)/(VSWR+1))²
VSWR传输效率100W 发射功率实际辐射
1.596%96W
2.089%89W
3.075%75W
5.056%56W

5G AAU 场景：64 根天线，每根 VSWR = 2.0，意味着每根天线有 11% 的功率被反射。64 根加起来就是 64 × 11% = 7 根天线的功率白白浪费。

4.2 功放损坏风险

反射功率回到功放输出端，会导致：

  • 热损伤：反射功率在功放输出级转化为热
  • 电压击穿：驻波最大值处电压可能超过功放耐压
  • 振荡：反射波与功放内部电路相互作用产生自激振荡
  • 互调失真：非线性效应产生杂散信号

VSWR 保护机制：

  • 现代功放内置 VSWR 保护电路
  • 检测到高 VSWR 时自动降低输出功率（foldback）
  • 极端情况下完全关闭输出

4.3 信号质量劣化

  • 多径效应：反射信号与直达信号叠加造成相位抵消
  • 误码率上升：信号幅度波动导致解调困难
  • 邻道干扰：互调产物落入相邻信道

4.4 测量误差

VSWR 会影响 S 参数测量的准确性：

  • 源端匹配不良 → 入射波幅度不确定
  • 负载匹配不良 → 反射波幅度不确定
  • 多次反射 → 测量误差增大

解决方法：使用隔离器/环形器，或在 VNA 校准中使用误差修正模型。

五、5G AAU 的 VSWR 要求

行业标准

设备类型VSWR 要求回波损耗要求反射功率
4G 扇区天线≤ 2.0≥ 9.5dB≤ 11%
5G AAU (Sub-6GHz)≤ 1.5≥ 14dB≤ 4%
5G AAU (毫米波)≤ 1.5≥ 14dB≤ 4%
卫星通信≤ 1.2≥ 20dB≤ 1%
雷达系统≤ 1.1≥ 26dB≤ 0.8%

为什么 5G AAU 要求更严？

  1. 功率更大：AAU 发射功率可达 200W+（4G RRU 通常 80-100W），反射功率 = 200W × 11% = 22W 的热损伤风险
  2. 集成度高：64 根天线紧密排列，互耦导致 VSWR 随波束指向变化
  3. 散热困难：AAU 内部空间紧凑，反射功率产生的热量难以散发
  4. O-RAN 互操作性：多厂商设备对接要求统一的 VSWR 标准

波束指向对 VSWR 的影响

Massive MIMO 的 VSWR 不是固定的——它随波束指向变化：

VSWR(θ, φ) = f(波束指向角 θ, φ)

原因：

  • 波束赋形改变了天线的电流分布
  • 电流分布改变 → 输入阻抗变化 → VSWR 变化
  • 边缘波束的 VSWR 通常比中心波束差 0.3-0.5

工程对策：

  • 在设计阶段仿真所有波束指向的 VSWR
  • 设置波束相关的 VSWR 保护阈值
  • 边缘波束自动降低功率（power back-off）

六、VSWR 测量方法

6.1 矢量网络分析仪（VNA）

原理：直接测量 S11（反射系数），然后换算为 VSWR。

VSWR = (1 + |S11|) / (1 - |S11|)

优势：

  • 同时获得幅度和相位信息
  • 全频段扫描（不是单频点）
  • 可通过 Smith Chart 直观看到阻抗位置
  • 误差修正后可达 ±0.01 VSWR 精度

典型流程：

  1. SOLT 校准（Short-Open-Load-Thru）
  2. 设置频率范围（如 3.3-3.8GHz）
  3. 扫描并记录 S11
  4. 换算为 VSWR 曲线
  5. 检查全频段 VSWR ≤ 1.5

6.2 定向耦合器 + 功率计

原理：分别测量正向功率和反射功率：

Γ = √(P_reflected / P_forward)
    VSWR = (1 + Γ) / (1 - Γ)

优势：

  • 设备简单（定向耦合器 + 两个功率计）
  • 可在系统运行时实时监测
  • 适合现场安装验收

局限：

  • 只能测单频点
  • 精度低于 VNA
  • 需要精确校准定向耦合器

6.3 基站内置 VSWR 监测

LG 专利方法（LG Information & Communications）：

Pd = PdB_REV - PdB_FWD  （反射功率与正向功率的差值）
    τ = 10^(Pd/20)            （反射系数）
    VSWR = (1 + τ) / (1 - τ)  （驻波比）

工作原理：

  1. 测量基站输出信号功率（P_FWD）
  2. 测量天线反射回来的信号功率（P_REV）
  3. 计算差值 Pd
  4. 换算为 VSWR

优势：无需外部设备，基站自带监测功能，可实时告警。

6.4 MAX2016 专用 VSWR 监测芯片

架构：

天线 → 环形器 → MAX2016（正向/反射功率检测）
                            ↓
                       MAX5402（数字电位器）
                            ↓
                       MAX1116/1117（ADC）
                            ↓
                       MCU（阈值比较 + 告警）

工作流程：

  1. MAX2016 同时检测正向和反射功率
  2. 输出两者的差值电压
  3. ADC 采样差值电压
  4. MCU 比较阈值，触发告警或降功率

七、电缆和连接器对 VSWR 的影响

电缆损耗的"伪装效应"

关键洞察：电缆损耗会"掩盖"天线的不良 VSWR。

在功放端测量的 VSWR ≠ 天线端的 VSWR

原因：反射功率经过电缆两次（去一次、回一次），每次都有损耗。

示例：

  • 天线端 VSWR = 10:1（很差）
  • 电缆损耗 = 6dB（往返 12dB）
  • 功放端测量 VSWR ≈ 1.5:1（看起来很好！）

结论：测量 VSWR 时，必须在天线连接器处测量，不能在功放端测量。否则会得到虚假的好结果。

连接器的 VSWR

每个连接器都会引入微小的阻抗不连续：

5G 毫米波挑战：28GHz 以上必须使用 2.92mm 或 1.85mm 连接器，N 型连接器在毫米波频段 VSWR 会急剧恶化到 2.0 以上。

八、改善 VSWR 的方法

8.1 阻抗匹配网络

L 型匹配：最简单，两个元件（L+C） Pi 型匹配：三个元件，带宽更宽 T 型匹配：三个元件，Q 值可控 Stub 匹配：微带线/同轴线常用

8.2 调整天线结构

  • 改变辐射体尺寸（调谐振频率）
  • 添加匹配枝节（stub）
  • 优化接地平面
  • 调整馈电点位置

8.3 使用隔离器/环形器

隔离器：只允许信号单向传输，反射功率被吸收在终端负载中。

效果：

  • 即使天线 VSWR = ∞（开路），隔离器输出端看到的 VSWR = 隔离器自身 VSWR（通常 1.1-1.3）
  • 代价：插入损耗 0.3-0.5dB，且只能用于单通道

5G AAU 不使用隔离器的原因：

  • 64 通道 × 64 个隔离器 = 成本爆炸
  • 隔离器体积大，不利于 AAU 小型化
  • 隔离器是非互易器件，需要磁性材料，高温性能差

8.4 预失真技术

原理：在发射端预先对信号进行"反向扭曲"，抵消匹配不良引起的失真。

适用场景：

  • 毫米波天线（VSWR 随频率变化大）
  • 宽带天线（VSWR 在频带边缘恶化）

九、给非专业读者的一句话总结

VSWR 就是"反射警报器"——VSWR = 1 表示 100% 功率传出去了，VSWR = 2 表示 11% 的功率弹回来了。5G 基站功率那么大，11% 的反射功率就是几十瓦的热量，能把功放烤熟。所以 5G AAU 要求 VSWR ≤ 1.5，把反射功率压到 4% 以下。电缆和连接器的每一个接头都会让 VSWR 变差——这就是为什么基站工程师最讨厌拧螺丝。

金句：VSWR 告诉你一个道理——最好的传输不是"拼命往前冲"，而是"不留任何回头路"。匹配好的系统，能量只进不出（反射为零）。
