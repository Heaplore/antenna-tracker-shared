方向性系数 Directivity D

方向性系数衡量天线将辐射能量"集中"到某个方向的能力——定义为指定方向上的辐射强度与同等辐射功率下理想各向同性天线的平均辐射强度之比，与天线效率无关。

类比入口

方向性系数就像手电筒的聚光能力。同样功率的灯泡，裸泡发光向四面八方（各向同性，D = 1），加上反光罩后光集中在一个方向（高方向性，D = 100+）。方向性系数就是"加了反光罩后有多聚光"的度量——它不管反光罩本身吸了多少光（那是效率的事），只管"光往哪个方向打、打了多集中"。

一、定义与物理意义

1.1 数学定义

D(θ, φ) = U(θ, φ) / U_avg

其中：

  • U(θ, φ) = 指定方向的辐射强度 [W/sr]
  • U_avg = P_rad / 4π（平均辐射强度，P_rad 为总辐射功率）

1.2 方向性系数的 dB 表示

D(dBi) = 10·log₁₀(D)

"i" 表示相对于各向同性辐射体（isotropic radiator）。

1.3 方向性系数 vs 天线增益

这是最容易混淆的概念：

方向性系数 D = 几何集中度（不考虑损耗）
    天线增益 G = 几何集中度 × 效率（考虑损耗）
    
    G(θ,φ) = η · D(θ,φ)
    G(dBi) = D(dBi) + 10·log₁₀(η)

其中 η 是天线总效率（0 < η ≤ 1）。

天线D(dBi)ηG(dBi)说明

实际半波偶极子2.15dBi95%2.07dBi导体/介质损耗

关键区别：

  • D 只取决于天线的方向图形状
  • G 取决于方向图形状 + 天线的效率
  • 同一个天线，D 是固定的，但 G 随效率变化（温度、老化、频率漂移）

二、计算公式

2.1 从方向图积分计算

D_max = 4π / Ω_A

其中 Ω_A 是波束立体角：

Ω_A = ∫∫_4π P_n(θ,φ)·sin(θ) dθ dφ

P_n(θ,φ) 是归一化方向图（最大值 = 1）。

2.2 近似公式（主瓣近似）

对于主瓣尖锐、旁瓣小的天线：

D ≈ 41253 / (HPBW_θ · HPBW_φ)   （HPBW 单位：度）

或更精确的：

D ≈ K / (HPBW_θ · HPBW_φ)

其中 K 是波束填充因子（0.8-1.0，取决于方向图形状）。

典型 K 值：

方向图K说明
高斯型0.96抛物面天线
sinc² 型0.88均匀照射矩形口径
Taylor 型0.92低旁瓣照射
Chebyshev0.90等旁瓣照射

2.3 与有效孔径的关系

D = 4π · A_e / λ²

其中 A_e 是有效孔径面积。

物理意义：方向性系数正比于"天线抓光的面积"与"波长大小的比值平方"。

2.4 阵列的方向性

N 元均匀线阵：

D_array ≈ N · D_element

N 元均匀面阵（M×N）：

D_array ≈ M·N · D_element

举例：

  • 64 单元阵列 × 单单元 D = 6dBi → 阵列 D ≈ 6 + 10·log₁₀(64) = 24dBi
  • 实际 G = 22-23dBi（扣除效率和互耦损耗）

四、测量方法

4.1 比较法（与标准增益天线对比）

步骤：
    1. 用标准增益天线（已知 G_std）测量接收功率 P_std
    2. 换上天线（AUT）测量接收功率 P_AUT
    3. D_AUT = D_std · (P_AUT / P_std) / η_AUT

4.2 三点法（获取完整方向图后积分）

步骤：
    1. 在消声室测量 AUT 的完整 3D 方向图
    2. 数值积分计算 Ω_A
    3. D = 4π / Ω_A

4.3 近场扫描法

步骤：
    1. 近场扫描获取 E-field 分布
    2. 傅里叶变换到远场
    3. 计算方向图积分 → D

优点：精度高，无需远场暗室 缺点：数据处理复杂

六、工程注意事项

6.1 D 和 G 的标注陷阱

很多数据手册只写 "Gain" 但不标明是 D 还是 G：

  • 如果标注 "Directivity" 或 "D" → 是方向性系数（不含效率）
  • 如果标注 "Gain" 或 "G" → 是增益（含效率）
  • 有些厂家故意用 D 冒充 G 来"美化"数据

判断方法：如果 Gain > Directivity，一定是标错了（物理上不可能）。

6.2 方向性随频率变化

D(f) ∝ (f/λ)² ∝ f²

对于固定尺寸的抛物面天线：

  • 频率加倍 → D 变为 4 倍（+6dB）
  • 但口径效率可能下降（表面精度要求更高）

6.3 方向性与波束宽度的互换

波束越窄 → D 越高 → 覆盖范围越小
    波束越宽 → D 越低 → 覆盖范围越大

这是天线设计的永恒权衡。

七、一句话总结

方向性系数是天线"聚光能力"的纯几何度量——它不管天线损耗多少，只看能量往哪个方向集中。记住 D 和 G 的区别：D 是"理想状态下有多聚光"，G 是"实际状态下有多聚光"。中间差的就是效率 η。有了 D，乘以效率就是 G；有了 G，除以效率就是 D。
