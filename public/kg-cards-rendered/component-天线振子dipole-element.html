天线振子 Dipole Element（天线单元） Dipole-Element

天线振子是天线辐射能量的最基本单元——半波偶极子是最经典的振子（增益 2.15dBi），贴片单元是毫米波最常用的振子（增益 6-8dBi）。所有天线阵列都是由这些基本单元"排列组合"出来的。

类比入口

天线振子就像乐高积木的基本块——单个积木块能力有限（增益只有 2-8dBi），但把它们按特定方式排列组合（阵列），就能搭出各种形状和功能的建筑（高增益相控阵天线）。

一、定义与功能

1.1 振子的角色

振子 = 天线的"最小辐射单元"
      - 接收：将电磁波转换为电信号
      - 发射：将电信号转换为电磁波
      
    阵列 = N 个振子 + 馈电网络 + 移相器

1.2 常见振子类型

类型尺寸增益带宽适用频段说明
半波偶极子λ/22.15dBi宽DC-6GHz经典
V 偶极子λ/2×22-5dBi宽1-6GHz基站常用
微带贴片λ/2×λ/26-8dBi窄> 3GHz5G 毫米波

2.3 关键参数

参数值说明
增益2.15dBi经典值
输入阻抗73+j42.5Ω谐振时 ≈ 73Ω
带宽~5-10%取决于粗细

2.4 阻抗匹配

偶极子阻抗 73Ω → 需要匹配到 50Ω
    
    匹配方法：
      1. 折叠偶极子：阻抗 ≈ 300Ω → 用 4:1 巴伦
      2. 加匹配网络：LC 匹配到 50Ω
      3. 调整臂长：微调谐振频率

三、微带贴片单元（Patch Element）

3.1 结构

    ┌──────────────────┐
        │     金属贴片      │  ← 辐射体（宽 W，长 L ≈ λ/2）
        ├──────────────────┤
        │     介质基板      │  ← ε_r = 2.2-10
        ├──────────────────┤
        │     接地层        │  ← 金属地
        └──────────────────┘

3.2 关键参数

参数值说明
增益6-8dBi（单贴片）比偶极子高
输入阻抗可调（馈电位置）30-200Ω
带宽1-5%（窄！）主要缺点
极化线极化取决于馈电
剖面< λ/10（超薄）主要优点

3.3 带宽提升方法

单贴片带宽：1-3%
    
    提升方法：
      1. 加厚基板 → 带宽 ↑，但可能出现表面波
      2. 降低 ε_r → 带宽 ↑，但尺寸 ↑
      3. 叠层贴片（stacked patch）→ 带宽 10-20%
      4. 边缘馈电 + 探针 → 带宽 5-10%
      5. 缝隙耦合 → 带宽 10-15%

3.4 5G 毫米波贴片单元

28GHz 贴片单元：
      尺寸：≈ 5.4mm × 5.4mm（λ = 10.7mm）
      增益：7-9dBi（单贴片）
      带宽：3-5%（足够 5G NR 的 400MHz 带宽）
      基板：LCP，h = 0.1mm，ε_r = 3.0

四、振子在阵列中的应用

4.1 阵列中的单元间距

d = λ/2（最常见）
    d = 0.7λ（需要宽扫描）
    d = 0.4λ（避免栅瓣，但互耦大）

4.2 有源单元方向图（AUP）

单个振子孤立时的方向图 ≠ 阵列中的方向图
    
    原因：互耦效应
      - 相邻振子反射功率
      - 表面波
      - 共同辐射阻抗
    
    AUP = 阵列中单个振子在有源环境下的方向图

4.3 单元增益 vs 阵列增益

单个振子增益：G_elem
    阵列增益：G_array = N × G_elem × η_array
    
    η_array = 阵列效率（馈电损耗、互耦、量化误差）
    
    64 单元阵列：
      G_elem = 7dBi（贴片）
      G_array = 7 + 10·log₁₀(64) + 损耗
              = 7 + 18 - 2（馈电损耗）
              = 23dBi（典型）

五、振子选择指南

5.1 按频段选择

f < 1GHz → 偶极子/折叠偶极子（尺寸合理）
    1-6GHz → 偶极子/V 偶极子/贴片
    6-30GHz → 贴片/喇叭单元
    > 30GHz → 贴片/SIW 单元/AiP

5.2 按应用选择

应用推荐振子原因
5G 基站贴片阵列高增益、超薄、易集成
WiFi偶极子/PCB 天线低成本、宽带
卫星螺旋/喇叭高增益、圆极化
车载雷达Vivaldi/贴片宽带、低剖面
手机贴片/LDS超薄、小尺寸

六、一句话总结

振子是天线的基本"砖块"——半波偶极子增益 2.15dBi、带宽宽，贴片单元增益 6-8dBi、超薄但带宽窄。5G 毫米波基站用贴片单元组成 64 元阵列，单单元增益 7dBi，阵列总增益 23-25dBi。记住：单个振子的性能决定了整个阵列的上限——再好的阵列算法也弥补不了差的单元。
