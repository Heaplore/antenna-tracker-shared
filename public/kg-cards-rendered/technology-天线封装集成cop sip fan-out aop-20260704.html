天线封装与集成技术 CoP/SiP/Fan-out

天线封装从 AiP（天线在封装内）演进到 AoP（天线在板上）再到 CoP（天线在芯片上）和 Fan-out 封装——毫米波时代天线离芯片越近越好，因为 28GHz 信号走 1mm 线就会损耗 3dB。

类比入口

天线封装就像"把天线搬到芯片家门口"——AiP 是天线的"隔壁房间"，AoP 是天线的"同栋楼"，CoP 是天线的"窗户就对着芯片"，Fan-out 是把天线"印在芯片基板上"。距离越近，信号损耗越小。

一、封装技术演进

1.1 为什么毫米波需要特殊封装？

频率与损耗的关系：
      Sub-6GHz（3.5GHz）：
        - PCB 走线损耗：~0.1dB/cm
        - 10cm 走线损耗：~1dB（可接受）
        
      毫米波（28GHz）：
        - PCB 走线损耗：~1.5dB/cm
        - 1cm 走线损耗：~1.5dB（已很高）
        - 3cm 走线损耗：~4.5dB（不可接受）
        
      太赫兹（140GHz）：
        - PCB 走线损耗：~10dB/cm
        - 1mm 走线损耗：~1dB（必须集成）
        
    结论：频率越高，天线越要靠近芯片！

1.2 封装技术代际

第一代：PCB 级封装（Sub-6GHz）
      天线 → PCB 走线 → 射频 IC
      距离：1-5cm
      损耗：0.5-2dB
      代表：传统基站天线
      
    第二代：AiP（Antenna-in-Package，封装内天线）
      天线 → 封装基板 → 射频 IC
      距离：1-5mm
      损耗：0.1-0.5dB
      代表：5G 手机毫米波天线
      
    第三代：AoP（Antenna-on-Package，封装上天线）
      天线印刷在封装基板表面
      距离：~0mm（直接在封装上）
      损耗：< 0.1dB
      代表：Intel 5G 毫米波模块
      
    第四代：CoP（Chip-on-Package / Chip-scale Antenna）
      天线直接集成在芯片衬底上
      距离：~0mm（芯片级）
      损耗：< 0.05dB
      代表：硅基天线/芯片级天线
      
    第五代：Fan-out 封装
      天线分布在扇出晶圆上
      距离：~0mm
      损耗：< 0.05dB
      代表：Apple iPhone 12/13 毫米波模组

二、AiP（天线在封装内）

2.1 技术原理

AiP 结构：
      ┌─────────────────────────────┐
      │   天线阵列（16/32/64 单元）  │
      │   LCP/玻璃/硅基板           │
      ├─────────────────────────────┤
      │   封装基板（Organic/陶瓷）   │
      ├─────────────────────────────┤
      │   RFIC（波束成形 IC）       │
      ├─────────────────────────────┤
      │   散热基板（AlN/DBC）        │
      └─────────────────────────────┘
      
    关键技术：
      1. 天线制造
         - LCP 薄膜蚀刻（线宽 < 20μm）
         - 玻璃基板 RIE 深槽
         - 硅基 SIW（基片集成波导）
         
      2. 互连技术
         - Flip-Chip 倒装（间距 < 50μm）
         - Cu-Cu 混合键合（间距 < 10μm）
         - 硅通孔 TSV（Through-Silicon Via）
         
      3. 热管理
         - GaN/SiGe 射频 IC 发热量大
         - 需要 AlN/DBC 散热基板
         - 热阻 < 1°C/W

2.2 AiP 封装材料

材料DkDf适用频段特点
LCP3.00.00228-100GHz低损耗，柔性
玻璃3.7-7.80.001-0.00528-140GHz平整度高，可布线
硅11.70.01-0.0260-300GHz与 IC 工艺兼容
BT 树脂4.40.005< 20GHz成本低，Sub-6GHz
Rogers3.5-100.001-0.00410-40GHz高频稳定

2.3 AiP 量产挑战

1. 良率问题
       - 天线阵列 64 单元，任一单元不良 → 整批报废
       - 目标良率：> 95%
       - 实际良率：85-92%（取决于工艺）
       
    2. 测试问题
       - 封装前测试（CP）：射频 IC 单独测试
       - 封装后测试（FT）：AiP 整体测试
       - 毫米波测试需要暗室 + 探头，成本高
       
    3. 散热问题
       - 64 通道毫米波模组功耗：20-40W
       - 需要高效散热设计
       - 热应力导致封装变形 → 性能下降
       
    4. 成本问题
       - LCP 材料成本高（~$50/m²）
       - 玻璃基板加工成本高
       - AiP 模组价格：$15-30/颗（28GHz 4×4）

三、AoP（封装上天线）

3.1 技术原理

AoP 结构：
      ┌─────────────────────────────┐
      │   天线印刷层（铜/银）         │
      │   直接在封装基板表面          │
      ├─────────────────────────────┤
      │   封装基板（Organic/陶瓷）   │
      ├─────────────────────────────┤
      │   RFIC（波束成形 IC）       │
      └─────────────────────────────┘
      
    与 AiP 的区别：
      - AiP：天线在封装内部（多层基板中）
      - AoP：天线在封装表面（单层印刷）
      
    优势：
      - 工艺简单（印刷/蚀刻即可）
      - 可灵活调整天线位置
      - 适合大规模阵列
      
    劣势：
      - 天线在表面，易受干扰
      - 散热路径长
      - 天线性能略逊于 AiP

3.2 典型应用

Intel 5G 毫米波模组（2019）：
      - 频段：24-39GHz
      - 天线数：64 单元（8×8）
      - 封装：Fan-out + AoP
      - 数据速率：10Gbps+
      
    Qualcomm 5G 毫米波模组（SM8250）：
      - 频段：24-39GHz
      - 天线数：32 单元（4×8）
      - 封装：AiP + LCP
      - 支持波束赋形/切换

四、CoP（芯片级天线）

4.1 技术原理

CoP 结构：
      ┌─────────────────────────────┐
      │   天线（直接刻蚀在芯片上）    │
      │   硅衬底/GaAs/GaN            │
      ├─────────────────────────────┤
      │   射频电路                   │
      ├─────────────────────────────┤
      │   硅衬底                     │
      └─────────────────────────────┘
      
    实现方式：
      1. 硅基天线（Silicon-based Antenna）
         - 在硅衬底上蚀刻天线结构
         - 与射频 IC 单片集成
         - 适合 60-300GHz
         
      2. GaAs/GaN 天线
         - 化合物半导体衬底
         - 与功率放大器集成
         - 适合 28GHz 以下
         
      3. 硅通孔天线（TSV Antenna）
         - 通过 TSV 实现垂直辐射
         - 节省芯片面积
         - 适合 MIMO 阵列

4.2 硅基天线挑战

1. 衬底损耗
       - 硅 Dk=11.7，介电常数高 → 场集中在衬底
       - 损耗角正切 ~0.015 → 信号衰减
       - 对策：使用高阻硅（ρ > 1000Ω·cm）
       
    2. 表面波
       - 高 Dk 导致表面波激发 → 互耦增加
       - 对策：使用 EBG/DGS 结构抑制
       
    3. 热传导
       - 硅导热性好（150W/m·K）→ 散热优势
       - 但天线在表面，热量不易散出
       - 对策：背面散热 + TSV 导热
       
    4. 封装成本
       - 硅基天线与 IC 单片集成 → 成本最低
       - 但良率挑战大（IC + 天线一起坏 = 全废）

五、Fan-out 封装

5.1 技术原理

Fan-out RDL（Redistribution Layer）：
      ┌─────────────────────────────┐
      │   天线层（RDL 最上层铜）      │
      │   扇出区域（超出芯片本身）    │
      ├─────────────────────────────┤
      │   RDL 布线层（2-4 层）       │
      ├─────────────────────────────┤
      │   芯片（RFIC/基带）          │
      │   嵌入在 Fan-out 区域内      │
      ├─────────────────────────────┤
      │   BGA 焊球                 │
      └─────────────────────────────┘
      
    与传统封装区别：
      - Wire Bond：芯片 → 引线键合 → 基板
      - Flip Chip：芯片 → 倒装 → 基板
      - Fan-out：芯片嵌入 → RDL 扇出 → 天线/RDL
      
    优势：
      - 天线可直接印刷在 RDL 上
      - 电气长度短（< 1mm）
      - 适合大规模量产
      
    劣势：
      - 翘曲控制难（Fan-out 面积大）
      - 成本高（需要新设备）
      - 良率挑战

5.2 典型应用

Apple iPhone 12/13/14 毫米波模组：
      - 供应商：Skyworks/Qorvo/高通
      - 封装：Fan-out RDL
      - 频段：24-39GHz（mmWave）
      - 天线：6×8 阵列
      - 特性：动态波束赋形 + AI 波束管理
      
    Samsung Galaxy S21/S22/S23 毫米波：
      - 封装：AiP + LCP
      - 频段：28GHz
      - 天线：4×4 阵列
      - 特性：波束切换（非动态赋形）

六、封装技术选型指南

6.1 按频段选择

频段推荐封装原因
Sub-6GHz（< 6GHz）PCB/Wire Bond损耗低，无需特殊封装
C-band（4-8GHz）Flip Chip + 封装天线适中
毫米波（24-40GHz）AiP/LCP低损耗，距离短
K-band（40-80GHz）AoP/CoP必须集成
太赫兹（> 100GHz）CoP/硅基唯一选择

七、一句话总结

毫米波时代，天线封装的核心矛盾是"信号损耗 vs 成本"——频率越高，天线离芯片越近越好，但封装成本也越高。Sub-6GHz 用 PCB 就够了，28GHz 毫米波必须用 AiP/LCP，太赫兹只能用 CoP/硅基。记住：Fan-out 封装是未来趋势（Apple 已经在用），但良率和翘曲是两大拦路虎。QMS 的关键指标是"封装良率"和"射频一致性"——这两项直接决定出货合格率。
