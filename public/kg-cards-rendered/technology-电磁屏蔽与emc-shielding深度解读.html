2.1 金属材料

材料 σ(S/m) μ_r 优点 缺点
铜 5.8×10⁷ 1 导电最好，R 大 重，贵
铝 3.5×10⁷ 1 轻，便宜 σ 较低

Mu-metal 1×10⁶ 20-100k 超低频磁屏蔽之王 极贵，难加工

2.2 导电聚合物/涂层

应用场景：塑料外壳喷导电漆（镍/铜/银）→ 减轻重量，降低成本。

典型 SE：30-60dB（1MHz-1GHz）

2.3 电磁屏蔽衬垫（Conductive Gasket）

作用：填补屏蔽罩接缝处的缝隙 → 防止泄漏。

类型 结构 压缩比 SE
指形簧片（fingerstock） 金属弹簧片 20-30% 30-60dB
导电泡沫 聚氨酯+镍/银涂层 30-50% 60-100dB
导电织物 金属丝编织 10-20% 40-80dB
真空保持衬垫（VGS） 软硅胶+金属纤维 15-25% 80-120dB

2.4 蜂窝通风面板（Honeycomb Vent）

原理：波导截止——孔径 < λ/2 时电磁波无法通过。

f_cutoff ≈ c / (2 × d)

其中 d 为六角形孔径。

典型规格： | 孔径 | f_cutoff | 适用 | |------|---------|------| | 1.6mm | 94GHz | 5G mmWave | | 3.2mm | 47GHz | WiFi 6E/7 | | 6.4mm | 23.5GHz | Sub-6GHz 5G | | 12.7mm | 11.8GHz | LTE/WiFi |

设计规则：蜂窝深度 ≥ 3×孔径直径

2.5 吸波材料（Absorber）

功能：吸收腔内反射波 → 降低多径干扰 + 抑制腔体共振。

** Laird 5G mmWave 吸波材料白皮书要点 **：

  • 屏蔽罩内贴薄层吸波材料 → 阻尼腔体共振
  • 吸波材料的 ε_r 和 μ_r 决定能量存储和耗散
  • 需通过 S11/S21 测量获得复数介电常数和磁导率
  • 消除天线附近物体的反射干扰

3.1 腔体共振（Cavity Resonance）

问题：屏蔽罩尺寸 > λ/2 时，内部形成谐振腔。

28GHz 毫米波：λ = 10.7mm → λ/2 = 5.35mm

后果：

  • 腔内能量振荡 → 电路工作异常
  • 屏蔽泄漏增加
  • 近场耦合增强

解决方案：

  1. 在罩内贴吸波材料（damping）
  2. 缩小腔体尺寸（< λ/2）
  3. 使用多腔分割设计

3.2 吸收主导的低反射屏蔽膜

2024 Nano-Micro Letters 突破：

SrM 铁氧体 + Cu 网格多层膜：

  • 多波段超低反射：39GHz 和 52GHz 反射率 < 5%（SER < 0.1dB）
  • 宽带高屏蔽：40-90GHz SE > 30dB（屏蔽 99.9% EMI）
  • 超薄：0.58mm（39/52GHz 版）/ 0.34mm（60/77GHz 版）
  • 原理：M 型铁氧体铁磁共振频率（FMR）可调 → 在目标频段吸收主导
  • 传输线模型：可理论设计屏蔽膜参数

应用：

  • 5G 电信：39GHz/52GHz 频段
  • 汽车雷达：60GHz/77GHz 频段

为什么重要：5G 组件间距极小 → 屏蔽膜反射会造成二次 EMI 污染 → 吸收主导型屏蔽是必然趋势。

3.3 MXene 屏蔽材料

MXene：二维过渡金属碳化物/氮化物（如 Ti₃C₂Tₓ）

特点：

  • 超高导电性 → R 大
  • 层状结构 → 多次内部反射 → A 大
  • 超薄柔性 → 可穿戴/折叠屏适用
  • SE 可达 90dB（10μm 厚度）

3.4 屏蔽罩（Shield Can）设计

板级屏蔽罩：

    ┌──────────┐
    │  屏蔽罩   │  ← 金属盖（Ni/Cu 电镀）
    ├──────────┤
    │ 吸波层   │  ← 抑制腔体共振
    ├──────────┤
    │  PCB     │
    │ [RFIC]   │  ← mmWave 电路
    └──────────┘

关键设计规则：

  1. 接地过孔间距 ≤ λ_g/10（guidewave 在罩内表面的波长）
  2. 多排接地孔 → 降低接地电感
  3. 罩内贴吸波材料 → 阻尼共振
  4. 接口处用导电衬垫 → 密封接缝

4.1 双天线法（Two-Antenna Method）

Setup：

[Tx Antenna] ====[SHIELD]===> [Rx Antenna]
       ↑                      ↑
    发射天线             接收天线

步骤：

  1. 无屏蔽时测量 S21₀
  2. 放入屏蔽材料后测量 S21
  3. SE = 20×log₁₀(|S21₀|/|S21|)

标准：IEC 61000-4-21 / ASTM D4935

4.2 混响室法（Reverberation Chamber）

适用：大体积屏蔽室/设备的统计 SE 测量 优势：快速，覆盖多入射角

4.3 屏蔽室测试

法拉第室：

  • 用于 EMC 预兼容测试
  • SE > 100dB（10kHz-18GHz）
  • 门缝用指形簧片 + 窗用蜂窝板

5.1 分层接地策略

电源地 ───┐
数字地 ───┤ 单点接地 ──→ 屏蔽壳
模拟地 ───┘

规则：

  • 电源/数字/模拟/RF 分区
  • 金属隔板分隔功能区
  • 接地前分层，单点汇接

5.2 屏蔽完整性

失效模式排名：

  1. 接缝泄漏（占 70%+ 失败原因）
  2. 通风孔/滤波器未屏蔽
  3. 电缆进出未滤波
  4. 显示屏窗口泄漏
  5. 壳体穿孔（螺丝孔/指示孔）

5.3 电缆屏蔽

关键：电缆是最佳"天线" → 必须屏蔽！

做法：

  • 360° 端接（clamp 式连接器，非螺旋缠绕）
  • 输入/输出加滤波器（feedthrough capacitor）
  • 屏蔽层可靠接地

六、给非专业读者的一句话总结

电磁屏蔽的本质是"堵"和"吸"——导体把电磁波反射回去（堵），损耗材料把电磁波变成热量（吸）。5G 毫米波带来两个全新挑战：一是屏蔽罩变成了"谐振腔"，里面的信号来回反弹；二是组件靠得太近，屏蔽膜的反射会造成二次干扰——所以需要"吸收主导"的低反射屏蔽膜。2024 年的 SrM 铁氧体方案做到了在 39/52GHz 反射率不到 5%，同时屏蔽 99.9% 的干扰。电磁屏蔽告诉我们：最好的防御不是硬碰硬地反弹，而是温柔地把能量消化掉。

金句：屏蔽的最高境界——不是把所有东西挡在外面，而是让进来的一切都安静下来。
