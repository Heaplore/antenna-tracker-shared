环形器/隔离器 Circulator / Isolator Circulator-Isolator

环形器是一个多端口非互易器件——信号从端口 1 到端口 2，从端口 2 到端口 3，从端口 3 到端口 1，反向不通。隔离器是环形器的特例（2 端口），只允许信号单向通过，反射信号被"吞"掉。

类比入口

环形器就像一个"单行道转盘"——车从 A 入口进来只能从 B 出口出去，从 B 进来只能从 C 出去，不能逆行。隔离器就是一个"单向阀门"——水只能从进口流向出口，逆流时阀门自动关闭。在射频系统中，隔离器保护 PA 不被天线的反射功率烧毁。

一、定义与工作原理

1.1 环形器（Circulator）

    Port 1 ──→ Port 2
        ↑              │
        │              ↓
        Port 3 ←── Port 2

信号流向：

P1 → P2 → P3 → P1 → ...（顺时针）
    反向不通：P2 ↛ P1, P3 ↛ P2, P1 ↛ P3

1.2 隔离器（Isolator）

隔离器 = 环形器 + 端接负载：

         ┌──[50Ω Load]──┐
    P1 ──→ P2 ──→ Output
    ↑              │
    │              ↓
    Isolation      P3（内部端接）

信号流向：

输入 → 隔离器 → 输出（正向通过）
    反射信号 ← 隔离器（被吸收，不回传输入端）

隔离度 > 20dB 意味着反射信号被衰减 100 倍以上。

1.3 工作原理：铁氧体 + 磁场

铁氧体材料 + 永久磁铁 → 非互易性 → 环形/单向传输

物理原理：

  • 外加磁场使铁氧体中的电子进动
  • 顺时针和逆时针极化的波经历不同的有效磁导率
  • 导致正向和反向传播的相位速度不同
  • 结果是信号只能单向通过

2.2 隔离器指标

指标符号典型值说明
插入损耗IL0.1-0.3dB正向损耗
隔离度ISO20-40dB反向隔离
电压驻波比VSWR1.05-1.2端口匹配
工作频段f_min - f_max1-18GHz 常见
功率容量P_max10W-500W

2.3 隔离度与功率容量的关系

隔离度越高 → 对 PA 的保护越好
    功率容量越大 → 能承受的输出功率越高

典型权衡：

  • 低功率隔离器（< 1W）：IL = 0.05dB，ISO = 30dB
  • 中功率隔离器（10W）：IL = 0.15dB，ISO = 25dB
  • 高功率隔离器（100W）：IL = 0.3dB，ISO = 20dB

三、主要类型

3.1 铁氧体环形器/隔离器

最传统、最成熟的类型：

结构：
      永久磁铁 → 提供偏置磁场
      铁氧体圆盘 → 核心非互易材料
      微带/波导结构 → 信号传输路径
参数典型值
频段DC-18GHz
IL0.1-0.5dB
ISO20-30dB
功率1W-1kW
体积较大

3.2 平面环形器（Microstrip/Monolithic）

集成于 PCB 或 MMIC：

结构：
      铁氧体薄膜 + 微带线
      或
      磁光材料 + 光波导（光环形器）
参数典型值
频段1-40GHz
IL0.5-2dB
ISO10-20dB
功率< 1W
体积很小（芯片级）

3.3 光环形器（Optical Circulator）

用于光纤通信：

波长 | 端口1 → 端口2 → 端口3
    1310nm | 输入 → 环行 → 输出
    1550nm | 反射 → 环行 → 监测

3.4 非互易相移器

利用磁光效应实现非互易相移：

正向相移：+45°
    反向相移：-45°

四、应用场景

4.1 PA 保护（隔离器）

PA → 隔离器 → 滤波器 → 天线
          ↑
       反射功率被吸收

为什么需要：

  • 天线 VSWR 不佳时，反射功率可能烧毁 PA
  • 隔离器阻止反射功率回传 PA
  • 保护 PA 免受驻波损坏

4.2 全双工通信（环形器）

Tx → 端口1 → 端口2 → 天线
    Rx ← 端口3 ← 天线（接收信号）

全双工挑战：

  • 发射信号泄漏到接收端（自干扰）
  • 需要环形器 + 自干扰消除
  • 隔离度要求 > 60dB（环形器本身只有 20-30dB）

4.3 T/R 模块中的收发隔离

PA → 环形器端口1 → 端口2 → 天线
    LNA ← 环形器端口3 ← 天线

4.4 雷达系统

脉冲雷达：
      发射 → 环形器 → 天线
      接收 ← 环形器 ← 天线

4.5 基站前端

AAU 中每个通道：
      PA → 隔离器 → 滤波器 → 天线
      保护 PA 免受天线失配损坏

五、与功分器的关系

环形器和功分器经常一起使用：

PA 输出 → 功分器（N 路）→ N 个 PA → N 个隔离器 → 合路器 → 天线

隔离器确保每个 PA 的反射不会影响其他 PA。

六、工程注意事项

6.1 温度对性能的影响

铁氧体材料的磁特性随温度变化：
    ΔISO/ΔT ≈ -0.1 to -0.5 dB/°C
    ΔIL/ΔT ≈ +0.01 to +0.05 dB/°C

高温环境（如密闭 AAU）需要特别注意。

6.2 功率容量与散热

高功率下铁氧体发热 → 磁特性退化 → 隔离度下降

散热措施：

  • 金属外壳导热
  • PCB 散热通孔
  • 强制风冷（高功率场景）

6.3 磁场屏蔽

环形器/隔离器内部的永久磁铁可能干扰周围电路：

外部磁场 < 100 Oe（避免影响铁氧体工作状态）

6.4 小型化趋势

传统铁氧体隔离器：10×10×5mm
    平面集成隔离器：1×1mm（芯片级）

5G 毫米波基站中，隔离器正逐渐集成到 T/R 模块内部。

八、一句话总结

环形器/隔离器是射频系统中的"单向阀门"——利用铁氧体的非互易特性，让信号只能单向通过。隔离器保护 PA 不被反射功率烧毁，环形器在 T/R 模块中实现收发隔离。核心指标：插入损耗（越小越好）、隔离度（越大越好）、功率容量（越大越好）。5G 毫米波时代，环形器正从分立器件走向集成化（芯片级）。
