方向图综合中的凸优化 Convex Optimization in Pattern Synthesis

方向图综合的凸优化方法将天线阵列的幅相加权问题转化为凸优化问题（如 SDP、SOCP），保证找到全局最优解，特别擅长处理多约束场景——同时压低旁瓣、放置零陷、控制波束宽度，而不像经典方法（Chebyshev/Taylor）那样只能优化单一指标。

类比入口

经典方向图综合像"调一首歌的音量"——你只能调整体音量大小。凸优化方向图综合像"调音台上的 20 个推子"——你可以独立控制低频、中频、高频，还可以静音某个频段。更形象地说：Chebyshev 是给阵列"穿上一件预设好的旁瓣抑制衣"，而凸优化是"量身定制一件衣服"——你要什么形状、什么旁瓣电平、要在哪个方向挖零陷，它都能满足。

一、基本定义与原理

1.1 方向图综合的核心问题

给定天线阵列的几何结构和期望的方向图，求解每个单元的激励幅度和相位：

已知：阵列几何 G = {r₁, r₂, ..., r_N}
    求解：激励向量 w = [w₁, w₂, ..., w_N]^T
    使得：AF(θ,φ) ≈ F_desired(θ,φ)

其中 AF 是阵列因子：

AF(θ,φ) = Σ(n=1 to N) w_n · exp(jk·r_n · [sinθcosφ, sinθsinφ, cosθ])

1.2 为什么经典方法不够？

方法优点缺点
傅里叶变换法简单快速旁瓣高（-13dB），无法控制零陷
Dolph-Chebyshev等旁瓣，旁瓣可控只能优化旁瓣，不能放零陷
Taylor旁瓣+零陷平衡参数选择复杂，零陷位置受限
Woodward-Lawson灵活旁瓣不均匀
凸优化多约束同时优化，全局最优计算量大

1.3 凸优化的核心优势

凸优化问题的定义：

minimize    f₀(x)        （凸目标函数）
    subject to  fᵢ(x) ≤ bᵢ   （凸不等式约束）
                Ax = c       （线性等式约束）

关键性质：任何局部最优解都是全局最优解。

对于方向图综合，这意味着：

  • 不会陷入局部最优（旁瓣假峰）
  • 可以在同一问题中同时优化多个指标
  • 有成熟的求解器（CVX、MOSEK、SeDuMi）

二、凸优化在方向图综合中的主要形式

2.1 半定松弛（Semidefinite Relaxation, SDR）

核心思想：将非凸的二次问题转化为凸的半定规划（SDP）。

原始问题（非凸）：

minimize    ||AF(θ) - F_desired(θ)||²
    subject to  |w_n|² ≤ P_max

SDR 变换：

令 R = wwᴴ    （秩为 1 的半正定矩阵）
    则 w_n·w_m* = R_nm

转化后的 SDP：

minimize    tr(A₀R) + b₀
    subject to  tr(AᵢR) ≤ bᵢ,  i = 1,...,m
                R ⪰ 0           （R 半正定）
                rank(R) = 1     （非凸，需松弛）

松弛掉 rank(R) = 1 约束后，问题变为凸 SDP，求解后再通过随机化或特征分解恢复 w。

2.2 二阶锥规划（Second-Order Cone Programming, SOCP）

适合处理范数约束和波束方向图约束：

minimize    t
    subject to  ||w||₂ ≤ P_max
                |AF(θ_null,i)| ≤ ε,  i = 1,...,K    （零陷约束）
                AF(θ_main,j) ≥ G_min,  j = 1,...,M  （主瓣约束）
                PSLL(θ) ≤ SLL_max                     （旁瓣约束）

每个约束都是二阶锥形式：

||Ax + b||₂ ≤ cᵀx + d

优势：

  • 比 SDP 计算效率高（无矩阵分解）
  • 求解器成熟（CVX + ECOS/SCS）
  • 适合实时波束赋形

2.3 线性规划（LP）

最简单的凸优化形式，适合处理绝对值约束：

minimize    Σ_n |w_n|
    subject to  AF(θ_null,i) = 0,  i = 1,...,K
                |AF(θ_sidelobe)| ≤ ε

优势：计算最快，适合大规模阵列（N > 1000）。 劣势：表达能力有限，只能处理线性约束。

2.4 二次规划（QP）

目标函数为二次型，约束为线性：

minimize    wᴴQw + cᵀw + d
    subject to  Aw ≤ b

适合最小化方向图与期望方向图的均方误差：

Q = ∫ AF(θ)·AF*(θ) dΩ  （方向图自相关矩阵）

三、典型应用场景

3.1 低旁瓣 + 零陷放置联合优化

问题：同时压低旁瓣并在干扰方向放置零陷。

minimize    max_θ∈SLL |AF(θ)|          （最小化最大旁瓣）
    subject to  AF(θ_interfere,i) = 0,    （零陷约束）
                i = 1,...,K
                |AF(θ_main)| ≥ G_min,     （主瓣增益约束）
                ||w||² ≤ P_max            （功率约束）

效果：

  • 旁瓣压低到 -30dB 或更低
  • 零陷深度 -60dB 或更低
  • 主瓣宽度可控

对比 Chebyshev：Chebyshev 只能优化旁瓣，零陷需要额外处理（如 Longman 方法），效果不如联合优化。

3.2 波束宽度 + 旁瓣联合优化

minimize    SLL_max + λ · BW          （加权多目标）
    subject to  ||w||² ≤ P_max

其中 BW 是 3dB 波束宽度，λ 是权重系数。

Pareto 前沿：通过调整 λ，可以得到一系列 (SLL, BW) 的 Pareto 最优解。

3.3 幅相误差鲁棒综合

实际阵列中，幅相误差不可避免：

w_actual = w_designed + Δw
    Δw ~ CN(0, σ²)

鲁棒优化考虑最坏情况：

minimize    max_{||Δw||≤ε} max_θ∈SLL |AF(θ, w+Δw)|

效果：综合出的方向图对幅相误差不敏感，旁瓣在误差存在时仍能保持低位。

3.4 近场方向图综合

近场波束赋形需要同时优化焦点位置和方向图：

minimize    ||AF_NF(r_focus, θ_focus) - F_desired||²
    subject to  ||w||² ≤ P_max

其中 AF_NF 是近场阵列因子（球面波模型）。

3.5 极化方向图综合

双极化阵列需要同时优化两个极化通道的方向图：

minimize    ||AF_V(θ) - F_V_desired||² + ||AF_H(θ) - F_H_desired||²
    subject to  |w_V,n|² + |w_H,n|² ≤ P_max,n

5.2 典型工作流程（MATLAB + CVX）

N = 64;                          % 阵列单元数
    theta_sll = [-90:-1:-30, 30:1:90];  % 旁瓣采样点
    theta_null = [-15, 25];            % 零陷方向
    theta_main = [-5:5];               % 主瓣采样点
    
    variables w_real(N) w_imag(N)
    w = w_real + 1j*w_imag;
    
    % 阵列因子
    AF_sll = zeros(length(theta_sll), 1);
    for i = 1:length(theta_sll)
        AF_sll(i) = sum(w .* exp(1j*2*pi*(0:N-1)*sin(theta_sll(i)*pi/180)));
    end
    
    AF_null = zeros(length(theta_null), 1);
    for i = 1:length(theta_null)
        AF_null(i) = sum(w .* exp(1j*2*pi*(0:N-1)*sin(theta_null(i)*pi/180)));
    end
    
    % 优化问题
    minimize( max(abs(AF_sll)) )
    subject to
        abs(AF_null) <= 1e-3;          % 零陷约束
        abs(sum(w)) >= sqrt(N)*0.8;    % 主瓣增益约束
        sum(abs(w).^2) <= N;           % 功率约束
    
    % 求解
    cvx_solver mosek
    cvx_begin
        % ... 同上 ...
    cvx_end
    
    disp(['Peak SLL: ', num2str(20*log10(cvx_optval), '%.2f'), ' dB']);

六、工程挑战与坑

6.1 离散化误差

方向图约束需要在连续角度上定义，但实际只能采样离散点：

θ_sampled = [-90, -89, ..., 90]  （1° 间隔）

如果实际旁瓣峰出现在 0.5° 处，采样点会错过它。

对策：

  • 加密采样（0.1° 间隔）
  • 随机采样 + 多次优化
  • 解析约束（不依赖采样）

6.2 非凸问题的松弛间隙

SDR 松弛掉 rank(R) = 1 约束后，可能存在松弛间隙（Relaxation Gap）：

R_opt 的特征值：[λ₁, λ₂, ..., λ_N]
    如果 λ₂ >> 0 → 松弛间隙大 → 恢复的 w 质量差

对策：

  • 使用随机化方法（Randomized Rounding）
  • 添加秩约束的凸近似
  • 改用 SOCP 形式（天然保证 rank-1）

6.3 实时性

凸优化求解通常需要秒级时间，不适合实时波束赋形：

对策：

  • 离线优化 + 在线查找（Codebook 预计算）
  • 热启动（Warm Start）：利用上一次解作为初始值
  • 一阶方法（ADMM）：适合大规模问题，收敛快但精度低

6.4 约束冲突

多个约束可能相互矛盾：

约束 1：旁瓣 < -30dB
    约束 2：零陷深度 < -60dB
    约束 3：波束宽度 < 2°

这三个约束可能无法同时满足。

对策：

  • 优先级排序：主瓣 > 零陷 > 旁瓣
  • 软约束：将硬性约束改为惩罚项
  • Pareto 优化：找最优折中

八、技术演进

  • 1940s-1950s：傅里叶变换法、 Woodward-Lawson 法
  • 1950s-1960s：Dolph-Chebyshev、Taylor 方法建立
  • 1970s-1980s：Longman 零陷方法、Bernstein 多项式
  • 1990s：Boyd 等人将凸优化引入天线设计
  • 2000s：SDR 在波束赋形中的应用
  • 2010s：SOCP 在实时波束赋形中的应用
  • 2020s：深度学习辅助凸优化（Learning to Optimize）
  • 2025年：AI + 凸优化混合方法，端到端可微优化

九、技术成熟度

  • 成熟度等级：大规模商用
  • 说明：凸优化方向图综合在雷达、卫星通信、5G/6G 基站中已有广泛应用。Chebyshev/Taylor 仍是工程首选（简单快速），凸优化在复杂约束场景中不可替代。

十、一句话总结

凸优化将方向图综合从"手工调参"升级为"自动求解"——你告诉优化器你要什么（旁瓣多少、零陷在哪、波束多宽），它给你全局最优的幅相加权。虽然计算代价比 Chebyshev 高，但在 6G 复杂波束赋形场景下，这是值得付出的成本。
