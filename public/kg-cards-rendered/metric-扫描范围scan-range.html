扫描范围 Scan Range（扫描角） Scan-Range

扫描范围是相控阵天线能够通过电子方式改变波束指向的角度范围——它告诉你天线能"看"到多广的视野，扫描范围越大，系统的灵活性和覆盖能力越强。

类比入口

扫描范围就像人的眼睛——你能转动眼球看到的范围大约 ±60°（不算头部转动）。相控阵天线就是"电子眼球"，不需要转动整个天线，只通过控制每个单元的相位就能"看"向不同方向。扫描范围就是这只"电子眼球"能转到的极限角度。

一、定义与物理意义

1.1 扫描范围的定义

扫描范围 = 波束能在多大角度范围内电子扫描
    通常用 ±θ_max 表示
    
    典型值：
      雷达：±60°
      5G 基站：±60°（水平）/ ±15°（垂直）
      卫星通信：±45°
      车载雷达：±60°

1.2 扫描角与相位梯度的关系

波束指向角 θ₀ 由移相器相位梯度决定：
    Δφ = (2πd/λ) · sin(θ₀)
    
    θ₀ = arcsin(Δφ · λ / (2πd))

最大扫描角限制：

当 Δφ = π（180°）时：
    sin(θ_max) = λ / (2d)
    
    d = λ/2 → θ_max = 90°（理论最大）
    d = 0.6λ → θ_max = arcsin(0.83) = 56°
    d = 0.7λ → θ_max = arcsin(0.71) = 45°

1.3 扫描范围 vs 扫描速度

扫描范围：能看多广（角度大小）
    扫描速度：转多快（切换时间）
    
    相控阵扫描速度：微秒级（远快于机械转动）

二、扫描范围的限制因素

2.1 栅瓣限制（Grating Lobe Constraint）

为避免栅瓣：
    d/λ ≤ 1 / (1 + |sin(θ_max)|)
    
    θ_max = 0° → d ≤ λ/2
    θ_max = 30° → d ≤ 0.67λ
    θ_max = 45° → d ≤ 0.59λ
    θ_max = 60° → d ≤ 0.5λ
    θ_max = 90° → d ≤ 0.5λ

核心约束：要实现 ±60° 扫描，单元间距必须 d ≤ λ/2。

2.2 扫描盲区（Scan Blindness）

扫描盲区：某些扫描角下天线无法辐射/接收
    原因：表面波激发、互耦共振、栅瓣进入可见区

扫描盲区条件：

d/λ = 1 - 1/N（N 为整数）
    在特定扫描角下出现盲点
    
    例如：d = 0.5λ 时，θ = ±90° 出现盲区

2.3 波束展宽（Beam Broadening）

扫描角越大，波束越宽：
    θ_scan = θ_broadside / cos(θ₀)
扫描角波束展宽增益损失
0°1.0×0dB
30°1.15×-0.6dB
45°1.41×-1.5dB
60°2.0×-3.0dB
70°2.92×-4.7dB

2.4 波束指向误差（Pointing Error）

移相器量化误差导致的波束指向偏差：
    Δθ ≈ Δφ / (k·d·cosθ₀)
    
    Δφ = 量化相位误差
    k = 2π/λ
移相器位数指向误差（典型）
3 bit±5°
4 bit±2.5°
5 bit±1.25°
6 bit±0.6°
8 bit±0.15°

四、扫描范围与系统性能

4.1 扫描范围对增益的影响

G(θ₀) = G(0°) · cos(θ₀) · η_scan(θ₀)
    
    G(0°) = 天线最大增益
    η_scan(θ₀) = 扫描效率（通常 0.7-0.95）

增益损失汇总：

扫描角几何损失扫描效率总损失
0°0dB0dB0dB
30°-0.6dB-0.2dB-0.8dB
45°-1.5dB-0.5dB-2.0dB
60°-3.0dB-1.0dB-4.0dB
70°-4.7dB-1.5dB-6.2dB

4.2 扫描范围与覆盖半径

R(θ) = R(0°) · cos(θ)
    
    θ = 60° → R = 0.5 · R(0°)
    θ = 45° → R = 0.71 · R(0°)

实际影响：扫描角越大，有效覆盖半径越小。

4.3 扫描范围与干扰抑制

干扰来自 θ_int 方向：
    抑制比 = FBR(θ_int) + SLL(θ_int)
    
    θ_int = 30° → 抑制比 ≈ 25dB
    θ_int = 60° → 抑制比 ≈ 15dB（恶化 10dB）

五、扩大扫描范围的技术

5.1 减小单元间距

d < λ/2 → 扫描范围 > ±60°
    d = 0.4λ → θ_max ≈ 75°
    d = 0.3λ → θ_max ≈ 85°

代价：互耦增强，设计复杂度 ↑

5.2 非均匀间距（Thinned Array）

均匀间距：d = λ/2
    非均匀间距：随机或优化的间距分布
    
    效果：
      - 栅瓣被"打散"到旁瓣水平
      - 扫描范围不受栅瓣严格限制
      - 但最大扫描角仍受 d_min 限制

5.3 宽带扫描技术

频率扫描阵列：
      通过改变频率实现波束扫描
      扫描范围受限于带宽
    
    相位扫描阵列（相控阵）：
      通过改变相位实现波束扫描
      扫描范围不受频率限制

5.4 多波束技术

同时生成多个波束：
      - 数字波束赋形：N 个独立波束
      - 模拟波束赋形：1 个波束，快速切换
      - 混合波束赋形：M 个波束（M < N）

六、工程注意事项

6.1 扫描范围 vs 带宽的权衡

宽扫描范围 → 需要小间距 → 带宽受限
    宽带宽 → 需要大间距 → 扫描范围受限

5G 毫米波的解决方案：

  • 使用宽频带天线单元（如 Vivaldi）
  • 单元间距 d = λ/2（在中心频率）
  • 通过数字波束赋形补偿扫描引起的带宽变化

6.2 扫描范围与天线罩

天线罩（Radome）对扫描范围有影响：
      - 倾斜入射时，天线罩引起相位误差
      - 扫描角 > 45° 时影响显著
      - 需优化天线罩曲面设计

6.3 扫描范围与互耦

互耦随扫描角变化：
      θ = 0° → 互耦最小
      θ = 60° → 互耦最大（表面波激发）
    
    互耦导致：
      - 扫描盲区
      - 方向图畸变
      - 增益波动

七、一句话总结

扫描范围是相控阵天线能"看"到的最大角度——它受栅瓣限制（d ≤ λ/2 才能扫到 ±60°）、波束展宽（扫描角越大波束越宽）、增益损失（60° 扫描损失 3-4dB）和扫描盲区的影响。记住三个关键：① 扫描角越大，性能越差（增益↓、波束宽↑）② 栅瓣是扫描范围的最大限制因素 ③ 5G 基站通常设计为 ±60° 水平扫描。
