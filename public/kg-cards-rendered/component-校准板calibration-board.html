校准板 Calibration Board

校准板是 AAU 内部校准的"桥梁"——它通过耦合器和环形器连接所有 T/R 模块，在主通道和从通道之间建立校准链路，确保 64 个通道的幅度和相位一致性。

一、校准板的角色

1.1 为什么需要校准

通道不一致的来源：
      1. T/R 模块个体差异（PA增益、LNA噪声）
      2. 温度漂移（不同通道温升不同）
      3. 频率响应差异
      4. 老化效应
      
    不校准的后果：
      - 波束指向偏差（误码率↑）
      - 旁瓣抬高（干扰↑）
      - MIMO 容量下降（速率↓）

1.2 校准链路架构

校准板连接方式：
      ┌──────────────────────────────────────┐
      │  主通道耦合器（Master Coupler）       │
      │  - 从 TX 路径采样信号                 │
      │  - 分发到从通道 RX                    │
      └──────────────────────────────────────┘
             │
      ┌──────┴──────────────────────────────┐
      │  从通道耦合器（Slave Couplers）       │
      │  - 从各 TX 路径采样信号               │
      │  - 发送到主通道 RX                   │
      └──────────────────────────────────────┘

二、校准方法

2.1 内部校准（推荐）

步骤：
      1. 通道 1 发射校准信号
      2. 其他通道通过耦合器接收
      3. 计算相对幅度和相位误差
      4. 更新通道 1 的校准系数
      5. 循环对所有通道执行
      
    优点：
      - 无需外部设备
      - 可在线校准
      - 精度高
      
    缺点：
      - 耦合器误差会影响精度
      - 需要额外的耦合器/环形器

2.2 外部校准（OTA）

步骤：
      1. AAU 放入暗室
      2. 标准天线接收信号
      3. 测量全方向图
      4. 计算校准系数
      5. 下发到 AAU
      
    优点：
      - 精度最高
      - 包含天线方向图效应
      
    缺点：
      - 需要暗室设备
      - 不能在线校准
      - 成本高

三、校准板硬件

3.1 核心组件

校准板组成：
      ├── 耦合器网络（Star Coupler / 环形器）
      ├── 开关矩阵（选择主/从通道）
      ├── 衰减器（保护接收链路）
      ├── 滤波器（抑制带外干扰）
      ├── MCU（校准算法执行）
      └── 存储（校准系数 Flash）

3.2 校准信号

校准信号类型：
      1. CW 信号（连续波）
         - 最简单
         - 适合幅度/相位校准
         
      2. PN 序列（伪随机）
         - 抗干扰
         - 适合噪声基底高的场景
         
      3. OFDM 符号
         - 频率选择性校准
         - 适合宽带系统

四、校准精度

4.1 精度要求

5G NR 要求：
      幅度一致性：±0.5dB
      相位一致性：±3°
      
    校准后精度：
      幅度：±0.1dB
      相位：±1°
      
    温度漂移：
      幅度：±0.1dB/°C
      相位：±0.5°/°C

4.2 校准周期

校准触发条件：
      1. 开机时（全通道校准）
      2. 温度变化 > 5°C（部分校准）
      3. 定时校准（每 6 小时）
      4. 网管手动触发
      
    校准耗时：
      全通道校准：~5 秒
      温度补偿校准：~1 秒

五、工程注意事项

5.1 耦合器设计

校准耦合器要求：
      1. 耦合度一致性：±0.1dB
      2. 相位一致性：±1°
      3. 温度稳定性：±0.01dB/°C
      4. 插入损耗：< 0.5dB

5.2 校准系数存储

存储位置：
      1. EEPROM（断电保存）
      2. Flash（可擦写）
      3. 云端（远程备份）
      
    更新策略：
      1. 全通道校准后更新
      2. 温度补偿后增量更新
      3. OTA 升级时同步更新

六、一句话总结

校准板是 AAU 通道一致性的保障——通过耦合器网络连接所有 T/R 模块，实现内部校准。精度要求：幅度 ±0.1dB、相位 ±1°。记住：校准不是一次性的，温度漂移和老化会导致精度退化，定期校准是波束赋形质量的保证。
