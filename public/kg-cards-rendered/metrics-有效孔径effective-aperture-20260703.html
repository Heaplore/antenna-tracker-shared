有效孔径 Effective Aperture（A_e） Effective-Aperture

有效孔径是天线的"捕光面积"——它表示天线从入射电磁波中"捕获"了多少功率，与物理尺寸不一定相等（通常更小），但与增益直接相关：A_e = G·λ²/(4π)。

类比入口

想象你在下雨天拿一个桶接雨水。桶口的物理面积是 100cm²，但因为你拿桶的手势不对（没有正对雨的方向），实际接到的雨量只相当于 60cm² 的面积。这 60cm² 就是你的"有效孔径"——它反映了天线"真正能用"的捕集能力，而不是它的物理大小。

一、定义与物理意义

1.1 数学定义

A_e = P_received / S_incident

其中：

  • P_received = 天线接收到的功率 [W]
  • S_incident = 入射波的功率密度 [W/m²]

1.2 有效孔径与增益的关系

这是最重要的公式：

A_e = G · λ² / (4π)

其中：

  • G 是天线的增益（线性值，不是 dB）
  • λ 是波长

推导：

P_received = S_incident · A_e
    同时：P_received = (P_transmitted · G_t · G_r · λ²) / ((4π)² · d²)  （Friis 公式）
    而 S_incident = P_transmitted / (4πd²)
    
    所以：A_e = G_r · λ² / (4π)

1.4 有效孔径的方向性

A_e 不是常数——它随入射波的方向变化：

A_e(θ, φ) = A_e,max · F(θ, φ) / F(θ_max, φ_max)

其中 F(θ, φ) 是归一化方向图。

物理意义：天线正对着信号源时"捕集"能力最强，偏离后减弱。

二、计算公式

2.1 从功率密度计算

A_e = P_r / (E² / (2η₀))

其中：

  • P_r = 接收功率
  • E = 入射电场幅值
  • η₀ = 自由空间阻抗 ≈ 377Ω

2.2 从增益计算

A_e = G · λ² / (4π)

举例（28GHz，G = 20dBi = 100）：

λ = c/f = 3×10⁸ / 28×10⁹ = 10.7mm
    A_e = 100 × (0.0107)² / (4π) = 0.0091 m² = 91 cm²

2.3 从口径场积分计算

A_e = |∫∫_A E_a(x,y) dx dy|² / (λ² ∫∫_A |E_a(x,y)|² dx dy)

其中 E_a(x,y) 是口径上的电场分布。

简化理解：分子是口径场的"相干叠加"，分母是口径场的"总能量"。口径场越均匀，A_e 越接近 A_phys。

2.4 抛物面天线的有效孔径

A_e = ε_a · π(D/2)²

其中 D 是抛物面直径，ε_a 是口径效率（通常 50-70%）。

口径效率分解：

ε_a = ε_tap · ε_spill · ε_phase · ε_pol · ε_obc
效率因子符号典型值说明
照射效率ε_tap0.7-0.9馈源照射均匀度
溢出效率ε_spill0.8-0.95能量不溢出抛物面边缘
相位效率ε_phase0.9-0.99口径相位不均匀度
极化效率ε_pol0.95-0.99极化匹配度
遮挡效率ε_obc0.85-0.95副镜/馈源支撑遮挡

2.5 阵列天线的有效孔径

N 元阵列：

A_e,array = N · A_e,element · ε_coupling

其中 ε_coupling 是互耦效率（0.7-0.95）。

四、测量方法

4.1 已知发射天线法

步骤：
    1. 用已知增益 G_t 的发射天线
    2. 测量接收功率 P_r
    3. 已知发射功率 P_t 和距离 d
    4. 计算：A_e = P_r · (4πd²) / (P_t · G_t)

4.2 增益比较法

步骤：
    1. 用标准增益天线测量参考功率 P_std
    2. 换上 AUT 测量 P_AUT
    3. 计算：A_e,std = G_std · λ²/(4π)
    4. A_e,AUT = A_e,std · (P_AUT / P_std)

4.3 暗室方向图积分法

步骤：
    1. 测量完整 3D 方向图
    2. 计算波束立体角 Ω_A
    3. D = 4π/Ω_A
    4. A_e = D · λ²/(4π) · η

六、工程注意事项

6.1 A_e 随频率变化

对于宽频天线（如喇叭、对数周期天线），A_e 随频率变化显著：

A_e(f) = G(f) · c² / (4π · f²)

设计宽频系统时，需要考虑 A_e 在整个频段内的变化。

6.2 有效孔径与接收灵敏度

接收信噪比 SNR ∝ A_e · P_t / (k · T_sys · B)

A_e 越大 → 接收灵敏度越高 → 可以接收更弱的信号。

这就是为什么射电望远镜要用巨大的口径——不是为了"放大"信号，而是为了"捕获"更多能量。

6.3 MIMO 中的有效孔径

MIMO 的空间分辨率取决于有效孔径的物理尺寸（不是单元数）：

空间分辨率 Δθ ≈ λ / A_e_physical

64 单元 MIMO 如果挤在小面积里，空间分辨率不如 8 单元但分布在大面积上的天线。

七、一句话总结

有效孔径是天线的"真实捕集面积"——它告诉你天线能从电磁波中"捞到多少功率"。核心公式 A_e = G·λ²/(4π) 把增益和有效孔径绑在一起：增益高的天线有效孔径大，但高频天线的 λ² 小，所以同样的增益下，高频天线的 A_e 反而更小。记住：A_e 不等于物理面积，它通常小于物理面积（口径效率 < 100%）。
