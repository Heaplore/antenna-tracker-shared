极化波束赋形 Polarization Beamforming

极化波束赋形是在传统空间波束赋形基础上，额外利用电磁波的极化自由度（如 ±45° 双极化、圆极化、任意椭圆极化）进行独立的波束控制和空间复用，从而在不增加天线数量的前提下将容量翻倍的技术。它是 6G XL-MIMO 的核心使能技术之一。

类比入口

传统波束赋形像是在一维的墙上打孔——你只能左右上下调整角度。极化波束赋形相当于在墙上开了两扇独立的门：一扇门处理"水平"信号，另一扇门处理"垂直"信号，两扇门可以指向不同方向、服务不同用户。这样，同样的天线阵列可以同时服务两倍的用户。

一、基本定义与原理

1.1 从极化分集到极化波束赋形

技术原理容量增益
极化分集利用正交极化降低信号相关性1.2-1.5×
极化复用同一频率、不同极化传输独立数据流2×
极化波束赋形对不同极化通道独立赋形，形成极化依赖的波束2-4×

极化波束赋形是上述技术的综合——它不仅利用双极化来倍增容量，还对每个极化通道独立进行波束指向控制。

1.2 ±45° 双极化阵列的基本模型

基站天线通常采用 ±45° 交叉极化配置：

E_total = E_V · p_V + E_H · p_H

其中 p_V 和 p_H 分别是 +45° 和 -45° 极化单位矢量。

对于 N 单元双极化阵列（N 个物理单元，每个单元有两个极化端口），波束赋形向量是 2N × 1 的复数向量：

w = [w_V1, w_V2, ..., w_VN, w_H1, w_H2, ..., w_HN]^T

1.3 极化波束赋形的核心思想

传统空间波束赋形只优化 w 的相位部分（控制波束指向），而极化波束赋形同时优化：

  1. 空间相位：控制波束指向角（θ, φ）
  2. 极化相位：控制 V/H 极化之间的相位差 → 决定辐射极化状态
  3. 极化幅度：控制 V/H 极化之间的幅度比 → 决定极化纯度

1.4 极化状态的数学描述

任意极化状态可以用极化椭圆参数描述：

椭圆率 χ = arctan(b/a)  （a=长半轴，b=短半轴）
    倾斜角 τ = arctan(E_H/E_V)
  • τ = 0° → 垂直极化
  • τ = 45° → +45° 线性极化
  • τ = -45° → -45° 线性极化
  • χ = ±45° → 右/左旋圆极化

通过控制 w_V 和 w_H 的幅度和相位，可以动态生成任意极化状态。

二、XL-MIMO 中的极化波束赋形

2.1 XL-MIMO 的极化挑战

6G 的 XL-MIMO（超大规模 MIMO）天线阵列可能有 1024-4096 个单元，双极化后达到 2048-8192 个端口。极化波束赋形在此规模下面临独特挑战：

挑战原因影响
XPD 空间变化阵列不同位置的单元 XPD 不同极化隔离度下降
近场效应XL-MIMO 的近场区可达数百米极化状态随距离变化
去极化传播环境导致极化状态改变极化复用增益下降
校准复杂度2N 个端口的幅相校准校准时间大幅增加

2.2 XPD（交叉极化鉴别率）的空间变化

在传统 MIMO 中，XPD 被视为一个标量值（通常 20-30dB）。但在 XL-MIMO 中，XPD 在阵列上是非均匀的：

XPD(x,y) = 10·log₁₀(P_co(x,y) / P_cross(x,y))

边缘单元的 XPD 可能比中心单元低 5-10dB，原因是：

  • 边缘单元的互耦环境不同
  • 近场效应在边缘更显著
  • 安装公差在边缘累积更大

影响：XPD 不均匀会导致极化复用串扰增加，容量下降 10-30%。

2.3 近场极化波束赋形

在 XL-MIMO 的近场区（Fresnel 区），电磁波是球面波而非平面波，极化状态随传播方向变化：

E_r(θ,φ) = E_θ · r̂_θ + E_φ · r̂_φ

其中 r̂_θ 和 r̂_φ 是球坐标系下的极化单位矢量，随 (θ, φ) 变化。

传统波束赋形假设：所有单元的极化方向一致（平面波近似）。 近场极化波束赋形：每个单元的极化方向需要根据其到用户的几何关系独立调整。

2.4 极化波束赋形的容量增益

双极化 XL-MIMO 相比单极化的容量增益：

C_dual-pol ≈ C_single-pol × (1 + ρ_xpol)

其中 ρ_xpol 是交叉极化通道的相关系数。

  • 理想情况（ρ_xpol = 0）：容量翻倍（2×）
  • 实际情况（ρ_xpol = 0.3-0.5）：容量增加 1.5-1.7×
  • 恶劣情况（ρ_xpol > 0.7）：容量增加 < 1.3×

关键：降低极化相关性是极化波束赋形发挥容量的前提。

三、极化波束赋形的方法

3.1 独立极化波束（Independent Polarization Beam）

为 V 和 H 极化通道分别设计独立的波束赋形向量：

w_V = [w_V1, w_V2, ..., w_VN]^T  → 波束指向 θ_V, φ_V
    w_H = [w_H1, w_H2, ..., w_HN]^T  → 波束指向 θ_H, φ_H

特点：

  • V 和 H 波束可以指向不同方向
  • 可以同时服务两个不同方向的用户
  • 极化隔离度要求较低

3.2 联合极化波束（Joint Polarization Beam）

V 和 H 极化波束指向同一方向，但极化状态不同：

w_V = w · cos(τ)
    w_H = w · sin(τ) · exp(j·Δφ)

其中 τ 是倾斜角，Δφ 是极化相位差。

特点：

  • 波束指向一致，利于覆盖优化
  • 通过调整 τ 和 Δφ 可以动态改变极化状态
  • 适合极化复用场景

3.3 自适应极化波束赋形

根据信道状态信息（CSI）动态调整极化波束：

maximize    C(w_V, w_H)
    subject to  ||w_V||² + ||w_H||² ≤ P_total
                XPD(w_V, w_H) ≥ XPD_min

优化变量：

  • w_V 和 w_H 的幅度
  • w_V 和 w_H 的相位
  • 极化倾斜角 τ
  • 极化相位差 Δφ

求解方法：

  • 交替优化（Alternating Optimization）
  • 凸优化（Convex Optimization）
  • 深度学习（Deep Learning）

3.4 极化波束量化

实际系统中，极化状态只能离散取值（受限于 BFIC 的移相器位数）：

极化状态τ (度)Δφ (度)对应极化
1450+45° 线性
2-450-45° 线性
34590右旋圆
445-90左旋圆
500垂直线性
6900水平线性

6-bit BFIC 可以提供 64 种极化状态，足以近似任意极化椭圆。

五、应用场景

5.1 6G XL-MIMO 基站

  • 配置：1024-4096 单元双极化阵列
  • 目标：容量相比 5G 提升 10-100 倍
  • 极化波束赋形作用：在不增加硬件的前提下，将空间自由度翻倍

5.2 卫星通信

  • 圆极化波束赋形：适应卫星姿态变化导致的极化旋转
  • 频率复用：同一频率、不同极化（RHCP/LHCP）实现两次容量

5.3 无人机通信

  • 极化自适应：无人机姿态变化时，自动调整极化状态保持最佳连接
  • 极化分集：多架无人机之间的极化正交复用

5.4 ISAC（通感一体化）

  • 不同极化波束分别服务于通信和感知
  • 感知波束使用圆极化（对目标极化不敏感）
  • 通信波束使用线性极化（高纯度）

六、工程挑战与坑

6.1 极化校准

双极化阵列的极化波束赋形需要精确的极化校准：

校准步骤：

  1. 测量每个单元的 S 参数矩阵（2N × 2N）
  2. 确定每个单元的极化倾斜角 τ 和极化相位差 Δφ
  3. 计算极化波束赋形向量 w_V 和 w_H
  4. 在线监测和定期重新校准

校准精度要求：

  • 幅度误差：< 0.2dB
  • 相位误差：< 2°
  • 极化角误差：< 1°

6.2 温度漂移

温度变化会导致：

  • 移相器相位漂移（±0.5°/°C）
  • 极化倾斜角漂移（±0.3°/°C）
  • XPD 变化（±1dB/°C）

对策：内置温度传感器 + 实时补偿算法。

6.3 极化失配

传播环境中的散射、反射和折射会改变信号的极化状态：

E_received = Γ_co · E_transmitted + Γ_cross · E_transmitted_perp

其中 Γ_co 是同极化传输系数，Γ_cross 是交叉极化传输系数。

去极化系数：

δ = |Γ_cross|² / |Γ_co|²

δ 越大，极化复用增益越低。在城市环境中 δ ≈ 0.1-0.3，在室内环境中 δ ≈ 0.3-0.5。

6.4 计算复杂度

N 单元双极化阵列的波束赋形向量有 2N 个复数自由度：

w ∈ ℂ^(2N)

对于 N = 1024 的 XL-MIMO，需要优化 2048 个复数变量——这比 5G Massive MIMO（64 通道）大了 32 倍。

降维策略：

  • 波束域（Beamspace）方法：在傅里叶域优化，自由度降低 8-16 倍
  • 两级波束赋形：模拟域（极化+粗波束）+ 数字域（精波束）
  • 分组优化：将阵列分成子阵，每阵独立优化

  • 1980s：双极化 MIMO 概念提出
  • 2000s：±45° 双极化成为基站天线标准配置

  • 2018-2020年：5G NR 双极化波束赋形标准化

  • 2025年：近场极化波束赋形算法提出
  • 2025-2026年：6G 极化波束赋形进入原型验证

八、技术成熟度

  • 成熟度等级：快速成长期

十、一句话总结

极化波束赋形是在空间维度之外打开的"第二维度"——同样的天线阵列，通过独立控制 V/H 极化波束，可以将容量提升 1.5-2 倍而不增加任何硬件成本。在 6G XL-MIMO 的时代，极化不再是简单的分集手段，而是与空间维度同等重要的波束控制自由度。
