漏波天线（Leaky-Wave Antenna, LWA） Leaky-Wave-Antenna

漏波天线是一种"边走边漏"的开放波导结构——电磁波在其中传播时，每走一步就泄漏一小部分能量辐射出去，所有泄漏波在远场相干叠加形成方向图。最大的亮点是频率扫描：改变工作频率就能自动改变波束指向，无需昂贵的移相器。

类比入口

想象一根水管，管壁上有一排小孔。水从一端注入，沿着管子流的过程中从小孔中喷出。如果水流速度（频率）加快，喷出的水角度就会改变。漏波天线就是这根"漏水的水管"——电磁波在里面传播时不断泄漏辐射，改变频率就等于改变了泄漏的角度。

一、基本原理

1.1 什么是漏波天线？

漏波天线是一种开放结构天线（Open-Structure Antenna），与传统的"封闭结构"（如微带贴片、偶极子）不同，LWA 的辐射体本身就是传输线：

信号输入 ──→ ════════════════════════→ 终端匹配
                  ↑↑↑ ↑↑↑ ↑↑↑ ↑↑↑ ↑↑↑
                  辐射  辐射  辐射  辐射  辐射
                  (沿传播方向不断泄漏)

核心区别：

封闭结构天线开放结构天线（LWA）
能量在辐射体上建立驻波能量沿传输线行波传播
辐射集中在天线表面辐射沿整个传输线分布
频率固定，波束固定频率改变，波束自动扫描
带宽较窄带宽较宽

1.2 色散关系与波束指向

LWA 的辐射方向由色散关系决定：

β(ω) = 传播常数（随频率变化）
    k₀(ω) = 自由空间波数 = ω/c

波束指向角：

sin(θ₀) = β(ω) / k₀(ω)

当频率 f 变化时，β 和 k₀ 都变化 → θ₀ 变化 → 波束扫描。

典型扫描特性：

频率β/k₀sin(θ₀)θ₀
f_low0.8k₀0.853°（后向）
f_mid0.3k₀0.317°（近 broadside）
f_high0.05k₀0.053°（前向）

1.3 向后辐射 vs 向前辐射

LWA 可以分为两种模式：

向后扫描（Backward Radiating）：

β > 0, 相速度 & 群速度反向
    sin(θ₀) = β/k₀ > 0 → θ₀ 从 180° 扫到 90°

低频时波束指向后方，频率升高波束逐渐移向 broadside。

向前扫描（Forward Radiating）：

β > 0, 相速度 & 群速度同向
    sin(θ₀) = β/k₀ > 0 → θ₀ 从 0° 扫到 90°

低频时波束指向前方，频率升高波束逐渐远离 broadside。

1.4 开放阻带（Open Stop Band, OSB）

当 β → 0 时（broadside 方向，θ₀ = 90°），LWA 出现"开放阻带"：

β = 0 → sin(θ₀) = 0 → θ₀ = 90°（broadside）

此时：

  • 波束无限宽（无法形成定向波束）
  • 增益骤降（5-10dB）
  • 输入匹配恶化

抑制 OSB 的方法：

  • 非周期性结构（打破 β = 0 的条件）
  • 弯曲波导（蛇形线）
  • 非对称缝隙
  • 慢波结构（增强 β，避免 β → 0）

二、LWA 的主要类型

2.1 SIW 漏波天线（Substrate Integrated Waveguide LWA）

目前最主流的毫米波 LWA：

┌─────────────────────────────┐
    │ ▒▒ ▒▒ ▒▒ ▒▒ ▒▒ ▒▒ ▒▒ ▒▒    │ ← 横向缝隙（辐射体）
    │ ════════════════════════    │ ← SIW 波导
    │ ●● ●● ●● ●● ●● ●● ●● ●●    │ ← 接地层通孔
    └─────────────────────────────┘

特点：

  • 集成于 PCB，成本低
  • 30-46GHz 工作（5G FR2）
  • 带宽：20-40%
  • 增益：10-18dBi
  • 扫描范围：±30°

2.2 微带漏波天线（Microstrip LWA）

┌──────────────────────┐
    │ ──┬──┬──┬──┬──┬──   │ ← 非对称缝隙
    │   │  │  │  │  │      │
    │ 微带线                │
    │   │  │  │  │  │      │
    │ ──┴──┴──┴──┴──┴──   │ ← 接地层
    └──────────────────────┘

特点：

  • 最简单结构
  • 带宽较窄（5-15%）
  • 增益较低（8-12dBi）
  • 适合低频段

2.3 共面波导漏波天线（CPW LWA）

信号线在顶层，接地在两侧，无底层地平面

特点：

  • 双面辐射
  • 易于集成有源器件
  • 损耗高于 SIW

2.4 光栅耦合漏波天线（Grating-Coupled LWA）

利用周期性光栅结构实现辐射：

周期 Λ → 满足光栅方程：
    sin(θ_m) = m·λ/Λ - β/k₀

特点：

  • 精确控制辐射方向
  • 可用于太赫兹频段
  • 加工精度要求高

2.5 人工表面表面波漏波天线（SSW LWA）

利用人工表面波（Surface Wave on Artificial Dielectric）：

金属柱阵列 → 等效表面波 → 周期性辐射

特点：

  • 超低剖面
  • 适合室内覆盖
  • 扫描范围大（±60°）

2.6 CRLH 漏波天线（复合左右手传输线）

同时支持正向波（Right-Handed）和反向波（Left-Handed）：

CRLH 单元：
      ┌──[C_R]──┬──[L_R]──┐
      │          │          │
      └──[L_L]──┬──[C_L]──┘

特点：

  • 双向扫描（backward + forward）
  • broadside 辐射无 OSB
  • 带宽极宽（> 100%）
  • 设计复杂

3.3 波束纯度

LWA 的旁瓣通常较高（-10 到 -15dB），因为辐射沿传输线均匀分布（类似均匀照射的口径）：

降低旁瓣的方法：

  • 非均匀单元设计（改变每个单元的泄漏率）
  • 振幅加权（渐削照射）
  • 多 LWA 组合

四、与相控阵的对比

特性LWA相控阵
扫描方式频率扫描相位扫描
扫描速度快（调频即可）快（移相器）
波束数量连续（频率连续可调）离散/连续
复杂度极低（无移相器）高（N 个移相器）
成本低高
功耗极低高
增益中等（10-20dBi）高（20-35dBi）
波束宽度较宽较窄
多波束困难容易
极化切换困难容易

核心优势：LWA 是最便宜的电子扫描天线——不需要移相器、不需要 T/R 模块、不需要波束赋形 IC，只需要一个可调频的信号源。

核心劣势：扫描时频率也在变 → 波束和频率耦合（frequency-scanning coupling），不适合需要频率固定的场景。

五、典型应用场景

5.1 5G 基站

  • 28/39GHz SIW LWA
  • 扇区覆盖（一个 LWA 覆盖一个 120° 扇区）
  • 频率扫描替代机械旋转
  • 低成本、低剖面

5.2 车载雷达

  • 77GHz LWA
  • 频率扫描实现方位角覆盖
  • 与毫米波 SoC 集成
  • 成本远低于相控阵

5.3 卫星通信

  • Ka 波段 LWA
  • 轻量、低剖面
  • 频率扫描替代机械转向

5.4 6G 太赫兹通信

  • 0.1-1THz LWA
  • 芯片级集成（SIW/WSIW）
  • 超宽带（> 100GHz 带宽）
  • 波束扫描 + 高速数据传输

六、工程挑战

6.1 开放阻带（OSB）

broadside 方向的 OSB 是 LWA 最大的痛点：

频率 f_OSbroadside → 波束无限宽 → 增益崩溃

抑制方法：

  • 蛇形波导（增加有效长度，避免 β = 0）
  • 非对称结构（打破对称性）
  • CRLH 结构（同时支持 forward/backward）

6.2 频率-波束耦合

LWA 的波束指向依赖于频率，这意味着：

  • 通信时频率变化 → 波束偏移 → 链路中断
  • 需要频率稳定或波束跟踪

解决方案：

  • 窄带通信（频率变化小）
  • 反馈控制（检测波束偏移并补偿）
  • 多 LWA 冗余覆盖

6.3 带宽与增益的权衡

带宽 ↑ → 单元数 ↓ → 增益 ↓

要同时获得宽频带和高增益非常困难。

6.4 加工精度

毫米波 LWA 的缝隙/周期结构精度要求高：

ΔL/L < 1% → Δθ < 1°

对于 30GHz（λ = 10mm），ΔL < 0.1mm。

七、技术演进

  • 1950s：LWA 理论建立（Watson, Oliner）
  • 1970s：周期性漏波结构实用化
  • 1990s：非均匀 LWA（降低旁瓣）
  • 2000s：CRLH LWA 提出
  • 2010s：SIW LWA 成为主流（毫米波）
  • 2020-2024年：太赫兹 LWA、芯片级集成
  • 2025年：拓扑光子学 LWA（72Gbps 芯片级）

八、技术成熟度

  • 成熟度等级：成熟（低频）→ 商用过渡（毫米波）
  • 说明：Sub-6GHz LWA 技术成熟。毫米波 SIW LWA 在 5G 基站和车载雷达中有小规模应用。太赫兹 LWA 处于实验室阶段。

九、一句话总结

漏波天线是天线界的"变色龙"——改变频率就改变波束方向，无需移相器、无需 T/R 模块，是最经济实惠的电子扫描天线。它的核心痛点是开放阻带（OSB）和频率-波束耦合，但在 5G/6G 毫米波场景下，这些缺点往往可以被其低成本和低复杂度所抵消。
