平板天线 Panel Antenna（基站天线） Panel-Antenna

平板天线是基站天线的"标准体型"——长方体外壳，正面辐射，背面安装支架。5G AAU 本质上是"更厚更重的平板天线"，内部集成了天线阵列+T/R模块+散热系统。

类比入口

平板天线就像一块"智能黑板"——正面是辐射面（写板书），背面是电子元件（藏起来），侧面是安装接口。4G 平板天线薄一些（里面只有偶极子+馈电网络），5G 平板天线厚一些（里面塞进了 64 个 T/R 模块和散热片）。

一、平板天线的基本结构

1.1 外形尺寸

4G 平板天线（2T2R）：
      尺寸：1200×220×120mm
      重量：8-12kg
      频段：700MHz-2.6GHz
    
    5G Sub-6GHz 平板天线（64T64R）：
      尺寸：1400×300×180mm
      重量：30-45kg
      频段：600MHz-3.5GHz
    
    5G 毫米波平板天线（64T64R @ 28GHz）：
      尺寸：600×300×80mm
      重量：10-15kg
      频段：24-39GHz

1.2 内部结构

平板天线分层：
      ┌──────────────────────┐ ← 前面板（辐射面）
      │  天线阵列层           │
      │  - 贴片/偶极子单元    │
      │  - 馈电网络           │
      ├──────────────────────┤
      │  T/R 模块层           │
      │  - 64 个 T/R 芯片     │
      │  - 移相器/衰减器      │
      ├──────────────────────┤
      │  散热层               │
      │  - 铝合金散热片       │
      │  - 导热垫             │
      ├──────────────────────┤
      │  控制层               │
      │  - RCU/波控板         │
      │  - 电源管理           │
      └──────────────────────┘ ← 后面板（安装面）

二、平板天线 vs AAU

2.1 4G 平板天线（无源）

结构：
      天线阵列（偶极子/贴片）
      + 馈电网络
      + 反射板
      = 纯无源器件
    
    特点：
      - 无源（不需要供电）
      - 重量轻（8-12kg）
      - 安装简单
      - 波束固定（机械下倾）

2.2 5G AAU（有源平板天线）

结构：
      天线阵列（贴片）
      + T/R 模块（64 通道）
      + 波束赋形 IC
      + 散热系统
      + 控制板
      = 有源设备
    
    特点：
      - 有源（需要供电 200-500W）
      - 重量重（30-60kg）
      - 需要前传光纤
      - 波束可调（电调下倾 + 波束赋形）

四、平板天线类型

4.1 按频段分类

低频平板（700MHz）：
      - 尺寸大（λ = 429mm）
      - 增益低（15-17dBi）
      - 覆盖广
      
    中频平板（1.8-2.6GHz）：
      - 尺寸中等
      - 增益 17-19dBi
      - 容量大
      
    高频平板（3.5GHz）：
      - 尺寸小
      - 增益 20-22dBi
      - 5G 主流频段
      
    毫米波平板（28GHz）：
      - 尺寸很小（λ = 10.7mm）
      - 增益 23-26dBi
      - 波束窄

4.2 按扇区分类

120° 扇区天线（主流）：
      - 每站 3 个扇区
      - 每个扇区 1 块平板
      - 覆盖 360°
      
    30° 窄波束天线（Massive MIMO）：
      - 每站 6-12 个扇区
      - 波束宽度 3-6°
      - 高密度覆盖

五、安装与维护

5.1 安装要求

抱杆直径：≥ 110mm
    抱杆壁厚：≥ 4mm
    安装高度：15-50m
    间距要求：
      - 水平：≥ 0.5m
      - 垂直：≥ 1.0m
      - 与障碍物：≥ 0.3m

5.2 维护要点

定期检查：
      1. 螺栓紧固（每年 1 次）
      2. 防水胶泥（每 2 年更换）
      3. PIM 测试（每半年 1 次）
      4. RET 校准（每年 1 次）
      5. 外观检查（腐蚀/变形）

六、一句话总结

平板天线是基站天线的"标准体型"——4G 是无源的（8-12kg），5G AAU 是有源的（30-60kg）。核心差异在于：5G 平板内部集成了 T/R 模块和波束赋形能力，实现了 3D 波束赋形和电调下倾。记住：5G AAU 的重量是 4G 的 3-5 倍，抱杆和支架必须加固。
