天线测量深度解读 Antenna Measurement

天线测量就是给天线"体检"——用矢量网络分析仪（VNA）测 S 参数（反射/传输），用消声室测方向图和增益，用 OTA 方法测总辐射功率（TRP）。5G 手机没有天线接口了，只能整个放在消声室里用探头"隔空"测量。毫米波频段更变态，VNA 能到 110GHz，但校准精度要求亚微米级。

一、天线测量的核心参数

1.1 S 参数（Scattering Parameters）

用 VNA 直接测量：

参数含义理想值
S11（Return Loss）端口 1 的反射≤ -10dB
S21（Insertion Loss）端口 1→2 的传输取决于设计
S12（Reverse Isolation）端口 2→1 的隔离越大越好
S22端口 2 的反射≤ -10dB

S11 与 VSWR 的转换：

VSWR = (1 + |Γ|)/(1 - |Γ|)
    Γ = S11（线性值，非 dB）

1.2 方向图（Radiation Pattern）

三维方向图：在球坐标系 (θ, φ) 上测量电场幅度。

关键参数：

  • HPBW（半功率波束宽度）
  • SLL（旁瓣电平）
  • FBR（前后比）
  • 零点位置（Null Direction）

1.3 增益（Gain）

测量方法：

  1. 比较法：DUT 与标准增益天线（如偶极子/喇叭）比较
  2. 三点法：已知两个标准天线，求第三个
  3. 增益公式：G = η_total × D

1.4 效率（Efficiency）

  • 辐射效率：Wheeler Cap 法 / 两点法 / 方向图积分法
  • 总效率：OTA-TRP 法（3GPP 标准）

1.5 轴比（Axial Ratio）

圆极化天线专用：用旋转源法（Rotation Source Method）或三点法测量。

二、测量环境

2.1 远场区（Far-Field Region）

Fraunhofer 距离 ：

R_ff ≥ 2D²/λ

其中 D 是天线的最大尺寸，λ 是波长。

典型值：

天线尺寸频率R_ff
0.1m2.4GHz8.3m
0.5m3.5GHz57m
2m28GHz57m
10m（雷达）10GHz2km

关键洞察：5G AAU（2m×1m）在 3.5GHz 需要 57 米 远场距离！大型消声室都未必够长。

2.2 近场区（Near-Field Region）

范围：0.62√(D³/λ) < R < 2D²/λ

特点：

  • 场结构复杂（辐射近场/菲涅尔区）
  • 不能用远场公式
  • 但可以通过傅里叶变换计算远场

2.3 电波暗室（Anechoic Chamber）

结构：

    ╔══════════════════╗
        ║  吸波材料（楔形海绵）║
        ║  ┃  ┃  ┃  ┃  ┃  ┃  ║
        ║  ┃  ┃  ┃  ┃  ┃  ┃  ║
        ║  ┃  ┃  ┃  ┃  ┃  ┃  ║
        ║                  ║
        ║  [DUT]    [Probe] ║
        ║                  ║
        ╚══════════════════╝

吸波材料：

  • 楔形碳海绵：低频（< 1GHz）长度 0.5-1m
  • 铁氧体片：低频吸收
  • 金属尖锥：高频（> 1GHz）

安静区（Quiet Zone）：

  • 直径 1-3m 的球形区域
  • 场均匀性：±0.5dB（Sub-6GHz）/ ±1.0dB（mmWave）

2.4 近场扫描系统（Near-Field Scanner）

原理：在 DUT 附近的平面上扫描探测场分布 → 傅里叶变换 → 远场方向图。

类型：

类型扫描面适用场景
平面近场平面高增益阵列（5G AAU）
圆柱近场柱面中等增益天线
球面近场球面全向天线

优势：

  • 只需几米距离（vs 远场几十米）
  • 精度高（Anritsu 数据：3λ 处计划近场扫描）
  • 数据处理快（FFT 算法）

2.5 缩尺暗室（Compact Range / CATR）

原理：用抛物面反射镜将馈源发出的球面波"变成"平面波。

        ╭─────╮
           ╭╯ 抛物面 ╰╮
          │    反射镜    │
           ╰─────╯
              │
           馈源天线
              │
           [DUT]

优势：

  • 在短距离内模拟远场条件
  • 安静区大（直径可达天线尺寸的 3-5 倍）
  • 适合大型天线（卫星通信、雷达）

2.6 混响室（Reverberation Chamber, RC）

原理：搅拌器（mode stirrer）改变边界条件 → 产生多个独立瑞利衰落样本 → 统计平均得到 TRP/TIS。

3GPP 规定：

  • ≥ 100 个独立样本
  • 可用于 > 3GHz 的 TRP/TRS 测试
  • 需定期与消声室"黄金设备"比对校准

优势：

  • 体积小（3×3×3m 可测到 40GHz）
  • 成本低（vs 消声室）
  • 高通量（适合产线测试）

三、OTA 测量（Over-The-Air）

3.1 为什么需要 OTA？

5G 趋势：

  • 天线集成到封装内（AiP）→ 无射频接口
  • 多天线 MIMO → 接口数量爆炸
  • 波束赋形 → 需要测量实际辐射方向图

3GPP TR 38.810 定义三种 OTA 方法：

  1. DFF（Direct Far-Field）：直接远场法
  2. IFF（Indirect Far-Field）：间接远场法（近场→远场变换）
  3. NFTF（Near-Field to Far-Field Transform）：近场到远场变换

3.2 TRP（Total Radiated Power）

定义：天线在所有方向上的总辐射功率。

TRP = ∫∫ U(θ,φ) sin(θ) dθ dφ

测量方法（CTIA 标准）：

  • 球面采样：每 15° 测一个点（θ 和 φ 轴）
  • 总共 266 个点（每极化）
  • 积分得到 TRP

3GPP TS 38.561 Release 18：

  • 消声室为参考方法
  • RC 为替代方法（> 3GHz 已 harmonized）
  • 需定期用 golden UE 比对

3.3 TIS / TRS（Total Isotropic Sensitivity）

TIS（Transmitter）：全向灵敏度 TRS（Receiver）：全向接收灵敏度

测量：DUT 作为发射机，探天线在所有方向接收 → 积分得到 TRP。 反过来，探天线发射，DUT 在所有方向接收 → 得到 TIS/TRS。

3.4 EIRP（Effective Isotropic Radiated Power）

EIRP(θ,φ) = P_input × G(θ,φ)

3GPP 新增要求：测量方向性 EIRP（而非仅 TRP），用于波束赋形验证。

四、VNA 校准技术

4.1 SOLT 校准（Short-Open-Load-Thru）

标准：短路（S）、开路（O）、负载（L）、直通（T）

特点：

  • 最常用，工业标准
  • 标准件需要精确表征
  • 无频率带宽限制
  • 精度：±0.01dB / ±0.5°（高端 PNA）

4.2 TRL 校准（Thru-Reflect-Line）

标准：直通（T）、反射（R）、线（L）

特点：

  • "自校准"方法，标准件不需完全表征
  • 精度高于 SOLT（多数情况）
  • 线标准限制带宽（~8:1 频率比）
  • 适用于波导/微带等非同轴接口
  • 毫米波/太赫兹首选

4.3 ECal（电子校准）

原理：VNA 内置开关矩阵 + 已知校准件 → 一键校准

特点：

  • 速度快（10 秒完成多端口校准）
  • 可重复性好
  • 精度略低于 TRL（±0.02dB / ±1°）
  • 适合产线测试

4.4 校准方法对比

方法精度速度带宽限制适用场景
SOLT高中无通用
TRL最高慢有（8:1）毫米波/波导
SOLR高中无单侧已知反射
ECal中高最快无产线/多端口

五、5G 毫米波天线测量特殊挑战

5.1 高频 VNA

频率范围：

  • 常规 VNA：Up to 50GHz（Keysight PNA-X）
  • 扩展 VNA：Up to 110GHz（W 频段）
  • 太赫兹：Up to 300GHz（用谐波混频器扩展）

5.2 波导接口

毫米波频段：同轴连接器损耗太大 → 改用波导（WR-15/WR-12）

波导类型：

频段波导型号尺寸
28GHzWR-4210.67×4.32mm
39GHzWR-287.11×3.40mm
75-110GHzWR-102.29×1.02mm

5.3 阵列天线测量

64T AAU 测量挑战：

  • 8×8 或更多端口 → 多端口 VNA（8-16 端口）
  • 波束赋形 → 需要动态扫描多个波束
  • 功耗高 → 需要大功率负载和散热

2026 IMS 进展：Antennex 混响室 EVM+TRP 联合测量，单次测量同时得到效率、失真、邻道泄漏比。

5.4 天线罩影响

测量时需扣除天线罩损耗：

IL_radome = -10×log₁₀(T²)
    T = 穿透率（通常 0.9-0.98）

典型影响：

  • 3mm PBT 罩 @28GHz：IL ≈ 0.1-0.3dB
  • 5mm 特氟龙罩 @28GHz：IL ≈ 0.2-0.5dB

六、测量不确定度

6.1 主要误差来源

误差源典型贡献控制方法
VNA 校准残差0.02dB / 1°高精度 TRL 校准
暗室反射0.1-0.5dB优质吸波材料 + 安静区验证
对准误差0.1-0.3dB精密转台（±0.1°）
电缆/连接器损耗0.05-0.2dB低损耗电缆 + 定期校准
环境扰动0.05-0.1dB温度/湿度控制
近场采样间隔0.1-0.5dBNyquist 采样准则

6.2 NIST 互比对

关键原则：测量不确定度描述必须基于与其他实验室的互比对结果。没有互比对的实验室，其测量结果是不完整的。

七、给非专业读者的一句话总结

天线测量就是给天线做"全身检查"。VNA 是听诊器（测反射和传输），消声室是 CT 机（看辐射方向图），TRP 测试是验血（看总辐射功率）。5G 毫米波时代，检查设备从 GHz 升级到 110GHz，检查精度从毫米级升级到微米级。最有趣的是 OTA 测试——手机没有天线接口了，只能整个泡在"射频消声室"里，用探头隔空测量，就像给人体做 MRI 一样。

金句：测量告诉你一个道理——看不见摸不着的东西，恰恰是最需要被精确量化的。无线电波看不见，但我们可以用数学（傅里叶变换）和物理（VNA 误差模型）把它"抓"出来。
