授时天线 GNSS Timing Antenna

授时天线是基站的"时间老师"——它接收 GPS/北斗/GLONASS/Galileo 卫星信号，为基站提供纳秒级时间同步，是 5G TDD 系统和电力物联网的必备设备。

类比入口

授时天线就像学校的"上课铃"——没有它，每个教室（基站）的时间都不一样，TDD 上下行时隙就会打架。授时天线确保全网基站"听同一个铃声"，同步精度达到纳秒级。

一、授时天线的基本原理

1.1 GNSS 卫星信号

四大卫星导航系统：
      GPS（美国）：L1 = 1575.42MHz
      北斗（中国）：B1 = 1561.098MHz
      GLONASS（俄罗斯）：L1 ≈ 1602MHz（FDMA）
      Galileo（欧洲）：E1 = 1575.42MHz
      
    共同点：
      - L1 频段接近（1561-1602MHz）
      - 信号功率极弱（-125 to -160dBm）
      - 需要高灵敏度天线

1.2 授时天线类型

1. 有源授时天线（主流）
       - 内置 LNA（低噪声放大器）
       - 增益：20-30dB
       - 噪声系数：< 1dB
       - 需要供电（通过同轴电缆）
       
    2. 无源授时天线
       - 无 LNA
       - 灵敏度低
       - 仅用于短距离

1.3 有源天线结构

┌─────────────────────┐
    │  陶瓷贴片（辐射体）   │  ← 右旋圆极化
    ├─────────────────────┤
    │  低噪声放大器（LNA）  │  ← 增益 25dB, NF < 1dB
    ├─────────────────────┤
    │  滤波器（L1 带通）    │  ← 抑制干扰
    ├─────────────────────┤
    │  同轴连接器（TNC/N）  │
    └─────────────────────┘
             │
          同轴电缆（供电+信号）

三、授时天线在基站中的应用

3.1 5G TDD 同步要求

5G TDD 时间同步：
      - 帧同步：< 1.5μs
      - 相位同步：< 0.4μs（±120ns）
      
    授时天线提供：
      - GPS 1PPS（每秒脉冲）→ 帧同步
      - GPS 时间码 → 相位同步
      
    精度：
      GPS 1PPS 精度：< 10ns
      满足 5G 同步要求

3.2 北斗授时

中国运营商要求：
      - 优先使用北斗
      - 支持 GPS+北斗双模
      
    北斗授时天线：
      - B1 频段：1561.098MHz
      - B2 频段：1207.14MHz
      - 支持 BDS-3（第三代北斗）

3.3 安装要求

安装位置：
      - 屋顶/塔顶（开阔天空视野）
      - 避免遮挡（仰角 > 5°）
      - 远离干扰源（雷达、发射天线）
      
    安装角度：
      - 天线面朝上（水平安装）
      - 底部接地（防雷）
      
    电缆长度：
      - 最大 50m（有源天线）
      - 超过 50m 需信号放大

四、授时天线 vs 普通天线

特性授时天线基站平板天线
频段1.5-1.6GHz（GNSS）700MHz-3.5GHz
极化右旋圆极化（RHCP）±45° 线极化
功率接收（-125dBm）发射（43dBm）
增益3-5dBi17-25dBi
尺寸小（100mm）大（1400mm）
安装朝上（看卫星）朝水平（看用户）

五、一句话总结

授时天线是基站的"时间老师"——接收 GPS/北斗卫星信号，提供纳秒级时间同步。有源授时天线内置 LNA（增益 20-30dB，NF < 1dB），通过同轴电缆供电。记住：授时天线必须朝上安装、视野开阔，否则同步精度会大幅下降。
