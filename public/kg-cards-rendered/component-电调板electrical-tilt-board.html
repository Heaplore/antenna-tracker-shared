电调板 Electrical Tilt Board

电调板是 RET 系统的"肌肉"——它接收 RCU 的下倾角指令，通过控制移相器改变各通道相位差，实现波束的电子倾斜。核心组件是移相器驱动电路和角度编码器。

一、电调板架构

电调板组成：
      ┌─────────────────────────────┐
      │  RS485 接口（接收 RCU 指令）  │
      ├─────────────────────────────┤
      │  MCU（STM32/Cortex-M0）      │
      │  - 解析 RET 命令              │
      │  - 控制移相器序列             │
      │  - 读取编码器反馈             │
      ├─────────────────────────────┤
      │  移相器驱动电路               │
      │  - 数字移相器控制（GPIO）     │
      │  - 模拟移相器控制（DAC）      │
      ├─────────────────────────────┤
      │  角度编码器（霍尔/光电）       │
      │  - 实时反馈当前下倾角          │
      ├─────────────────────────────┤
      │  电源管理（5V→3.3V/1.8V）     │
      └─────────────────────────────┘

二、移相器控制

2.1 移相器类型对比

类型控制方式精度损耗响应时间适用场景
数字移相器GPIO + 开关电容5.6°（6-bit）0.5-1dB< 10μsSub-6GHz
模拟移相器DAC + 变容管连续可调1-2dB< 1ms毫米波
MEMS 移相器静电驱动0.01°< 0.5dB1-10ms高端 AAU

2.2 移相器控制算法

下倾角 θ → 各通道相位差 Δφ：
    
    Δφ_n = -2π·n·d·sin(θ)/λ
    
    n = 0, 1, 2, ..., N-1（通道序号）
    d = 单元间距（通常 λ/2）
    λ = 波长
    
    例：8 通道天线，θ = 5°, f = 2.6GHz
      Δφ_0 = 0°
      Δφ_1 = -36.5°
      Δφ_2 = -73.0°
      ...
      Δφ_7 = -255.5°

三、角度编码与反馈

3.1 编码器类型

1. 光电编码器
       - 分辨率：0.01°
       - 寿命：100 万次
       - 抗干扰：好
       
    2. 霍尔编码器
       - 分辨率：0.1°
       - 寿命：50 万次
       - 成本低
       
    3. 电阻式电位器
       - 分辨率：1°
       - 寿命：10 万次
       - 简单但易磨损

3.2 闭环控制

PID 控制算法：
      设定值：θ_set（RCU 下发）
      反馈值：θ_fb（编码器读取）
      误差：e = θ_set - θ_fb
      
      输出：u = Kp·e + Ki·∫e·dt + Kd·de/dt
      
      响应时间：
        数字移相器：~100μs
        模拟移相器：~10ms
        MEMS：~5ms

五、一句话总结

电调板是 RET 的执行层——MCU 解析 RCU 指令，驱动移相器改变相位差，编码器反馈角度形成闭环。数字移相器最快（10μs），MEMS 最准（0.01°）。记住：电调板的精度决定了整个 RET 系统的可靠性，编码器选型是关键。
