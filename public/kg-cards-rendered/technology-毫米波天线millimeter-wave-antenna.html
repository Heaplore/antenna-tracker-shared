毫米波天线 Millimeter-Wave Antenna

毫米波天线工作在 30-300GHz 频段（5G FR2 主要为 24.25-52.6GHz），面临极短的传播距离、严重的自由空间损耗和介质损耗，必须依靠大规模阵列和高增益设计来补偿——这也是为什么毫米波天线几乎总是以 AiP（封装天线）或 PCB 阵列的形式出现。

类比入口

如果把 Sub-6GHz 天线比作一个人在空旷广场上喊话，声音能传很远；毫米波天线就像在同一个广场上用手指轻轻敲击——声音（信号）非常微弱，传不了多远。为了弥补这个差距，你需要让 100 个人同时敲击（大规模阵列），并且用扩音器聚焦方向（高增益波束赋形）。毫米波天线设计的核心就是"人多力量大 + 方向要精准"。

1）大气吸收峰：

23.8 GHz  → H₂O 吸收峰
    60 GHz    → O₂ 吸收峰（最强）
    118.75 GHz → O₂ 吸收峰

2）穿透损耗：

障碍物Sub-6GHz 损耗毫米波损耗
玻璃窗2-3dB5-10dB
树叶1-2dB10-30dB
人体1-2dB15-30dB
墙壁10-20dB> 40dB

3）趋肤效应：

δ_skin = √(2ρ/ωμ)

28GHz 时铜的趋肤深度仅 ~0.35μm，而 3.5GHz 时为 ~1.1μm。这意味着：

  • 导体损耗大幅增加
  • 镀金/镀银层厚度必须 > 3δ ≈ 1μm
  • PCB 表面处理工艺至关重要（ENEIG vs HASL）

二、毫米波天线的主要类型

2.1 微带贴片阵列（Microstrip Patch Array）

最主流的毫米波天线类型：

┌──────────────────────────────┐
    │  ┌──┐ ┌──┐ ┌──┐ ┌──┐       │
    │  │▓▓│ │▓▓│ │▓▓│ │▓▓│  ← 辐射贴片（4×4阵列）
    │  └──┘ └──┘ └──┘ └──┘       │
    │        介质基板              │
    │        [GND]                 │
    └──────────────────────────────┘

特点：

  • 28GHz 增益：15-20dBi（4×4 阵列）
  • 带宽：2-5%（窄带）
  • 剖面薄：1-2mm
  • 可集成于 PCB 或 AiP

馈电方式：

  • 边缘馈电：简单，但损耗大
  • 探针馈电：隔离好，但加工难
  • 耦合馈电：带宽略宽
  • 波导馈电：损耗最低，适合基站

2.2 缝隙天线（Slot Antenna）

在金属接地层上刻蚀缝隙作为辐射体：

GND 层:  ──╫══╫══╫══╫──
               ▒  ▒  ▒  ▒     ← 缝隙（辐射体）
    Dielectric
    Signal:   ─────────────   ← 微带馈线

特点：

  • 双面辐射（上下对称）
  • 易于集成于多层 PCB
  • 带宽优于贴片（5-10%）
  • 增益：10-15dBi（4×4 阵列）

2.3 波导缝隙阵列（Waveguide Slot Array）

毫米波基站天线的主流选择：

┌──────────────────────┐
    │  ═══ ═══ ═══ ═══     │  ← 波导壁上的缝隙
    │  ┌────────────────┐  │
    │  │  波导腔体       │  │
    │  └────────────────┘  │
    │  馈电端口              │
    └──────────────────────┘

特点：

  • 损耗极低（波导导体损耗 < 贴片 PCB 的 1/10）
  • 增益：25-35dBi
  • 带宽：5-10%
  • 体积大、成本高
  • 典型应用：5G 基站 AAU、卫星通信

2.4 漏波天线（Leaky-Wave Antenna）

详见 技术概念-漏波天线Leaky-Wave-Antenna

特点：

  • 频率扫描波束（无需移相器）
  • 高增益（15-25dBi）
  • 宽带（10-40%）
  • 集成度高（SIW 实现）

2.5 封装天线 AiP（Antenna-in-Package）

详见 技术概念-封装天线Antenna-in-Package-AiP

特点：

  • 天线集成于芯片封装
  • 单元尺寸极小（毫米级）
  • 需要 LCP/玻璃基板
  • 5G 手机主流方案

2.6 平面倒 F 天线（PIFA）

  • 常用于毫米波手机内部
  • 尺寸紧凑
  • 带宽窄（1-3%）
  • 增益：3-6dBi

三、毫米波天线的关键设计挑战

3.1 基板损耗

毫米波频段的介质损耗角正切（tanδ）影响巨大：

基板材料ε_rtanδ @ 28GHz适用场景
FR44.40.02-0.03❌ 不适用
Rogers RO30033.00.0013✅ 基站
Rogers RO4350B3.480.0037✅ 通用
LCP3.00.002-0.004✅ 手机 AiP
玻璃5-70.005-0.01✅ 透明天线

导体损耗计算：

α_c = (R_s / (Z_0 · w)) · (l/g)  [dB/cm]

其中 R_s 是表面电阻：

R_s = √(πfμ/σ)

28GHz 时铜的 R_s ≈ 0.027Ω，比 3.5GHz 时的 0.009Ω 高 3 倍。

3.2 互耦加剧

毫米波波长极短（28GHz 时 λ ≈ 10.7mm），单元间距仅需 λ/2 ≈ 5.3mm。密集排列导致：

  • 表面波耦合增强
  • 馈电网络互耦显著
  • 需要精密的去耦设计

3.3 热管理

毫米波功放效率低（10-20%），大量能量转化为热量：

P_heat = P_RF / η_PA - P_RF

对于 100W RF 输出、η_PA = 15%：

P_heat = 100/0.15 - 100 = 567W

散热方案：

  • 金属散热底座（Cu-W, Cu-Mo）
  • 热通孔阵列（Thermal Via Array）
  • 微通道液冷（高端基站）

3.4 加工精度

毫米波天线对加工公差的敏感度极高：

频率λ允许公差（相位误差 < 10°）
3.5GHz86mm±0.5mm
28GHz10.7mm±0.06mm
39GHz7.7mm±0.04mm
77GHz3.9mm±0.02mm

±0.04mm 的公差需要精密 CNC 加工或光刻工艺，PCB 厂的标准公差（±0.1mm）不够用。

3.5 馈电网络损耗

大规模阵列的馈电网络本身就是损耗大户：

总损耗 = 分配损耗 + 导体损耗 + 接头损耗

对于 64 单元阵列：

  • 树状分配：6 级（64 = 2⁶）
  • 每级 Wilkinson 损耗：~0.1dB
  • 总分配损耗：0.6dB
  • 导体损耗（28GHz）：1-3dB
  • 接头/过渡损耗：0.5-1dB
  • 总计：2-5dB

四、毫米波天线阵列设计

4.1 阵列拓扑

Level 0:  输入
              │
         ┌────┴────┐
    Level 1:  1→2    1→2
              │      │
         ┌───┴──┐ ┌──┴───┐
    Level 2: 1→2 1→2  1→2 1→2
              │   │   │   │
         ┌────┴───┴───┴────┐
    Level 3: 8 路输出（每个再分 8 路 → 64 单元）

两种主要拓扑：

拓扑优点缺点
树状（Corporate）设计简单级数多时损耗大
并行（Parallel）损耗低隔离度差
混合（Hybrid）兼顾设计复杂

4.2 波束赋形网络

毫米波天线必须配合波束赋形才能工作：

数字域：BFIC（波束成形IC）
             ↓
    模拟域：移相器 + 功分器
             ↓
    天线阵列（64 单元）

关键参数：

  • 移相器位数：3-8 bit
  • 幅度控制：4-8 bit
  • 波束切换时间：< 1μs
  • 波束指向精度：< 0.5°

4.3 通道校准

毫米波阵列需要定期校准以补偿通道间偏差：

Δphase_per_channel < 5°
    Δgain_per_channel < 0.5dB

校准方法：

  • 内置校准网络（Coupler + Directional Detector）
  • 外部互易校准（利用信道互易性）
  • 数字域自适应校准（基于导频信号）

五、典型应用场景

5.1 5G 基站（gNB）

  • 天线类型：波导缝隙阵列 / 贴片阵列
  • 单元数：64-512（大规模 MIMO）
  • 增益：25-35dBi
  • 波束数：8-64 个独立波束
  • 典型厂商：华为、中兴、爱立信、诺基亚

5.2 5G 手机

  • 天线类型：AiP / PIFA / 贴片
  • 频段：n257/n258/n260/n261
  • 单元数：4-16（受限于机身空间）
  • 增益：3-10dBi
  • 关键挑战：人手遮挡、电池/屏幕干扰

5.3 V2X 车联网

  • 频段：76-81 GHz（4D 成像雷达）
  • 天线类型：SiGe MMIC 集成贴片/缝隙
  • 增益：15-20dBi
  • 波束扫描：电子扫描 ±60°

5.4 卫星通信（Ka 波段）

  • 频段：20-40 GHz
  • 天线类型：抛物面 / 相控阵
  • 增益：35-45dBi
  • 波束指向：机械 + 电子混合

六、技术演进

  • 2010s 初期：毫米波天线研究起步（WiGig 60GHz）
  • 2015-2018年：5G 毫米波标准化（3GPP R14-R15）
  • 2019-2021年：商用部署（Samsung Galaxy S10 5G、Verizon 5G UW）
  • 2022-2024年：LCP/AiP 成熟、SIW 漏波天线普及
  • 2025-2026年：6G FR2-2（52-71GHz）预研、太赫兹天线实验

七、技术成熟度

  • 成熟度等级：大规模商用
  • 说明：28GHz 毫米波天线已大规模商用（5G 基站、手机），39GHz 逐步铺开。77GHz 车载雷达天线也非常成熟。52-71GHz（FR2-2）仍处于预研阶段。

八、一句话总结

毫米波天线是 5G/6G 的"高频武器"——带宽大、天线密度高、波束窄且精准，但传播距离短、损耗大、加工难。它的设计核心是"用阵列换距离、用波束换覆盖"，而封装技术（AiP）和基板材料（LCP/玻璃）是决定成败的关键。
