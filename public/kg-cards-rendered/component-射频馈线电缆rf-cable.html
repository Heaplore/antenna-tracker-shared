射频馈线/馈缆 RF Cable

射频馈线是 AAU 内部和 AAU 与外部天线之间的"信号高速公路"——从同轴电缆到波导再到柔性扁平电缆（FFC），核心指标就一个：损耗越低越好。5G 时代 Sub-6GHz 主流是低损耗同轴电缆（发泡聚乙烯绝缘），毫米波开始用波导和 FFC。

类比入口

馈线就像"水管"——信号是水，水管越长越细、内壁越粗糙，水就漏得越多。5G AAU 里 64 个通道各有自己的"水管"，每根水管的损耗差异不能超过 0.1dB，否则 64 个通道的"水量"就不均衡了，波束指向就会歪。

一、馈线/馈缆分类

1.1 同轴电缆（Coaxial Cable）—— 最主流

结构：
      中心导体（铜/铜包钢）
      ── 绝缘层（PE/发泡 PE/PTFE）
      ── 外导体（铜编织/铜管）
      ── 护套（PVC/LSZH）
    
    类型对比：
    
      1. 硬同轴电缆（Semirigid）
         绝缘：实心 PE
         外径：1/8" (3.2mm) / 1/4" (6.4mm) / 7/16" (11mm)
         损耗：极低
         适用：AAU 内部短距离连接
         特点：不可弯曲，需专用压接端子
         
      2. 半柔性电缆（Semi-flexible）
         绝缘：实心 PE/PTFE
         外径：0.085" (2.16mm) / 0.141" (3.56mm) / 0.242" (6.15mm)
         适用：测试仪器连接线
         特点：可轻微弯曲，多次插拔
         
      3. 柔性同轴电缆（Flexible）
         绝缘：发泡 PE/PTFE
         外径：1/2" (12.7mm) / 7/8" (22.2mm) / 1-5/8" (41mm)
         适用：AAU 到天线阵列的长距离馈线
         特点：可弯曲安装
         
      4. 波纹电缆（Corrugated）
         外导体：不锈钢波纹管
         绝缘：发泡 PE/空气
         外径：1-5/8" (主流基站)
         适用：RRU 到天线（长距离，> 20m）
         特点：弯曲半径小，损耗低，抗风载

1.2 波导（Waveguide）—— 毫米波主流

结构：
      矩形/圆形金属管道
      信号在管道内以电磁波形式传播
      
    尺寸（3.5GHz）：
      WR-284：72.14mm × 34.30mm
      
    尺寸（28GHz）：
      WR-28：7.11mm × 3.40mm
      
    优势：
      - 损耗极低（同轴的 1/5-1/10）
      - 功率容量大
      - 屏蔽性好
      
    劣势：
      - 体积大、重量大
      - 不可弯曲
      - 加工精度要求高
      
    适用场景：
      - 毫米波 AAU 内部馈线
      - 卫星通信
      - 雷达系统

1.3 柔性扁平电缆（FFC/FPC）

结构：
      多层 LCP/PI 薄膜
      + 铜箔蚀刻传输线（带状线/微带线）
      
    优势：
      - 超薄（0.5-2mm）
      - 重量极轻
      - 适合密集布线
      - 阻抗可控（50Ω）
      
    劣势：
      - 功率容量低（< 1W）
      - 长度受限（< 1m）
      
    适用场景：
      - AAU 内部 T/R 模块到天线阵列的短连接
      - 毫米波 AiP 封装内部互联

二、关键指标

2.1 插入损耗（Insertion Loss）

损耗公式（同轴电缆）：
      α = α_conductor + α_dielectric
      
    α_conductor（导体损耗）∝ √f
    α_dielectric（介质损耗）∝ f·Df
      
    典型损耗对比（3.5GHz, 10m 长度）：
    
      电缆类型          损耗(dB/10m)
      ──────────────────────────────
      RG-213 (PE)       ~6.0
      LMR-400 (发泡PE)   ~3.5
      Times HiFlex 1/2"  ~2.8
      Times Superflex 7/8" ~2.0
      LDF5-50A (1-5/8")  ~1.2
      波导 (WR-284)      ~0.3
      
    结论：
      长距离馈线（> 10m）必须用大直径低损耗电缆
      1-5/8" 超柔电缆是 RRU-天线连接的主流

三、5G AAU 中的馈线应用

3.1 Sub-6GHz AAU

AAU 内部馈线架构：
      
      RCU（主控）
        │
        ├─→ 背板（PCB 带状线）
        │     │
        │     ↓
        │   T/R 模块（64 通道）
        │     │
        │     ↓
        │   柔性扁平电缆（FFC）← 模块到天线阵列
        │     │
        │     ↓
        │   天线振子（64 单元）
        
    关键设计：
      - T/R 模块到天线阵列：FFC（< 200mm，损耗 < 0.3dB）
      - 背板传输线：PCB 微带线（RO4350B，损耗 < 0.5dB）
      - 整机馈线总损耗：< 2dB（含连接器）

3.2 RRU 到天线（分离式架构）

传统 4G/5G 分离架构：
      
      RRU（机房）
        │
        └─→ 1-5/8" 低损耗同轴电缆（20-50m）
              │
              ↓
            天线（铁塔）
              
    5G AAU 一体化：
      AAU = RRU + 天线（集成在一起）
      → 无长距离馈线
      → 损耗降低 1-2dB
      → 这是 5G 相比 4G 的核心优势之一

3.3 毫米波 AAU

28GHz AAU 内部：
      
      BFIC
        │
        ├─→ FFC（LCP 基板，50mm）
        │     ↓
        │   波导（100mm）← 低损耗传输
        │     ↓
        │   AiP 天线阵列（256 单元）
        
    挑战：
      - 毫米波馈线损耗急剧增加
      - 1m 普通同轴电缆在 28GHz 损耗 > 20dB
      - 必须用波导或 FFC（< 300mm）

4.2 低 PIM 连接器

5G 基站 PIM 要求：
      双三载波：< -150dBc
      
    连接器 PIM 来源：
      1. 接触面氧化（铜/银氧化）
      2. 螺纹加工精度不足
      3. 密封圈材料非线性
      
    低 PIM 连接器设计：
      - 镀镍+镀金双层（防氧化）
      - 精密螺纹（±0.01mm）
      - 硅胶密封圈（低 PIM 材料）
      - 扭矩扳手安装（精确压力）

五、一句话总结

馈线是 AAU 的"血管"——Sub-6GHz 用同轴电缆（1/2"-7/8"），毫米波用波导或 FFC。5G AAU 一体化设计的核心优势之一就是消除了 RRU 到天线的长距离馈线，节省 1-2dB 损耗。记住：馈线损耗 = 系统噪声系数恶化 + 发射功率下降，每一分损耗都要精打细算。
