交叉极化比 XPD Cross-Polarization Discrimination

XPD 衡量天线辐射波中"期望极化"相对于"交叉极化"的纯净程度——定义为同极化增益与交叉极化增益之比，单位 dB。XPD 越高，极化越纯，极化复用带来的容量增益越大。

类比入口

想象两个人在不同频道说话：你在听 A 频道的广播（同极化），但背景里隐约能听到 B 频道的声音（交叉极化）。XPD 就是"你想听的音量"除以"背景噪音音量"的比值。XPD = 30dB 意味着背景噪音只有你想听的声音的千分之一——干净极了。XPD = 10dB 意味着背景噪音是主信号的三分之一——你会觉得吵。

一、定义与物理意义

1.1 XPD 的数学定义

XPD = G_co-pol / G_cross-pol
    XPD(dB) = 10·log₁₀(G_co-pol / G_cross-pol)
            = G_co-pol(dBi) - G_cross-pol(dBi)

其中：

  • G_co-pol：期望极化方向的增益（如 +45° 线极化）
  • G_cross-pol：正交极化方向的增益（如 -45° 线极化）

关键：两个增益必须在同一方向测量（通常是波束中心 θ = 0°）。

1.2 XPD vs 端口隔离度（Port-to-Port Isolation）

这是最容易混淆的两个概念：

特性XPD（交叉极化比）端口隔离度
测量位置远场辐射方向图端口 S 参数（VNA）
测量对象辐射场的极化纯度端口间的电气泄漏
典型值20-40dB25-45dB
影响因素天线结构对称性、加工精度馈电网络布局、屏蔽
关系XPD 是端口隔离的"辐射表现"端口隔离是 XPD 的"内部基础"

关系：端口隔离度好是 XPD 好的必要条件，但不是充分条件。即使端口隔离 40dB，如果天线结构不对称，远场 XPD 也可能只有 25dB。

1.3 XPD 与极化类型的关系

二、计算公式

2.1 从电场分量计算

E_co = 同极化电场分量
    E_cross = 交叉极化电场分量
    
    XPD = |E_co|² / |E_cross|² = (|E_co| / |E_cross|)²
    XPD(dB) = 20·log₁₀(|E_co| / |E_cross|)

2.2 XPD 与轴比的关系（圆极化）

对于圆极化天线，XPD 与轴比 AR 直接相关：

AR = √(G_RHCP / G_LHCP)
    XPD = G_RHCP / G_LHCP = AR²
    XPD(dB) = 20·log₁₀(AR) = AR(dB)

即：圆极化天线的 XPD 等于其轴比（dB 值相同）。

举例：

  • AR = 1（0dB，理想圆极化）→ XPD = ∞（无穷大）
  • AR = 1.41（3dB）→ XPD = 3dB
  • AR = 3.16（10dB）→ XPD = 10dB

2.3 XPD 随角度变化

XPD 不是常数——它随角度（θ, φ）变化：

XPD(θ, φ) = G_co(θ,φ) / G_cross(θ,φ)

通常：

  • 波束中心 XPD 最高
  • 偏离中心后 XPD 下降
  • 在旁瓣区域 XPD 可能降至 10dB 以下

四、测量方法

4.1 远场方向图测量法（标准方法）

设置：
    - 消声室暗室
    - 发射天线（可切换 ±45° 极化）
    - 接收天线（AUT，固定在转台上）
    - 距离：远场条件 2D²/λ
    
    步骤：
    1. 发射天线设为 +45°，旋转 AUT 测量 E_+45(θ,φ)
    2. 发射天线设为 -45°，旋转 AUT 测量 E_-45(θ,φ)
    3. 在波束中心方向：
       G_co = 最大同极化增益
       G_cross = 正交极化增益
    4. XPD = G_co - G_cross [dB]

4.2 交叉极化隔离度法（双天线法）

设置：
    - 参考天线 A：+45° 极化（发射）
    - AUT：+45° 极化（接收）
    
    步骤：
    1. 测量 S21_co：A(+45°) → AUT(+45°)
    2. 交换极化：A(-45°) → AUT(+45°)
    3. 测量 S21_cross：A(-45°) → AUT(+45°)
    4. XPD = S21_co - S21_cross [dB]

4.3 旋转极化参考天线法

设置：
    - AUT：固定极化（如 +45°）
    - 参考天线：可旋转极化的线极化天线
    
    步骤：
    1. 旋转参考天线，找到功率最大位置（极化匹配）
    2. 旋转 90°，找到功率最小位置（极化正交）
    3. XPD = P_max - P_min [dB]

4.4 近场扫描法（高精度）

设置：
    - 近场扫描系统（平面/圆柱面扫描）
    - 二维探头（同时采集 co-pol 和 cross-pol）
    
    步骤：
    1. 近场扫描获取完整电磁场分布
    2. 傅里叶变换到远场
    3. 直接提取 co-pol 和 cross-pol 方向图
    4. 计算 XPD(θ,φ) 全空间分布
    
    优点：无需远场暗室，精度高
    缺点：设备复杂，数据处理量大

六、工程注意事项

6.1 加工精度对 XPD 的影响

天线结构的不对称性是 XPD 恶化的头号杀手：

尺寸偏差 Δ → XPD 下降 ≈ 20·log₁₀(1 + Δ/λ)
频段λ允许偏差预期 XPD
3.5GHz86mm±0.5mm> 35dB
28GHz10.7mm±0.1mm> 25dB
60GHz5mm±0.05mm> 20dB

6.2 安装对准对 XPD 的影响

基站天线安装时如果极化角偏差 α：

XPD_effective ≈ XPD_intrinsic - 20·log₁₀(tan(45° + α))

偏差 1° → XPD 下降约 0.17dB 偏差 3° → XPD 下降约 0.53dB 偏差 5° → XPD 下降约 0.9dB

6.3 温度对 XPD 的影响

温度变化导致结构膨胀/收缩，改变对称性：

ΔXPD/ΔT ≈ -0.02 to -0.1 dB/°C

对于室外基站（温差 -30°C 到 +60°C），XPD 可能下降 2-8dB。

6.4 多径环境中的 XPD 恶化

实际传播环境中，多径反射会改变极化状态：

直射波：XPD = 30dB（天线固有）
    多径叠加后：XPD_effective = 10-20dB（信道效应）

这就是为什么 MIMO 在 rich scattering 环境中性能更好——多径虽然"污染"了极化，但也增加了信道独立性。

七、一句话总结

XPD 是天线的"极化洁癖指数"——它告诉你天线辐射的波有多"纯"。XPD = 30dB 意味着交叉极化污染只有主极化的千分之一。在 5G 极化分集和 6G 极化复用中，XPD 直接决定了你能从极化维度榨出多少额外容量。记住：XPD 和端口隔离度是两回事——一个看远场辐射，一个看端口泄漏，但好 XPD 一定需要好端口隔离打底。
