Doherty-PA-DPD预失真
天线技术/技术概念Doherty-PADPD预失真PA线性化5G基站GaN
一句话: Doherty PA 是 5G 基站 PA 的"效率之王"——通过主从两级结构，在 6-8dB 的回退范围内保持 50-60% 的 PAE（比传统 Class AB 高 20-30%）。DPD（数字预失真）是 Doherty 的"拍档"——没有 DPD，Doherty 的线性度根本达不到 5G NR 的要求。
类比: Doherty PA 就像"双人自行车"——低速时只有一个人蹬（主通道），高速时两个人一起蹬（主通道+辅助通道），这样在低速时也不浪费体力。
1.1 为什么需要 Doherty markdown
5G NR 信号特性：
  峰均比（PAPR）：8-12dB（4x4 MIMO）
  平均功率：比峰值低 6-8dB
  
传统 Class AB PA：
  峰值效率：~50%
  平均效率（回退 6dB）：~15%
  → 功耗大、散热难
  
Doherty PA：
  峰值效率：~50%
  平均效率（回退 6dB）：~40-50%
  → 效率提升 2-3 倍
1.2 Doherty 架构 markdown
Doherty PA 结构：
  
  输入信号
    │
    ├─→ 功率分路器（1:√2:1）
    │     │         │
    │     ↓         ↓
    │   主通道     辅助通道
    │   (Carrier)  (Peaker)
    │     │         │
    │     ↓         ↓
    │   Class AB   Class C
    │     │         │
    │     ↓         ↓
    │   合成器（λ/4 传输线）
    │     │
    │     ↓
    │   输出信号
  
关键：辅助通道通过 λ/4 传输线连接到主通道
  低功率时：辅助通道关闭，主通道独立工作
  高功率时：辅助通道导通，两通道并联工作
1.3 工作原理 markdown
阶段 1：低功率（P < P_th）
  - 辅助通道关闭（Class C 偏置在截止区）
  - 主通道独立工作（Class AB）
  - 效率：~30-40%
  
阶段 2：中等功率（P_th < P < P_sat）
  - 辅助通道开始导通
  - λ/4 传输线实现阻抗变换
  - 主通道看到的负载阻抗下降
  - 效率：~40-50%
  
阶段 3：峰值功率（P = P_sat）
  - 两通道都饱和
  - 效率：~50-60%
  
效率提升原理：
  辅助通道"借用"主通道的负载阻抗
  使主通道在高回退时仍保持高效率
2.1 关键指标
指标典型值说明
PAE（峰值） 50-60% 功率附加效率
PAE（6dB 回退） 40-50% 5G NR 典型工作点
增益 15-25dB 小信号增益
带宽 100-400MHz 5G NR 信道带宽
线性度（ACLR） <-45dBc 需要 DPD 辅助
工作频段 2.6/3.5/6.5GHz Sub-6GHz

工艺PAE成本适用场景

3.1 为什么需要 DPD markdown
Doherty PA 的非线性来源：
  1. Class AB 主通道：AM-AM/AM-PM
  2. Class C 辅助通道：开关非线性
  3. λ/4 合成器：频率依赖性
  4. 温度漂移：效率随温度变化
  
5G NR 线性度要求：
  ACLR（Adjacent Channel Leakage Ratio）：<-45dBc（30MHz BW）
  EVM（Error Vector Magnitude）：<3.5%（256QAM）
  
没有 DPD 的 Doherty PA：
  ACLR：<-30dBc（不达标）
  EVM：>8%（不达标）
  
有 DPD 的 Doherty PA：
  ACLR：<-50dBc（达标）
  EVM：<2.5%（达标）
3.2 DPD 架构 markdown
DPD 系统：
  
  输入信号 → DPD 预失真器 → PA → 输出信号
                          ↓
                      采样电路
                          ↓
                      误差计算器
                          ↓
                  自适应算法（RLS/LMS）
                          ↓
                  更新预失真器系数
3.3 DPD 算法 markdown
主流 DPD 模型：
  1. Volterra 级数
     - 通用非线性模型
     - 计算量大
     
  2. 多项式模型
     - AM-AM + AM-PM
     - 计算量中等
     
  3. 查找表（LUT）
     - 直接存储预失真值
     - 计算量小，实时性好
     
  4. 神经网络 DPD
     - 深度学习模型
     - 适应性强
     - 5G-A/6G 趋势
3.4 DPD 实现位置 markdown
DPD 部署位置：
  1. 基带 DPD（BBU/DU 内）
     - 数字域预失真
     - 灵活性高
     - 5G AAU 主流方案
     
  2. 射频 DPD（AAU 内）
     - 模拟/混合预失真
     - 延迟小
     - 适合高频段
     
  3. 分布式 DPD
     - BBU 粗 DPD + AAU 细 DPD
     - 前传带宽优化
4.1 Sub-6GHz AAU markdown
典型配置：
  频段：3.5GHz（n78）
  通道：64T64R
  PA：GaN Doherty × 64
  DPD：数字 DPD（每通道独立）
  带宽：100MHz（单载波）
  功率：200-300W（总输出）
  
效率：
  峰值 PAE：~55%
  平均 PAE（6dB 回退）：~40%
  对比传统 Class AB：节能 30-40%
4.2 U6G AAU（6.5GHz） markdown
典型配置：
  频段：6.5GHz（n104）
  通道：64T64R/192T192R
  PA：GaN Doherty × 64/192
  DPD：数字 DPD
  带宽：400MHz（单载波）
  功率：100-200W（总输出）
  
挑战：
  - 6.5GHz GaN 成本高
  - DPD 带宽需求大（400MHz）
  - 散热更严峻
