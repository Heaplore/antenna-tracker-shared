Doherty-PA-DPDйў„еӨұзңҹ
еӨ©зәҝжҠҖжңҜ/жҠҖжңҜжҰӮеҝөDoherty-PADPDйў„еӨұзңҹPAзәҝжҖ§еҢ–5Gеҹәз«ҷGaN
дёҖеҸҘиҜқ: Doherty PA жҳҜ 5G еҹәз«ҷ PA зҡ„"ж•ҲзҺҮд№ӢзҺӢ"вҖ”вҖ”йҖҡиҝҮдё»д»ҺдёӨзә§з»“жһ„пјҢеңЁ 6-8dB зҡ„еӣһйҖҖиҢғеӣҙеҶ…дҝқжҢҒ 50-60% зҡ„ PAEпјҲжҜ”дј з»ҹ Class AB й«ҳ 20-30%пјүгҖӮDPDпјҲж•°еӯ—йў„еӨұзңҹпјүжҳҜ Doherty зҡ„"жӢҚжЎЈ"вҖ”вҖ”жІЎжңү DPDпјҢDoherty зҡ„зәҝжҖ§еәҰж №жң¬иҫҫдёҚеҲ° 5G NR зҡ„иҰҒжұӮгҖӮ
зұ»жҜ”: Doherty PA е°ұеғҸ"еҸҢдәәиҮӘиЎҢиҪҰ"вҖ”вҖ”дҪҺйҖҹж—¶еҸӘжңүдёҖдёӘдәәи№¬пјҲдё»йҖҡйҒ“пјүпјҢй«ҳйҖҹж—¶дёӨдёӘдәәдёҖиө·и№¬пјҲдё»йҖҡйҒ“+иҫ…еҠ©йҖҡйҒ“пјүпјҢиҝҷж ·еңЁдҪҺйҖҹж—¶д№ҹдёҚжөӘиҙ№дҪ“еҠӣгҖӮ
1.1 дёәд»Җд№ҲйңҖиҰҒ Doherty markdown
5G NR дҝЎеҸ·зү№жҖ§пјҡ
  еі°еқҮжҜ”пјҲPAPRпјүпјҡ8-12dBпјҲ4x4 MIMOпјү
  е№іеқҮеҠҹзҺҮпјҡжҜ”еі°еҖјдҪҺ 6-8dB
  
дј з»ҹ Class AB PAпјҡ
  еі°еҖјж•ҲзҺҮпјҡ~50%
  е№іеқҮж•ҲзҺҮпјҲеӣһйҖҖ 6dBпјүпјҡ~15%
  вҶ’ еҠҹиҖ—еӨ§гҖҒж•Јзғӯйҡҫ
  
Doherty PAпјҡ
  еі°еҖјж•ҲзҺҮпјҡ~50%
  е№іеқҮж•ҲзҺҮпјҲеӣһйҖҖ 6dBпјүпјҡ~40-50%
  вҶ’ ж•ҲзҺҮжҸҗеҚҮ 2-3 еҖҚ

  - дё»йҖҡйҒ“зӢ¬з«Ӣе·ҘдҪңпјҲClass ABпјү
  - ж•ҲзҺҮпјҡ~30-40%
  
йҳ¶ж®ө 2пјҡдёӯзӯүеҠҹзҺҮпјҲP_th < P < P_satпјү
  - иҫ…еҠ©йҖҡйҒ“ејҖе§ӢеҜјйҖҡ
  - О»/4 дј иҫ“зәҝе®һзҺ°йҳ»жҠ—еҸҳжҚў
  - дё»йҖҡйҒ“зңӢеҲ°зҡ„иҙҹиҪҪйҳ»жҠ—дёӢйҷҚ
  - ж•ҲзҺҮпјҡ~40-50%

  - дёӨйҖҡйҒ“йғҪйҘұе’Ң

3.1 дёәд»Җд№ҲйңҖиҰҒ DPD markdown
Doherty PA зҡ„йқһзәҝжҖ§жқҘжәҗпјҡ
  1. Class AB дё»йҖҡйҒ“пјҡAM-AM/AM-PM
  2. Class C иҫ…еҠ©йҖҡйҒ“пјҡејҖе…ійқһзәҝжҖ§
  3. О»/4 еҗҲжҲҗеҷЁпјҡйў‘зҺҮдҫқиө–жҖ§
  4. жё©еәҰжјӮз§»пјҡж•ҲзҺҮйҡҸжё©еәҰеҸҳеҢ–
  
5G NR зәҝжҖ§еәҰиҰҒжұӮпјҡ
  ACLRпјҲAdjacent Channel Leakage Ratioпјүпјҡ<-45dBcпјҲ30MHz BWпјү
  EVMпјҲError Vector Magnitudeпјүпјҡ<3.5%пјҲ256QAMпјү
  
жІЎжңү DPD зҡ„ Doherty PAпјҡ
  ACLRпјҡ<-30dBcпјҲдёҚиҫҫж Үпјү
  EVMпјҡ>8%пјҲдёҚиҫҫж Үпјү
  
жңү DPD зҡ„ Doherty PAпјҡ
  ACLRпјҡ<-50dBcпјҲиҫҫж Үпјү
  EVMпјҡ<2.5%пјҲиҫҫж Үпјү
