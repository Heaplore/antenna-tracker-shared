毫米波天线Millimeter-Wave-Antenna
天线技术/技术概念毫米波天线Millimeter-WaveAntenna28GHz39GHz5GFR2基板损耗
一句话: 毫米波天线工作在 30-300GHz 频段（5G FR2 主要为 24.25-52.6GHz），面临极短的传播距离、严重的自由空间损耗和介质损耗，必须依靠大规模阵列和高增益设计来补偿——这也是为什么毫米波天线几乎总是以 AiP（封装天线）或 PCB 阵列的形式出现。
类比: 如果把 Sub-6GHz 天线比作一个人在空旷广场上喊话，声音能传很远；毫米波天线就像在同一个广场上用手指轻轻敲击——声音（信号）非常微弱，传不了多远。

2.1 微带贴片阵列（Microstrip Patch Array）
内容
最主流的毫米波天线类型：
`
│ ┌──┐ ┌──┐ ┌──┐ ┌──┐ │
│ │▓▓│ │▓▓│ │▓▓│ │▓▓│ ← 辐射贴片（4×4阵列）
│ └──┘ └──┘ └──┘ └──┘ │
│ 介质基板 │
│ [GND] │
`
特点：
28GHz 增益：15-20dBi（4×4 阵列）
带宽：2-5%（窄带）
剖面薄：1-2mm
可集成于 PCB 或 AiP
馈电方式：
边缘馈电：简单，但损耗大
探针馈电：隔离好，但加工难
耦合馈电：带宽略宽
波导馈电：损耗最低，适合基站
2.2 缝隙天线（Slot Antenna）
2.2 缝隙天线（Slot Antenna）
在金属接地层上刻蚀缝隙作为辐射体： ` GND 层: ──╫══╫══╫══╫── ▒ ▒ ▒ ▒ ← 缝隙（辐射体） Dielectric Signal: ───────────── ← 微带馈线 ` 特点： 双面辐射（上下对称） 易于集成于多层 PCB 带宽优于贴片（5-10%） 增益：10-15dBi（4×4 阵列）
2.3 波导缝隙阵列（Waveguide Slot Array）
内容
毫米波基站天线的主流选择：
`
│ ═══ ═══ ═══ ═══ │ ← 波导壁上的缝隙
│ ┌────────────────┐ │
│ │ 波导腔体 │ │
│ └────────────────┘ │
│ 馈电端口 │
`
特点：
损耗极低（波导导体损耗 < 贴片 PCB 的 1/10）
增益：25-35dBi
带宽：5-10%
体积大、成本高
典型应用：5G 基站 AAU、卫星通信
2.4 漏波天线（Leaky-Wave Antenna）
2.4 漏波天线（Leaky-Wave Antenna）
详见 [[技术概念-漏波天线Leaky-Wave-Antenna]] 特点： 频率扫描波束（无需移相器） 高增益（15-25dBi） 宽带（10-40%） 集成度高（SIW 实现）
2.5 封装天线 AiP（Antenna-in-Package）
2.5 封装天线 AiP（Antenna-in-Package）
详见 [[技术概念-封装天线Antenna-in-Package-AiP]] 特点： 天线集成于芯片封装 单元尺寸极小（毫米级） 需要 LCP/玻璃基板 5G 手机主流方案
2.6 平面倒 F 天线（PIFA）
2.6 平面倒 F 天线（PIFA）
常用于毫米波手机内部 尺寸紧凑 带宽窄（1-3%） 增益：3-6dBi

导体损耗计算： ` α_c = (R_s / (Z_0 · w)) · (l/g) [dB/cm] ` 其中 R_s 是表面电阻： ` R_s = √(πfμ/σ) ` 28GHz 时铜的 R_s ≈ 0.027Ω，比 3.5GHz 时的 0.009Ω 高 3 倍。
3.2 互耦加剧
3.2 互耦加剧
毫米波波长极短（28GHz 时 λ ≈ 10.7mm），单元间距仅需 λ/2 ≈ 5.3mm。密集排列导致： 表面波耦合增强 馈电网络互耦显著 需要精密的去耦设计
3.3 热管理
3.3 热管理
毫米波功放效率低（10-20%），大量能量转化为热量： ` P_heat = P_RF / η_PA - P_RF ` 对于 100W RF 输出、η_PA = 15%： ` P_heat = 100/0.15 - 100 = 567W ` 散热方案： 金属散热底座（Cu-W, Cu-Mo） 热通孔阵列（Thermal Via Array） 微通道液冷（高端基站）
3.4 加工精度
频率λ允许公差（相位误差 < 10°）
3.5GHz 86mm ±0.5mm
28GHz 10.7mm ±0.06mm
39GHz 7.7mm ±0.04mm
77GHz 3.9mm ±0.02mm
±0.04mm 的公差需要精密 CNC 加工或光刻工艺，PCB 厂的标准公差（±0.1mm）不够用。
3.5 馈电网络损耗
3.5 馈电网络损耗
大规模阵列的馈电网络本身就是损耗大户： ` 总损耗 = 分配损耗 + 导体损耗 + 接头损耗 ` 对于 64 单元阵列： 树状分配：6 级（64 = 2⁶） 每级 Wilkinson 损耗：~0.1dB 总分配损耗：0.6dB 导体损耗（28GHz）：1-3dB 接头/过渡损耗：0.5-1dB 总计：2-5dB

4.2 波束赋形网络
4.2 波束赋形网络
毫米波天线必须配合波束赋形才能工作： ` 数字域：BFIC（波束成形IC） ↓ 模拟域：移相器 + 功分器 ↓ 天线阵列（64 单元） ` 关键参数： 移相器位数：3-8 bit 幅度控制：4-8 bit 波束切换时间：< 1μs 波束指向精度：< 0.5°
4.3 通道校准
4.3 通道校准
毫米波阵列需要定期校准以补偿通道间偏差： ` Δphase_per_channel < 5° Δgain_per_channel < 0.5dB ` 校准方法： 内置校准网络（Coupler + Directional Detector） 外部互易校准（利用信道互易性） 数字域自适应校准（基于导频信号）

5.1 5G 基站（gNB）

5.2 5G 手机

5.3 V2X 车联网
