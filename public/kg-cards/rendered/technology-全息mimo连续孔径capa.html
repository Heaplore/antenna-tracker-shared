全息MIMO连续孔径CAPA
天线技术/技术概念全息MIMO连续孔径CAPA6G电磁AI
一句话: 全息 MIMO（Holographic MIMO）和连续孔径阵列（CAPA）是 6G 的终极天线形态——它用一块连续的、可编程的电磁表面替代了离散的天线单元阵列，实现了理论上无限多的空间自由度。这不是"更多的天线"，而是"没有天线"的天线：电磁波直接在连续表面上被编程和控制。
类比: 传统天线阵列像钢琴——有固定的黑白键（天线单元），你能弹的音符是有限的。
1.1 从离散到连续：天线范式的根本转变
维度 离散阵列（SPDA） 连续孔径（CAPA）
辐射体 离散的天线单元（N 个） 连续的电磁表面
自由度 N 个（有限） 无限个（理论）
波束控制 通过离散单元的幅相加权 通过表面电流分布 J(x,y)
方向图 AF(θ) = Σ I_n · exp(j·n·ψ) F(θ) = ∬ J(x,y) · exp(j·k·r) dx dy
物理实现 天线单元 + 移相器 超表面 / RIS / 可编程金属表面
1.2 数学本质：从傅里叶级数到傅里叶变换

离散阵列的方向图是傅里叶级数：

AF(θ) = Σ(n=-N/2 to N/2) [I_n · exp(j·n·k·d·sinθ)]

连续孔径的方向图是傅里叶变换：

F(k_x, k_y) = ∬∬ J(x,y) · exp[j·(k_x·x + k_y·y)] dx dy

其中 J(x,y) 是连续表面上的电流分布，k_x = k·sinθ·cosφ，k_y = k·sinθ·sinφ。

关键洞察：傅里叶变换有无限多的频率分量，而傅里叶级数只有 N 个。这意味着 CAPA 可以形成无限多的独立波束（理论上）。

1.3 全息原理

全息 MIMO 的名称来源于光学全息技术——全息图记录的是光波的完整振幅和相位信息，而不是简单的强度分布。

在天线领域，"全息"意味着：
- 表面不仅控制辐射强度
- 还精确控制每个点的辐射相位
- 通过干涉和衍射原理，在空间中"重建"出任意波前

2.1 CAPA 的信道模型

传统 MIMO 信道矩阵 H 是 N_t × N_r 的离散矩阵。CAPA 的信道是连续信道函数 h(x,y; x',y')：

y(x',y') = ∬ h(x,y; x',y') · J(x,y) dx dy + n

其中：
- J(x,y) 是发射表面的电流分布
- h(x,y; x',y') 是两点之间的信道传递函数
- y(x',y') 是接收表面的感应电流

2.2 CAPA 的容量上限

传统 MIMO 的容量：

C = log₂ det(I + (ρ/N_t) · H · Hᴴ)

CAPA 的容量是上述公式的连续极限：

C_CAPA = lim(N→∞) C_MIMO(N)

研究表明，在相同面积和功率下，CAPA 的容量可以比最优离散 MIMO 高 2-5 倍，原因是：
- 无限自由度可以精确匹配信道特征函数
- 没有单元间距限制（无栅瓣）
- 近场波前可以精确建模

2.3 波束赋形的电磁优化

CAPA 的波束赋形不是简单的向量乘法，而是电磁场的优化问题：

maximize    接收功率 = ||∬ J(x,y) · h(x,y) dx dy||²
subject to  ∬ |J(x,y)|² dx dy ≤ P_max
            J(x,y) ∈ 可实现集合

可实现集合的约束取决于物理实现：
- 超表面：|J(x,y)| = 常数，相位可调（0-2π）
- 可编程金属：|J(x,y)| 和相位都可调
- RIS：|J(x,y)| = 0 或 1（被动反射）

2.4 模式分割复用（Pattern-Division Multiplexing, PDM）

CAPA 的核心创新——不再用空间位置或极化来区分用户，而是用波束形状来区分：

用户 1: J₁(x,y) → 波束形状 F₁(θ,φ)
用户 2: J₂(x,y) → 波束形状 F₂(θ,φ)
...
用户 K: J_K(x,y) → 波束形状 F_K(θ,φ)

只要 {F₁, F₂, ..., F_K} 在接收端正交，就可以在同一频率、同一时间、同一空间位置服务 K 个用户。

关键公式（正交条件）：

∬ F_i(θ,φ) · F_j*(θ,φ) dΩ = 0,  i ≠ j
2.5 全息波束赋形设计

给定目标波前 W_target(θ,φ)，求解表面电流分布 J(x,y)：

J(x,y) = F⁻¹[W_target(k_x, k_y)]

其中 F⁻¹ 是逆傅里叶变换。这就是"全息"的含义——通过逆傅里叶变换"重建"出产生目标波前的表面电流分布。

4.1 6G 基站
  • 用一块 2m × 2m 的超表面替代传统 AAU
  • 同时服务数十个用户（每个用户一个独立波束）
  • 波束形状可动态调整（适应不同场景）
4.2 室内覆盖
  • 天花板安装可编程电磁表面
  • 自动跟踪用户位置，形成"跟随波束"
  • 无需传统 AP/RRU
4.3 ISAC（通感一体化）
  • 同一块超表面同时完成通信和感知
  • 通信波束和感知波束在电磁域联合优化
  • CAPA 的无限自由度可以最大化 ISAC 的 Pareto 最优区域
4.4 卫星通信
  • 轻量化、超薄型的 CAPA 天线
  • 可展开的柔性连续孔径（FCAPA）
  • 在轨可重构波束
4.5 智能反射面（RIS）增强覆盖
  • 在信号盲区放置 RIS 表面
  • 动态调整反射相位，将信号"拐"到盲区
  • 低成本、低功耗

6.2 控制复杂度

N 个离散单元的波束赋形需要 N 个复数控制信号。CAPA 有"无限"个控制点——如何控制？

实际方案：
- 将连续表面离散化为 M 个控制单元（M >> N）
- 使用压缩感知/稀疏优化减少控制信号数量
- AI 驱动的控制：神经网络直接从目标波前映射到控制信号

6.3 互耦效应

在 CAPA 中，相邻单元之间的互耦不再是小扰动，而是主导因素：

  • 每个单元的响应都受到所有其他单元的影响
  • 传统的"独立单元 + 阵列因子"模型失效
  • 需要全电磁仿真 + 互耦矩阵补偿
6.4 校准

CAPA 的校准比离散阵列复杂得多：

  • 离散阵列：校准 N 个通道的幅相
  • CAPA：校准 M 个单元（M >> N）的幅相
  • 校准时间：M 次测量 vs N 次测量

对策：
- 分组校准（将 M 个单元分成若干组）
- 在线自校准（利用通信信号）
- AI 辅助校准（学习校准映射）

6.5 功耗

虽然 CAPA 本身（被动超表面）功耗极低，但其控制系统（FPGA/ASIC）的功耗可能不低：

  • M 个单元的实时相位控制需要 M 个 DAC
  • M = 1000 时，DAC 功耗可能超过传统 Massive MIMO

对策：
- 事件驱动控制（只激活活跃的单元）
- 低功耗 MEMS 移相器
- 模拟域波束赋形（避免数字域的大规模运算）
