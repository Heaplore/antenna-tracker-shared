3GPP TR38.901信道模型
天线技术/技术概念3GPP信道模型TR38.901近场信道6G信道DTC数字孪生信道
一句话: 3GPP TR 38.901 是 5G NR 信道建模的标准——定义了从 0.5-100GHz 的路径损耗、时延扩展、RMS 角度扩展、到达/离开角度分布。Rel-17 新增了"近场信道模型"（平面波模型），为 XL-MIMO/CAPA 做准备。
类比: 信道模型就像"天气预报模型"——根据历史数据和物理规律，预测信号在空间中的"天气"。
1.1 适用范围
3GPP TR 38.901 V17.0.0（2022）：

频率范围：0.5 - 100 GHz
场景：
  - UMi（Urban Microcell，城市微蜂窝）
  - UMa（Urban Macrocell，城市宏蜂窝）
  - Rural（乡村）
  - Indoor（室内）
  - 室外到室内（O2I）

天线模型：
  - 远场（球面波 SWP）
  - 近场（平面波 PW，Rel-17 新增）
1.2 路径损耗模型（PL）
总路径损耗 = PL_BSD + PL_IN + PL_O2I

1. BSD（Base Station to Device）路径损耗：

   LOS（视距）：
    PL_BSD_LOS = max(PL_FS, PL_BSD_LOS_min) + X

  NLOS（非视距）：
    PL_BSD_NLOS = max(PL_FS + L_OsNLoS, PL_BSD_NLOS_min) + X

  其中：
    PL_FS = 20log₁₀(4πd·f/c)  （自由空间损耗）
    d = 传输距离（m）
    f = 频率（GHz）
    X = 阴影衰落（高斯分布）

  LOS/NLOS 判定：
    根据 BS 和 UE 高度、距离、城市环境
    → 计算 LOS 概率

2. IN（Indoor）路径损耗：
   - 墙壁穿透损耗：15-25dB（混凝土）
   - 楼层损耗：10-20dB/层
   - 室内传播损耗：30-40dB/10m

典型值对比（d=100m, f=3.5GHz）：
  ┌─────────────────────────────────┐
  │ 场景    │ PL(dB) │ 备注         │
  ├─────────────────────────────────┤
  │ 自由空间 │ 80     │ 理论下限     │
  │ UMi LOS │ 88     │ 城市视距     │
  │ UMi NLOS│ 112    │ 城市非视距   │
  │ UMa LOS │ 85     │ 宏站视距     │
  │ UMa NLOS│ 118    │ 宏站非视距   │
  └─────────────────────────────────┘
1.3 时空信道参数（TSCS）
38.901 定义的完整信道参数：

1. 时延扩展（Delay Spread）：
   RMS 时延扩展：0.5-15ns（城市）

2. 角度扩展（Angular Spread）：
   - Azimuth Angle Spread（AAS）：水平角度扩展
   - Elevation Angle Spread（EES）：垂直角度扩展

3. 到达/离开角度（AoA/AoD）：
   - 水平 AoA/AoD：均匀分布 + 主瓣
   - 垂直 AoA/AoD：高斯分布

4. 多普勒扩展（Doppler Spread）：
   - 静止：0Hz
   - 步行（3km/h）：~100Hz @3.5GHz
   - 驾车（60km/h）：~200Hz @3.5GHz
   - 高铁（300km/h）：~1000Hz @3.5GHz
2.1 为什么需要近场模型？
传统 38.901（Rel-15/16）：
  假设球面波（Spherical Wave, SWP）
  → 适用于远场（d > 2D²/λ）
  → D = 天线口径，λ = 波长

6G XL-MIMO / CAPA：
  天线口径极大（D = 10-100m）
  → 近场区域延伸到 km 级！
  → 球面波假设不准确

解决方案：
  平面波（Plane Wave, PW）分解模型
  → 近场用多个平面波叠加
  → 更准确地描述近场传播
2.2 近场信道特征
近场 vs 远场：

  远场：
    - 波前 = 球面（近似平面）
    - 角度与距离解耦
    - 一个 AoA 对应一个方向

  近场：
    - 波前 = 明显球面
    - 角度与距离耦合
    - 同一个 AoA 在不同距离有不同的相位分布
    → 需要同时估计 AoA + 距离

影响：
  - 近场波束赋形 = 球面波赋形（不是平面波）
  - 信道矩阵不再是 Toeplitz 结构
  - 容量计算需要新方法
