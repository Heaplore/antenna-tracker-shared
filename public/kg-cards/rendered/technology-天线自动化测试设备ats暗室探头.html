天线自动化测试设备ATS暗室探头
天线技术/技术概念OTA测试暗室探头ATS自动化测试系统3GPP测试
一句话: 天线自动化测试系统（ATS）= 暗室/转台/探头/软件 组成的"天线体检中心"。5G Massive MIMO 时代，64 通道天线的人工测试不现实——必须用自动化系统，30 分钟内测完一个 AAU 的 200+ 项指标。
类比: ATS 就像"汽车排放检测线"——车开上去，机器自动测尾气、测动力、测噪音，出一本报告。

1.2 关键设备参数
设备参数5G AAU 要求
VNA 频率范围 DC-6.5GHz（Sub-6GHz）
VNA 端口数 4-8 端口（多端口校准）
信号发生器 频率 DC-6.5GHz
信号发生器 通道数 ≥ 64（Massive MIMO）
转台 精度 < 0.01°
转台 承重 > 100kg（AAU 很重）
探头天线 频率 多频段探头
探头天线 极化 ±45° 可切换
暗室 截止频率 > 6.5GHz
暗室 尺寸 ≥ 6m × 4m × 3m
2.1 远场 OTA 测试（主流） markdown
原理：
  DUT（待测天线）与探头天线距离 > 2R²/λ
  R = 天线最大尺寸
  λ = 波长
  
典型距离：
  64T64R AAU（1.2m）@ 3.5GHz：
  R_far = 2 × 1.2² / 0.086 = ~34m
  
  → 远场暗室需要 34m 以上！
  → 实际用 15-20m + 近远场变换
  
测试流程：
  1. 安装 DUT 到转台
  2. VNA/SG 连接 DUT 的 RF 端口
  3. 探头天线接收/发射信号
  4. 转台旋转（水平 0-360°）
  5. 升降台移动（垂直 -90° to +90°）
  6. 采集每个角度的 S 参数
  7. 软件处理 → 方向图/增益/隔离度
2.2 近场测试 markdown
原理：
  探头靠近 DUT（< R_far）
  测量近场 → 数学变换 → 远场
  
优势：
  - 暗室可以很小（3-5m）
  - 适合实验室
  
劣势：
  - 变换算法复杂
  - 精度略低
  - 不适用于有源天线（需要激励所有通道）
2.3 混响室测试 markdown
原理：
  金属腔体内搅拌电磁场
  形成"统计均匀"的多径环境
  
测量：
  - TRP（总辐射功率）
  - TIS（总接收灵敏度）
  - MIMO 容量
  
优势：
  - 测试速度快（分钟级）
  - 暗室小（2-3m）
  - 适合产线批量测试
  
劣势：
  - 不能测方向图
  - 不能测增益
  - 只能测"总量"
3.1 测试项目清单 markdown
AAU 出厂测试（200+ 项）：

S 参数测试（VNA）
   - S11/S22（端口回波损耗）
   - S12/S21（插入损耗）
   - Sij（端口间隔离度）
   
有源测试（SG + 探头）
   - 增益（Gain）
   - 方向图（Pattern）
   - 波束宽度（Beamwidth）
   - 前后比（FBR）
   - 旁瓣电平（SLL）
   
MIMO 测试
   - 通道间幅度一致性（< 0.5dB）
   - 通道间相位一致性（< 5°）
   - 空间流数验证
   - 预编码矩阵验证
   
PIM 测试
   - 双三载波 PIM
   - 要求：< -150dBc
   
极化测试
   - 交叉极化隔离度（> 25dB）
   - 轴比（如适用）
   
温度循环测试
   - -40°C to +85°C
   - 各温度点复测关键指标
3.2 测试时间 markdown
单台 64T64R AAU 测试时间：

  安装定位：5min
  S 参数测试：10min
  有源测试（全方向图）：15min
  MIMO 测试：10min
  PIM 测试：5min
  极化测试：5min
  数据分析和报告：5min
  
  总计：~55min
  
产线节拍：
  目标：2 台/小时
  → 需要 2 套 ATS 并行
