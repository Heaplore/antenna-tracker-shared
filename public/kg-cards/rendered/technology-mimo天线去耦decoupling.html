MIMO天线去耦Decoupling
天线技术/技术概念MIMO去耦端口隔离互耦抑制DGSEBG极化分集
一句话: MIMO天线去耦技术是通过各种电磁结构设计或信号处理手段，降低多输入多输出天线系统中相邻端口之间互耦（Mutual Coupling）的方法集合。它是实现高密度 MIMO 和 Massive MIMO 的关键使能技术，直接决定了空间复用的增益能否在现实中兑现。
类比: 想象两个人在狭小的房间里同时打电话——如果房间太小，A 说的话会干扰 B 的通话质量。
1.1 互耦的本质

当两个天线单元靠得很近时（通常间距 < 2λ），一个单元辐射的电磁场会在相邻单元中感应出电流，这种现象称为互耦（Mutual Coupling）。互耦用 S 参数表示为 S_ij（i ≠ j），即从端口 j 到端口 i 的耦合系数。

互耦的危害：

  • 端口间信号串扰 → MIMO 空间复用增益下降
  • 输入阻抗变化 → 匹配恶化 → VSWR 升高
  • 方向图畸变 → 波束指向偏移
  • 相关性增加 → ECC（Envelope Correlation Coefficient）升高 → 分集增益下降
1.2 去耦技术的两大类别
类别 方法 原理
内部去耦 空间分集、极化分集、频率分集 让天线"天生"就不耦合
外部去耦 DGS、EBG、中和线、寄生单元、超材料 在天线之间加"隔离墙"

2.1 空间分集（Spatial Diversity）

原理：将天线单元物理间距拉大，降低互耦。

隔离度 ≈ 20·log₁₀(λ / (4π·d))

典型效果：

  • d = 0.5λ → S21 ≈ -14dB
  • d = λ → S21 ≈ -28dB
  • d = 2λ → S21 ≈ -34dB

优点：最简单、最有效、无额外损耗 缺点：占用面积大，不适合手机等小型设备 适用场景：基站 AAU（单元间距通常 ≥ 0.5λ）、户外 MIMO

2.2 极化分集（Polarization Diversity）

原理：使用正交极化（如 ±45° 双极化）的天线单元，正交极化之间的天然隔离度可达 25-35dB。

关键参数：XPD（Cross-Polarization Discrimination，交叉极化鉴别率）

XPD = 10·log₁₀(P_co / P_cross)

典型效果：

  • ±45° 双极化：隔离度 25-35dB
  • 水平/垂直极化：隔离度 20-30dB
  • 左旋/右旋圆极化：隔离度 20-25dB

优点：不增加天线间距，可在同一位置放置两个正交极化天线 缺点：传播环境会改变极化状态（去极化），实际隔离度可能降至 15-20dB 适用场景：基站双极化天线（±45° 是行业标准）、手机 MIMO

2.3 频率分集（Frequency Diversity）

原理：不同频段的天线之间天然隔离（工作频率不同，互耦可忽略）。

适用场景：多频段 MIMO（如 4G + 5G 共存）、载波聚合

2.4 模式分集（Mode Diversity）

原理：利用天线 Different Radiation Modes（如 TE10 和 TE20 模式）的正交性实现去耦。

典型应用：波导 MIMO、介质谐振器天线（DRA）

3.1 缺陷接地结构（Defected Ground Structure, DGS）

原理：在接地层上蚀刻特定形状的缺口（如 U 形、V 形、环形），破坏表面波的传播路径。

效果：

  • 隔离度改善：5-15dB
  • 工作频带：DC-40GHz
  • 尺寸：缺口尺寸 ≈ λ/2

典型形状： | 形状 | 特点 | 适用场景 | |------|------|---------| | U 形 | 最简单，易加工 | Sub-6GHz MIMO | | V 形 | 可调谐频带 | 多频段 | | 环形 | 多频带抑制 | 多频段 MIMO | | 十字形 | 宽频抑制 | 宽带 MIMO |

3.2 电磁带隙结构（Electromagnetic Bandgap, EBG）

原理：周期性排列的金属贴片或孔洞，在特定频段内形成"带隙"，阻止表面波传播。

效果：

  • 隔离度改善：10-25dB
  • 带宽：通常 10-20%（相对带宽）
  • 尺寸：周期 ≈ λ/2

EBG 的变种：

  • Mushroom EBG：金属柱 + 顶部金属贴片，最常用
  • Yagi EBG：类似 Yagi-Uda 结构，宽带效果好
  • UC-EBG（Ultra-Compact）：紧凑版 EBG，尺寸减半
3.3 中和线（Neutralization Line, NL）

原理：在天线单元之间连接一条金属线（或微带线），产生一个与互耦电流反向的电流，抵消互耦。

效果：

  • 隔离度改善：15-30dB
  • 带宽：较窄（5-10%）
  • 设计复杂度高（需要精确计算中和线长度和位置）

适用场景：紧密间距的 MIMO 天线（d < 0.3λ）

3.4 寄生去耦单元（Parasitic Decoupling Element, PDE）

原理：在天线之间放置一个无源（不连接馈线）的金属单元，通过感应电流产生反向耦合场。

效果：

  • 隔离度改善：10-20dB
  • 带宽：10-15%
  • 设计相对简单

变种：

  • 单寄生单元：简单但带宽窄
  • 多寄生单元：带宽更宽
  • 可调寄生单元（含 varactor）：可动态调谐
3.5 超材料去耦（Metamaterial Decoupling）

原理：使用人工超材料（如 SRR 谐振环、NRI 材料）的特殊电磁特性来抑制互耦。

效果：

  • 隔离度改善：15-35dB
  • 带宽：5-20%（取决于设计）
  • 可设计性强，但加工复杂

典型结构：

  • SRR（Split Ring Resonator）：适用于 10-60GHz
  • ELC（Electric LC Resonator）：适用于 Sub-6GHz
  • 超表面（Metasurface）：覆盖大面积，宽带效果好
3.6 去耦网络（Decoupling Network）

原理：在天线馈线之间插入无源网络（变压器、LC 网络、传输线组合），直接在电路层面抵消耦合。

效果：

  • 隔离度改善：20-40dB
  • 带宽：取决于网络设计
  • 不改变天线结构，可后加装

典型方案：

  • 正交模耦合器（OMC）：用于双极化天线
  • 传输线变压器：宽带去耦
  • LC 谐振网络：窄带但高效
3.7 去耦与图案塑造表面（DPSS）

原理：2025 年最新技术，在天线阵列上方放置周期性金属贴片表面，同时实现去耦和方向图塑造。

效果：

  • 隔离度：< -20dB
  • 效率：> 75%
  • 同时可塑造方向图（降低特定方向的旁瓣）

特点：通过遗传算法优化贴片连接状态，将问题转化为二进制优化问题。

5.1 规模效应

64T64R AAU 有 128 个端口（64TX + 64RX），端口之间的互耦组合多达 128×127 = 16,256 对。即使每对的耦合只有 -15dB，累积效应也会导致严重的性能下降。

5.2 双极化 + 空间密集排列

典型 4V8H 配置（4 垂直 × 8 水平 = 32 单元，双极化 → 64 端口）：

  • 水平间距：≈ 0.5λ
  • 垂直间距：≈ 0.5λ
  • 极化隔离：25-30dB
  • 互耦隔离：15-20dB（同极化相邻单元）

6.1 耦合矩阵补偿

通过测量得到 N×N 的耦合矩阵 C，在基带对接收信号进行补偿：

x_clean = C⁻¹ · x_measured
6.2 自适应波束成形

MVDR（最小方差无畸变响应）波束成形器可以自动在互耦方向上放置零陷：

w_opt = R⁻¹ · a(θ₀) / (a(θ₀)ᴴ · R⁻¹ · a(θ₀))

其中 R 是协方差矩阵（包含互耦信息），a(θ₀) 是导向矢量。

6.3 AI 辅助去耦

2024-2026 年的研究趋势：用深度学习模型直接从天线几何结构预测最优去耦方案，或将互耦补偿学习为端到端的神经网络。

七、技术演进
  • 1990s：空间分集和极化分集是主要的去耦手段
  • 2000s：DGS 和 EBG 被提出用于微带天线去耦
  • 2010s：中和线和寄生去耦单元在 4G MIMO 中广泛应用
  • 2015-2018年：超材料去耦成为研究热点
  • 2018-2022年：5G Massive MIMO 推动多技术组合去耦方案
  • 2023-2024年：去耦网络 + 数字域补偿成为基站 AAU 标准方案
  • 2025年：DPSS（去耦与图案塑造表面）首次提出
  • 2025-2026年：AI 辅助去耦设计进入工程验证阶段
