天线极化基础Polarization-Fundamentals
天线技术/技术概念极化Polarization线极化圆极化椭圆极化轴比交叉极化
一句话: 天线极化描述了电磁波电场矢量在空间中随时间变化的轨迹——可以是直线（线极化）、圆（圆极化）或椭圆（椭圆极化），极化匹配程度直接决定了收发天线之间的功率传输效率，错极化可导致 20-30dB 的功率损失。
类比: 想象你用一根绳子的一端拴在墙上，另一端在你手里抖动。
1.1 极化的数学定义
1.1 极化的数学定义
电磁波的电场可以分解为两个正交分量： ` E(z,t) = E_x·cos(ωt - kz + φ_x)·x̂ + E_y·cos(ωt - kz + φ_y)·ŷ ` 极化由三个参数决定： 幅度比：E_y / E_x 相位差：Δφ = φ_y - φ_x 传播方向：k（通常取 +z 方向）
1.2 极化椭圆的形成
1.2 极化椭圆的形成
当 E_x 和 E_y 都存在且 Δφ ≠ 0, π 时，电场矢量的末端在固定位置随时间画出一个椭圆： ` (E_x/E_0x)² + (E_y/E_0y)² - 2(E_x/E_0x)(E_y/E_0y)cos(Δφ) = sin²(Δφ) ` 这就是极化椭圆方程。
1.3 极化椭圆的几何参数
参数符号含义
倾角（Tilt Angle） τ 椭圆长轴与 x 轴的夹角，-90° ≤ τ ≤ 90°
轴比（Axial Ratio） AR 长轴/短轴之比，线性极化 AR=∞，圆极化 AR=1
旋向（Handedness） LHCP/RHCP 顺时针/逆时针旋转
轴比的 dB 表示： ` AR_dB = 20·log₁₀(AR) ` AR = 1（线性）→ AR_dB = 0dB ❌ 不对，AR=∞ 是线极化 AR = 1（圆）→ AR_dB = 0dB ✅ AR = 3 → AR_dB = 9.5dB ❌ 也不对 纠正： ` AR = 长轴 / 短轴 AR = ∞ → 线极化（短轴=0） AR = 1 → 圆极化（长轴=短轴） AR = 1.5 → 椭圆极化（接近圆） ` AR_dB 的定义有两种常见方式： 线性尺度：AR（无量纲比值） dB 尺度：AR_dB = 20·log₁₀(AR)，圆极化 = 0dB，线极化 = ∞dB 工程中常用"轴比 ≤ 3dB"作为圆极化天线的验收标准。
2.1 线极化（Linear Polarization） markdown
Δφ = 0 或 π
E_y / E_x = 常数


电场矢量始终沿一条直线振荡：


      ↑ E
      │
      │
──────┼────── z（传播方向）
      │
      ↓


分类：
| 类型 | Δφ | 方向 | 示例 |
|------|-----|------|------|
| 水平线极化（HPol） | 0 | 平行于地面 | FM 广播 |
| 垂直线极化（VPol） | 0 | 垂直于地面 | 对讲机 |
| ±45° 线极化 | 0 或 π | 倾斜 45° | 5G MIMO 基站 |

特点：
最简单、最常用
基站天线几乎都用 ±45° 线极化
极化隔离度：理论 ∞（正交极化完全不干扰），实际 20-35dB
2.2 圆极化（Circular Polarization） markdown
E_x = E_y
Δφ = ±π/2


电场矢量末端画出一个圆：


      ↺ RHCP（右手圆极化）
    ← ↑ →
      ↻


旋向定义（IEEE 标准）：
RHCP（Right-Hand CP）：观察者面向波传播方向，电场顺时针旋转
LHCP（Left-Hand CP）：观察者面向波传播方向，电场逆时针旋转

注意：天线学界有两种定义方式（IEEE vs 光学），务必确认标准！

圆极化的优势：
不受天线取向影响：接收天线旋转不影响信号强度
抑制多径干扰：反射波会翻转旋向，RHCP 天线接收 LHCP 反射波时被抑制
雨衰较小：对降水引起的极化旋转不敏感

应用场景：
GPS/GLONASS/Galileo（卫星信号）
卫星通信（DVB-S2）
气象雷达（区分雨滴和冰雹）
RFID（标签取向不确定）
军事通信（抗干扰）
2.3 椭圆极化（Elliptical Polarization）
内容
最一般的情况：
`
E_x ≠ E_y 且 Δφ ≠ 0, π/2, π
`
电场矢量末端画出一个椭圆：
`
╱───╲
╱ ╲
│ ● │ ← 椭圆
╲ ╱
╲───╱
`
特点：
线极化和圆极化都是椭圆极化的特例
实际天线辐射的几乎都是椭圆极化（不完美）
轴比 AR 是衡量"有多椭圆"的指标
3.1 极化匹配因子（Polarization Match Factor）
3.1 极化匹配因子（Polarization Match Factor）
收发天线之间的功率传输效率： ` PLF = |ρ̂_t · ρ̂_r|² ` 其中 ρ̂_t 和 ρ̂_r 分别是发射和接收天线的极化单位矢量。 典型值： | 发射 × 接收 | PLF (线性) | PLF (dB) | 功率损失 | |------------|-----------|---------|---------| | VPol × VPol | 1.0 | 0dB | 0dB | | HPol × HPol | 1.0 | 0dB | 0dB | | VPol × HPol | 0 | -∞dB | 完全隔离 | | ±45° × ∓45° | 0 | -∞dB | 完全隔离 | | RHCP × RHCP | 1.0 | 0dB | 0dB | | RHCP × LHCP | 0 | -∞dB | 完全隔离 | | VPol × RHCP | 0.5 | -3dB | 一半功率 | | HPol × RHCP | 0.5 | -3dB | 一半功率 |
3.2 极化隔离度（Polarization Isolation）
极化类型典型隔离度
±45° 线极化 20-35dB
圆极化（同旋向 vs 反旋向） 20-30dB
高隔离度基站天线 > 30dB
3.3 交叉极化（Cross-Polarization）
场景XPD
理想天线 > 30dB
实际基站天线 20-25dB
雨中对流层链路 10-15dB
电离层传播 5-10dB
4.1 极化分集（Polarization Diversity）
4.1 极化分集（Polarization Diversity）
利用两个正交极化在同一频率上传输独立数据流： ` 天线 A：+45° 线极化 → 数据流 1 天线 B：-45° 线极化 → 数据流 2 ` 效果： 容量翻倍（在低互耦条件下） 无需额外频谱 实际互耦 20-30dB → 容量提升 30-50% 已有笔记：[[技术概念-极化分集Polarization-Diversity]]
4.2 MIMO 中的极化
内容
4G/5G 基站广泛使用双极化天线：
`
AAU 天线单元：
│+45° │ ← 端口 A
│──────│
│-45° │ ← 端口 B
`
为什么用 ±45° 而不是 V/H：
±45° 在用户手持设备中更鲁棒（手机旋转时两个极化都有信号）
V/H 极化在手机横屏/竖屏时可能一个极化完全对准、另一个完全正交
±45° 提供更稳定的分集增益
4.3 极化复用（Polarization Division Multiplexing, PDM）
4.3 极化复用（Polarization Division Multiplexing, PDM）
光纤通信和卫星通信中广泛使用： ` 发送端： 数据 A → 水平极化光 数据 B → 垂直极化光 接收端： 分离 H 和 V 极化 → 分别解调 ` 频谱效率提升：2×（同一频率传两路数据）
4.4 卫星通信中的圆极化
4.4 卫星通信中的圆极化
GPS 信号使用 RHCP： ` 卫星 → RHCP 信号 → 地面接收天线（RHCP） ↓ 反射地面 → LHCP 信号（旋向翻转）→ 被 RHCP 天线抑制 ` 好处： 卫星姿态变化不影响信号（圆极化不受旋转影响） 多径反射被抑制（反射改变旋向） 室内/城市峡谷中仍有较好接收
5.1 XL-MIMO 中的极化维度
内容
传统 MIMO 利用空间维度（多个物理位置的天线单元）：
`
空间 MIMO：
│ │ │ │ │ │ │ │ ← 每个单元单极化
`
极化 MIMO 在每个空间位置增加极化维度：
`
极化 MIMO：
│││ │││ │││ │││ ← 每个单元双极化（+45°/-45°）
`
容量提升：2×（理论上）
5.2 独立极化波束 markdown
波束 1（+45° 极化）→ 用户 A
波束 2（-45° 极化）→ 用户 B


两个用户可以使用相同频率、相同波束指向，通过极化隔离实现复用。
5.3 自适应极化波束赋形
5.3 自适应极化波束赋形
根据信道状态动态调整极化方向： ` 信道测量 → 最优极化方向 → 调整天线极化 → 最大化信噪比 ` 已有笔记：[[技术概念-极化波束赋形Polarization-Beamforming]]
6.1 雨致极化旋转
6.1 雨致极化旋转
降雨会使线极化波发生旋转： ` 旋转角 ∝ 降雨率 × 传播距离 × f² ` 影响： Ku 波段（12GHz）：小雨可忽略，大雨旋转 5-10° Ka 波段（30GHz）：小雨旋转 10-20° 100GHz+：旋转可达 45°+ 对策：使用圆极化（不受旋转影响）或自适应极化补偿
6.2 电离层法拉第旋转
6.2 电离层法拉第旋转
低频（< 1GHz）信号穿过电离层时极化面旋转： ` 旋转角 ∝ TEC × f⁻² ` TEC 是总电子含量，随太阳活动变化。 影响： GPS L1（1.57GHz）：旋转 5-20° 使用 RHCP 可消除影响
6.3 多径极化散射
6.3 多径极化散射
信号经建筑物/地面反射后极化状态改变： ` 入射 VPol → 反射 → VPol + 少量 HPol（交叉极化） 入射 HPol → 反射 → HPol + 少量 VPol（交叉极化） ` 反射面越粗糙，交叉极化分量越大。
7.1 轴比测量
7.1 轴比测量
旋转天线法： 固定发射天线（RHCP） 旋转接收天线，记录功率变化 最大功率与最小功率之比 = AR² 两点法： 测量两个正交线极化分量 计算相位差和幅度比 推导轴比
7.2 极化隔离度测量
7.2 极化隔离度测量
发射天线设为 +45° 线极化 接收天线分别设为 +45° 和 -45° 测量 S21_co 和 S21_cross PI = S21_co - S21_cross

7.3 极化方向图测量
在暗室中使用可旋转极化的参考天线： ` 方位角 θ → 旋转接收天线极化 → 记录 co-pol 和 cross-pol 方向图 ` 输出： co-pol 方向图（期望极化） cross-pol 方向图（交叉极化） 轴比方向图（全空间 AR 分布）
