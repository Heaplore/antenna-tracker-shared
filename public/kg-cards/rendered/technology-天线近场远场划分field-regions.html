天线近场远场划分Field-Regions
天线技术/技术概念近场远场Near-FieldFar-Field辐射场反应场天线测量
一句话: 天线周围的电磁场可以划分为三个区域：反应式近场（储能为主）、辐射式近场（Fresnel 区，球面波）和远场（Fraunhofer 区，平面波）。划分的边界决定了天线测量的条件、波束赋形的方法和信道模型的选择。
类比: 想象一个喷泉：喷口附近的水花乱溅、方向不定（反应式近场）；稍微远一点，水柱开始成形但还没完全展开（辐射式近场/Fresnel 区）；再远一点，水柱变成稳定的直线射流（远场/Fraunhofer 区）。
1.1 反应式近场（Reactive Near-Field）

定义：紧贴天线表面的区域，电磁场以储能为主（电场能和磁场能交替交换，不向外辐射）。

边界：

r < 0.62·√(D³/λ)   （最严格的定义）
或
r < λ/2π ≈ 0.16λ   （单极子/偶极子的简化定义）

其中 D 是天线的最大尺寸。

特征：
- 场强随距离快速衰减（1/r² 或 1/r³）
- 电场和磁场相位相差 90°（无功功率）
- 方向图不稳定，随距离变化剧烈
- 无法进行有意义的方向图测量

工程意义：
- 天线与馈线之间的过渡区
- 近场天线耦合（如 NFC/RFID）
- 人体贴近天线时的 SAR（比吸收率）评估

1.2 辐射式近场 / Fresnel 区（Radiative Near-Field）

定义：波已经开始辐射但仍保持球面波前特性的区域。

边界：

0.62·√(D³/λ) < r < 2D²/λ

特征：
- 场强随距离按 1/r 衰减（辐射场）
- 波前为球面（相位非线性）
- 方向图随距离变化（但未稳定）
- 可以进行波束聚焦（beamfocusing）

子区域（2024 年新研究补充）：

子区域 距离范围 特征
深度聚焦区 r < λ/(10·NA²) 径向波束聚焦可行
深度聚焦区以外 λ/(10·NA²) < r < 2D²/λ 角度波束赋形为主

其中 NA = D/λ 是数值孔径。

工程意义：
- 6G XL-MIMO 的主要工作区
- 近场波束赋形、波束聚焦
- 近场天线测量

1.3 远场 / Fraunhofer 区（Far-Field）

定义：波前近似为平面、方向图稳定的区域。

边界（Rayleigh 距离）：

r > 2D²/λ

更严格的定义（相位误差 < π/8）：

r > 2D²/λ

相位误差更宽松的定义（相位误差 < π/2）：

r > D²/(2λ)

特征：
- 场强按 1/r 衰减
- 波前为平面（相位线性）
- 方向图稳定，不随距离变化
- 角度和幅度独立

工程意义：
- 传统天线测量的标准条件
- 5G Massive MIMO 的典型工作区
- 卫星通信、雷达的远场目标

2.1 Rayleigh 距离（Fraunhofer 距离）
R_F = 2D²/λ

推导：考虑孔径边缘和中心的最大路径差：

Δr_max = √(r² + (D/2)²) - r ≈ D²/(8r)  （当 r >> D）

要求 Δr_max < λ/16（相位误差 < π/8）：

D²/(8r) < λ/16  →  r > 2D²/λ

典型值：

天线 D 频率 λ R_F
5G AAU (64T) 1m 3.5GHz 0.086m 23m
6G XL-MIMO 10m 3.5GHz 0.086m 2.3km
6G XL-MIMO 20m 3.5GHz 0.086m 9.3km
卫星抛物面 (3m) 3m 12GHz 0.025m 288m
车载毫米波雷达 0.1m 77GHz 0.004m 5m
2.2 Fresnel 距离（Reactive/Radiative 边界）
R_Fresnel = 0.62·√(D³/λ)

推导：基于近场 reactive 和 radiative 功率的平衡点。

典型值：

天线 D 频率 λ R_Fresnel
5G AAU 1m 3.5GHz 0.086m 2.1m
6G XL-MIMO 10m 3.5GHz 0.086m 20m
卫星抛物面 3m 12GHz 0.025m 33m
2.3 非辐射距离（Non-Radiating Distance）
R_nr ≈ λ/(4π)

在这个距离以内，无功功率（stored reactive power）大于有功功率（radiated power）。

典型值：

频率 λ R_nr
3.5GHz 0.086m 6.8mm
28GHz 0.011m 0.8mm
100GHz 0.003m 0.2mm
2.4 径向聚焦距离（Radial-Focal Distance）

2024 年新提出的概念——近场波束聚焦可行的最大距离：

R_rf ≈ D²/(4λ) = R_F/8

在这个距离以内，可以使用径向波束聚焦（beamfocusing in the radial domain）。

3.1 远场测量

条件：r > 2D²/λ

方法：
- 室外开阔场（OATS）
- 室内暗室（anechoic chamber），满足 r > 2D²/λ
- 缩距暗室（利用透镜压缩等效距离）

优点：平面波近似成立，测量简单
缺点：需要很长的测量距离（XL-MIMO 需要数公里）

3.2 近场测量

条件：R_Fresnel < r < R_F

方法：
- 平面近场：在平面上扫描，通过傅里叶变换得到远场方向图
- 圆柱近场：在圆柱面上扫描
- 球面近场：在球面上扫描，最精确但最复杂

原理：

E_FF(θ,φ) = FT[E_NF(x,y)]

其中 FT 是傅里叶变换。

优点：测量距离短（只需几米）
缺点：数据处理复杂，需要精密扫描机构

3.3 OTA 测量（Over-The-Air）

条件：适用于无法通过电缆连接的天线（如手机、IoT 设备）

方法：
- DFF（Direct Far-Field）：在远场直接测量
- 缩距法：利用多探头阵列在近场合成远场

4.1 远场波束赋形

模型：平面波

w_FF(θ) = steer_vec_FF(θ) = [1, exp(j·ψ), ..., exp(j·(N-1)·ψ)]^T

适用场景：
- 5G Massive MIMO（用户通常在远场）
- 卫星通信
- 雷达（目标通常在远场）

4.2 近场波束赋形

模型：球面波

w_NF(r, θ) = steer_vec_NF(r, θ) = [1/r₁·exp(-jk·r₁), ...]^T

适用场景：
- 6G XL-MIMO（大部分用户在近场）
- 近场成像
- 能量收集

4.3 混合场区波束赋形

当阵列同时服务近场和远场用户时：

w_hybrid = α·w_NF + (1-α)·w_FF

其中 α 根据用户距离自适应调整。

5.1 远场信道模型
h = Σ_l α_l · a_R(θ_l) · exp(-jk·r_l) · a_T(φ_l)

其中 a_T 和 a_R 是远场导向矢量（平面波假设）。

5.2 近场信道模型
h = Σ_l α_l · a_R(r_l, θ_l) · exp(-jk·r_l) / r_l · a_T(φ_l)

其中 a_T 和 a_R 是近场导向矢量（球面波假设），多了 1/r_l 的幅度衰减项。

5.3 3GPP 信道模型

3GPP TR 38.901 v19 新增了近场信道模型：

  • 远场：平面波模型（PWM）
  • 近场：球面波模型（SWM）
  • 边界：R_F = 2D²/λ
6.1 测量距离不足

在暗室中测量大孔径天线时，如果距离不够远（r < 2D²/λ），平面波近似不成立，测量结果会有系统性误差：

相位误差 Δφ ≈ k·D²/(8r) = π·D²/(4λr)

对于 D = 1m, λ = 0.086m, r = 10m：

Δφ ≈ π·1/(4·0.086·10) ≈ 0.91 rad ≈ 52°

52° 的相位误差会导致方向图严重失真。

对策：
- 使用近场测量 + 傅里叶变换
- 使用缩距暗室（透镜）
- 增加测量距离

6.2 多径干扰

近场测量更容易受到反射和多径干扰的影响，因为反射波的路径差较小：

Δτ_reflected ≈ 2·d_mirror / c

对策：
- 高质量的吸波材料（反射损耗 > 40dB）
- 近场扫描时使用门控技术
- 多径消除算法

6.3 空气衰减

在太赫兹频段，空气对水的吸收会导致近场测量的显著衰减：

α_air ≈ 10-20 dB/km @ 200GHz

对于近距离测量（< 100m），影响较小。但对于近场扫描（需要大范围移动），累积衰减可能不可忽视。

6.4 6G XL-MIMO 的特殊挑战

XL-MIMO 的 R_F 可达数公里，这意味着：
- 传统远场测量方法几乎不可行
- 近场测量成为唯一选择
- 需要新的测量标准和协议

七、技术演进
  • 1950s：近场/远场划分的经典理论建立
  • 1970s：近场测量技术（平面/圆柱/球面）发展
  • 2000s：OTA 测量成为手机天线标准
  • 2010s：缩距暗室和 MIMO OTA 成熟
  • 2020-2022年：XL-MIMO 近场测量需求推动新技术
  • 2023-2024年：3GPP TR 38.901 v19 纳入近场信道模型
  • 2025年：径向聚焦距离概念正式提出
  • 2025-2026年：太赫兹近场测量标准制定中
