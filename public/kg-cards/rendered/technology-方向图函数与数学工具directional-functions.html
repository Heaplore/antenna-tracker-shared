方向图函数与数学工具Directional-Functions
天线技术/技术概念方向图函数特殊函数贝塞尔ChebyshevLegendre数学工具
一句话: 天线方向图的背后是一套丰富的数学工具——sinc 函数描述矩形孔径、贝塞尔函数描述圆形孔径、Chebyshev 多项式控制旁瓣、Legendre 多项式描述球面波。掌握这些函数的数学特性，就能理解和预测天线方向图的行为。
类比: 天线方向图就像一首乐曲的"声音指纹"，而这些特殊函数就是乐谱上的音符。
1.1 sinc 函数（抽样函数）
1.1 sinc 函数（抽样函数）
定义： ` sinc(x) = sin(x)/x （数学定义） 或 sinc(x) = sin(πx)/(πx) （工程定义，零点在整数处） ` 在天线中的应用： 矩形孔径均匀照明 → 方向图 = sinc 函数 线性阵列均匀激励 → 阵列因子 = Dirichlet 核（周期 sinc） 关键性质： ` 零点：x = ±π, ±2π, ±3π, ... 第一旁瓣：-13.2 dB 旁瓣滚降：∝ 1/x（缓慢） ` 图形特征： ` 主瓣宽度（-3dB）：Δx ≈ 0.886·π 主瓣宽度（零点到零点）：Δx = 2π `
1.2 贝塞尔函数（Bessel Function）
1.2 贝塞尔函数（Bessel Function）
定义（一阶）： ` J₁(x) = Σ(k=0 to ∞) [(-1)ᵏ · (x/2)^(2k+1)] / [k! · (k+1)!] ` 在天线中的应用： 圆形孔径均匀照明 → 方向图 = 2·J₁(x)/x 抛物面天线的馈源方向图 关键性质： ` 第一零点：x ≈ 3.832 第一旁瓣：-17.6 dB（比矩形孔径更高） 旁瓣滚降：∝ 1/√x（比 sinc 慢） ` 圆形孔径方向图： ` F(θ) = 2·J₁(ka·sinθ) / (ka·sinθ) ` 其中 a = 孔径半径，k = 2π/λ。 3dB 波束宽度： ` HPBW ≈ 1.02·λ/D （D = 2a 是直径） ` 对比矩形孔径：HPBW ≈ 0.886·λ/L
1.3 Chebyshev 多项式
1.3 Chebyshev 多项式
定义： ` T_n(x) = cos(n·arccos(x)), |x| ≤ 1 T_n(x) = cosh(n·arccosh(x)), |x| > 1 ` 递推关系： ` T_0(x) = 1 T_1(x) = x T_{n+1}(x) = 2x·T_n(x) - T_{n-1}(x) ` 关键性质： 在 [-1, 1] 区间内，|T_n(x)| ≤ 1（等振荡） 在 |x| > 1 时，T_n(x) 单调递增 等旁瓣特性 → 用于 Dolph-Chebyshev 天线设计 在天线中的应用： Dolph-Chebyshev 阵列的方向图： ` AF(ψ) = T_{N-1}[cos(ψ₀)·cos(ψ/2)] / T_{N-1}[cos(ψ₀)] ` 其中 ψ₀ 控制旁瓣电平： ` SLL = 20·log₁₀(T_{N-1}(cos(ψ₀))) `
1.4 Legendre 多项式
1.4 Legendre 多项式
定义： ` P_n(x) = (1/(2ⁿ·n!)) · dⁿ/dxⁿ[(x²-1)ⁿ] ` 前几个： ` P_0(x) = 1 P_1(x) = x P_2(x) = (3x²-1)/2 P_3(x) = (5x³-3x)/2 ` 在天线中的应用： 球面波展开 球面天线（如卫星天线）的方向图分析 球坐标系中的角度函数
1.5 Dirichlet 核（周期 sinc）
1.5 Dirichlet 核（周期 sinc）
定义： ` D_N(x) = sin((N+½)x) / sin(x/2) ` 在天线中的应用： 均匀线性阵列的方向图（有限单元数 N） 当 N → ∞ 时，D_N(x) → 周期 sinc 函数 关键性质： ` 主瓣宽度：4π/N 第一旁瓣：-13.2 dB 旁瓣数量：N-1 个 `
