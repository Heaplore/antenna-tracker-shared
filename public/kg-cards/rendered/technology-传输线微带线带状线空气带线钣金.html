传输线微带线带状线空气带线钣金
天线技术/技术概念微带线带状线空气带线钣金传输线馈电网络传输线
一句话: 传输线是 AAU 内部信号分配的"血管"——从 PCB 上的微带线到金属腔体内的空气带线/钣金带状线，核心矛盾是"损耗 vs 体积 vs 功率容量"。Sub-6GHz 用微带线够用，毫米波和大功率开始转向空气带线和钣金结构。
类比: 传输线就像"水管"——信号是水，水管的形状决定了水流稳不稳。
1.1 结构
微带线结构：
  ┌──────────────┐ ← 信号线（铜，宽度 W）
  │              │
  │  介质基板    │ ← Dk=3.48（RO4350B）
  │              │
  └──────────────┘ ← 接地层（铜）

关键尺寸：
  W/H（线宽/基板厚）→ 决定特性阻抗 Z₀
  Z₀ = 50Ω（通信标准）

典型参数（RO4350B, H=0.81mm）：
  50Ω 微带线：W ≈ 1.5mm
  3.5GHz 损耗：~0.5dB/10cm
1.2 特点
特性 值/说明
损耗 中等（0.3-1dB/10cm @3.5GHz）
功率容量 低（< 1W）
尺寸 最小（适合高密度）
成本 最低
加工 最简单（PCB 蚀刻）
适用频段 DC-6.5GHz
1.3 在 AAU 中的应用
微带线用于：
  - T/R 模块内部信号分配
  - 馈电网络（功分器/移相器）
  - PCB 振子之间的连接

局限：
  - 损耗随频率升高急剧增加
  - 28GHz 时 10cm 微带线损耗 > 5dB
  - 不适合毫米波长距离传输
2.1 结构
带状线结构：
  ┌──────────────┐ ← 上层接地
  │              │
  │  ───────     │ ← 信号线（嵌在介质中间）
  │  ───────     │   （宽度 W，离上下接地距离 H/2）
  │              │
  └──────────────┘ ← 下层接地

特点：
  - 信号线被接地层上下包围
  - 电磁场完全封闭在介质内
  - 无辐射损耗
  - 比微带线损耗低 20-30%
2.2 微带线 vs 带状线
特性 微带线 带状线
结构 信号线在上层 信号线在介质中间
损耗 较高（有辐射） 较低（无辐射）
尺寸 更小 更大（需要多层板）
耦合 有边缘耦合 隔离度更好
加工 简单 复杂（需要多层压合）
适用 Sub-6GHz Sub-6GHz 高密度
2.3 在 AAU 中的应用
带状线用于：
  - 64T64R AAU 的背板（多层 RO4350B）
  - 需要高隔离度的信号分配
  - 高密度布线（64 通道）

优势：
  - 通道间串扰 < -30dB
  - 比微带线更紧凑
  - 适合 64 通道以上的密集布线
3.1 结构
空气带线结构：
  ┌──────────────┐ ← 金属盖板
  │              │
  │  空气（Dk≈1）│ ← 介质是空气！
  │              │
  │  ───────     │ ← 信号线（铜）
  │  ───────     │
  └──────────────┘ ← 金属底板（接地）

关键：
  - 介质是空气（Dk=1.0）
  - 有效 Dk ≈ 1.0-1.5（取决于盖板高度）
  - 信号在空气中传播 → 损耗极低
3.2 特点
特性 值/说明
损耗 极低（0.05-0.2dB/10cm @3.5GHz）
功率容量 高（空气散热好）
尺寸 中等（比微带线大）
成本 中（需要精密加工）
适用频段 DC-10GHz
3.3 在 AAU 中的应用
空气带线用于：
  - 毫米波 AAU 内部馈线（28/39GHz）
  - 高功率 Sub-6GHz 分配网络
  - 低损耗相位一致性要求高的场景

优势：
  - 损耗比微带线低 5-10 倍
  - 温度稳定性好（空气 Dk 不随温度变）
  - 功率容量大（空气不发热）
4.1 结构
钣金带状线结构：
  ┌──────────────────┐ ← 金属上盖（铝/铜）
  │                    │
  │  ┌──────────────┐  │
  │  │ 信号铜条     │  │ ← 信号导体（铜排）
  │  │              │  │
  │  │ 介质支撑柱   │  │ ← PTFE/陶瓷支撑
  │  │              │  │
  │  └──────────────┘  │
  │                    │
  └────────────────────┘ ← 金属底板（接地）

加工方式：
  - 钣金折弯/冲压形成金属腔体
  - 铜排蚀刻/冲压形成信号线
  - PTFE 支撑柱固定间距

特点：
  - 全金属结构 → 屏蔽极佳
  - 功率容量大（铜排散热好）
  - 损耗低（空气+PTFE 介质）
4.2 特点
特性 值/说明
损耗 低（0.1-0.3dB/10cm @3.5GHz）
功率容量 很高（10-100W）
尺寸 较大（适合机架内）
成本 中（钣金加工）
适用频段 DC-6.5GHz
隔离度 > 40dB
4.3 在 AAU 中的应用
钣金传输线用于：
  - 4G/5G 基站馈电网络（功分器/合路器）
  - RRU 到 AAU 的内部连接
  - 大功率分配网络

优势：
  - 比 PCB 微带线功率容量高 10-100 倍
  - 比同轴电缆更紧凑
  - 适合批量生产（钣金+组装）

典型应用：
  - 64 通道功分器（钣金腔体 + PCB 电路）
  - 双频/三频合路器
  - 滤波器腔体

六、5G AAU 传输线选型指南
Sub-6GHz AAU（3.5GHz）：
  T/R 模块内部 → 微带线（PCB 集成）
  64 通道分配 → 带状线（多层背板）
  功分器/合路器 → 钣金传输线（功率大）

毫米波 AAU（28GHz）：
  BFIC 到 AiP → 空气带线/FFC（低损耗）
  腔体内 → 波导（功率大/损耗最低）

关键决策因素：
  1. 频率 → 越高越需要低损耗结构
  2. 功率 → 越大越需要金属结构
  3. 尺寸 → 越小越需要 PCB 集成
  4. 成本 → 越低越需要批量工艺
