A+P混合天线Active-Plus-Passive
天线技术/技术概念A+P天线Active-Plus-Passive混合天线频段共存Sub-6GHz
一句话: A+P 天线是 5G 基站天线的"混搭方案"——有源层（AAU）负责高频段（3.5GHz）提供容量，无源层（平板天线）负责低频段（700/2.6GHz）提供覆盖，两层共用一个支架，节省站址资源。
类比: A+P 就像双层汉堡包——上层 AAU 是"肉饼"（高频高容量），下层平板是"面包"（低频广覆盖），共用一个"托盘"（安装支架）。
1.1 分层结构 markdown
顶层：AAU（有源层）
  - 频段：3.5GHz（n78）
  - 通道：64T64R
  - 功能：高容量数据
  - 供电：48V DC，200-500W
  
中层：馈电/隔离层
  - 频段隔离（700MHz ↔ 3.5GHz）
  - 结构支撑
  
底层：平板天线（无源层）
  - 频段：700MHz（n28）+ 2.6GHz（n41）
  - 通道：2T2R 或 4T4R
  - 功能：广覆盖 + 容量补充
  - 供电：不需要
1.2 天线共装 markdown
传统方案（分开安装）：
  700MHz 天线 → 独立抱杆
  3.5GHz AAU → 独立抱杆
  占用 2 个站址或 2 组抱杆
  
A+P 方案（共装）：
  700MHz 天线 + 3.5GHz AAU → 同一抱杆
  占用 1 个站址，1 组抱杆
  节省站址成本 30-50%
2.1 隔离需求 markdown
700MHz ↔ 3.5GHz 隔离度要求：
  - 互调抑制：> 80dB
  - 耦合抑制：> 60dB
  - 防止 700MHz 信号进入 3.5GHz 接收机
  
2.6GHz ↔ 3.5GHz 隔离度要求：
  - 互调抑制：> 70dB
  - 耦合抑制：> 50dB
2.2 隔离方法 markdown
物理间距
   - 700MHz ↔ 3.5GHz：≥ 0.5m
   - 2.6GHz ↔ 3.5GHz：≥ 0.3m
   
极化隔离
   - 底层：±45° 极化
   - 顶层：±45° 极化（错开 90°）
   
滤波隔离
   - 底层加带通滤波器
   - 顶层加带通滤波器
3.1 优势
优势说明
站址节省 共装一套抱杆，节省 30-50% 站址成本
覆盖+容量 低频广覆盖 + 高频高容量
运维简化 一套站点，统一运维
快速部署 5G 快速覆盖，无需新建站址
3.2 挑战
挑战说明
重量负担 总重可达 50-70kg，抱杆需加固
风载增加 迎风面积增加 50-100%，风载加倍
频段隔离 需要足够的物理间距和滤波
散热 AAU 发热 + 密闭空间 = 散热挑战
安装难度 需要更大抱杆和更强支架
4.1 中国移动 A+P markdown
底层：700MHz n28（中国移动）
  - 天线：2T2R 平板
  - 增益：17dBi
  - 功率：43dBm
  
顶层：3.5GHz n78（中国电信/联通）
  - AAU：64T64R
  - 增益：24dBi
  - 功率：200W（每通道）
  
间距：≥ 0.8m
4.2 欧洲 A+P markdown
底层：800MHz（运营商 A）
  - 2T2R 平板
  - 覆盖层
  
中层：1.8GHz（运营商 A）
  - 4T4R 平板
  - 容量层
  
顶层：3.5GHz（运营商 B）
  - 64T64R AAU
  - 5G 容量层
  
间距：≥ 0.5m（每层）
