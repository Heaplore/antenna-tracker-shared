频率选择表面FSS
天线技术/技术概念频率选择表面FSSFrequency-Selective-Surface超表面滤波天线罩
一句话: FSS 是"频率滤镜"——它只对特定频率的电磁波透明，对其他频率反射或吸收。用在天线罩上就是"滤波天线罩"，用在多频共存天线中间就是"频段隔离层"。
类比: FSS 就像百叶窗——阳光（电磁波）从外面照过来，但百叶窗只让特定颜色的光通过，其他颜色的光被挡住。
1.1 结构 markdown
FSS 单元结构（周期性排列）：
  ┌──────────────────┐
  │  金属贴片/缝隙    │  ← 谐振结构
  ├──────────────────┤
  │  介质基板         │  ← ε_r = 2.2-10
  ├──────────────────┤
  │  金属贴片/缝隙    │  ← 谐振结构
  └──────────────────┘
  
单元间距：d < λ/2（避免栅瓣）
1.2 工作机理 markdown
共振频率 f_r：
  f_r = c / (2π√(ε_eff) · L)
  
L = 单元尺寸
ε_eff = 有效介电常数
  
当 f = f_r：
  - 带通型 FSS：透射（天线罩应用）
  - 带阻型 FSS：反射（隔离应用）
  
当 f ≠ f_r：
  - 带通型 FSS：反射
  - 带阻型 FSS：透射
2.1 带通型 FSS（Pass-Band） markdown
结构：金属贴片阵列
功能：只让目标频段通过
应用：滤波天线罩、多频天线隔离

典型参数：
  中心频率：3.5GHz
  带宽：100MHz（2.8% FBW）
  插入损耗：< 0.1dB
  阻带抑制：> 30dB（@ 700MHz）
2.2 带阻型 FSS（Stop-Band） markdown
结构：缝隙阵列
功能：阻挡特定频段
应用：天线隔离、EMI 屏蔽

典型参数：
  中心频率：2.4GHz
  带宽：200MHz（8% FBW）
  插入损耗：< 0.5dB（通带）
  阻带抑制：> 40dB
2.3 双频/多频 FSS markdown
结构：多层 FSS（每层不同频率）
功能：多层频段选择
应用：A+P 天线频段隔离

典型参数：
  层数：2-4 层
  频段：700MHz + 2.6GHz + 3.5GHz
  层间间距：λ/4
3.1 滤波天线罩 markdown
传统天线罩：
  全频段透明（300MHz-6GHz）
  损耗：0.5-1dB
  
FSS 滤波天线罩：
  只透射目标频段（如 3.5GHz）
  阻挡干扰频段（700MHz、2.6GHz）
  损耗：< 0.1dB（通带）
  抑制：> 30dB（阻带）
3.2 A+P 天线频段隔离 markdown
问题：
  700MHz ↔ 3.5GHz 互调
  
解决方案：
  在两层天线之间加 FSS 隔离层
  
效果：
  700MHz 信号被 FSS 反射
  3.5GHz 信号透过 FSS
  互调抑制：> 80dB
3.3 多频共口径天线 markdown
结构：
  上层：3.5GHz 贴片天线
  中层：FSS 隔离层（透 3.5GHz，阻 700MHz）
  下层：700MHz 偶极子天线
  
效果：
  两层天线共用同一口径
  频段隔离 > 40dB
  节省站址空间
