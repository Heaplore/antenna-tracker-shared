FD-MIMO全维度MIMO
天线技术/技术概念FD-MIMO全维度MIMO3D波束赋形双极化天线5GMassiveMIMO
一句话: FD-MIMO（Full Dimension MIMO）= 3D 波束赋形 = 在水平和垂直两个维度同时形成波束。传统 MIMO 只能控制水平方向，FD-MIMO 通过垂直阵列 + 双极化天线，可以同时区分不同楼层的用户，实现真正的三维空间复用。
类比: 传统 MIMO 像"手电筒照水平面"——只能控制左右方向；FD-MIMO 像"探照灯"——既能控制左右，又能控制上下。
1.1 从 2D 到 3D markdown
传统 MIMO（2D）：
  天线阵列：水平排列（如 2T2R, 4T4R）
  波束控制：只能控制水平方向
  空间复用：水平维度
  
FD-MIMO（3D）：
  天线阵列：水平 + 垂直排列（如 64T64R, 128T128R）
  波束控制：水平 + 垂直双维度
  空间复用：3D 空间维度
  
关键差异：
  垂直维度的引入 → 可以区分不同高度的用户
  → 高层建筑覆盖质量大幅提升
  → 小区容量提升 2-3 倍

2.1 垂直波束倾斜 markdown
传统电调倾角（RET）：
  整个天线阵面作为一个整体倾斜
  → 只能调整一个倾角
  → 无法对不同高度用户差异化覆盖
  
FD-MIMO 3D 波束赋形：
  垂直维度每个单元独立控制相位
  → 可以同时形成多个垂直波束
  → 例如：一个波束对准 3-5 楼，一个对准 10-15 楼
  
垂直波束宽度：
  8 单元垂直阵列 → HPBW ≈ 70°/8 = ~9°
  → 可以区分不同楼层
  
波束赋形公式（3D）：
  AF(θ,φ) = Σ_n Σ_m w_nm · e^(j·k·d_n·sin(θ)cos(φ)) · e^(j·k·d_m·sin(θ)sin(φ))
  
  其中：
  θ = 俯仰角（垂直维度）
  φ = 方位角（水平维度）
  w_nm = 第 n 行第 m 列的复数权重
2.2 3D 波束赋形优势 markdown
垂直维度复用
   同一频率，不同楼层 = 不同用户 = 空间复用
   → 容量提升 2-3 倍
   
覆盖优化
   高层建筑边缘用户信号改善 5-10dB
   → 减少掉话率
   
干扰抑制
   垂直方向零陷 → 减少小区间干扰
   → 边缘用户 SINR 提升
   
动态调整
   根据用户分布实时调整 3D 波束
   → 白天写字楼（中高层密集）
   → 晚上住宅区（高层密集）

四、FD-MIMO 与 Massive MIMO 的关系
` Massive MIMO： 广义概念 → 大规模天线阵列（> 8T8R） 可以是 2D 也可以是 3D FD-MIMO： 狭义概念 → 特指 3D 波束赋形 强调垂直维度的引入 关系： FD-MIMO ⊂ Massive MIMO 所有 FD-MIMO 都是 Massive MIMO 但不是所有 Massive MIMO 都是 FD-MIMO 5G 基站现状： 64T64R AAU 几乎都是 FD-MIMO（3D 波束赋形） → 垂直+水平双维度波束控制 `
