漏波天线Leaky-Wave-Antenna
天线技术/技术概念漏波天线Leaky-WaveAntennaLWA频率扫描SIW开放结构
一句话: 漏波天线是一种"边走边漏"的开放波导结构——电磁波在其中传播时，每走一步就泄漏一小部分能量辐射出去，所有泄漏波在远场相干叠加形成方向图。最大的亮点是频率扫描：改变工作频率就能自动改变波束指向，无需昂贵的移相器。
类比: 想象一根水管，管壁上有一排小孔。
1.1 什么是漏波天线？
封闭结构天线开放结构天线（LWA）
能量在辐射体上建立驻波 能量沿传输线行波传播
辐射集中在天线表面 辐射沿整个传输线分布
频率固定，波束固定 频率改变，波束自动扫描
带宽较窄 带宽较宽

1.3 向后辐射 vs 向前辐射
LWA 可以分为两种模式： 向后扫描（Backward Radiating）： ` β > 0, 相速度 & 群速度反向 sin(θ₀) = β/k₀ > 0 → θ₀ 从 180° 扫到 90° ` 低频时波束指向后方，频率升高波束逐渐移向 broadside。 向前扫描（Forward Radiating）： ` β > 0, 相速度 & 群速度同向 sin(θ₀) = β/k₀ > 0 → θ₀ 从 0° 扫到 90° ` 低频时波束指向前方，频率升高波束逐渐远离 broadside。
1.4 开放阻带（Open Stop Band, OSB）
当 β → 0 时（broadside 方向，θ₀ = 90°），LWA 出现"开放阻带"： ` β = 0 → sin(θ₀) = 0 → θ₀ = 90°（broadside） ` 此时： 波束无限宽（无法形成定向波束） 增益骤降（5-10dB） 输入匹配恶化 抑制 OSB 的方法： 非周期性结构（打破 β = 0 的条件） 弯曲波导（蛇形线） 非对称缝隙 慢波结构（增强 β，避免 β → 0）

2.4 光栅耦合漏波天线（Grating-Coupled LWA）
利用周期性光栅结构实现辐射： ` 周期 Λ → 满足光栅方程： sin(θ_m) = m·λ/Λ - β/k₀ ` 特点： 精确控制辐射方向 可用于太赫兹频段 加工精度要求高
2.5 人工表面表面波漏波天线（SSW LWA）
利用人工表面波（Surface Wave on Artificial Dielectric）： ` 金属柱阵列 → 等效表面波 → 周期性辐射 ` 特点： 超低剖面 适合室内覆盖 扫描范围大（±60°）

3.3 波束纯度
LWA 的旁瓣通常较高（-10 到 -15dB），因为辐射沿传输线均匀分布（类似均匀照射的口径）： 降低旁瓣的方法： 非均匀单元设计（改变每个单元的泄漏率） 振幅加权（渐削照射） 多 LWA 组合

5.1 5G 基站
28/39GHz SIW LWA 扇区覆盖（一个 LWA 覆盖一个 120° 扇区） 频率扫描替代机械旋转 低成本、低剖面
5.2 车载雷达
77GHz LWA 频率扫描实现方位角覆盖 与毫米波 SoC 集成 成本远低于相控阵
5.3 卫星通信
Ka 波段 LWA 轻量、低剖面 频率扫描替代机械转向
5.4 6G 太赫兹通信
0.1-1THz LWA 芯片级集成（SIW/WSIW） 超宽带（> 100GHz 带宽） 波束扫描 + 高速数据传输
6.1 开放阻带（OSB）
broadside 方向的 OSB 是 LWA 最大的痛点： ` 频率 f_OSbroadside → 波束无限宽 → 增益崩溃 ` 抑制方法： 蛇形波导（增加有效长度，避免 β = 0） 非对称结构（打破对称性） CRLH 结构（同时支持 forward/backward）
6.2 频率-波束耦合
LWA 的波束指向依赖于频率，这意味着： 通信时频率变化 → 波束偏移 → 链路中断 需要频率稳定或波束跟踪 解决方案： 窄带通信（频率变化小） 反馈控制（检测波束偏移并补偿） 多 LWA 冗余覆盖
6.3 带宽与增益的权衡 markdown
带宽 ↑ → 单元数 ↓ → 增益 ↓


要同时获得宽频带和高增益非常困难。
6.4 加工精度
毫米波 LWA 的缝隙/周期结构精度要求高： ` ΔL/L < 1% → Δθ < 1° ` 对于 30GHz（λ = 10mm），ΔL < 0.1mm。
七、技术演进
1950s：LWA 理论建立（Watson, Oliner） 1970s：周期性漏波结构实用化 1990s：非均匀 LWA（降低旁瓣） 2000s：CRLH LWA 提出 2010s：SIW LWA 成为主流（毫米波） 2020-2024年：太赫兹 LWA、芯片级集成 2025年：拓扑光子学 LWA（72Gbps 芯片级）
