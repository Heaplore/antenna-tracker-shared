透明天线Transparent-Antenna
天线技术/技术概念透明天线TransparentAntennaITO石墨烯导电聚合物
一句话: 透明天线使用透明导电材料（ITO、石墨烯、银纳米线、导电聚合物等）替代传统金属导体，使天线能够嵌入玻璃、OLED 屏幕或车窗中——在保持视觉透明度的同时实现无线信号收发，是 6G 智能窗户、透明显示屏和车载天线的前沿方向。
类比: 现在的智能手机屏幕是不透明的，因为下面铺满了金属电路。

1.2 ITO（Indium Tin Oxide）
1.2 ITO（Indium Tin Oxide）
最常用的透明导电材料： ` 化学式：In₂O₃:SnO₂（90:10 摩尔比） 晶体结构：立方晶系（Bixbyite） 沉积方法：溅射、蒸镀、旋涂 ` 电学-光学权衡： ` 载流子浓度 N ∝ 电导率 σ 等离子体频率 ω_p ∝ √N ` 当 N 太高时，ω_p 进入可见光范围 → 材料变反光（不透明）。 当 N 太低时，σ 不够 → 天线效率下降。 最佳窗口：光学透明度 85% 时，方块电阻 ≈ 10-30Ω/□。
1.3 石墨烯
1.3 石墨烯
单层石墨烯的理论透明度：97.7%（每层吸收 2.3% 可见光）。 ` 电导率 σ_graphene = 4e²/h ≈ 1.5×10⁻⁴ S ` 优势： 超高透明度 超高载流子迁移率（200,000 cm²/V·s） 柔性、轻薄 化学稳定性好 劣势： 大面积制备困难（CVD 转移易破损） 方块电阻高（> 100Ω/□） 与射频电路集成复杂

2.4 ScreenAnt（屏幕天线）
内容
2025 年提出的创新概念——将整个手机屏幕变成天线阵列：
`
│ ITO 透明导电薄膜（全覆盖）│ ← 整个屏幕作为辐射体
│ ┌──┐ ┌──┐ ┌──┐ ┌──┐ │
│ │▓▓│ │▓▓│ │▓▓│ │▓▓│ │ ← 电极化图案（透明）
│ └──┘ └──┘ └──┘ └──┘ │
│ │
│ OLED 显示层 │
`
核心创新：
天线从"边缘移到屏幕中央"——有效孔径增大 10 倍
使用 ITO 薄膜覆盖整个前玻璃
通过电极化图案定义辐射单元
波束赋形通过屏幕上的透明电极独立控制
性能（28GHz）：
频谱效率比边缘天线高 50%（有遮挡）或 30%（无遮挡）
人手遮挡影响降低 80%
透明度：75-85%
性能（300GHz 太赫兹）：
频谱效率提升 150%

5.3 光学-电磁联合仿真
5.3 光学-电磁联合仿真
透明天线需要同时优化光学和电磁性能： ` 目标函数： J = w₁·(1 - η_EM) + w₂·(T_target - T_actual)² + w₃·(f_res - f_target)² ` 其中 η_EM 是电磁效率，T 是光学透明度，f_res 是谐振频率。
6.1 ITO 的脆性问题
6.1 ITO 的脆性问题
ITO 在弯曲时容易开裂： ` 临界应变：~0.5-1% ` 对于柔性设备，需要使用银纳米线或石墨烯替代。
6.2 大面积均匀性
6.2 大面积均匀性
透明导电薄膜在大面积上的方块电阻不均匀： ` ΔR_square / R_square ≈ 5-15%（大面积） ` 导致天线性能在屏幕上不均匀。
6.3 与显示技术的兼容
6.3 与显示技术的兼容
透明天线需要与 OLED/LCD 显示层共存： OLED 的金属电极可能与透明天线耦合 显示刷新可能产生电磁干扰 需要电磁隔离设计
6.4 耐候性
6.4 耐候性
户外使用的透明天线（车窗、幕墙）需要抵抗： UV 老化（ITO 降解） 温度循环（-40°C 到 +85°C） 湿度（水汽渗透） 刮擦
