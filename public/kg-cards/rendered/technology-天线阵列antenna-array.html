天线阵列Antenna-Array
天线技术/技术概念天线阵列相控阵波束赋形阵列因子
一句话: 天线阵列是将多个独立天线单元按一定几何排列组合，通过控制各单元的幅度和相位，使辐射方向图按需求成形和指向的综合性天线系统。它是现代通信、雷达、卫星导航等领域几乎所有高级天线技术的基础。
类比: 想象一群人站在广场上各自喊话——如果每个人都随意喊，声音杂乱无章；但如果大家按指挥的节奏同时开口，声波就能叠加成一个方向的声音，让对面的人听得特别清楚。
1.1 定义
1.1 定义
天线阵列（Antenna Array）是由两个或多个相同或不同的辐射单元按照一定的几何形状和排列方式组成的天线系统。阵列的核心思想是空间信号处理——利用多个天线单元的空间分布特性，通过幅度加权（Amplitude Weighting）和相位控制（Phase Control），实现对辐射方向图的灵活调控。
2.2 基本原理：阵列因子理论
2.2 基本原理：阵列因子理论
天线阵列的总辐射方向图由两个部分组成： ` 总方向图 = 单元方向图 × 阵列因子（Array Factor, AF） ` 单元方向图（Element Pattern）：单个天线单元自身的辐射方向特性 阵列因子（AF）：由多个单元的空间排列、激励幅度和相位决定的方向图 阵列因子的数学表达式（均匀线性阵列）： ` AF(θ) = Σ(n=0 to N-1) [An · exp(j·n·ψ)] ` 其中： N = 阵列单元数量 An = 第n个单元的激励幅度 ψ = kd·cos(θ) + α，k为波数（2π/λ），d为单元间距，α为相邻单元间的相位差 这个公式揭示了天线阵列的本质：它是一个空间采样系统，单元间距d决定了空间采样率，相位差α决定了波束指向。
1.3 核心物理机制
1.3 核心物理机制
相长干涉：当电磁波到达某方向时相位相同，信号叠加增强 相消干涉：当电磁波到达某方向时相位相反，信号相互抵消 波束扫描：通过改变馈电相位梯度，电子控制波束指向，无需机械转动
2.1 关键公式
2.1 关键公式
| 公式 | 含义 | 说明 | |------|------|------| | 波束指向角 | θ₀ = arccos(-α / kd) | α为相邻单元相位差，d为单元间距 | | 半功率波束宽度 | HPBW ≈ 0.886·λ / (N·d·cosθ₀) | N为单元数，HPBW越小波束越窄 | | 阵列增益 | G ≈ N·Ge | 理想情况下增益与单元数成正比（Ge为单元增益） | | 栅瓣条件 | d/λ ≥ 1/(1+sinθₛ) | 超过此间距会出现栅瓣（Grating Lobe） | | 第一零点波束宽度 | FNBW ≈ 2λ / (Nd) | 主瓣两侧第一个零点之间的角度 |
2.2 阵列权重函数（窗函数）
2.2 阵列权重函数（窗函数）
与光学中的傅里叶变换类似，阵列单元的幅度分布决定了旁瓣电平： 均匀分布（所有单元等幅）：波束最窄，但旁瓣最高（-13.2 dB） 泰勒分布（Taylor Weighting）：旁瓣可控（-20 至 -40 dB），波束稍宽 切比雪夫分布（Chebyshev）：等旁瓣电平最优分布 汉宁/海明分布（Hanning/Hamming）：旁瓣抑制好，但波束展宽明显
2.3 互耦效应（Mutual Coupling）
2.3 互耦效应（Mutual Coupling）
阵列中相邻单元之间通过近场耦合相互影响，导致： 单元输入阻抗随扫描角变化（Active Impedance Variation） 方向图畸变 增益下降 互耦效应在密集阵列（d < 0.5λ）中尤为显著，是工程设计的核心挑战之一。
3.1 按几何排列分类
3.1 按几何排列分类
| 类型 | 结构特征 | 典型应用 | |------|---------|---------| | 线性阵列（Linear Array） | 单元沿直线排列 | 基站扇区覆盖、雷达扫描 | | 平面阵列（Planar Array） | 单元在二维平面上排列 | 卫星通信、5G Massive MIMO | | 圆形阵列（Circular Array） | 单元分布在圆周上 | 全向扫描、测向系统 | | 球形阵列（Spherical Array） | 单元分布在球面上 | 360°全覆盖、近场测量 | | 共形阵列（Conformal Array） | 贴合载体曲面 | 飞机/导弹外壳天线 |
3.2 按波束形成方式分类
3.2 按波束形成方式分类
| 类型 | 原理 | 优缺点 | |------|------|--------| | 模拟波束赋形（Analog BF） | 用移相器和功分器在RF域形成波束 | 硬件简单、成本低，但波束灵活性差 | | 数字波束赋形（Digital BF） | 每个单元配独立RF链，在基带数字域处理 | 波束灵活、可多波束，但成本高、功耗大 | | 混合波束赋形（Hybrid BF） | 模拟+数字两级处理，折中方案 | 5G mmWave主流方案，平衡成本和性能 |
3.3 按有源/无源分类
3.3 按有源/无源分类
无源阵列（Passive Array）：只有一个发射机/接收机，通过移相网络分配到各单元 有源阵列（Active Array / AESA）：每个单元或子阵配有独立的发射/接收模块（T/R Module） - T/R模块通常采用GaAs或GaN工艺 - 支持真正的多波束同时工作 - 单点故障不影响整体性能（冗余设计）

| 指标 | 定义 | 典型要求 | |------|------|---------| | 阵列增益（Array Gain） | 相比单单元的定向增强倍数 | 每倍频程约+3dB | | 波束宽度（Beamwidth） | 主瓣半功率点之间的角度 | 基站天线3-15° | | 旁瓣电平（SLL） | 最大旁瓣与主瓣之比 | 通常<-15dB，雷达<-20dB | | 栅瓣（Grating Lobe） | 除主瓣外出现的强方向瓣 | 需避免，d≤λ/2 | | 扫描范围（Scan Angle） | 波束可电子扫描的最大角度 | ±60°常见 | | 极化隔离度 | 双极化阵列中两正交极化间的隔离 | >25dB | | 端口隔离度 | 多端口阵列中相邻端口间的泄漏 | >20dB | | 增益平坦度 | 扫描范围内增益的变化 | <3dB |
6.1 栅瓣（Grating Lobe）问题
6.1 栅瓣（Grating Lobe）问题
当单元间距 d > λ/2 时，在波束扫描过程中会出现与主瓣强度相当的"镜像瓣"，导致能量浪费和方向混淆。工程铁律：在最大扫描角θ_max处，需满足 d ≤ λ/(1+sinθ_max)。对于±60°扫描的阵列，间距需≤λ/2。
6.2 互耦引起的阻抗失配
6.2 互耦引起的阻抗失配
阵列中单元越密，互耦越强，导致： 单元输入阻抗偏离设计值（通常从50Ω漂移） VSWR恶化，反射增大 实际增益低于理论值 解决方案：去耦结构设计（如defected ground structure、电磁带隙EBG结构）、单元间距优化。
6.3 相位误差累积
6.3 相位误差累积
实际系统中，每个T/R通道的相位和幅度不可能完全一致： 电缆长度差异引入相位偏差 温度变化导致移相器漂移 ADC/DAC量化精度有限 相位误差会导致波束指向偏移、旁瓣升高。校准是关键环节，通常在出厂时做一次全通道校准，运行中定期自校准。
6.4 散热问题
6.4 散热问题
有源阵列（尤其AESA）的T/R模块功耗巨大。一个64T64R的5G AAU，整机功耗可达数百瓦，散热设计直接影响可靠性和寿命。
6.5 校准复杂度
6.5 校准复杂度
大规模阵列的校准是一个"鸡生蛋"问题：要校准就需要知道真实方向图，但方向图本身又受校准精度影响。业界通常采用基于近场测量的迭代校准方案。

Küpfmüller首次提出通过相位控制实现波束扫描的理论 1944年：德国Mammut雷达建成首批相控阵天线（3D防空雷达） 1956-1961年：MIT林肯实验室开发AN/FPS-85（首批大型相控阵雷达），Hughes研发T/R模块概念 1970s-1980s：AESA技术成熟，AN/APG-63成为首款量产战斗机AESA雷达；AN/SPY-1相控阵雷达装备宙斯盾战舰 1990s：卫星通信相控阵终端商业化；自适应阵列（LMS算法）广泛用于抗干扰 2000s：标准化T/R模块（SMTR®）出现，GaN技术大幅提升效率 2010s：5G Massive MIMO兴起，相控阵从军事走向民用大规模商用 2016年：诺基亚和爱立信推出首批5G相控阵基站 2019年：SpaceX Starlink相控阵用户终端量产 2020s：6G太赫兹阵列、AI辅助波束管理、智能超表面（RIS）与阵列融合 2025-2026年：数字智能阵列、全平面覆盖天线、共享孔径技术成为研究热点
