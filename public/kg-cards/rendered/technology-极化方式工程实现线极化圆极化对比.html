极化方式工程实现线极化圆极化对比
天线技术/技术概念极化方式线极化圆极化双极化±45°轴比天线工程
一句话: 基站天线 99% 用 ±45° 线极化（双极化天线），卫星/GNSS 用圆极化。极化方式选择取决于"空间复用效率 vs 实现复杂度"——±45° 线极化在移动场景中表现最好，交叉极化隔离度 > 25dB 就能实现双流复用。
类比: 极化就像"光的偏振方向"——太阳镜的偏振片只让特定方向的光通过。
1.1 ±45° 双极化（基站主流） markdown
结构：
  每个天线单元 = 两根互相垂直的振子
  振子 1：+45° 方向
  振子 2：-45° 方向
  
实现方式：
  1. 交叉偶极子（Crossed Dipole）
     两根半波振子成 90° 交叉
     馈电点偏移 45° 实现 ±45° 极化
     
  2. 微带贴片（Patch）
     两个馈点激励正 TM 模
     馈电相位差 90° → ±45°
     
  3. 波导缝隙（Waveguide Slot）
     波导壁上切两组正交缝隙
     分别激励 +45°/-45°

关键指标：
  - 极化隔离度（Cross-Pol Isolation）：> 25dB
  - 轴比（Axial Ratio）：< 3dB（线极化接近 0°）
  - 极化纯度：> 95%

优势：
  - 结构简单，成本低
  - 极化隔离度好（25-30dB）
  - 适合空间复用（双流 MIMO）
  - 5G 基站标配

劣势：
  - 多径衰落时极化失配
  - 终端天线极化不确定时性能下降
1.0°/90° 垂直极化 markdown
应用：
  - 卫星通信（部分）
  - 雷达
  - 老旧 2G/3G 基站
  
特点：
  - 极化隔离度好
  - 但受终端姿态影响大
  - 5G 已基本不用
2.1 原理 markdown
圆极化：
  电场矢量端点随时间画圆
  
  右旋圆极化（RHCP）：顺时针旋转
  左旋圆极化（LHCP）：逆时针旋转
  
产生方式：
  1. 正交线极化 + 90° 相位差
     E_x = E₀·cos(ωt)
     E_y = E₀·cos(ωt + 90°) = -E₀·sin(ωt)
     → 合成矢量端点画圆
     
  2. 螺旋天线（Helical Antenna）
     导线绕成螺旋形
     轴向辐射 = 圆极化
     
  3. 微带贴片 + 偏心馈电
     方形贴片两个对角馈电
     相位差 90° → 圆极化
2.2 关键指标 markdown
轴比（Axial Ratio, AR）：
  衡量圆极化"圆"的程度
  AR = 长轴/短轴
  
  AR = 1（0dB）→ 完美圆极化
  AR = 3dB → 可接受
  AR > 6dB → 不适合圆极化应用
  
带宽：
  圆极化带宽通常比线极化窄
  ±45° 线极化：~100% 相对带宽
  圆极化：~10-20% 相对带宽
2.3 应用场景
应用极化方式原因
GNSS（GPS/北斗） RHCP 卫星信号经过大气反射后极化不变
卫星通信 RHCP/LHCP 减少极化失配（卫星姿态变化）
RFID RHCP 标签姿态不确定
车载毫米波雷达 圆极化 多径反射抑制
5G 基站 ❌ 不用 线极化更高效
