近场波束赋形Near-Field-Beamforming
天线技术/技术概念近场波束赋形Near-FieldBeamformingXL-MIMO6G球面波
一句话: 近场波束赋形是 6G XL-MIMO 时代的核心变革——当天线阵列大到近场区延伸到数百米时，传统的"平面波"假设不再成立，必须改用"球面波"模型，利用距离维度实现真正的三维波束聚焦（而不仅是波束指向）。
类比: 传统波束赋形像手电筒——光束朝哪个方向打，就是哪个方向。
1.1 从远场到近场：传播模型的范式转移
特性 远场（FF） 近场（NF）
波前模型 平面波（Plane Wave） 球面波（Spherical Wave）
相位分布 线性变化 二次方变化（抛物线）
幅度分布 恒定（归一化） 随距离衰减（1/r）
自由度 角度（θ, φ） 角度 + 距离（θ, φ, r）
波束行为 平行波束 聚焦波束
1.2 近场区的定义

传统天线理论中，近场/远场的分界是 Rayleigh 距离：

R_Rayleigh = 2D² / λ

其中 D 是天线孔径，λ 是波长。

对于 5G Massive MIMO（D = 1m, f = 3.5GHz, λ = 0.086m）：

R_Rayleigh = 2 × 1² / 0.086 ≈ 23m

对于 6G XL-MIMO（D = 20m, f = 3.5GHz, λ = 0.086m）：

R_Rayleigh = 2 × 20² / 0.086 ≈ 9.3km

关键洞察：6G XL-MIMO 的近场区可以延伸到数公里！大部分用户都在近场内。

1.3 近场区的细分

近场本身还可以分为两个子区域：

区域 距离范围 特征
菲涅尔区（Fresnel / Radiative Near-Field） λ < r < 2D²/λ 辐射场，波束聚焦可用
夫琅禾费区（Fraunhofer / Far-Field） r > 2D²/λ 平面波近似成立

注意：有些文献将近场进一步细分为：

  • 反应式近场（Reactive Near-Field）：r < λ/2π，储能为主，无辐射
  • 辐射式近场（Radiative Near-Field / Fresnel）：λ/2π < r < 2D²/λ
  • 远场（Fraunhofer）：r > 2D²/λ

通信中主要关注辐射式近场。

1.4 球面波模型

远场平面波模型：

steering_vector_FF(θ) = [1, exp(j·ψ), exp(j·2ψ), ..., exp(j·(N-1)ψ)]^T

其中 ψ = (2πd/λ)·sinθ。

近场球面波模型：

steering_vector_NF(r, θ) = [1/r₁·exp(-jk·r₁), 1/r₂·exp(-jk·r₂), ..., 1/rₙ·exp(-jk·rₙ)]^T

其中 rₙ = √(r² + (n·d)² - 2rd·sinθ) 是第 n 个单元到用户的距离。

关键差异：球面波模型中，每个单元的相位是距离 r 的二次函数（抛物线相位），而远场模型中相位是线性函数。

2.1 波束聚焦（Beamfocusing）

远场波束赋形只能控制波束的指向角，所有距离上的增益相同。近场波束赋形可以控制波束的焦点位置：

w_focus(r₀, θ₀) = steer_vec_NF*(r₀, θ₀)

这个波束在 (r₀, θ₀) 处达到最大增益，离开焦点后增益迅速下降。

效果：

  • 焦点处增益：比远场波束高 3-6dB
  • 非焦点处干扰：降低 10-20dB
2.2 距离维度复用（Range-Dimension Multiplexing）

远场波束赋形中，两个在同一角度但不同距离的用户无法区分。近场波束赋形可以利用距离维度同时服务这两个用户：

用户 A: (r_A, θ)
用户 B: (r_B, θ)

远场：无法区分，只能分时复用。 近场：可以形成两个不同的聚焦波束，同时服务。

关键公式（距离分辨率）：

Δr_min ≈ λ·r / (2D·√SNR)

对于 D = 20m, λ = 0.086m, r = 100m, SNR = 20dB：

Δr_min ≈ 0.086 × 100 / (2 × 20 × 10) ≈ 0.02m = 2cm

这意味着可以在 100m 距离上区分相距 2cm 的两个用户！

2.3 距离-角度解耦波束赋形（Range-Angle Decoupled, RAD）

2025 年提出的新框架——将波束赋形分为两步：

第一步：距离聚焦（Range-Focusing）

w_range = diag(exp(jk·r_n²/(2r₀)))

补偿球面波的二次相位。

第二步：角度零陷（Angle-Nulling）

w_angle = ZF([w_range · steer_vec_FF(θ₁), ...])

在角度域做干扰消除。

优势：

  • 反馈开销从 O(N) 降到 O(1)（只需反馈 r 和 θ 两个标量）
  • 实现与最大比值传输（MRT）等效的单用户增益
2.4 近场波束赋形的容量增益

研究表明，近场波束赋形相比远场波束赋形：

场景 容量增益 说明
单用户 1-3dB 聚焦增益
多用户（同角度不同距离） 2-5× 距离维度复用
多用户（不同角度） 0-1dB 增益有限
高密度用户 3-10× 三维空间复用
3.1 基于码本的波束训练

与传统远场波束训练不同，近场波束训练需要在角度-距离二维网格上搜索：

Codebook = { (θ₁, r₁), (θ₁, r₂), ..., (θ_K, r_S) }

挑战：

  • 远场码本大小：K 个角度 → O(K)
  • 近场码本大小：K 个角度 × S 个距离 → O(K·S)
  • S 通常取 10-100，码本大小暴增 10-100 倍

解决方案：

  • 两级码本：粗角度搜索 + 精距离搜索
  • 网格-free 方法：压缩感知、原子范最小化
  • 深度学习：直接预测 (θ, r)
3.2 基于优化的波束赋形

直接优化波束赋形向量：

maximize    Σ_k log₂(1 + SINR_k(r_k, θ_k))
subject to  ||w||² ≤ P_max

求解方法：

  • 交替优化（AO）：固定 w，优化 (r, θ)；固定 (r, θ)，优化 w
  • 半定松弛（SDR）
  • 梯度下降 + 流形优化
3.3 近场-远场统一波束赋形

2025 年的研究趋势——设计一个统一的波束赋形框架，同时适用于近场和远场用户：

w_unified = α · w_NF + (1-α) · w_FF

其中 α 是根据用户距离自适应调整的权重：

  • α → 1：用户在近场 → 使用近场波束赋形
  • α → 0：用户在远场 → 使用远场波束赋形
3.4 近场波束赋形与全息 MIMO 的结合

CAPA（连续孔径阵列）天然支持近场波束赋形——因为连续表面可以生成任意波前，包括球面波前：

J(x,y) = A(x,y) · exp(-jk·√((x-x₀)² + (y-y₀)² + r₀²))

这个电流分布在 (x₀, y₀, r₀) 处形成聚焦波束。

6.1 信道估计复杂度

近场波束赋形需要在二维（角度 + 距离）上估计信道：

估计参数：θ₁, r₁, θ₂, r₂, ..., θ_L, r_L + 复数路径增益

对于 L = 5 条路径，需要估计 10 + 5 = 15 个参数。而远场只需要 5 + 5 = 10 个参数。

挑战：

  • 二维搜索空间大
  • 距离参数的量化误差直接影响聚焦性能
  • 需要更长的导频和更多的天线
6.2 相位误差的敏感性

近场波束赋形对相位误差极其敏感——因为依赖精确的二次相位补偿：

相位误差 Δφ → 聚焦增益下降 ∝ exp(-Δφ²/2)
相位误差 RMS 聚焦增益损失
1° < 0.1dB
5° 0.5dB
10° 1.5dB
20° 4dB
45° 10dB

对比：远场波束赋形对相位误差的容忍度更高（10° 误差仅损失 ~0.3dB）。

6.3 用户移动性

用户移动时，焦点位置 (r, θ) 随之变化，需要实时更新波束赋形向量：

更新频率 f_update ≥ 2 · v_max / λ_focusing_spot

对于 v_max = 10m/s（行人），λ_focusing_spot = 1m：

f_update ≥ 20Hz

这意味着每 50ms 就要重新计算波束赋形向量——对基站的计算能力提出了很高要求。

6.4 近场-远场混合场景

实际网络中，既有近场用户也有远场用户：

用户 A: r = 50m → 近场
用户 B: r = 5km → 远场

如何在一个波束赋形框架下同时服务？

方案：

  • 统一波束赋形（α 混合）
  • 分层波束赋形：远场用户用平面波，近场用户用球面波
  • 码本设计：二维码本（角度 + 距离）覆盖近场，一维码本（角度）覆盖远场
6.5 多径效应

近场多径比远场更复杂——每条径有不同的距离和角度：

h_NF = Σ_l α_l · steer_vec_NF(r_l, θ_l)

其中 l 遍历所有径，每条径有自己的 (r_l, θ_l)。

影响：

  • 信道秩增加（近场 LoS 路径的秩 > 1）
  • 波束赋形需要考虑多径合并
  • 干扰管理更加复杂
