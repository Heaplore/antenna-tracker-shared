天线馈电网络Feed-Network-Power-Divider
天线技术/技术概念馈电网络FeedNetworkPowerDividerWilkinsonCorporateTree波导馈电
一句话: 馈电网络是大功率天线阵列的"血液循环系统"——它把一路输入信号均匀分配到 N 个天线单元，同时保证每个单元获得相同幅度和相位（或所需相位差）的能量。从 Sub-6GHz 的微带功分器到毫米波的波导馈电网络，再到 AiP 中的 Chip-on-Board 馈线，馈电网络的设计直接决定了阵列的性能上限。
类比: 馈电网络就像城市的自来水管网：水源（信号发生器）进来后，需要通过主干管（输入传输线）分流到各个社区支管（功分器级联），最终到达每家每户（天线单元）。
1.1 功率分配
1.1 功率分配
最基本的功能：将输入信号分成 N 路等幅（或不等幅）输出： ``` 输入 1 路 → 输出 N 路 ``` **等幅分配**（最常见）： - 所有输出端口功率相等 - 用于均匀照射的阵列 - 典型应用：波束赋形阵列、相控阵 **不等幅分配**（渐削照射）： - 中心单元功率高，边缘单元功率低 - 用于降低旁瓣电平 - 典型应用：射电望远镜、卫星通信
1.2 相位控制
1.2 相位控制
某些馈电网络需要引入特定的相位差： ``` Δφ_n = n · d · sin(θ₀) · 2π/λ ``` 其中 d 是单元间距，θ₀ 是目标波束指向。 **用途**： - 波束指向控制（相控阵） - 波束倾斜（机械/电子倾角） - 多波束生成
1.3 阻抗匹配
1.3 阻抗匹配
确保从输入到每个输出的阻抗匹配： ``` Z_in = Z₀（输入匹配） Z_out_n = Z₀（输出匹配） ``` 匹配不好会导致： - 反射增大 → S11 恶化 - 功率损耗 → 效率下降 - 谐波失真 → 信号质量下降
1.4 端口隔离
1.4 端口隔离
输出端口之间需要足够的隔离度： ``` S_ij < -20dB（i ≠ j，输出端口间） ``` 隔离度差会导致： - 互耦增大 → 方向图畸变 - 功率在端口间泄露 → 效率下降 - 多径效应加剧 → MIMO 性能下降
Wilkinson 功分器（最经典）
``` Z₀·√2, θ₀ Input ────┤╬├──── Output 1 │ │ [R] ← 隔离电阻（2Z₀） │ │ Output 2 ─┤╬├──── Output 3 Z₀·√2, θ₀ ``` **原理**： - 两段 λ/4 阻抗变换线（Z = Z₀√2） - 中心隔离电阻 R = 2Z₀ = 100Ω（50Ω 系统） - 当两输出端匹配时，电阻上无电流 → 不耗散功率 - 当输出失配时，电阻吸收反射功率 → 提高隔离 **性能指标**： | 参数 | 典型值 | |------|--------| | 插入损耗 | 0.1-0.3dB | | 隔离度 | > 20dB | | 带宽 | 10-20% | | 幅度平衡 | ±0.5° | | 相位平衡 | ±0.1dB | **多路扩展**（树状结构）： ``` 1→2 功分器 ├─ 1→2 → 输出 1,2 └─ 1→2 → 1→2 → 输出 3,4 └─ 1→2 → 输出 5,6 └─ 1→2 → 输出 7,8 ``` N 路需要 log₂(N) 级 Wilkinson。
T 结功分器（T-Junction Divider）
最简单的结构，但隔离度差： ``` Input ──┬── Output 1 └── Output 2 ``` - 插入损耗：0.5-1dB - 隔离度：-3dB（很差） - 带宽：宽（DC 到 GHz） - 适用：对隔离要求不高的场景
2.2 波导馈电网络（Waveguide Feed Network）
内容
毫米波基站的主流选择：
```
Input ──┤
├─── 臂 B ──→ 单元 9-16
├─── 臂 C ──→ 单元 17-24
```
**优势**：
- 损耗极低（比微带低 5-10 倍）
- 功率容量大（kW 级）
- 高频性能好（毫米波/太赫兹）
- 屏蔽性好（无辐射损耗）
**劣势**：
- 体积大、重量大
- 加工成本高
- 难以集成
**常见波导功分器**：
- E 面 T 接头
- H 面 T 接头
- 交叉波导（Magic Tee）
- Rat-Race 环形电桥
2.3 带状线馈电网络（Stripline Feed Network）
2.3 带状线馈电网络（Stripline Feed Network）
介于微带和波导之间： ``` Top Microstrip Dielectric Middle Stripline ← 馈电网络藏在此层 Dielectric Bottom Microstrip ← 天线单元在此层 ``` **优势**： - 双面布局（馈线和天线分开） - 屏蔽好（无辐射损耗） - 适合多层 PCB
2.4 共面波导馈电网络（CPW Feed Network） markdown
Signal ────┬──── Signal
Ground ────┘ ──── Ground


- 无需过孔接地
- 易于并联有源器件
- 损耗高于微带和波导
2.5 硅通孔馈电网络（Through-Silicon Via, TSV）
内容
用于 AiP 和芯片级集成：
```
Chip Top: 天线单元
│
TSV: ▼▼▼▼▼▼ ← 垂直互连
Chip Mid: 馈电网络（硅基底内）
│
Chip Bot: 驱动 IC（BFIC）
```
**优势**：
- 超薄剖面
- 高频损耗低
- 适合大规模量产
2.6 激光诱导技术馈电网络（LIFT / LIGA）
2.6 激光诱导技术馈电网络（LIFT / LIGA）
用于毫米波/太赫兹的高精度馈线： ``` 高精度 LIGA 工艺 → 深硅刻蚀 → 金属填充 ``` - 线宽精度 ±1μm - 适合 100GHz+ 频段 - 成本高
2.7 X 型全波导馈电网络（X-Type Full-Corporate Waveguide）
2.7 X 型全波导馈电网络（X-Type Full-Corporate Waveguide）
2022 年提出的新型毫米波馈电拓扑： **特点**： - 级数减半（相比传统树状） - 带宽提升 2 倍 - 插入损耗降低 3dB - 适合 28/39GHz 大规模阵列
3.1 树状馈电（Corporate Feed）
内容
最经典的分配方式：
```
Input
│
│ 1→2 │
│ 1→2 │ │ 1→2 │
│1→2 │ │1→2 │
1 2 3 4 （4 单元阵列）
```
**优缺点**：
| 优点 | 缺点 |
|------|------|
| 设计简单 | 级数多时损耗大 |
| 幅度/相位一致性好 | 级联越多，累积误差越大 |
| 易于仿真和优化 | 故障扩散（一级坏了全废） |
3.2 并行馈电（Parallel Feed / Bus Feed）
内容
所有单元并联到同一根馈线上：
```
Input ────────────────────────────────
│ │ │
Unit 1 Unit 2 Unit 3
```
**优缺点**：
| 优点 | 缺点 |
|------|------|
| 结构简单 | 馈线损耗大（沿程衰减） |
| 无级联损耗 | 幅度不一致（边缘单元弱） |
| 故障不影响其他 | 阻抗变化大 |
3.3 混合馈电（Hybrid Feed）
3.3 混合馈电（Hybrid Feed）
树状 + 并行结合： ``` 输入 → 树状分配（4 路）→ 每路并行馈给 8 个单元 ``` **优点**：兼顾简单性和均匀性 **缺点**：设计复杂
3.4 波导混合馈电（Waveguide Corporate + Parallel） markdown
波导分配 → 微带功分 → 天线单元


**典型应用**：5G 基站 AAU
4.1 插入损耗（Insertion Loss） markdown
IL_total = IL_splitter + IL_line + IL_connector


| 频率 | 微带线损耗 | 波导损耗 | LCP 损耗 |
|------|----------|---------|---------|
| 3.5GHz | 0.1 dB/m | 0.02 dB/m | 0.08 dB/m |
| 28GHz | 1.5 dB/m | 0.2 dB/m | 1.0 dB/m |
| 39GHz | 2.5 dB/m | 0.3 dB/m | 1.8 dB/m |
| 77GHz | 5.0 dB/m | 0.8 dB/m | 4.0 dB/m |
4.2 幅度平衡（Amplitude Balance） markdown
ΔP = |P_out1 - P_out2| < 0.5dB（典型要求）


幅度不平衡会导致：
- 旁瓣电平升高
- 波束指向偏移
- 极化隔离度下降
4.3 相位平衡（Phase Balance） markdown
Δφ = |φ_out1 - φ_out2| < 5°（典型要求）


相位不平衡的后果更严重：
- 波束指向误差：Δθ ≈ Δφ / (180°/N)
- 64 单元阵列中，Δφ = 5° → Δθ ≈ 4.5°
- 可能导致链路中断
4.4 隔离度（Isolation） markdown
S_isolation = 20·log₁₀(|V_reflected| / |V_incident|) < -20dB


隔离度不足的影响：
- 单元间互耦
- 方向图畸变
- MIMO 信道相关性增加
4.5 带宽（Bandwidth） markdown
BW = (f_max - f_min) / f_center × 100%


| 馈电类型 | 典型带宽 |
|---------|---------|
| Wilkinson | 10-20% |
| Rat-Race | 10-15% |
| Marchand Balun | 20-40% |
| 波导 | 10-30% |
| 多节变换 | > 50% |
5.1 导体损耗剧增
5.1 导体损耗剧增
趋肤深度 δ 随频率升高而减小： ``` δ = √(2ρ/ωμ) ``` 28GHz 时铜的 δ ≈ 0.35μm，意味着： - PCB 铜厚至少 3δ ≈ 1μm（镀金层） - 表面粗糙度影响显著 - Ra < 0.5μm 是目标
5.2 尺寸缩小
5.2 尺寸缩小
波长缩短 → 所有传输线尺寸按比例缩小： ``` λ_28GHz = λ_3.5GHz / 8 ``` - 微带线宽度从 ~1mm 缩到 ~0.13mm - 波导尺寸从 ~70mm 缩到 ~9mm - 加工精度要求从 ±0.1mm 提升到 ±0.02mm
5.3 多路功分器的级联损耗
5.3 多路功分器的级联损耗
N = 64 单元需要 6 级 Wilkinson： ``` 总损耗 = 6 × 0.15dB（每级）= 0.9dB ``` 加上传输线损耗（假设平均每路 10cm，28GHz 下 1.5dB/m）： ``` 总损耗 ≈ 0.9 + 0.15 = 1.05dB ``` 对于 256 单元（8 级）： ``` 总损耗 ≈ 8 × 0.15 + 0.2 = 1.4dB ```
5.4 双频/多频馈电
5.4 双频/多频馈电
5G 毫米波通常覆盖多个频段（如 28GHz + 39GHz）： ``` 双频 Wilkinson 设计： - 使用多节阻抗变换 - 每节针对一个频段优化 - 带宽扩展技术（inter-stage phase optimization） ``` 2024 年天津大学提出的 16 路双频功分器： - 28GHz 带宽：23-32GHz（32.1%） - 39GHz 带宽：34-46.5GHz（32.1%） - 采用级间相位优化技术
6.1 5G 基站 AAU 馈电网络 markdown
BFIC → 波导分配 → 微带功分 → 天线单元（64-256）


- 波导部分：低损耗主干分配
- 微带部分：精细阻抗匹配
- 总损耗：< 2dB
- 隔离度：> 25dB
6.2 手机 AiP 馈电网络 markdown
BFIC → LCP 微带 → 天线单元（4-16）


- 基板：LCP 或玻璃
- 线宽：50-100μm
- 总损耗：< 1.5dB
- 尺寸：< 20mm × 20mm
6.3 卫星通信馈电网络 markdown
LNA → 波导分配 → 喇叭馈源 → 龙勃透镜


- 功率容量：kW 级
- 损耗：< 0.5dB
- 可靠性：MTBF > 100,000 小时
