一句话: 贴片天线就是"电路板上的金属方块"——一块铜皮贴在介电基板上面，下面铺满接地层，中间用探针或缝隙馈电。它薄如纸、轻如羽，增益 6-8dBi，天然适合做阵列。5G 毫米波基站里成百上千个贴片堆在一起，每个都精确到微米级别，这就是 Massive MIMO 的物理基础。
一句话版本
> 贴片天线就是"电路板上的金属方块"——一块铜皮贴在介电基板上面，下面铺满接地层，中间用探针或缝隙馈电。它薄如纸、轻如羽，增益 6-8dBi，天然适合做阵列。5G 毫米波基站里成百上千个贴片堆在一起，每个都精确到微米级别，这就是 Massive MIMO 的物理基础。 类比入口：微波炉。微波在金属腔体内来回反射，形成驻波模式。贴片天线类似——电磁波在金属贴片和接地层之间来回反射，形成谐振腔模式，能量从边缘缝隙"泄漏"出来变成辐射。 ---
1.1 几何结构 markdown
┌─────────────────────┐
    │                     │
    │    金属贴片         │  ← 辐射体（铜皮）
    │   (W × L)          │
    │                     │
    ├─────────────────────┤  ← 介电基板（h, ε_r）
    │                     │
    │     接地层          │  ← 全覆铜
    │                     │
    └─────────────────────┘
         │
      馈电点


关键尺寸：

L ≈ λ_g / 2 = c / (2f√ε_eff)
W ≈ c / (2f) × √(2/(ε_r+1))  （优化辐射效率）


其中 ε_eff 是有效介电常数：

ε_eff = (ε_r + 1)/2 + (ε_r - 1)/2 × [1 + 12(h/W)]^(-1/2)
1.2 工作原理：谐振腔模型
1.2 工作原理：谐振腔模型
物理图像： 贴片和接地层构成一个"开放谐振腔" 电磁波在贴片边缘来回反射 当腔长 = λ_g/2 时，形成谐振 能量从贴片边缘的缝隙辐射出去 电场分布（矩形贴片 TM₁₀ 模）： ` E_z(x) ∝ cos(πx/L) ` 磁场分布： ` H_y(x) ∝ sin(πx/L) `
1.3 辐射机制：缝隙模型
1.3 辐射机制：缝隙模型
每个短边等效为一个缝隙： ` E_edge ∝ V_gap / d ` 两个缝隙的辐射叠加： 相位相同 → 法向（broadside）辐射最强 方向图 = 单缝隙方向图 × 阵列因子 ---
2.1 探针馈电（Probe/Coaxial Feed）
内容
结构：同轴电缆内导体穿过基板连接到贴片。
`
│ Patch│
├────●──┤ ← 探针
│ │
├───────┤
│ GND │
`
特点：
优点：结构简单，易于加工，阻抗匹配好
缺点：引入寄生电感，带宽窄（1-2%）
适用：单贴片、小规模阵列
阻抗变换：通过调节探针插入位置（从中心到边缘，阻抗从 ~300Ω 降到 ~50Ω）。
2.2 微带线边缘馈电（Edge/Microstrip Feed）
内容
结构：微带传输线直接连接到贴片边缘。
`
│ Patch│───┐
├───────┘ │
│ Feed │
│ Line │
`
特点：
优点：与 PCB 工艺兼容，易于集成馈电网络
缺点：表面波损耗，谐波辐射
适用：阵列天线，5G 毫米波主流方案
2.3 耦合缝隙馈电（Aperture-Coupled Feed）
内容
结构：贴片和馈线之间隔一层基板，通过缝隙耦合。
`
│ Patch│ ← 辐射层
├───────┤
│ Gap │
├───────┤
│ Slot │ ← 耦合缝隙
├───────┤
│ GND │
├───────┤
│ Feed │ ← 馈线层（反向）
`
特点：
优点：无表面波，隔离度高，可多层设计
缺点：结构复杂，加工成本高
适用：双极化天线，MIMO 阵列
2.4 近场耦合馈电（Proximity-Coupled Feed）
内容
结构：类似耦合缝隙，但不用金属缝隙，而是靠近场电磁耦合。
`
│ Patch│
├───────┤
│ Sub │
├───────┤
│Couple │ ← 耦合微带线（不接触）
├───────┤
│ GND │
`
特点：
优点：无穿孔，表面平整
缺点：设计复杂
适用：高频（> 30GHz）

3.2 带宽瓶颈
3.2 带宽瓶颈
根本原因：贴片天线的 Q 值很高（储能大，辐射弱）。 ` Q ≈ √ε_r × (λ₀/h) BW ≈ f₀/Q ` 提高带宽的方法： 增加基板厚度 h：Q 降低 → BW 增加（但引入表面波） 降低介电常数 ε_r：Q 降低 → BW 增加 使用激子层（Exciter Layer）：双层贴片 开槽/刻蚀：引入多谐振模式 使用 EBG 结构：抑制表面波，提高效率

5.1 互耦问题
5.1 互耦问题
原因：贴片之间的近场电磁耦合。 影响： 阻抗失配（S11 恶化） 方向图畸变 MIMO 相关性增加 → 容量下降 耦合强度： ` |S₂₁|_coupling ∝ exp(-d/δ_c) ` 其中 d 是间距，δ_c 是耦合衰减长度。

6.5 柔性贴片天线
6.5 柔性贴片天线
基板：PI（聚酰亚胺）、PET、LCP（液态晶体聚合物） 特点： 可弯曲、可折叠 适用于可穿戴设备 损耗略高于刚性基板 ---
优点
优点
超低剖面：高度 0.01-0.05λ，几乎不占空间 轻量化：PCB 材料，克级重量 易集成：与微波集成电路（MIC/MMIC）兼容 低成本：标准 PCB 工艺，批量生产极便宜 灵活极化：矩形/圆形/交叉贴片实现线极化/圆极化 阵列友好：天然适合大规模阵列（Massive MIMO）
缺点
缺点
窄带宽：基本结构仅 1-5% 低功率容量：PCB 材料耐热性差（< 10W） 表面波损耗：高 ε_r 基板损耗大 互耦严重：密集阵列时性能下降 制造公差敏感：毫米波频段尺寸误差影响大 ---
八、给非专业读者的一句话总结
> 贴片天线就是"电路板上的铜皮方块"——薄、轻、便宜、好量产。单个增益只有 7dBi，但把 64 个堆在一起就是 25dBi 的 5G 基站天线。它的带宽窄（像收音机只能收一个台），但阵列化之后什么都能覆盖。5G 毫米波基站里密密麻麻的都是这种"铜皮方块"，每个只有米粒大小，但加起来就是高速网络的基石。 金句：贴片天线告诉你一个道理——个体渺小没关系，团结起来就是力量。一个贴片 7dBi，64 个贴片就是 25dBi。 ---
