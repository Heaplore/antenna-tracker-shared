天线技术/技术概念有源单元方向图AEP扫描阻抗互耦TARC扫描盲区
一句话: 有源单元方向图（AEP）是指在大规模阵列中，当只有一个单元被激励而其他所有单元保持匹配负载时，该单元在整个空间中的辐射方向图。它包含了互耦效应和扫描角依赖的全部信息，是分析和设计相控阵天线最核心的工具之一。
类比: 想象一个合唱团的歌手——当每个人都孤立地唱歌时，声音传播模式是固定的（这就是"无源单元方向图"，即单个天线单元自身的方向图）。
1.1 定义

有源单元方向图（Active Element Pattern, AEP），也称为扫描单元方向图（Scanned Element Pattern），是相控阵天线分析中的一个核心概念。它定义为：

在无限大或有限大阵列中，当第 n 个单元被激励（施加信号），而阵列中所有其他单元都端接匹配负载（通常为 50Ω）时，该单元在整个空间中的归一化辐射方向图。

AEP 与无源单元方向图（Isolated Element Pattern）的区别在于：AEP 包含了互耦效应（Mutual Coupling）的影响，因此它随扫描角变化而变化，而无源单元方向图是固定不变的。

1.2 与阵列因子理论的关系

天线阵列的总方向图可以分解为：

总方向图 = AEP(θ,φ,θ₀,φ₀) × AF(θ,φ)

注意这里的关键区别：

  • 经典分解：总方向图 = 单元方向图 × 阵列因子（AF）—— 这只在无互耦的理想情况下成立
  • 实际分解：总方向图 = 有源单元方向图 × 阵列因子 —— AEP 已经包含了互耦效应

这意味着 AEP 本质上是一个"修正因子"，它将理想单元方向图修正为考虑了互耦后的实际方向图。

1.3 扫描角依赖性

AEP 不是固定的，它强烈依赖于阵列的波束指向角（θ₀, φ₀）：

  • 当波束指向 broadside（垂直于阵列平面）时，AEP 对应于 broadside AEP
  • 当波束扫描到某个角度 θ₀ 时，AEP 会随之变化
  • 扫描角度越大，AEP 畸变越严重

这种扫描角依赖性正是互耦效应在方向图上的体现——每个单元周围的"电磁环境"随扫描角而变化。

2.1 AEP 的数学推导

基于散射参数（S-parameters），AEP 可以严格推导如下：

对于一个 N 单元阵列，第 i 个单元的有源辐射场可以表示为：

E_i(θ,φ; θ₀,φ₀) = √(4π/G₀) · Γ_i(θ₀,φ₀) · exp(j·k·r_i·sinθ·cos(φ-φ₀))

其中：

  • G₀ 是参考单元增益
  • Γ_i(θ₀,φ₀) 是第 i 个单元在扫描角 (θ₀, φ₀) 下的有源反射系数
  • r_i 是第 i 个单元的位置矢量
2.2 有源反射系数与 TARC

有源反射系数（Active Reflection Coefficient, ARC） 是 AEP 理论中最关键的参数之一：

Γ_active,i = b_i / a_i | (a_j=0, j≠i)

即在只有第 i 个单元激励的情况下，该端口的反射系数。它不同于传统的 S11，因为考虑了其他端口的匹配负载产生的反射回来信号。

总有源反射系数（Total Active Reflection Coefficient, TARC） 是多端口阵列的推广：

TARC = √(Σ|b_i|²) / √(Σ|a_i|²) | 所有端口按波束指向角加权激励

TARC 的物理意义：

  • TARC = 0：所有功率都被天线吸收（完美匹配）
  • TARC = 1：所有功率都被反射（完全失配）
  • 工程接受标准：TARC ≤ -10 dB（通常要求 ≤ -15 dB）
2.3 扫描阻抗（Scan Impedance）

AEP 与扫描阻抗直接相关：

Z_active(θ₀,φ₀) = Z₀ · (1 + Γ_active) / (1 - Γ_active)

扫描阻抗随扫描角变化是阵列设计中最头疼的问题之一——阻抗匹配网络在 broadside 调好了，扫描到大角度就失配了。

2.4 互耦矩阵与补偿

对于 N 单元阵列，互耦可以用 N×N 的互阻抗矩阵 Z_mutual 描述：

V = Z_self · I + Z_mutual · I

其中 V 是端口电压向量，I 是端口电流向量，Z_self 是自阻抗对角矩阵。

互耦补偿矩阵 C 可以通过求逆得到：

I_corrected = C⁻¹ · I_measured

补偿后可以显著降低旁瓣电平（SLL）和提高波束指向精度。

2.5 AEP 与单元方向图的关系

AEP(θ,φ; θ₀,φ₀) = E_isolated(θ,φ) · F_coupling(θ,φ; θ₀,φ₀)

其中 F_coupling 是互耦修正因子，它包含了：

  • 相邻单元的散射效应
  • 阵列底板的表面波模式
  • 介质基板的导波模式

3.2 相位归一化 AEP（PNAEP）

在异质架构（如混频架构）阵列中，本振信号的相位不确定性会影响 AEP 测量。解决方法是引入相位归一化 AEP：

PNAEP = AEP / exp(j·φ_LO)

其中 φ_LO 是本振信号的相位。PNAEP 消除了 LO 相位不确定性的影响，可以同时表征幅度和相位。

3.3 有限阵列 vs 无限阵列
  • 无限阵列模型：假设阵列尺寸无穷大，边缘效应可忽略。计算效率高，适合初步设计
  • 有限阵列模型：考虑边缘效应，更贴近实际。边缘单元的 AEP 与中心单元不同

在大型阵列（N > 100）中，中心单元的 AEP 与无限阵列模型吻合良好，但边缘单元的 AEP 会有明显差异（约 1-2 dB 增益偏差）。

6.1 扫描盲区（Scan Blindness）

扫描盲区是 AEP 理论中最严重的现象之一：

  • 定义：在某些特定的扫描角度，阵列的有源反射系数急剧增大（接近 1），导致大部分功率被反射而不是辐射出去
  • 成因：阵列中的 Floquet 模式（导波模式）与表面波模式耦合，形成谐振
  • 影响：在该扫描角附近，阵列几乎"失效"，无法正常工作
  • 典型场景：微带阵列在介质基板较厚时更容易出现扫描盲区

避免扫描盲区的方法：

  • 减小单元间距（但会增加互耦）
  • 使用薄介质基板（但会降低增益）
  • 在阵列上方添加吸波层（增加重量和成本）
  • 使用波导或槽缝阵列（天然不易出现盲区，但体积大）
6.2 边缘效应（Edge Effect）

有限尺寸阵列的边缘单元与中心单元的 AEP 不同：

  • 中心单元：周围被其他单元包围，互耦对称
  • 边缘单元：一侧没有相邻单元，互耦不对称
  • 结果：边缘单元的有源阻抗与中心单元不同，导致方向图不对称

工程对策：

  • 在阵列边缘添加"死单元"（dummy elements），模拟无限阵列环境
  • 通过数字波束赋形补偿边缘效应
6.3 互耦导致的增益 paradox（增益悖论）

Hannan 悖论：当扫描角增大时，单个单元的有源增益可能增加，但整个阵列的总增益却下降。这是因为：

  • 有源增益增加是因为单元方向图在主瓣方向变得更窄
  • 但阵列因子在主瓣方向也变窄，导致总的 HPBW 展宽
  • 结果是总增益下降

这个悖论提醒我们：不能单独看 AEP 增益，必须结合阵列因子一起分析。

6.4 温度漂移对 AEP 的影响

T/R 模块的温度变化会导致：

  • 移相器相位漂移（约 0.5-2°/°C）
  • 功放增益变化（约 0.05-0.1 dB/°C）
  • 介质基板介电常数变化 → 谐振频率偏移

这些变化会导致 AEP 随温度漂移，进而影响波束指向精度。

对策：

  • 内置温度传感器，实时补偿
  • 使用温度稳定的介质材料（如 RT/duroid 系列）
  • 定期在线校准
6.5 AEP 测量的复杂性

测量一个 N 单元阵列的完整 AEP 需要：

  • N 次单独测量（每次激励一个单元）
  • 每个测量需要在球面上扫描 θ 和 φ 两个角度
  • 对于 256 单元的 5G AAU，意味着 256 × 360 × 180 ≈ 1.6 亿个数据点

这在实际操作中是不可行的，因此业界通常采用：

  • 抽样测量（只测关键角度）
  • 近场扫描 + FFT 外推
  • 电磁仿真 + 少量实测校准
七、技术演进
  • 1960s：AEP 概念首次提出，Allen 和 Hannan 分别独立发表相关论文
  • 1970s：无限阵列模型建立，AEP 理论系统化
  • 1980s：扫描盲区现象被深入研究，Floquet 模式理论应用于阵列分析
  • 1990s：TARC 概念提出，成为多端口阵列的标准化评价指标
  • 2000s：近场扫描技术成熟，AEP 测量自动化
  • 2010s：5G Massive MIMO 兴起，AEP 在 OTA 测试中广泛应用
  • 2014年：混频架构阵列的 PNAEP（相位归一化 AEP）方法提出
  • 2017年：基于 AEP 的双基 retrodirective 方向图计算方法
  • 2024年：TARC、ACI（阵列耦合指数）和互耦补偿矩阵的综合分析方法
  • 2025-2026年：AI 辅助 AEP 预测和互耦补偿自动化
